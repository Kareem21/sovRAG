Add New Job Wizard

This appears when you click the New button in Agent Job Scheduler screen to assist on setting up a new job. In this wizard, you will see the following tabs:

  • Setup - this is where you specify the thread and job information

    • Thread Number - Select the thread.

    • Job Type - Select the job type.

      • Account Posting Rule [POSR]

      • AdobeSign - Download and Check In Signed Documents [ESAS]

      • CDDS Integration [CDDS]

      • Connect - Password Activation [VPAP]

      • Connect - Reset Password [VRSP]

      • Convert Image to PDF [IPDF]

      • Customisable SQL Driven Task [CSDT]

      • Deactivate Idle User Account [DEAC]

      • Document Virus Scan [VRSC]

      • DocuSign - Check In Sign Docs (JWT Grant)[ESCL]

      • Email Notification - Unread Notification Summary [EUNS]

      • Engage Servicing Portal - Activation E-mail [CPLA]

      • Engage Servicing Portal - Auto Load Profile [CPAP]

      • Engage Servicing Portal - Calendar Update [CPUC]

      • Engage Servicing Portal – Daily Report [CPDR]

      • Engage Servicing Portal - Debtor Ageing [CPDA]

      • Engage Servicing Portal - Document Sharing [CSPD]

      • Engage Servicing Portal - Entity Update [CPEU]

      • Engage Servicing Portal – Follower Notification [CPFN]

      • Engage Servicing Portal - Get Notification [CPGN]

      • Engage Servicing Portal - Payment Record Cleanup [CPPC]

      • Engage Servicing Portal - Register Printing [CPPR]

      • Engage Servicing Portal - Reset Password E-mail [CPLE]

      • Engage Servicing Portal – Weekly Report [CPWR]

      • External Program Execution [EXTP]

      • External REST API Integration [EAPI]

      • Global File Review [GLFR]

      • Name Check Service - Check Status/Change Status [NCCS]

      • Name Check Service - Submit Name [NCCN]

      • Notification - Delete Read/Unflagged Notification [DNOT]

      • OCR Extraction [OCRJ]

      • Queue Invoice [QINV]

      • Replication - Download Import File [REDL]

      • Replication - Export [REEX]

      • Replication - Import [REIM]

      • Report Queue Generation [RPQG]

      • Skribble - Download and Check In Signed Documents [ESSK]

      • SSRS Report - Subscriptions and Delivery [SSRS]

      • Timesheet Reminder [TSRE]

      • Reminders Review, To Do Summary Recalculation [TDRE]

      • Reminder Notifications [TDNT]

    • Description - This defaults to the description of the selected job type but can be changed.

    • Note - Enter additional notes.

    • Active - This will be ticked by default.

    • Retry Execution on Job Failure - Tick this to have Task Scheduler rerun the job if the job encountered any error. Otherwise, you would need to click Reset Job Status in order for failed job to be re-run.

    • Advanced Settings - Tick this to configure using tag values. This is optional for the following job types:

      • Engage Servicing Portal - Calendar Update and Register Printing

      • Replication - Download Import File

      • Report Queue Generation

      • Timesheet Reminder

      • Reminders Review

  • Tag Values – Available when Advanced Settings is ticked. Enter tags for additional configuration for the job.

  • Criteria – Enter the job criteria for the available mandatory/non-mandatory fields. Fields displayed are dependant on the job type selected in Setup.

  • Notification – Enable and configure e-mail notifications for successful job completion.

    • Enable E-mail Notification - Tick this option to enable and configure e-mail notification. This allows for one e-mail notification type which will be sent out upon completion of the job. This option is ticked by default when the job type is SQL-driven such as Timesheet Reminder [TSRE].

    • Advanced - Tick this to set up more than one type of e-mail notification. Depending on the job type, you may have two or three types. Each will appear in the Condition column.

    • Condition - The contents of this column depend on the job type and whether or not the Advanced option is ticked. When there is more than one type, it means that you can have different e-mail notifications for the job.

    • To, CC, BCC, Reply To - Enter the relevant e-mail addresses or click the Browse button to access the Parameter Wizard to make your selections.

    • Subject - Enter the subject line for your e-mail. If you want to include information on the job, click the Browse button to access the Parameter Wizard to get the relevant tag code.

    • Use template - Tick this if you want the e-mail to be based on an existing template. If you tick this, Template field will be activated while the Body field is deactivated.

    • Template - Select the template to use. Default template will be automatically selected for the following when 'Use Template' checkbox is enabled in Wizard Settings.

      • Timesheet Reminder [TSRE]

      • Reminders Review [TDRE]

      • Engage Servicing Portal - Calendar Update [CPUC]

      • Engage Servicing Portal - Register Printing [CPPR]

      • Deactivate Idle User Account [DEAC]

      • Report Queue Generation [RPQG]

    • Body - If you are not using a template, enter the e-mail body by typing in your message. If you want to include information on the job, click the Browse button to access the Parameter Wizard to get the relevant tag code.

  • Schedule - This is to specify when and how often the job should be performed.

Note: For e-mail notifications, ensure that the SMTP server is configured in System Defaults > SMTP Server Configuration.
