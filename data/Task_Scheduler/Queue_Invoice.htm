Queue Invoice

This job type automate the Queue Invoice function in Posting of WIP screen. It will process the invoices that have been queued from Posting of WIP. This job type able to create new draft invoice, approve and post the invoice as final as well as queue the invoice. This job type must first be configured in order for the Queue Invoice function can be used in the Posting of WIP screen.

 

Setting up Task Scheduler

  1. Go to Configuration > Business Parameters > Task Scheduler > Agent Job Scheduler.

  2. Click New.

  3. Select the Job Type 'Queue Invoice [QINV]'. You may enter the Note if any.

  4. Ensure that the ‘Active’ checkbox is enabled.

  5. Click Next >.

  6. If an e-mail notification is required, tick Enable E-mail Notification and specify details for the e-mail.

  7. Click Next >.

  8. In Schedule, enter the job frequency.

  9. Click Done.
