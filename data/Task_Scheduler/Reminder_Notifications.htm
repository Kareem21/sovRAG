Reminder Notifications

This job type will send e-mails to users with outstanding Reminder. The job can be set up for all Reminder or for specific Reminder based on the relevant Review Rule. When it is run, it first checks the Reminders to determine if a user will receive the notification.

To ensure that the reminders are correct, the Reminders must be up-to-date either by:

  • Having users review the Reminders manually on a regular basis.

  • Automating the Reminders review with the ‘Reminders Review [TDRE]’ job type.

If the latter, ensure that the job is scheduled to run earlier than the Reminder Notifications job type.

 

Setting up Task Scheduler

  1. Go to Configuration > Business Parameters > Task Scheduler > Agent Job Scheduler.

  2. Click New and select the Job Type ‘Reminder Notifications [TDNT]’.

  3. Click Next >.

  4. In Criteria, you may specify values for the following:

    • No Operation Time – This is to specify the interval between e-mail notifications in hours.

    • Reminder Code – This is to specify the Reminder Rules for the job. If nothing is specified, the job applies to all active Reminder Rules. To select a Reminder Rule, tick the checkbox for the Reminder Rule. You can also tick ‘(Select All)’ to select all listed Reminder Rules.

  5. Click Next >.

  6. In Notification, you need to tick Enable E-mail Notification and specify details for the email

    such as the recipient(s), subject line and the message body. If you want different types of e-mail notification, tick the Advanced option and you will see the following:

    • First Notification – This is for the first e-mail notification that will be sent.

    • First Reminder – This is for the second e-mail notification that will be sent if the user still fails to close the outstanding item.

    • Second Reminder – This is for the third e-mail notification that will be sent if the user still fails to close the outstanding item.

  7. Click Next >.

  8. In Schedule, enter the job frequency.

  9. Click Done.

Note: You can get the relevant tags for the e-mail notifications from the Parameter Wizard.
