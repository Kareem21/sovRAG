Deactivate Idle User Accounts

This is to automatically deactivate user accounts that have not been used for a specified amount of time except for the user account ‘SYSMANAGER’.

 

Setting up Task Scheduler

  1. Go to Configuration > Business Parameters > Task Scheduler > Agent Job Scheduler.

  2. Click New and ensure that the Job Type is set to 'Deactivate Idle User Account [DEAC]'.

  3. Click Next >.

  4. Enter the number of days allowed for the account to be idle before it will be deactivated. This must be greater than zero.

  5. Click Next >.

  6. If an e-mail notification is required, tick Enable E-mail Notification and specify details for the e-mail.

  7. Click Next >.

  8. In Schedule, enter the job frequency.

  9. Click Done.

In the Agent Job Log screen, you can see a log of the job with the Process Log showing user accounts that were deactivated. You can view the details for each record by double-clicking or clicking Details.

If e-mail notification was set up, this will be sent out as well.
