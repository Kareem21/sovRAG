Account Posting Rule

This job type automates the Accounting Posting Rules 'Run Rule' function. It supports all Posting types and runs all rules per job run.

When the Task Scheduler job is executed, it will run all active rules defined for the specific Account File as indicated by the 'Run-all' checkbox in File Maintenance > Accounting > Processing > Posting Rules.

 

Task Scheduler Job Run Criteria

The Task Scheduler job will check the Posting Rule’s last run ‘To’ date before running the Posting Rule. If it is earlier than the Task Scheduler job 'Next Run' date, the Posting Rules will be run.

Upon completion of the job run, the Posting Rules run records will be created in File Maintenance > Accounting > Processing > Posting Rules > 'Posting Rules Run' grid and journal will Post as Final.

 

Setting up Task Scheduler

  1. Go to Configuration > Business Parameters > Task Scheduler > Agent Job Scheduler.

  2. Click New.

  3. Select the Job Type 'Account Posting Rule [POSR]'. You may enter the Note if any.

  4. Ensure that the ‘Active’ checkbox is enabled.

  5. Click Next >.

  6. In Criteria, enter the File Code. This refers to the Account File code.

  7. Click Next >.

  8. If an e-mail notification is required, tick Enable E-mail Notification and specify details for the e-mail.

  9. Click Next >.

  10. In Schedule, enter the job frequency.

  11. Click Done.
