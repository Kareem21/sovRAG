Convert Image to PDF

The 'Conversion Image to PDF [IPDF]' job type converts stored images to PDF based on selected criteria of number of files, image type, and conversion order. All image files that have been successfully converted to PDF will remain in the holder. For images in revision groups, only files that are primary will be converted.

 

 

Setting up Task Scheduler

  1. Go to Configuration > Business Parameters > Task Scheduler > Agent Job Scheduler.

  2. Click New and ensure that the Job Type is set to 'Convert Image to PDF [IPDF]'.

  3. Click Next >.

  4. In the 'Criteria' tab, enter the values and select the options for the following mandatory fields:

    • Total Image File - Enter a numerical value for the number of image files to include in the PDF conversion process. For example, if the value is 1000, the PDF conversion will be performed on one thousand stored image files.

    • Image File Types - Select the file formats to include in the PDF conversion process.

    • Image File Conversion Order - Select 'Oldest to Newest' to perform PDF conversion starting from the earliest image file stored or 'Newest to Oldest' to start from the latest image file stored. Images already converted will not be included in this selection.

  5. Click Next >.

  6. If an e-mail notification is required, tick Enable E-mail Notification and specify details for the e-mail.

  7. Click Next >.

  8. In Schedule, enter the job frequency.

  9. Click Done.
