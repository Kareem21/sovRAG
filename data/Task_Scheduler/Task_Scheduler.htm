Task Scheduler and Notification Agent

This feature is an additional option and is only available if included in your licence.

 

The Viewpoint Task Scheduler and Notification Agent (VPTSNA) is to carry out jobs in an unattended mode. This is done by setting up the job schedule in the Agent Job Scheduler screen after you have installed and configured VPTSNA. In order to use this, you must ensure that the following has been completed:

  • Viewpoint Connect REST Services is set up through ViewPointDbManagement.exe.

  • Viewpoint Task Scheduler and Notification Agent is installed and configured through ViewPointDbMgt.exe.

  • Task schedule is set up through Agent Job Scheduler screen.
