Reminders Review, To Do Summary Recalculation

This job includes two options, which is to review Reminder and to recalculate the items in To Do Summary.  You can set up to have this job automated based on a set frequency.

 

Setting up Task Scheduler

  1. Go to Configuration > Business Parameters > Task Scheduler > Agent Job Scheduler.

  2. Click New and ensure that the Job Type is set to 'Reminders Review, To Do Summary Recalculation [TDRE]'.

  3. Click Next >.

  4. In Criteria, select the job as Reminders Review or To Do Summary Recalculation and specify if it applies to multiple users or only specific users.

  5. Click Next >.

  6. If an e-mail notification for the job is required, tick Enable E-mail Notification and specify details for the e-mail. If you want different types of e-mail notification, tick Advanced and you will get the following:

    • Success - this is the notification when the job is completed successfully

    • Fail - this is the notification when the job is not completed

  7. In Schedule, enter the job frequency.

  8. Click Done.

The Reminders Review job will review the Reminders screen for items with due date up to the 'Review Horizon' as set in 'Reminders Options' in Viewpoint Home > Configuration > Business Parameters > General > Reminder Rules - Options.

 

Note: If you ticked Advanced Settings in Setup tab, you can use the following tags as well as MAIL tags.

Tag

Explanation

Example

[/SELTYPE=<Value>/]

This is to define ‘whose’ Reminder to review. You can set it to a particular user, User Group, Administrator Group or for all users.

Values for SelType:

  • 2048 - To review for user.

  • 4096 - To review for Administrator Group.

  • 8192 - To review for User Group.

  • 16384 - To review for ALL users.

  • 32768 - To review for primary Administrator Group.

[/SELTYPE=2048/]

[/SELCODE=<Code>/]

This is to define the selection based on the selection type by using the code for the user, Administrator Group or User Group.

If SelType=16384, you can leave the value as blank.

[/SELCODE=SM/]

[/SELCODE='SM','Anne']

[/SELCODE='GROUP1','GROUP2','GROUP3'/]

[/JOBTYPE=<value>/]

 

This is to define the job type to for the action. The values are:

  • 0 - Reminders Review

  • 1 - To Do Summary Recalculation

[/JOBTYPE=0/]
