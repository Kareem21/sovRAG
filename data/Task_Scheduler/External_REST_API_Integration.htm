External REST API Integration

This job type pushes the contents to the external REST API based on the Service Name defined in the Change Link.

 

Setting up Task Scheduler

  1. Go to Configuration > Business Parameters > Task Scheduler > Agent Job Scheduler.

  2. Click New and ensure that the Job Type is set to ‘External REST API Integration [EAPI]'.

  3. Click Next >.

  4. Under the ‘Criteria’ tab, enter the value accordingly.

    Fields

    Description

    Token URL

    This refers to the token URL generated by the external REST API.

    Client ID

    This refers to the Client credential ID for API Service authentication.

    Client Secret

    This refers to the Client credential secret for API Service authentication.

    Service URL

    This refers to the service URL generated by the external REST API.

    Service Name

    This is the service name that links to the Change Link.

    Number of Attempt(s)

    This is the service name that links to the Change Link.

     

  5. Click Next >.

  6. If an e-mail notification is required, tick Enable E-mail Notification and specify details for the e-mail.

  7. Click Next >.

  8. In Schedule, enter the job frequency.

  9. Click Done.

Agent Job Log

Viewpoint Connect program will execute the scheduled task as per the configured file path and program parameters. It logs the executed tasks in the Agent Job Log. The log can be viewed in Home > Configuration > Business Parameters > Task Scheduler > Agent Job Log.

To view the log, select the task and click Details.
