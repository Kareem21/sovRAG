Agent Job Log

This screen contains logs of the jobs and e-mails for the jobs. When first accessed, you will see the selection criteria first. Specify the criteria and click Apply.

 

Filter

This allows you to view jobs that encountered errors, completed jobs or all.

 

Select button

This allows for additional criteria to apply to the Job Log.

 

Task Log

This shows all jobs attempted by the Task Scheduler and Notification agent.

 

Command buttons

For records in the Job Log, click on the relevant command button to perform the necessary.

  • Details - Click view additional details of the record.

  • Select All - Click this to select all records in the grid. This is only enabled for records with Job Status – ‘Error - see job log’.

  • Deselect All - Click this to deselect all records in the grid. This is only enabled for records with Job Status – ‘Error - see job log’.

  • Resend - Click this to Resend e-mails with the following conditions. This is only enabled for records with Job Status – ‘Error - see job log’ and have e-mail items/notifications.

    • E-mails that are not sent successfully.

    • E-mails that remain as ‘In Progress’ for more than 24 hours.

Task Notification Log

This shows e-mail sent for a selected job. For each record, you can view additional details by clicking Details.
