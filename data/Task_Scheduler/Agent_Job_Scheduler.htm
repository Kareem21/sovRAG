Agent Job Scheduler

All jobs for the Task Scheduler and Notification agent is to be set up in this screen.

 

Thread Number

Click to view jobs that have been allocated to the selected thread. By default, you will see Thread 1 but if additional threads have been set up in Agent Job Threads [AJTH] lookup table, you will be able to see them.

 

Job list

The list shows all the jobs that have been set up.

 

Command buttons

Click on the relevant command button to perform the necessary.

  • New - Click this to set up a new job in the Add New Job wizard.

  • Edit - Click this to make changes to the selected job. Editing the job might risk concurrency error when the job is running at the same time. It is recommended to stop the Task Scheduler and Notification Agent in Viewpoint Management before editing.

  • Delete - Click this to remove the selected job.

  • Reset Job Status - Click this to set the status for the selected job to Idle/Completed.

  • Run Now - Click this to execute the selected job immediately. The 'Next Run' date will not be updated when job is triggered by 'Run Now'.

  • Job Logs - Click this to navigate to Agent Job Log screen.
