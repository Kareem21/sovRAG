Timesheet Reminder

If you are using Time Management, you may want a reminder to be sent out to users to inform them to submit their timesheet. This reminder will be in the form of  an e-mail notification.

 

Setting up Task Scheduler

  1. Go to Configuration > Business Parameters > Task Scheduler > Agent Job Scheduler.

  2. Click New and ensure that the Job Type is set to 'Timesheet Reminder [TSRE]'.

  3. Click Next >.

  4. In Criteria, you can specify the interval between e-mail notifications by entering the number of hours in the No Operation Time field.

  5. Click Next >.

  6. In Notification, you need to tick Enable E-mail Notification and specify details for the e-mail such as the recipient(s), e-mail subject and the message body. If you want different types of e-mail notification, tick Advanced and you will be able to specify different e-mail details for the following:

    • First Notification - This is for the first e-mail notification that will be sent.

    • First Reminder - This is for the second e-mail notification that will be sent if the user still fails to submit the timesheet.

    • Second Reminder - This is for the third e-mail notification that will be sent if the user still fails to submit the timesheet.

  7. Click Next >.

  8. In Schedule, enter the job frequency.

  9. Click Done.
