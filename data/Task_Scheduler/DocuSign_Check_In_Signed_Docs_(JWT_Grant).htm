DocuSign - Check In Signed Docs (JWT Grant)

The task scheduler job 'DocuSign - Check In Sign Docs (JWT Grant) [ESCL]' downloads and checks in signed documents from DocuSign into Viewpoint, hence users do not need to manually check in the documents .

 

Setting up Task Scheduler

  1. Go to Configuration > Business Parameters > Task Scheduler > Agent Job Scheduler.

  2. Click New and ensure that the Job Type is set to 'DocuSign - Check In Signed Docs [ESCL]'.

  3. Click Next >.

  4. In the 'Criteria' tab, enter the  API Username and RSA Private Key of the DocuSign account.

    API Username – This value can be retrieved from the DocuSign account user profile. Select the API Username of the user for this job.

    RSA Private Key – This value is generated from the DocuSign account using the generate RSA function.

  5. Click Next >.

  6. If an e-mail notification is required, tick Enable E-mail Notification and specify details for the e-mail.

  7. Click Next >.

  8. In Schedule, enter the job frequency.

  9. Click Done.
