Payee/Payer Bank Accounts

This screen allows for bank accounts of payees and payers to be set up for use by Practice Management component. Alternatively, the records can also be created in:

  • File Maintenance screen:

    • Click Sales Ledger Ribbon button and select Invoicing > Payer Bank Accounts from the navigation listing in Sales Ledger Overview.

    • Click Purchase Ledger Ribbon button and select Invoicing > Payee Bank Accounts from the navigation listing in Purchase Ledger Overview.

 

Filters

The filters apply to the bank account list.

 

Select button

This allows for additional criteria to apply to the bank account list.

 

Bank account list

This list shows all the bank accounts set up for a payee/payer.

 

Command buttons

Click on the relevant command button to perform the necessary.

  • New - Click this to create a new record.

  • Edit - Click this to edit the details of a selected record. If an approved payee/payer record is edited, it needs to be re-approved.

  • Delete - Click this to delete the selected record.

  • Approval - Click this to approve the selected record. The approval process depends on how the approval card for payee/payer approval was set up.

    This will open the Approval screen allowing you to review the details and approve the payee by clicking Approve.
