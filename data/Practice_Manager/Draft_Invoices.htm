Draft Invoices

Draft invoices have not yet been posted to the invoice ledger and can therefore be edited or deleted without affecting the ledger. Once they are posted, you will not be able to edit or delete them; you would need to create a credit note to offset the amount of the invoice you want to edit or delete.

The Draft Invoices screen provides a snapshot of all draft invoices that have been created. This screen can be accessed through:

  • Viewpoint - Home screen:

    • Click Sales Ledger Ribbon button and select Sales Ledger > Accounts Receivable > Draft Invoices from the navigation listing in Sales Ledger Overview.

    • In Section Menu, click Home > Practice Management > Sales Ledger > Accounts Receivable > Draft Invoices.

  • File Maintenance screen:

    • Click Sales Ledger Ribbon button and select Invoicing > Draft Invoices from the navigation listing in Sales Ledger Overview.

    • In Section Menu, click File > Sales Ledger > Invoicing > Draft Invoices.

Filters

You can filter the list of draft invoices by the status of the draft invoice - approved or not.

 

Draft Invoices list

This shows the draft invoices that have been created. You can select a draft invoice by clicking on it and this will show the invoice lines in the Invoice Line Table. To select multiple draft invoices, tick the checkbox for the draft invoice.

 

Command buttons

Click on the relevant command button to perform the necessary.

  • Edit - Click this to edit the selected draft invoice. This will go to the Create/Edit Invoice screen.

  • Select All - Click this to select all draft invoices listed.

  • Deselect All - Click this to deselect the selected records.

  • Approve - Click this to approve the draft invoice. This needs to be done before you can post a draft to an invoice.

  • Undo Approve - Click this to reverse the approval.

  • Reopen - Click this to reopen a locked/closed draft invoice. Draft invoices are locked when you click Print button.

  • Delete - Click this to delete the selected draft invoice. Viewpoint will validate if the draft can be deleted: if (partial) payments have been made, of if the draft is approved, it cannot be deleted.

  • Print - Click this to print the selected draft invoices. The draft invoices are subsequently locked to prevent new items being added to it. You can reopen them in Create/Edit Draft Invoices if you need to edit the detail lines.

  • Invoice - Click this to invoice the selected draft invoices.

    1. In the Invoice Posting window that appears, complete the following information:

      • Invoice Date, Journal Date and Overwrite for (existing) Draft Invoices - When you post an invoice, the invoice and journal dates default to the date when the draft was created.     Note:

    When 'Use Recurring Services for Time Postings' and 'Use Recurring Services for Disbursement Postings' options are enabled in Home > Configuration > System Defaults > Time and Disbursements, the invoice will be created with the following type:

    • If Entity Administration component is activated - an invoice with document type 'Recurring Fee Invoice Entity [A_CS]' will be created.

    • If Entity Administration component is not activated -  an invoice with document type 'Recurring Fee Invoice Non-Entity [A_GE]' will be created.

    When the options are disabled, the invoice will be created with the type 'Debit Note [DR_N]'.

I've invoiced my draft. Where is it now?

Draft invoices that have been posted are removed from this screen, and can be found in the Accounts Receivable section.

 

 

Invoice Line list

This shows the lines of the selected draft invoice. If you have selected multiple draft invoices, this shows the lines for the draft invoice that is highlighted in blue.

 

Note: If you are using Accounting component and the option 'Generate Journals Automatically' in System Defaults > Accounting, this will create journal entries in the relevant account based on your Account Mapping.
