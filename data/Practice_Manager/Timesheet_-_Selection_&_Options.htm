Selection & Options Folder

In this folder, you can choose what you want to see in Timesheet folder.

 

Day/Week/Month grouping and date range

Click to select the grouping for the timesheet. You can select Day, Week or Month and enter the date range in the Date - from and - to fields. The selection here will have an impact on the command buttons seen in the Timesheet folder in Viewpoint - Home.

Note: The maximum is 31, when selecting:

  • Day -  The date range entered must not exceed 31 days.

  • Week - The date range entered must not exceed 31 weeks.

  • Month - The date range entered must not exceed 31 months.

 

Filter

In Viewpoint - Home screen, if you have the authority, you can choose to filter the timesheet list by Billing File, Client/Group, Administrator, Organisational Unit, Service, Revenue Entity or view all entries. However, in File Maintenance screen, it will display entries for the selected file only.

If the security model used by your organisation is 2 - Administrator + User Group Model or 3 - Access + User Group Model, access is restricted to your Administrator Group and Group Access authority.

  • Billing File: You are allowed to select Billing Files based on your Group Access and Billing File Administrator Group settings. The Administrator of the Billing File will not be verified.

  • Client/Group: You are allowed to select Client Files based on your Group Access and Client File Administrator Group settings. The Administrator of the Client File will not be verified.

 

Timesheet Columns

Select from the columns you want to display.

 

Show in Timesheet

Select the unit of the timesheet. Timesheets can be displayed in units, hours, amounts charged or written off.

 

Show Invoiced Time Entries

Tick to view invoiced time entries as well. If you do this, then note that the load time will be considerably longer.

 

Show Only My Time Entries

Tick this to view only your own time entries. Note that this option is only available in File Maintenance screen.

 

Show Totals of Last Week in Timesheet Folder

Tick this if you also want to see the totals for the prior week in Timesheet. This will add an additional column before the Total column in Timesheet folder. Note that this option is only available in Viewpoint - Home screen.

Note: This applies when you choose Day option and prior week is one week before the specified date range.

 

Hide Saturdays

Tick this to hide Saturdays from  the Timesheet. Note that this option is only available in Viewpoint - Home screen.

 

Hide Sundays

Tick this to hide Sundays from the Timesheet. Note that this option is only available in Viewpoint - Home screen.

 

More Options...

Tick this to see additional options. Note that these options are only available in Viewpoint - Home screen.

Use Context Sensitive Defaults for Timesheet Folder

This applies to time entries created in the Timesheet folder. When ticked, you can create a new time entry by double-clicking on an existing item and its details will follow the item you double-clicked on.

 

Use Context Sensitive Defaults for Time Entries and Disbursements folders

This applies to time and disbursement entries created in the Time Entries and Disbursements folder respectively. When ticked, you can create a new time entry by selecting an existing entry and clicking New. The new entry will contain details of the entry that you selected.

 

Do NOT Default the Date on the Time Entries and Disbursements folders

Tick this if you do not want today's date to default to the date field in these folders.

 

Separate Section for Non-chargeable Time

Tick this if you want to separate non-chargeable time from chargeable time in Timesheet folder.

 

Show Chargeable Amount/Unit Targets

Tick this to show chargeable units, chargeable amount and the target achieved percentage in the Timesheet. When the user's target Chargeable Units or Chargeable Amount is not defined in the user's Administrator and Rates setting in Configuration > Users & Security > Users, it will not show the respective lines.

The % Achieved is calculated as follows:

  • % Achieved for Chargeable Units = (the entered Chargeable Units / target Chargeable Units) * 100

  • % Achieved for Chargeable Amount = (the entered Chargeable Amount / target Chargeable Amount) * 100

 

Separate Section for Charges against Recurring Fees

This option is only available when 'Use Recurring Services for Time Entries' is enabled in the 'Time and Disbursements' setting screen.

Enabling this will add a row 'Charges against Recurring Fees' in the Timesheet. Time entries that meet the following criteria will be included into this row:

  • Service Class is not non-chargeable time AND

  • Recurring service having a non-zero value in 'Number of Units' field and 'Unit Price' field.

 

Save as Default Settings

If you have made any changes in this screen, click this to save the changes.
