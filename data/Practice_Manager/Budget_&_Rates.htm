Budget & Rates

This screen is where you set up time rates for a Billing File for a specific service. The time rate can then be applied to selected Matter or all Matters of the Billing File. This is used when the fixed time rate that has been agreed for the Billing File differs from the standard rate entered in the Service Rates screen.

Note that the rates set up here apply only to Administrator Rate times (i.e. services with Service Class 'TA') and the following needs to be done in order to use this feature:

  • Tick the option 'Use Budgets & Rates for Administrators' in System Defaults > Time and Disbursements.

  • Enable the Budget & Rates screen in Configuration > Settings > Windows Menu.

  • Set up the time rate in Budget & Rates screen.

 

Rate List

This shows the rate that you have set up for the Billing File.

 

Command buttons

Click on the relevant command buttons to perform the necessary.

  • New - Click this to add a time rate.

    Enter the required details. Click Update when done.

  • Edit - Click this to update existing time rate.

  • Delete - Click this to remove existing time rate.
