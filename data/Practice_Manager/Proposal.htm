Proposal

Payments of purchase invoices can be placed in a proposal list for approval before they are released. Prior to approval, the invoices will not be reflected as 'Paid'. You can view this screen through:

  • Viewpoint - Home screen:

    • Click Purchase Ledger Ribbon button and select Purchase Ledger > Accounts Payable > Proposal from the navigation listing in Purchase Ledger Overview.

    • In Section Menu, click Home > Practice Management > Purchase Ledger > Accounts Payable > Proposal.

  • File Maintenance screen:

    • Click Purchase Ledger Ribbon button and select Accounts Payable > Proposal from the navigation listing in Purchase Ledger Overview.

    • In Section Menu, click File > Purchase Ledger > Accounts Payable > Proposal.

I don't see this screen. What do I need to do?

Enable the feature by ticking the Batch/Approval of AP payment checkbox in Configuration > Settings > System Defaults > Practice Management.

 

Enable Purchase Order Proposal Screen:

  1. Go to Configuration > Users & Security > User Roles.

  2. Click Authorities.

  3. Ensure the appropriate User Group has the appropriate authority level for the Proposal screen in both Home and File windows menu.

  4. Click OK.

 

Filter

The filter allows you to view the filtered proposals.

 

Select button

This allows for additional criteria to be applied into the proposal list. This is available only in Viewpoint - Home.

 

Proposal list

This lists the existing payment proposals that have not been posted.

 

Command buttons

Click on the relevant command button to perform the necessary.

  • Edit - Click to edit the payment proposal.

  • Select All/Deselect All - Click to select/deselect all payment proposals.

  • Approve/Undo Approve - Click to approve/unapprove selected proposals.

  • Delete - Click to delete the payment proposal.

  • Post - Click to post approved payments.
