Corporate Fees

This function works based on the tags that are placed in the Tag Values field of the fee. You can have a fee set up for each entity type in a jurisdiction. This means, for instance, that you can only create one fee for the International Business Company (IBC) entity type in the British Virgin Islands, but you can set up another fee for the IBC entity type in Seychelles. The information used to calculate the corporate (or government) fee is placed in the Tag Values field.

 

Corporate Fees list

This shows all the fees that have been set up.

 

Command buttons

Click on the relevant command button to perform the necessary.

  • New - Click to add a new corporate fee.

    1. Enter the details:

      • Jurisdiction - Select the country of incorporation.

      • Entity Type - Select the entity type. The options available depend on the selected jurisdiction.

      • Description - Enter description for the type of fee you are creating.

      • Incorporation Template - This field does not have any impact and can be left blank.

      • Government Fee - Select the service. The services should be set up prior to this.

      • Recurring Fee Template - Select a recurring fee template, if applicable. When Cloning or File Review is performed, the services specified in the template will be added to the Billing File's Recurring Fees screen.

      • Tag Values - Enter the tags used to calculate the corporate fee.

        The available tags are for corporate fee calculations similar to the tags used in Service Master (please refer to the calculation for Fees Payable such as corporate fee calculations, due dates and others in Service Master - Tags).

        If the tags here include ‘[FEECALC_BASIS=x]’ tag, then the tag configuration here will prevail. However, if the ‘[FEECALC_BASIS=x]’  tag is not included, then the system will ignore the other tags and refer to the Service Master's Tag Values for the service.

        Note: When both Service Master and Corporate Fees contain ‘[FEECALC_BASIS=x]’ tag, then the tag configuration in Corporate Fees will prevail.

      • Note - Add any note that may be important to the fee structure or setup.

    2. Click Update.

  • Edit - Click this to edit the corporate fee.

  • Delete - Click this to delete the fee.
