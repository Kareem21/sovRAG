Disbursement Folder

The Disbursement folder is where you add, edit, review and delete disbursement entries. Disbursements can be posted to invoices.

 

Disbursements list

This displays disbursement entries that have not been invoiced:

  • For the period of the selected timesheet in Summary folder; or

  • That fulfils the criteria in Selection & Options folder.

 

Command buttons

Click on the relevant command button to perform the necessary.

  • New - Click this to add a disbursement entry. Find out more here.

  • Edit - Click this to make changes to a selected entry.

  • Copy - Click to copy the selected disbursement entry. You can also copy approved entries but if approval is required, the entry will require manual approval.

  • Select All/De-select All - Click to select/deselect all records.

  • Approve - Click to approve the disbursement. Approved entries cannot be edited nor deleted from the table. Once a disbursement is approved it can be posted to a draft invoice. This button is enabled only if time and disbursement entries require approval before they are posted to a draft invoice. User must have authority level 8 - Power User.

  • Undo Approve - Click to remove the approval on a selected record. An entry that has been posted to a draft invoice cannot be unapproved.

  • Delete - Click this to delete the selected record. This button is only activated if the selected record has not been approved and user has authority level 8 - Power User.

  • Special - Click this to open the Special functions window. This is available only if you have the proper authority. Note that whether or not you can use this on approved time entries depend on the setting in System Defaults > Time and Disbursements.

    The Special Functions screen provides options that you can perform on unapproved and unposted charges.

     

    Set entries to NO CHARGE

    Select this option to write off the Total Charge amount.

     

    Transfer to another Billing File and/or Matter

    Select this to transfer the service to another Billing File or Matter. Select the Billing File and/or Matter you want to transfer to. If 'Use Recurring Services for Time Entries' and/or 'Use Recurring Services for Disbursement Entries' is enabled in the Time and Disbursements settings, this option will appear as 'Transfer to another Billing File, Matter and/or Service Code' and additional fields are seen:

    • 'Transfer to (service code)' field - Select the new service code from the list.

    • 'Recalculate the unit price, total amount, unit cost and total cost' option - Once enabled, the system will perform recalculation for the entry. Otherwise, only the service code is changed.

    • 'Overwrite current description' option - This option will overwrite the current description with the description of the newly selected Service Code. User can also enter a customised description in the Description box.

    Transfer to another Billing File and/or Activity

    Select this to transfer the service to another Billing File or Activity. Select the Billing File and/or Activity you want to transfer to.

    Note: This option will only be available when the 'All Time/Disbursement Entries Link to an Activity' option is enabled in the Time and Disbursements settings and it will replace the 'Transfer to another Billing File and/or Matter' option.

     

    Proportionally change the 'Total Charge' by entering a new total charge

    Enter the new amount chargeable or percentage for mark up/down. If you enter a new amount, then the system will calculate the percentage, and vice versa.

     

    Reset 'Total Charge' to 'Units' x 'Unit Price'

    Reset the total charge to units multiplied by unit price according to the default.

     

    Change Service Code

    Select this option to change the service code for the time or disbursement entries. Once selected, the following options are seen:

    • 'Transfer to (service code)' field - Select the new service code from the list.

    • 'Recalculate the unit price, total amount, unit cost and total cost' option - Once enabled, the system will perform recalculation for the entry. Otherwise, only the service code is changed.

    • 'Overwrite current description' option - This option will overwrite the current description with the description of the newly selected Service Code. User can also enter a customised description in the Description box.

    Note: This is not available in Time Entries if 'Use Recurring Services for Time Entries' is enabled in the Time and Disbursements settings, and it is not available in Disbursements if  'Use Recurring Services for Disbursement Entries' is enabled. For disbursement entries, it is not possible to change services from Service Class DI to DM and vice versa.

     

    Change Analysis Code

    Select this option to change the analysis code. Enter the new analysis code.

    Note: This option is only available if 'Enable Analysis Code' is enabled in the Time and Disbursements settings, and it is not available in Disbursements.

     

    Recalculate Entry with New Unit Price

    Select this option to recalculate the entries based on a new unit price. Non-chargeable part in the entry, if any, will be recalculated with the same chargeable ratio.

     

 

How do I print the sheet?

Right-click your mouse on the table and select Print...
