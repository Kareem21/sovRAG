Matters

A Matter can be regarded as a project or job defined for a particular Billing File. It can be used as a definition for the type of work done for a client, for example company secretarial work, or it could be a specific project, for example finalisation of accounts for a certain period or to hold funds on behalf of the client.

Separate invoices are issued for different Matters, these can be settled directly with a receipt or payment from the client or can be offset against advance payments already entered for the Matter. Payments to third parties can also be recorded and a running balance of the Matter will be maintained. Matters can be revalued to handle exchange rate differences.

The usage of Matters is enabled by ticking 'Use Matters' option in System Defaults > Practice Management. If you opt not to use Matters, you must still create a general Matter for your Billing Files.

You can set up the Billing File's Matter(s) in the Matters screen.

Matter List

Command buttons

These apply to the Matter list. Click the relevant command button to perform the necessary.

  • New - Click this to add a Matter manually. You can also add a Matter using the Copy Fees command button.

    1. Enter details for the new Matter.

      • Matter Type - Select the Matter type. The selections available are set up in the Matter Type Codes [MTTR] lookup table.

      • Extension - Enter an extension of up to ten characters if you are using the same Matter type more than once. For example, you might want to use a general Matter for different departments and add the department abbreviation to the Matter type, for instance General CS for General Matter for the Company Secretarial department.

      • Matter User Combo - This is a user-definable field that can be used as necessary. Customise the caption and add items for selection in the Matter User Defined Combo [MTUC] lookup table.

      • Description - The description field defaults to the Matter type and extension. Change this if necessary and enter a suitable description for the Matter you are creating. This is the description that is seen by users entering time against the Matter, so it should be as concise and accurate as possible.

      • Billing Currency - Select the invoice currency for the Matter. This is the default currency for invoices raised against this Matter. If your environment is enabled for multi-currency rates, then all time and disbursement entries against the Matter will follow this currency.

      • Revenue Entity - Select a Revenue Entity for the Matter. Revenue Entities must be set up in advance.

        Where do I set up Revenue Entities?

        This is done in Configuration > Business Parameters > Practice Management > Setup > Revenue Entities.

      • Tax Charge Type - Select the tax applicable to services charged to that Matter. This will be applied during Billing Run and ad hoc invoicing. Options are to be set up in the Tax (VAT/GST etc) Charge Types [TCHT] lookup table.

      • Invoice Dispatch - This field can be used to record how the Billing File prefers to receive the invoice. The options are to be set up in the Matter Invoice Despatch [MTCL] lookup table. Note that the field is turned off by default.

      • Remit from - This field is for recording purposes only and options are bank accounts that have been set up and approved in the Billing File's Payer Bank Accounts screen. The field is inactive by default.

      • Remit to - This field is for recording purposes only and options are bank accounts that have been set up and approved in the Payer Bank Accounts screen for the Revenue Entity linked to the Matter.

      • Organisational Unit - Select the organisational unit for the Matter. The options available for selection are maintained in the Organisational Units [ORGUNITS] user table. If you are using WIP Accrual Accounting, then it should be noted that you may link more than one Matter to the same Revenue Entity, but each Matter may have a different organisational unit. Find out more about this in the Accounting help file under the topic Posting Rules - WIP Accrual Posting.

      • Rate List - Select the Rate List for the Matter. This will determine the rate that is used for every service charged to this Matter.

      • Bill WIP at <base currency> - Enter the minimum amount works in progress must reach before they can be posted to an invoice. You can see how this amount affects the invoicing process when you propose drafts of works in progress.

      • Time Charge Limit - Enter the maximum amount that can be charged to the Matter based on the Client Fee Agreement. This amount is reflected in the Time Entries screen within Time and Disbursements component. When creating time entries, the system will warn the user if the total value has exceeded the available amount by highlighting the available value in red. Note that this is not a cumulative limit: once works in progress have been posted to an invoice, the available limit is reset to the maximum amount.

      • Created - Enter the creation date of the Matter. This defaults to today's date.

      • (Expected) Completion - You can enter a completion date for the Matter if it is used for a special project.

      • Administrator - Select the Administrator who created or is in charge of the Matter, if required.

      • Settle Invoices Automatically - Check this box to indicate that if there are funds in hand, you may settle any outstanding invoices. Note that the settlement process must still be done in the usual manner, as Viewpoint will not automatically settle any outstanding invoices.

      • Disbursements are NOT Chargeable - Check this box if you want to record disbursements, but you do not want to charge it to your client.

      • Time is NOT Chargeable - Check this box if you want to record time spent, but you do not want to charge it to your client.

      • Invoice through other - Tick this option if you want invoices for this Matter to be raised under a different Billing File and/or Matter.

      • Invoice Billing File - Specify the Billing File for raising of invoice when the option 'Invoice through other' is used. This field is only enabled and mandatory when the option 'Invoice through other' is ticked.

      • Invoice Matter - Specify the Matter for raising of invoice when the option 'Invoice through other' is used. This field is only enabled and mandatory when the option 'Invoice through other' is ticked.

      • Status - The Matter status enables you to stop using the Matter. If the Matter is on hold, suspended or closed, no draft invoices can be created and existing drafts cannot be posted.

      • Closed - Enter the date that the Matter is closed.

      • Manager - Select the Manager in charge of the Matter, if required.

      • Responsible - Use this field to record persons or entities responsible for the Matter. You can, for example, use this field to record the bank account to which payments for the Matter are made. This field can be used to filter the display results in Settlements - Invoice.

      • Approval/Account Code - Enter the approval or account code for the Matter, if applicable.

      • Unit Code 4 - This field can be turned on to record the organisational code. The options are maintained in the Organisational Codes [ORGCODES] user table which must be defined in Table Code for Unit Code 4 in Accounting > Account File > Unit Codes.

      • Note - Enter any additional notes.

    2. Click Update.

     

    Note: Matter status can also be updated by updating the Billing File status through Status > Billing File screen.

  • Edit - Click this to edit a selected Matter.

  • Delete - Click this to delete the selected Matter from the table. If the Matter is in use by services, or has time or disbursements charged to it, or if there are draft invoices linked to it, you will have to delete them before you can delete the Matter. Once an invoice has been posted to a Matter, you will not be able to delete it. If you want to prevent further charges being made to the Matter, change the Matter status from Open to Closed, Suspended or On Hold.

  • Set as Default - Click this to set the selected Matter as the default. This means every time you create an invoice manually, this Matter will default to the Matter field.

  • Statement - Click this to generate the Matter statement.

  • Copy Fees - Click this to copy Recurring Fees from an existing Billing File. This will open the Copy Fees screen. By default, this command button is inactive.

 

User Fields - Matter Account
