WIP - Select

This is where you select the Matter and date range to retrieve information for the draft invoices.

 

Filter

Select the filter you want to apply and then select the value from the list.

 

Matter Type

Select the Matter you want to retrieve information on.

 

Date - from and To

Enter the date range.

 

Service Type

Select the service type. If no Service Type is selected, then it will include all Service Types.

 

Include -

Tick the boxes for the service classes you want to include in the draft invoice.

  • Recurring Fees -Tick this to retrieve recurring fees that are due.

  • Time Charges - Tick this to retrieve approved time charges entered via the Time Entry screen. Note that the date range entered in the Date - from and - to above will be posted to the Date - from and - to fields in the Detail Lines folder when the entries are posted to a draft invoice. However, if the 'Use Recurring Services for Time Postings' option in System Defaults > Time and Disbursements is ticked, then the Date - from field will be calculated based on the 'Next Invoicing Due' date and the frequency set for 'Charge Per' in the Recurring Fees screen and will not be later than the earliest Time entry.

  • Disbursement Charges - Tick this to retrieve approved Disbursement Charges entered via the Disbursements screen. Note that the date range entered in the Date - from and - to above will be posted to the Date - from and - to fields in the Detail Lines folder when the entries are posted to a draft invoice. However, if the 'Use Recurring Services for Disbursement Postings' option in System Defaults > Time and Disbursements is ticked, then the 'Fee Period From Date' field will be calculated based on the 'Next Invoicing Due' date and the frequency set for 'Charge Per' in the Recurring Fees screen and will not be later than the earliest Disbursement entry.

  • Include Time and/or Disbursement only if Recurring Fees are due - Tick this if you want the Time and Disbursement charges to be posted only if there is also a recurring fee due.

  • Link (and post with invoice) non-chargeable Time and/or Disbursements - Select this to post approved non-chargeable entries from the Time Entries and Disbursement screens so that they are no longer seen in Time and Disbursements. These entries are not posted to any invoice.

  • Roll the next due date forward for 0 value Time and/or Disbursements fees to align with the posting period - This option applies only when you have enabled 'Use Recurring Services for Time Entries' and 'Use Recurring Services for Time Postings' or have enabled 'Use Recurring Services for Disbursement Entries' and 'Use Recurring Services for Disbursement Postings' in System Defaults > Time Management. If ticked, it will update the 'Next Invoicing Due' date field for Time or Disbursements services when the Time or Disbursements service is not invoiced. The update is done when you click on Propose Drafts.

  • Time Cut-off - Enter the cut-off date for Time Charges. Time entries later than this date will not be included. This date will be used as the '-to' date for the invoice detail line for the Time charges. This field is mandatory when the 'Set Time Cut-off' option is enabled.

  • Set Time Cut-off - Tick this to filter approved Time Charges using a cut-off date. The date is to be entered in the Time Cut-off field. Any approved Time Entries after that date will not be included. For more information, click here.
