Settlements - Transactions Folder

Receipts, payments and Matter transactions are displayed here. The screen is divided into three parts. The first part is where you enter your search criteria. The transactions that match the criteria are shown in the table in the middle part of the screen together with command buttons that are related to them. Action, Matter and Line details related to the transaction can be seen by selecting the appropriate tab in the bottom part of the screen.

 

Search and Summary

Enter your search criteria here. The Summary box on the right of the search fields provides a snapshot of the matter balances, outstanding invoices and outstanding drafts for the selected Billing File.

  • Filter - This defaults to the selected Billing File, but you may also choose to display information based on the Billing Address, Client or Revenue Entity in Viewpoint - Home. Otherwise, this is fixed to the selected Billing File.

  • Receipts/Payments/Matter Transactions - Tick the checkboxes that correspond to the information you want to display.

  • Transaction No - Select these option and enter the transaction number you want to look for. For multiple values, separate with a comma.

  • Search/Refresh - Click this to refresh the list displayed in the Transaction table.

 

Transaction list

The transactions that match the search criteria are shown here.

 

Command buttons

Click on the relevant command button to perform the necessary. These apply to the selected transaction from in the Transaction List.

  • Cancel - Click this to cancel the selected receipt, payment or matter transaction.

  • Print - Click to print the receipt.

  • View Journal - Click to display the journal details.

 

For the selected invoice, you can select the tabs below to view additional information:

  • Actions - This tab shows the actions for the transaction, if any. You can also add new action. Refer to  Action Management.

  • Transactions - This tab does not display information for the Transactions folder.

  • Matters - This tab will display the Matter information for the selected transaction.

  • Lines (Details) - This tab will display the line information for the selected transaction.
