Sales Ledger

This section is seen if you ticked the option Time & Billing (Sales Ledger) in Master File > Setup screen to record time and disbursement entries so invoices can be created to charge the Master File. As such, you need to set it up as a Billing File in this section.

For new files, you will only see the following:

  • Cloning - Click this to copy information based on setup in First Bill Cloning and Corporate Fees.

    Information that will be cloned is displayed. Click Update to complete the process.

    The cloning function will complete the following fields:

    General screen

    • Send Invoice to - This defaults to the Billing File but you may change this after the Billing File has been set up.

    • Administrator and Manager - These follow the Administrator and Manager of the clone file.

    • Administrative Mark-up and Standard Payment Term - These are completed based on the settings in the clone file.

     

    Matters screen

    The Matter selected in the First Bill Cloning will be copied here.

     

    Recurring Fees screen

    Government Fees and nominee fees based on the selections made in the Corporate Fees and Nominees setup are copied.

     

    Note: If there is any error, it will be shown in red. You need to fix this before you can continue with the cloning.

  • General - Click this to manually enter Billing File details in the General screen.

Once you have completed the details, you will then see additional folders in Sales Ledger. If you choose to enter the details manually in General screen, you need to also complete the information for the following screens:

  • Matters

  • Recurring Fees, if applicable

  • User Fields, if applicable
