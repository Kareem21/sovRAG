Copy Fees

This function allows for Recurring Fees set up for an existing Billing File to be copied over to a new Billing File. You can access this function through:

  • In File Maintenance screen, click on Sales Ledger Ribbon button and select Billing File > Matters from the navigation listing in Sales Ledger Overview.

  • In File Maintenance screen, click on Sales Ledger Ribbon button and select Billing File > Recurring Fees from the navigation listing in Sales Ledger Overview.

 

Copy From

Select the Master File for the existing Billing File you want to copy from. It is possible to also copy from the same Billing File so you can copy existing Recurring Fees set up for the Billing File for another Matter.

 

Effective Date

Enter the effective date for the recurring fee. This will update the 'Effective Date' column in the Recurring Fee table for the listed recurring fee(s).

 

Add the Fees to the selected Matter

This option can be ticked only if you are performing Copy Fees from Matters screen where you have selected a Matter for the new Billing File. If the Billing File does not have any Matter, then this option cannot be ticked. Instead, you can choose from the Matters table. 

When performing Copy Fees from the Recurring Fees screen, this option also cannot be ticked.

 

Matter Table

If there is no Matter set up for the new Billing File, this lists Matter(s) from the source Billing File. You can therefore tick to select the applicable Matter(s) and the function will copy the Matter(s) as well.

If there is a Matter set up for the new Billing File, then this will be empty.

 

Recurring Fee Table

This lists the Recurring Fees from the source Billing File. Tick to select the Recurring Fees that you want to copy.

For the selected item:

  • Enter the date for 'Next Invoicing Due'

  • The 'From Date' will be the same as the 'Effective Date'

  • The 'To Date' will be calculated based on the setting of 'Charge Per'. If this is One Off [T]', then this be will the same as 'Next Invoicing Due'

 

Command buttons

Click the relevant command button to perform the necessary.

  • Update - Click this to start the process.

  • Cancel - Click this to cancel the process.
