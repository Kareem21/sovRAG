Time Management

Time spent by administrators can be tracked using timesheet which will then be reviewed and either approved or rejected by their manager or supervisor. To start, the period for the timesheet must first be created in the Time Management screen. After this, administrators will then enter time entries which will be included in their timesheet. At the end of the period, the timesheet can then be submitted for approval.

 

Time Management Screen

 

This screen can be accessed through:

    • Viewpoint - Home screen:

      • Click Sales Ledger Ribbon button and select Sales Ledger > Time Management from the navigation listing in the Sales Ledger Overview.

      • In Section Menu, click Home > Practice Management > Sales Ledger > Time Management.

You can click on the following to minimise or expand the sections in this screen:

  • - To minimise this section and hide the command buttons. In the grid, only the selected row will be seen.

  • - To expand the section and show the command buttons.

Filter

The first filter applies to the period list.

  • Status - This refers to the status of the periods:

    • Open - Shows all periods which allows creation of timesheets.

    • Locked - Shows periods which are locked to prevent creation of timesheets in those periods.

    • Approved & Closed - Shows periods where all timesheets in those periods have been approved.

    • All - Shows all existing periods.

 

Period list

This list shows all the timesheet period that has been created.

 

Command buttons

These apply to the period list. Click on the relevant command button to perform the necessary.

  • Create Periods - Click this to create periods for timesheet. The period will be identified as Week-Year and newly created period will be set to Open for its status.

    1. Enter the start date which must be a Monday and the number of weeks to create the period(s).

    2. Then click Start. This will create timesheet for administrators (users) that have been set up in Configuration > Users & Security > Users screen. Timesheet period(s) will only be created for the user if the entered date range falls within the user's 'date from' and 'date to' in the 'Administrators and Rates' grid.

  • Edit - This is to change the status of the selected period.

  • (Re)Open - Click this to re-open closed periods. The button will only be enabled for Approved and Closed periods.

  • Lock - This is active if the selected period's status is Open. Click to set the status to Locked and it will apply to timesheets in the period so no change would be possible for the timesheets.

  • Close - This is active if the selected period's status is not Closed. You can only close the period if all timesheets for the period has been approved and closed. If there are pending timesheets, you would not be able to close the period.

  • Validate - This is used to update the selected period. When clicked, you will see the following options:

    • Create Records / Update Target - Tick this to create timesheet for new administrators as well as to update weekly targets for the selected period. It will create timesheet period(s) if the date range is within the administrator 'date from' and 'date to' in the 'Administrators and Rates' user setting.

    • Delete Not Used - Tick this to remove timesheet that is not required for the period e.g. administrator has resigned or when the period is out of the administrator 'date from' and 'date to' range. This will remove only the timesheet period(s) with the status 'Open [0]'.

    • Recalculate Total - Tick this to recalculate the total units in time entries and posted time entries for the selected period.

  • Global Add - Click this to create non-chargeable time entries for all administrators for the selected period. This can be used to add time entries to record bank holidays for all administrators.

Filter

The second filter applies to the administrator timesheet list allowing you to filter by the status of the timesheet.

 

Administrator timesheet list

The list shows timesheets that has been created for active administrators in the selected period.

 

Command buttons

These apply to the administrator timesheet list. Click on the relevant command buttons to perform the necessary.

  • View - Click this to view the selected timesheet. It will bring you to the Timesheet Folder.

  • Edit - Click this to change the status for the selected timesheet.

  • Select All - Click this to select all timesheets.

  • Deselect All - Click this to deselect selected timesheets.

  • (Re)Open - This is active only if the selected timesheet's status is NOT set to Open. Click to set the status to Open so that it is possible to edit the timesheet (by adding, editing or deleting time entries that are included in the timesheet).

  • Lock - This is active if the selected timesheet's status is Open. Click to set the status to Locked and prevent further changes to the timesheet.

  • Validate - This is available only if the status is set to Open. This Validate function only validates against the selected period instead of validate for all the active periods. When clicked, you will see the following options:

    • Create Records / Update Target - Tick this to create timesheet for new administrators as well as to update weekly targets for the selected period. It will create timesheet period(s) if the date range is within the administrator 'date from' and 'date to' in the 'Administrators and Rates' user setting.

    • Delete Not Used - Tick this to remove timesheet that is not required for the period e.g. administrator has resigned or when the period is out of the administrator 'date from' and 'date to' range. This will remove only the timesheet period(s) with the status 'Open [0]'.

    • Recalculate Total - Tick this to recalculate the total units in time entries and posted time entries for the selected administrator.

  • Approve - Click to approve the timesheet. It will set the status to Approved and Closed. Self-approval is possible if user has the authority 'Approve/Undo Approval of Own T&D'.

  • Reject - Click to reject the timesheet. It will set the status to Rejected. Rejection of own timesheet is possible if user has the authority 'Approve/Undo Approval of Own T&D'.

  • Global Add - Click this to create non-chargeable time entries for the selected administrator(s). This can be used to add time entries to record bank holidays for multiple administrators.

    1. Tick the administrators.

    2. Click Global Add.

    3. Enter details for the time entry.

    4. Click Update.

     

 

Note: The periods are required only if you enable Time Management in System Defaults > Time and Disbursements.
