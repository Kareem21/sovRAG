Nominees

You can charge your clients for any nominee services that you provide. In this screen, you can name the persons who are acting as nominees, and the services that will be added to a Billing File's recurring fees if these persons are nominated.

 

Nominee list

This shows all the nominees that have been set up.

 

Command buttons

Click on the relevant command button to perform the necessary.

  • New - Click to add a new nominee charge.

    1. Enter the details:

      • Nominee - Select the nominee.

      • Jurisdiction - Select the Billing File's country of incorporation.

      • Nominee Type - Select the nominee type from the list. This is maintained in the lookup value table Nominee Types [BRNT].

      • Service - Select the service.

      • Currency - Select the currency for the fee.

    2. Click Update.

  • Edit - Click to edit a selected nominee charge.

  • Delete - Click to delete the selected nominee charge.
