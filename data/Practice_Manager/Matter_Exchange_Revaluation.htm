Matter Exchange Revaluation

This menu allows users to perform Matter Exchange revalue. The 'New Exchange' rate will be populated from the exchange rate master table based on the Revaluation Date. The revalue will be logged in the change log with Change Code 'BMEV'.

The controls in this screen as listed:

  • Revaluation Date - This refers to the date where the 'New Exchange' rate is retrieved from.

  • Revenue Entity - If specified, the system will load all the matters related to the revenue entity. If this is not defined, it will load all the matters.

  • Load Currencies and Rates - Based on the ‘revaluation date’, and ‘revenue entity’ fields defined, system will populate all the records. The system will calculate the below:

    • Exchange column = [CCY Balance] / Base Balance

    • Exchange Gain/Loss = (Matter Balance * [New Exchange]) - (Base Balance)

  • Select All - This button will selects all records in the grid.

  • Deselect All - This button will de-selects all records in the grid.

  • Revalue - This button will post the ‘revaluation receipt’. When clicked, the system will show a ‘Note’ column indicating that the record is successfully posted/failed.

Note: When right-mouse click on the screen, there will be four control options which are Print Preview, Print, Copy and Export to Excel.
