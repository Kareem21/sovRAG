Receivables

The Receivables folder provides you with the ability to record receipts for invoices. This screen can be accessed through:

  • Viewpoint - Home screen:

    • Click Sales Ledger Ribbon button and select Sales Ledger > Accounts Receivable > Receivables from the navigation listing in Sales Ledger Overview.

    • In Section Menu, click Home > Practice Management > Sales Ledger > Accounts Receivables > Receivables.

  • File Maintenance screen:

    • Click Sales Ledger Ribbon button and select Accounts Receivable > Receivables from the navigation listing in Sales Ledger Overview.

    • In Section menu, click Sales Ledger >  Accounts Receivables > Receivables.

You can click on the following to minimise or expand the sections in this screen:

  • - To minimise this section and hide the command buttons. In the grid, only the selected row will be seen.

  • - To expand the section and show the command buttons. If the subtotal line is enabled, it will only be displayed in the expanded view.

Invoices list

This shows sales invoices that have been posted. You can select an invoice by clicking on it and this will show the invoice lines in the Lines (Details). To select multiple invoices, tick the checkbox for the invoice.

 

Command buttons

Click on the relevant command button to perform the necessary.

  • Enter Receipt - Click to enter receipt for the selected invoice. The Receipt on Dr/Cr Note screen will open in a separate window for you to enter the receipt details.

    1. In the Receipt on Dr/Cr Note screen, enter the receipt details.

    2. Click Update.

     

  • Cancel D/C Note - Click this to cancel D/C Note. Either no settlement make before or had outstanding amount and invoice is Debit note (drCr > 0) or document type = CR_N (credit note) with Invoice matter status in 'Open', 'Blocked Time Entry', or 'On Hold'. 

    Credit Note Wizard will appear when user clicks this button, meanwhile Debit Note Wizard will appear when cancelling 'Credit Note [CR_N]' invoice type. User can choose the option to:

    • Create Credit (Debit) Note - Tick to automatically create the credit/debit note. This is ticked by default and will remain ticked.

    • Prompt Posting screen to post the Credit (Debit) Note - Tick to view the posting screen. This allows for you to change the invoice date and/or journal date if necessary. You can also choose to send credit/debit note to Document Queue to be printed.

    • Settle both the original and newly created Credit (Debit) Note - Tick to automatically settle both the invoice and the credit/debit note.

  • Select All - Click this to select all records listed in the list.

  • Deselect All - Click this to deselect the selected records.

  • Print Statement - Click this to print a statement.

    The options that are available depend on the document template:

    • Template with e-mail tags
      Deliver to the Document Queue

      Select this to send the report to Document Queue.

       

      Create an Outlook e-mail and: <option>

      This allows you to generate the document as an e-mail message and:

      • Send e-mail straightaway

      • Place the e-mail in Outlook's Drafts folder

      • Open the e-mail message

       

      View and/or Edit the generated document(s) in Word

      Select this to generate the document and open it in Microsoft Word program.

       

      Remove all e-mail tags and e-mail body from generated document(s)

      Select this to remove e-mail tags and section in the generated Word document. This function will be disabled if the 'Convert e-mail formats and...' or 'Add the document(s) to the E-mail Bin' options are ticked.

       

    • Template with Delivery Manager task
      Deliver to the Document Queue

      The document needs to be generated from the Document Queue screen.

       

    • Template without e-mail tags and Delivery Manager task
      Deliver to the Document Queue

      Select this to send the report to Document Queue.

       

      Generate the document and: <option>

      This allows you to generate the document and have the program:

      • Convert it to PDF format

      • Open the document for viewing

      • Add the document to E-mail Bin

      • Copy the document

      • Open the folder to where the document is sent to (Define this in Location)

       

      Print the generated document to the default printer

      Select this to send the document to be printed.

  • Reprint Invoice - Click this to reprint an invoice for the selected record.

    Deliver to the Document Queue

    This function will deliver the invoice to the Document Queue.

  • View Journal - Click to display the journal details.

Where do I amend the statement template?

Statement templates are found in Configuration > Document Libraries > Billing Templates > General Statement [GS] library.
