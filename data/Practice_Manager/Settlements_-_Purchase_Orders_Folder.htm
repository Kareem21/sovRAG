Settlements - Purchase Orders Folder

Purchase orders that have not been posted are listed here. You can opt to view the invoices according to those that are outstanding, those that are overpaid and those that have been settled. Note that if you are using the Batch/Approval of AP Payment option, settlements will not be reflected as 'Paid' until the proposal has been approved and posted in the Proposal screen. You can edit the purchase order details in the Create/Edit Purchase Orders screen.

 

Search and Summary

Enter your search criteria here. The Summary box on the right of the search fields provides a snapshot of the matter balances, outstanding invoices and outstanding drafts for the selected Supplier File.

  • Filter - The list of items displayed defaults to the selected Supplier File, but you may also choose to filter the information based on the Supplier/Invoice (this is the Master File specified in the Received from field in the Header - General folder when you created the purchase order), or for the Revenue Entity. Refer to Create/Edit Purchase Orders

  • Outstanding, Overpaid and Settled - Tick the checkboxes that correspond to the information you want to display.

  • Invoice No. - This cannot be used here as the purchase orders have not been posted.

  • Draft No. and Reference - Select any of these options and enter the purchase order number or reference you want to look for. For multiple values, separate with a comma.

  • Search/Refresh - Click this to refresh the list displayed in the invoice table.

 

Invoice list

The invoices that match the search criteria are shown here.

 

Command buttons

Click on the relevant command button to perform the necessary.

  • Payment/Receipt - Click this to record a payment.

    1. In the Payment on Cr/Dr Note screen, enter the payment details.

    2. Click Update when done.

     

  • Invoice - Click to print the selected draft purchase order. Once printed, it will be locked for editing. You need to reopen the draft to make further changes.

    The options that are available depend on the document template:

    • Template with e-mail tags

      Deliver to the Document Queue

      Select this to send the report to Document Queue.

       

      Create an Outlook e-mail and: <option>

      This allows you to generate the document as an e-mail message and:

      • Send e-mail straightaway

      • Place the e-mail in Outlook's Drafts folder

      • Open the e-mail message

       

      View and/or Edit the generated document(s) in Word

      Select this to generate the document and open it in Microsoft Word program.

       

      Remove all e-mail tags and e-mail body from generated document(s)

      Select this to remove e-mail tags and section in the generated Word document. This function will be disabled if the 'Convert e-mail formats and...' or 'Add the document(s) to the E-mail Bin' options are ticked.

       

    • Template with Delivery Manager task

      Deliver to the Document Queue

      The document needs to be generated from the Document Queue screen.

       

    • Template without e-mail tags and Delivery Manager task

      Deliver to the Document Queue

      Select this to send the report to Document Queue.

       

      Generate the document and: <option>

      This allows you to generate the document and have the program:

      • Convert it to PDF format

      • Open the document for viewing

      • Add the document to E-mail Bin

      • Copy the document

      • Open the folder to where the document is sent to (Define this in Location)

       

      Print the generated document to the default printer

      Select this to send the document to be printed.

    See also:

    • Document Queue

     

  • Cancel D/C Note - This button is inactive in this folder. To cancel a draft, you need to delete it in Create/Edit Purchase Orders.

  • Print - Click to print the invoice. It will be sent to the Document Queue.

  • Notes - Click to add notes related to provisions made for that particular invoice. Refer to  Settlements - Hold Payment.

    You can record the settlement and bad debt details in the Notes screen. The information entered here can be picked up in Accounting for the Posting Rule function.

     

    Note (general)

    Enter general details for the invoice including notes about the bad debt.

     

    Hold Amount in EUR

    Enter the hold amount.

     

    Hold Payment Note

    Enter the hold payment note.

     

    Update

    Click this to save the information.

     

  • Copy to New PO - Click to copy the Purchase Orders to a new draft Purchase Orders.

 

For the selected draft invoice, you can select the tabs below to view additional information:

  • Transactions - This tab shows the payment/receipt transactions related to the purchase order.

  • Purchase Orders (Details) - The individual detail lines of the selected purchase order are displayed here. Double click a line to see the line information, such as the VAT %, exchange rate and discount given. Refer to Create/Edit Purchase Orders.
