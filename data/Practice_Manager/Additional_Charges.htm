Additional Charges

Apart from taxes, sometimes, there are extra charges that need to be included in the invoices. This is done by using Charge tags which needs to be set up in Configuration > Business Parameters > Practice Management > Setup > Revenue Entities.

The tags allow for you to define:

  • The service that will be added as an additional charge.

  • The percentage or amount of the charge.

  • The items that will be exempted from the charge.

  • Whether or not the charge can be edited or removed.

You can add up to nine charges per revenue entity. If the charge is a percentage, then the calculation will be as follows:

 

First charge = (Total chargeable services + Total VAT amount) x % of charge

Second charge = (Total chargeable services + Total VAT amount + First charge) x % of charge

Third charge = Total chargeable services + Total VAT amount + First charge + Second charge ) x % of charge

 

... and so on.

 

Example:

[Charge1_Service=CHRG1]

[Charge1_Descr=Service Tax]

[Charge1_Percent=12]

[Charge1_InclTax=Y]

[Charge1_Rounding=N0]

 

The tags above will add an invoice line with the description 'Service Tax' in the invoice. This will be calculated as 12% of the chargeable services and VAT amount of the invoice.
