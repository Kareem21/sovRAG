Summary Folder

In this folder, you can see list of the timesheets for selected user(s) and filter the list by the timesheet status. This is seen only if you enable Time Management in System Defaults > Time and Disbursements.

 

Click this to select a different administrator, Administrator Group, User Group, or Organisational Unit.

    • Favourites - Click the desired option.

Filter

You can filter the list by the status.

The available statuses:

  • Open - Shows timesheets that have not been submitted for approval.

  • Rejected - Shows timesheets that have been submitted for approval and rejected.

  • Waiting for Approval - Shows timesheets that have been submitted for approval.

  • Locked - Shows timesheets that have been locked to prevent changes in terms of new time entries being added or existing time entries being edited.

  • Approved and Closed - Shows timesheets that have been approved and closed.

  • All - Shows all existing timesheets.

 

Timesheet list

This list shows timesheet periods that match your filter and selection criteria.

 

Command buttons

Click on the relevant command button to perform the necessary.

  • Timesheet - Click this to view the selected timesheet in Timesheet folder.

  • Submit - Click this to submit selected timesheet for approval. If the target is not met for the selected timesheet, you will receive a prompt to inform you. The timesheet status will be set to:

    • Waiting for Approval if timesheet approval is required.

    • Approved and Closed if timesheet approval is not required.

  • Approve - Click this to approve the selected timesheet. This is seen only if the status is set to 'Waiting for Approval' and user has authority level 8 - Power User.

  • Reject - Click this to reject the selected timesheet. This is seen only if the status is set to 'Waiting for Approval' and user has authority level 8 - Power User.

Note: Users can submit timesheet for others if the authority 'Allow Selection of Other/All Administrators' is enabled for the users' User Group. Users can approve/reject own timesheet if the authority 'Approve/Undo Approval of Own T&D' is enabled for the users' User Group.
