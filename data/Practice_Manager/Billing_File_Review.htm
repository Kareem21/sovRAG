Billing File Review

The File Review function checks whether a Billing File has been correctly set up especially if it is affected by the settings in Review Settings. It is available after a new Billing File has been cloned or set up.

This review can be done at any time to ensure changes in Review Settings take effect. When performing the review, the system will check the file setup and any changes to be made or errors in setup will be shown. These errors can be fixed and the file review can be performed again. When there is no error listed, you will be able to click Update to update the file details.

You can access File Review through:

  • Viewpoint Home screen:

    • Click Sales Ledger Ribbon button and select Sales Ledger > Settings (PM) > File Review from the navigation listing in Sales Ledger Overview.

    • In Section Menu, click Home > Practice Management > Settings (PM) > File Review.

  • File Maintenance screen:

    • Click Sales Ledger Ribbon button and select Billing File > File Review from the navigation listing in Sales Ledger Overview.

    • In Section Menu, click File > Sales Ledger > Billing File > File Review.

In Viewpoint - Home screen, you are able to perform the review on multiple files:

  • Selection folder - Enter details of the files you want to review:

    • Last Selection - Click to show results of the last selection that was reviewed.

    • Entities - Click to perform review for managed entities. This enables the Master File field so you can select the file or Administrator of the file to be reviewed.

      • Automatically Post (Correct/Create) review items - Tick if you want to post review items that are outstanding. This is not available if you choose Last Selection.

      • Entity Review - Select the areas you want to review. You can select from New Entity cloning and Billing, Billing File/Matter/Recurring Fee review and Inactive Entity Files.

    • Billing Files - Click to perform review for Billing Files. This enables the Master File field so you can select the file or Administrator of the file to be reviewed.

      • Automatically Post (Correct/Create) review items - Tick if you want to post review items that are outstanding. This is not available if you choose Last Selection.

      • Billing File review - Select the area to be reviewed: Billing File/Matter Setup or Recurring Fee charge rates.

    • >> Review - Click to begin the review process.

  • Details folder - The review results will be shown here. Select an item to see the results of the review for that Billing File. Errors will be shown in red. You can:

    The results of your selection and review will be displayed in the Details folder. The files that have been reviewed are listed on the top. If there are errors, these are displayed in red at the bottom half.

    • Select All - Click to select all files.

    • Post - Click to post the outstanding items. For example, if the Billing File has a recurring fee that has not been created, clicking Post will create it.

    • Reject - Click to reject the proposed change.

    • Undo Reject - Click to remove the reject on the proposed change.

In File Maintenance, when you click File Review, the system will check only the selected file's setup, and any changes to be made or errors in setup will be shown.
