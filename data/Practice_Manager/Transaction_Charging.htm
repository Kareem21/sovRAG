Transaction Charging

The Transaction Charging screen enables user to maintain fees where the Service Class is time and disbursements-related and the Matter Type is as set in the ‘Matter Type Codes [MTTR]’ Lookup Value. This screen allows user with restricted authority on Recurring Fees screen to set up Time and Disbursements transaction services. The Transaction Charging screen is inactive by default.

Note: This menu is recommended to use with the 'Use Recurring Services for Time Entries' or 'Use Recurring Services for Disbursement Entries' ticked in Configuration > Settings > System Defaults > Time and Disbursements.

Additionally, when ‘Link Services to Matter Types’ is enabled in Configuration > Settings > System Defaults > Practice Management screen, it will limit the services to those linked to the selected Matter Type only.

 

Filter

You can choose to filter the list of Transaction Charging by its status.

Active - Click to view only active recurring fees.

Inactive - Click to view recurring fees that have been deactivated.

All - Click to view all recurring fees.

 

Transaction Charging List

This lists all the Transaction Charging fees set up for the Billing File.

 

Command buttons

Click the relevant command button to perform the necessary.

  • New - Click this to enter a new Transaction Charging.

  • Edit - Change the details of the selected Transaction Charging.

  • Delete - Delete a Transaction Charging from the table. If the Transaction Charging entry has been in used for invoicing, you will not be able to delete the entry.

  • Fee History - Click this to redirect to the Fee History screen.

  • Deactivate/Activate - Click this to deactivate/activate the selected recurring fee. You can only deactivate an active recurring fee to prevent generating future invoices if there is no pending WIP transactions. When deactivating a recurring fee, you must specify the reason in the Reason of Termination field.

  • Select All/Deselect All - Click this to select or deselect all records in the Transaction Charging list.

  • Invoice - Click this to create a draft invoice for the selected fee(s). If the option 'Use Recurring Fees for Time Entries' is ticked, it will post the draft invoice with fixed fee on the Time service in the Transaction Charging. Similarly, if the option 'Use Recurring Fees for Disbursement Entries' is ticked, it will also post the draft invoice with fixed fee on the Disbursement service in the Transaction Charging.
