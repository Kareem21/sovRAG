Review Settings

This section is where you set up the cloning and file review parameters. There are five screens in this section:

  • Corporate Fees - The fees set up here are used to calculate government fees based on an entity's share capital, which will then be added to the Billing File's Recurring Fees screen. If the entity increases or decreases its share capital, the File Review function will pick up the new rate and adjust the recurring fee rate accordingly

  • First Bill Cloning - First Bill clones are set up here for each jurisdiction and may relate to the jurisdiction in general or to a specific client. The default Matter for a new Billing File is taken from the clone. The default Matter is added to the file's Matters screen.

  • Client Fee Agreements - Client fee agreements over-ride the service rates set up in the Service Master, and are used to record preferential rates for a preferred client. This preferential rate is used when setting up a recurring fee, or when creating an ad hoc invoice using that service.

  • Registered Office - This is used if you want to charge clients for using a specified address as the registered office address of their entities. The system checks the address set up in Entity Administration, and if the address is being used, a recurring fee will be set up.

  • Nominees - Viewpoint will verify if nominee services are being provided, and if so, the corresponding service will be added to the Billing File's Recurring Fees screen.
