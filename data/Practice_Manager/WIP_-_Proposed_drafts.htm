WIP - Proposed Drafts

This folder displays the results of the proposed drafts based on the criteria entered in the Select folder.

 

Over WIP limit/All selected BF/Matters

This allows to filter the list based on items that are over the WIP limit or not.

 

Proposed draft list

This shows items that can be invoiced according to the criteria specified in the Select folder.

 

Command buttons

Click on the relevant command button to perform the necessary.

  • Recurring Fees - Click to view the Recurring Fees record for the selected Billing File. The information will be shown at the bottom of the screen.

  • Activity Products - Click to view the Activity record that is not in Time or Disbursements service class for the selected Billing File. The information will be shown at the bottom of the screen.

  • Time Charges - Click to view the Time Entry record for the selected Billing File. The information will be shown at the bottom of the screen.

  • Disbursements - Click to view the Disbursements record for the selected Billing File. The information will be shown at the bottom of the screen.

  • Print - Click to print a report on the selected records.

  • Select All - Click to select all proposed drafts.

  • Deselect All - Click to deselect the selected proposed drafts.

  • Prepare Draft(s) - Click to prepare draft invoices for the selected records.
    If the status for Billing File or the selected matter does not allow for invoicing, you will see the total charges but the draft invoice will not be created. There will be an alert to let you know that you need to reopen the matter or Billing File to create the invoice.

  • Queue Invoice - Click this to queue the proposed drafts. This will generate list of invoices to be processed. You can choose to manually process or wait for the Task Scheduler to automatically process it.

    Invoice Date

    Define the invoice date in the draft.

     

    Journal Date

    Define the journal date in the draft.

     

    Overwrite for (existing) Draft Invoices

    Tick this to overwrite the invoice date or journal date for existing draft invoices.

     

    Add to Existing open Draft Invoice

    Tick this to add to existing draft invoice.

     

    Add Administrative Mark-up

    Tick this to apply administrative markup if applicable.

     

    Invoice Deliver Queue
    Create Draft Invoice

    Select this to create draft invoice.

     

    Create and Approve Draft Invoice

    Select this to create and approve draft invoice.

     

    Create and Post Invoice

    Select this to create, approve and post invoice as final.

     

    Generate and Deliver Draft Invoice

    Tick this to send document to Document Queue. This will have another TSNA task to generate and deliver invoice. Note: This option will be ticked by default for the first time you perform Queue Invoice action. The option in the next action will be determined by the last action selection.

     

  • Post Non-Ch. T&D - Click to post non-chargeable time and disbursement entries to move the entries from the TimeEntry and DisbEntry tables in the database to the TimePost and DisbPost tables. This is done in the back-end and is transparent to the user, but should be done on a regular basis to remove non-chargeable items in the TimeEntry and DisbEntry tables so that they are no longer seen by the user in Time Entries and Disbursement screens.
