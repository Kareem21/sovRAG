Settlements

In Settlements, you can process the draft invoices and invoices raised, and record receipts or payments on Matters. This screen can be accessed through:

  • Viewpoint - Home screen:

    • Click Sales Ledger Ribbon button and select Sales Ledger > Accounts Receivable > Settlements from the navigation listing in Sales Ledger Overview.

    • In Section Menu, click Home > Practice Management > Sales Ledger > Accounts Receivables > Settlements.

  • File Maintenance screen:

    •  Click Sales Ledger Ribbon button and select Accounts Receivable > Settlements from the navigation listing in Sales Ledger Overview.

    • In Section menu, click File > Sales Ledger >  Accounts Receivables > Settlements.

The screen comprises five folders:

  • Invoices - This allows you to search and select an invoice, record a receipt or write off an invoice, cancel an invoice, or settle an invoice to Matter.

  • Drafts - This allows you to search and select a draft invoice, record a receipt or write off a draft invoice, cancel a draft invoice, settle a draft invoice to Matter, or post the draft invoice.

  • Matters - This allows you to search and select a Matter, create a receipt or payment on the Matter, or revalue foreign currency Matters.

  • Transactions - This allows you to search and select a receipt, payment or Matter transaction, or cancel a transaction.

  • Actions - This allows you to view actions for the selected file as well as create new action, edit/delete an existing action, or indicate an action has been completed. Refer to Action Management.

Note: It is possible to set the authority level for the Print button seen in Invoice, Drafts, Matter and Transactions folders to control which users can print. This is to be done in Configuration > Settings > Customise > Field Settings.
