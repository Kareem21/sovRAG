Recurring Fees - Time and Disbursements Services

The option 'Use Recurring Services for Time Entries' or 'Use Recurring Services for Disbursement Entries' in System Defaults > Time and Disbursements allows for filtering of Time or Disbursement services per Billing File. This means that when creating Time or Disbursement entries for a Billing File, users can select only services that are being provided to that Billing File instead of all active Time or Disbursement services.

 

Note:

  • This can be in addition to filtering by Matter through enabling the option 'Link Services to Matter Types' in System Defaults > Practice Management.

  • The system does not allow user to create or edit the Time entry/Disbursement entry with the date later than the Recurring Fee's 'Date Terminated'.

 

Set up the Recurring Fee

When setting up the time or disbursement service, the following fields must be set up as follows:

  • Service - This will be your Time or Disbursement service.

  • Number of Units - Enter '0' for this field.

  • Unit Price - Enter '0' for this field.

  • Next Invoicing Due - Enter the Next Invoicing Due date for this field. Note: If the Next Invoicing Due date is before the Fee Period From Date, it is automatically set as 'Advance'. If the Next Invoicing Due date is after the Fee Period To Date, it is automatically set as 'Arrears'.

 

Calculating the Number of Units to be Charged

The calculation of the number of units to be charges for the Time and Disbursement services depends on whether or not you have the option 'Use Recurring Services for Time Postings' or 'Use Recurring Services for Disbursement Postings' ticked in System Defaults > Time and Disbursements:

  • When the option is not ticked, the draft invoice will include Time or Disbursement entries that are within the date range for Posting of WIP.

  • When the option is ticked, the draft invoice will only include approved Time or Disbursement entries dated before the 'Next Invoicing Due' specified in the Recurring Fees screen. It will not include Time or Disbursement entries that are after the 'Next Invoicing Due' even if they are within the date range set for Posting of WIP.
