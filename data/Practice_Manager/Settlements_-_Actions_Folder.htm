Settlements - Actions Folder

All actions that have been created will be displayed here. The screen is divided into three parts. The first part is where you enter your search criteria. The results from the search is shown in the middle part of the screen. Transaction, Matter and Draft/Invoice details related to the action can be seen by selecting the appropriate tab at the bottom part of the screen.

 

Search and Summary

Enter your search criteria here. The Summary box on the right of the search fields provides a snapshot of the matter balances, outstanding invoices and outstanding drafts for the selected Billing File.

  • Filter - This defaults to the selected Billing File, but you may change this in Viewpoint - Home. Otherwise, this is fixed to the selected Billing File.

  • Search/Refresh - Click this to refresh the list displayed in the Action table.

 

Action list

This shows the actions for the selected file.

 

Command buttons

Click on the relevant command button to perform the necessary. These apply to the selected transaction from the Transaction List.

  • New - Click this to set up a new action.

    1. Enter the required information:

      • File Code - Select the Master File the action is for.

      • Matter Code - Select the Matter the action is for, if applicable.

      • Draft Invoice Number - If the action is not for a specific invoice, enter 0. If any other number is entered, system will check if the number is a valid draft number based on the File Code selected. If the number is a valid number,  the action is deemed to be for the draft invoice of that number. However, if the system is not able to locate any draft invoice with the number, clicking Update will result in the error message 'Invalid Draft Number'.

      • Receipt/Payment Transaction Number - Similar to Draft Invoice Number, enter 0 if the action is not for a specific transaction. Otherwise, enter the receipt or payment transaction number the action is for.

      • Category - Select a category for the action. This is for classification. New categories can be set up in the Practice Management Action Categories [PACT] lookup table.

      • Type - Select the type of action. This depends on the category you have selected. New action types can be added in the Practice Management Action Types [PMAC] lookup table.

      • Description - Enter a description for the action.

      • Date Created - This defaults to today's date.

      • Created by - This defaults to the logged in user.

      • Assigned to - Select the user in charge of the action. The action item will be seen in the assignee's Reminders.

      • Date Due - Enter the due date for the action. This is mandatory if the Assigned to field is completed.

      • Date Completed - This is to be completed when the action is done.

      • Completed - This is ticked when the action is done.

      • Currency - Enter the currency code.

      • Currency Amount - Enter the amount.

      • Base Currency Amount - Enter the reporting currency code.

      • Note - Enter additional notes for the actions.

    2. Click Update.

  • Edit - Click this to edit the selected action.

  • Complete - Click this to indicate the action has been completed.

    1. Enter the Date Completed.

    2. Click Update.

  • Select All - Click this to tick the Select checkboxes for all listed items.

  • Deselect All - Click this to clear the Select checkboxes for all listed items.

  • Delete - Click this to delete the selected action.

 

For the selected Action, you can select the tabs below to view additional information

  • Transactions - This tab shows the receipt/payment transactions related to the Action.

  • Matters - This tab shows the active Matters of the Billing File.

  • Draft/Invoice (Details) - This tab will display the line information for the selected action.
