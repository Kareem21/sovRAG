Timesheet Folder

The Timesheet folder gives an overview of time entries based on the selections you made in the Selection & Options folder. You can enter time in this folder as well as in the Time Entries folder, but you cannot edit nor delete records here.

If you have set up context sensitivity, when you double-click on a particular Billing File and service line, some of the fields in the Time Entries window will be completed for you. You can also click New (see below) to add a new time entry.

 

Timesheet

This shows the selected timesheet. The information here can be pre-set in Selection & Options folder.

 

Command buttons

Click on the relevant command button to perform the necessary.

  • New - Click this to add a new time entry.  Find out more here.

  • Refresh - Click this to refresh the screen.

  • Service Command Buttons - There are three service command buttons that you can click to create a time entry where the service defaults to a predefined service code in System Defaults. These command buttons are available only if Day grouping is selected in Selection & Options. When Week or Month is selected, these buttons will not be seen. For the time entries to have a default Billing File and/or Matter, this is done by using tags for the user account.

    • Public Holiday - This is available only if you have set a default service for Public Holiday in System Defaults > Time Management. Click to indicate as Public Holiday. This will create time entry/entries with the default service for Public Holiday for the selected date.

      1. In the Time Entry screen, note the following:

        • Service field is set to the default service for Public Holiday.

        • Charge/<Unit of measure> field is Nil.

        • Total <Currency> field is Nil.

      2. Complete the rest of the required fields.

      3. Click Update.

      Note: To mark multiple days as Public Holiday, click and drag to select the date range and click Public Holiday. This will create a time entry for each day.

    • Leave - This is available only if you have set a default service for Leave in System Defaults > Time Management. Click to indicate as being on leave. This will create time entry/entries with the default service for Leave.

      1. In the Time Entry screen, note the following:

        • Service field is set to the default service for Leave.

        • Charge/<unit of measure> field is Nil.

        • Total <Currency> field is Nil.

      2. Indicate if it is full day or half day leave using the No. <unit of measure> field.

      3. Complete the rest of the required fields.

      4. Click Update.

      Note: To mark multiple days, click and drag to select the date range and click Leave. This will create a time entry for each day.

    • Internal Meetings - This is available only if you have set a default service for Internal Meetings in System Defaults > Time Management. Click to record when you attend a meeting. This will create time entry/entries with the default service for Internal Meetings.

      1. In the Time Entry screen, note the following:

        • Service field is set to the default service for Internal Meetings.

        • Charge/<unit of measure> field is Nil.

        • Total <Currency> field is Nil.

      2. Indicate the duration of the meeting using the No. <unit of measure> field.

      3. Complete the rest of the required fields.

      4. Click Update.

      Note: To mark multiple days, click and drag to select the date range and click Internal Meetings. This will create a time entry for each day.

  • Submit - Click this to submit the timesheet for approval. This is available only in Viewpoint - Home and if you:

    • access Timesheet folder from Summary folder

    • access Timesheet folder from Selection & Options folder and Day grouping is selected

  • Approve - Click this to approve the timesheet. This is available only in Viewpoint - Home and  requires authority level 8 - Power User. It is not seen if you access the Timesheet tab from Selection & Options tab and Week or Month grouping is selected. If the option 'Require Approval for Weekly Timesheet' in System Defaults - Time Management is not ticked, there is no need for approval so this will not be available.

  • Reject - Click this to reject the timesheet. This is available only in Viewpoint - Home and requires authority level 8 - Power User. It is not seen if you access the Timesheet tab from Selection & Options tab and Week or Month grouping is selected. If the option 'Require Approval for Weekly Timesheet' in System Defaults - Time Management is not ticked, there is no need for approval so this will not be available.

 

Can I print this out?

You can right-click and select Print Preview. Alternatively, you can also export this to Microsoft Excel - right-click and select Export to Excel.
