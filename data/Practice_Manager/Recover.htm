Recover

You can recover supplier charges from your clients by using the Recover function. Access the function from the Create/Edit Purchase Order screen or the Recoverables screen. You can recover the charges from one purchase order from different Billing Files. These recovery entries will appear in the Billing File's disbursement items, and will be picked up at the next Posting of WIP run.

The Recovery screen opens as a separate window.

 

Disbursement Entry/Invoiced Disbursement

You can filter the recovery entries according to those that have not yet been invoiced; or those that have already been invoiced.

 

Disbursements

This lists the disbursements created for the recovery.

 

Mark as Recovered

This checkbox is automatically ticked when the amount recovered equals or exceeds the amount on the purchase order. You can also set this to recovered even if the actual amount has not been recovered. Marking an item as recovered does not have any bearing on the system.

 

Command buttons

Click the relevant command button to perform the necessary.

  • New - Click this to create a new Disbursement entry. Find out more here.

    When you click New to create a new disbursement line, the Disbursement screen will appear as a separate window for data entry.

     

    Administrator

    The name of the administrator defaults to the current user and cannot be changed.

     

    Date

    This defaults to today's date. Change this if necessary.

     

    Billing File

    Select the Billing File the recovery amount is to be charged to.

     

    Matter

    This field will display the default Matter of the selected Billing File. Change this if necessary.

     

    Service

    This defaults to the service of the selected purchase order or invoice and cannot be changed.

     

    Currency

    The currency of the selected Matter defaults to this field.

     

    No. of Units

    Enter the number of units/cost to be recovered.

     

    Exchange

    The exchange rate from the Matter to billing currency is shown here. This field is disabled.

     

    Charge/Unit

    Enter the charge/unit or mark up amount.

     

    Total

    The total amount of the recovery is automatically calculated and displayed here.

     

    Written Off

    This field is disabled.

     

    Unit Cost

    Enter the cost/unit if relevant.

     

    Total Cost

    The total cost is automatically calculated and displayed here if relevant.

     

    Description

    The description of the service defaults to this field. Change this if necessary.

     

  • Add -  Click this to edit the selected item. Find out more here.

    This feature is used to add/assign existing Disbursement(s) that are mistakenly assigned to other PO.

     

    Billing File

    Select the Billing File to search.

     

    Date -from

    Enter the 'date from' for the Disbursement search.

     

    To

    Enter the 'date to' for the Disbursement search.

     

    Disbursement Entry

    Select this option to search disbursement entries that are yet to be invoiced.

     

    Invoiced Disbursement

    Select this option to search disbursement entries that have been invoiced.

     

    Select

    Click this to 'link' the Disbursement to the PO.

     

    Cancel

    Click this to close the 'Select Disbursement Entry' box.

     

  • Edit - Click this to edit the selected item.

  • Remove - Click this to remove the selected item.
