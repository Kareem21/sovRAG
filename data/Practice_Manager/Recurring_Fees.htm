Recurring Fees

This screen enables you to maintain fees that are billed at regular intervals such as annual fees, bi-annual fees, quarterly fees, monthly fees and so on. The term recurring fee is used for these regularly charged fees.

The recurring fees feature includes:

  • The option to automatically add Time and Disbursement charges to a recurring fee invoice.

  • Pro-rata billing.

  • The ability to set up different templates for managed and non-managed entities.

  • Currency conversion for the recurring fee item if the currency is different from the invoice currency. The conversion rate is based on the exchange rates set up in the Exchange Rate Master.

  • Automatic updating of the Next Invoicing Due date for the fee when an invoice is posted, to prevent invoicing errors.

However, it is also possible to use this screen to filter the list of Time and Disbursement services applicable for a Billing File. This requires for the option 'Use Recurring Services for Time Entries' or 'Use Recurring Services for Disbursement Entries' to be enabled in System Defaults > Time and Disbursements. This way, when users create Time or Disbursement entries, they will only see the Time or Disbursement services for the selected Billing File.

You can click on the following to minimise or expand the sections in this screen:

  • - To minimise this section and hide the command buttons. In the grid, only the selected row will be seen.

  • - To expand the section and show the command buttons.

 

Filter

You can choose to filter the list of recurring fees by its status.

Active - Click to view only active recurring fees.

Inactive - Click to view recurring fees that have been deactivated.

All - Click to view all recurring fees.

 

Recurring Fee List

This lists all the recurring fees for the Billing File. You can filter the list by recurring fee status.

Active - Click to view only active recurring fees.

Inactive - Click to view recurring fees that have been deactivated.

All - Click to view all recurring fees.

 

Command buttons

These apply to the recurring fees list. Click on the relevant command buttons to perform the necessary.

  • New - Click this to add new recurring fee. Find out more here.

  • Edit - Click this to change the details of the selected fee.

  • Delete - Click this to delete the selected recurring fee.

  • Fee History - Click this to redirect to the Fee History screen.

  • Deactivate/Activate - Click this to deactivate/activate the selected recurring fee. You can only deactivate an active recurring fee to prevent generating future invoices if there is no pending WIP transactions. When deactivating a recurring fee, you must specify the reason in the Reason of Termination field.

  • Copy Fees - Click this to copy Recurring Fees from an existing Billing File. This will open the Copy Fees screen. By default, this command button is inactive.

  • Select All/Deselect All - Click this to select or deselect all recurring fees.

  • Invoice - Click this to create a draft invoice for the selected fee(s). If the option 'Use Recurring Fees for Time Entries' is ticked, it will post the draft invoice with fixed fee on the Time service in the Recurring Fees. Similarly, if the option 'Use Recurring Fees for Disbursement Entries' is ticked, it will also post the draft invoice with fixed fee on the Disbursement service in the Recurring Fees. However, for Recurring Fees service (that is overdue with amount), it will always create a draft invoice regardless of whether 'Use Recurring Fees for Time Entries' or 'Use Recurring Fees for Disbursement Entries' is ticked/unticked. Note: This does not apply to Ad Valorem Fee Service Class.

 

Service Rate Table

This table will show a history of the effective dates and unit prices for the selected recurring fee. You can add a new record by using the Global Change function in the Service Master, or by using the New function (see below). You need to enable the 'Use Recurring Fee - Unit Price/Effective Date' option in Configuration > Settings > System Defaults > Practice Management in order to enable the function buttons below the table.

How do I add time and disbursement charges to an existing recurring fee draft invoice?

In order to do this, the time charges and disbursements must be in the same Matter as the recurring fee. Tick the Add to existing open Draft Invoice check box when preparing the draft. Find out more here.

 

How do I invoice more than one fee at the same time?

  1. Select the fees you want to invoice by ticking the checkbox next to those fees.

  2. Click Invoice.

  3. If the selected fees are linked to the same Matter, they will all be placed in the same draft invoice. If the selected fees are from different Matters, then you will get one invoice per Matter.

Where do I set up the templates that I want to use for recurring fees?

This is done in Configuration > Business Parameters > Practice Management > Document Templates. Make sure that you have already set up the template you want to use in the Configuration > Document Libraries > Billing Templates - Debit/Credit Note [DC] section.
