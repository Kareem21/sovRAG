First Bill Cloning

The First Bill Cloning feature is used to automatically set up Billing Files and Matters for new entities, effectively cutting down the time required on a manual process. This function works only for new Billing Files, as it sets up the information in the General screen, and creates a default Matter to which any recurring fees can be added. You will need a source Billing File from which to copy the information, so you might want to set up a specific file for this purpose.

 

File clone list

This shows all the cloning files that have been set up.

 

Command buttons

Click on the relevant command button to perform the necessary.

  • New - Click to create a new clone.

    1. Enter the details:

      • Jurisdiction - Select the country of incorporation.

      • All Other Clients or Client/Group - If the clone is being set up for all clients, tick this and the Client/Group field will be disabled. If the clone is for a specific client, select the client's name. Viewpoint will first check for a client-specific record before using the 'All other Clients' record.

      • Billing File - Select the Billing File to be used as the source for the cloning.

      • Matter - The matters set up for the source Billing File will be shown here. Select the matter you want to clone. Only one matter will be set up when you clone.

      • Tags - This is not in use.

    2. Click Update.

  • Edit - Click to edit the clone information.

  • Delete - Click to delete the clone.
