Draft Invoices - Creation

There are several ways that you can create draft invoices.

  • Invoicing Recurring Fees - This can be done by selecting the fees you want to invoice from Sales Ledger > Recurring Fees screen and clicking Invoice.

  • Invoicing Time and Disbursements - The invoicing of time charges and disbursements entered in the Time and Disbursements component is done in the Posting of WIP screen. Once you have retrieved the information on the outstanding works in progress, you can select the ones you want to invoice and click the Prepare Draft(s) button.

  • Manual/Ad hoc invoicing - This is done in the Sales Ledger > Invoicing > Create/Edit Invoice screen.

  • Invoicing Recurring Fee penalties - This can be done in the Fees Payable screen or the Invoice and Drafts folders of the Settlements screen.

 

If you create a draft in the Recurring Fees screen or in the Propose Drafts folder of the Posting of WIP screen, then you will see a separate window with options for creating the draft. You can edit informations if the draft invoice is created through Posting of WIP or Invoice command button in the Recurring Fees screen.

 

Invoice Date, Journal Date and Overwrite for (existing) Draft Invoices

When you create a new draft invoice, the invoice and journal dates default to the date when the earliest fee or charge is due. You may overwrite these dates by entering the date you want in the Invoice Date and Journal Date fields, and ticking the corresponding Overwrite for (existing) Draft Invoices checkboxes.

 

Print Document (to document queue)

Tick this checkbox to send a copy of the draft to the Document Queue for printing or editing.

 

Add to existing open Draft Invoice

Tick this checkbox to add this to an existing open draft invoice of the same Matter. If you do this, the invoice follows the date of the existing draft and not the date of the fee or charge. You can add items to an existing draft provided the following criteria are met:

  • The existing draft has the same matter as the WIP items.

  • The document type code for the existing draft does not start with 'M', for instance Matter - Advance Payment [M_AP], and is not a credit note.

  • Recurring fee items will only add to existing recurring fee invoice drafts of the document type A_CS or A_GE. However, Debit Note [DR_N] will be used when A_CS or A_GE are inactive.

  • The existing draft has not yet been approved, if the approval feature has been switched on.

  • The existing draft is not still open, i.e. it has not been printed.

 

Add Administrative Mark-up

Tick this checkbox to add an administrative mark-up to the invoice.

 

Start

Click Start to begin the process.

 

I have created a draft invoice. How do I edit the details of the invoice?

This is done in the Sales Ledger > Invoicing > Create/Edit Invoice screen. Find out more here.

 

There are some mistakes in the draft of my recurring fee invoice and I want to regenerate the invoice. However, nothing seems to happen when I click the Invoice button. What do I need to do?

If the fee has already been posted to a draft invoice, you need to delete that draft first, then regenerate the invoice.
