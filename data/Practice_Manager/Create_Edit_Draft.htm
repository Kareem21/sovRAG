Create/Edit Draft

This is only available in Viewpoint - Home screen and allows you to:

  • Manually create ad hoc invoices.

  • Edit or delete draft invoices that have not been approved as they have not yet been posted to the invoice ledger so the changes will not affect the ledger.

Once a draft invoice has been posted, you will not be able to edit or delete them; you would need to create a credit note to offset the amount of the invoice you want to edit or delete.

Draft invoice list

Lists all draft invoices created for multiple Billing Files.

 

Command buttons

Click the relevant command button to perform the necessary.

  • New - Click this to create an ad hoc invoice.

    1. Enter the details for the invoice.

    2. Click Update.

    3. Enter the lines by clicking New in Details table.

  • Edit - Click this to edit the selected draft invoice.

  • Reopen - Click this to reopen a locked/closed draft invoice. Draft invoices are locked when you click Print button.

  • Delete - Click this to delete the selected draft invoice.

    Click Yes to confirm the deletion.

  • Print - Click this to print the selected draft invoice. Once a draft has been printed, it will be locked for editing. You need to reopen the draft to make further changes.

    The options that are available depend on the document template:

    • Template with e-mail tags

      Deliver to the Document Queue

      Select this to send the report to Document Queue.

       

      Create an Outlook e-mail and: <option>

      This allows you to generate the document as an e-mail message and:

      • Send e-mail straightaway

      • Place the e-mail in Outlook's Drafts folder

      • Open the e-mail message

       

      View and/or Edit the generated document(s) in Word

      Select this to generate the document and open it in Microsoft Word program.

       

      Remove all e-mail tags and e-mail body from generated document(s)

      Select this to remove e-mail tags and section in the generated Word document. This function will be disabled if the 'Convert e-mail formats and...' or 'Add the document(s) to the E-mail Bin' options are ticked.

       

    • Template with Delivery Manager task

      Deliver to the Document Queue

      The document needs to be generated from the Document Queue screen.

       

    • Template without e-mail tags and Delivery Manager task

      Deliver to the Document Queue

      Select this to send the report to Document Queue.

       

      Generate the document and: <option>

      This allows you to generate the document and have the program:

      • Convert it to PDF format

      • Open the document for viewing

      • Add the document to E-mail Bin

      • Copy the document

      • Open the folder to where the document is sent to (Define this in Location)

       

      Print the generated document to the default printer

      Select this to send the document to be printed.

    See also:

    • Document Queue

     

  • Invoice - Click this to invoice the selected draft invoices.

    1. In the Invoice Posting window that appears, complete the following information:

      • Invoice Date, Journal Date and Overwrite for (existing) Draft Invoices - When you post an invoice, the invoice and journal dates default to the date when the draft was created. You may overwrite these dates by entering the date you want in the Invoice Date and Journal Date fields, and ticking the corresponding Overwrite for (existing) Draft Invoices checkboxes.

      • Print Document (to document queue) - Tick this checkbox to send a copy of the invoice to the Document Queue for printing or editing.

    2. Click Start to begin the process.

    Note:

    When 'Use Recurring Services for Time Postings' and 'Use Recurring Services for Disbursement Postings' options are enabled in Home > Configuration > System Defaults > Time and Disbursements, the invoice will be created with the following type:

    • If Entity Administration component is activated - an invoice with document type 'Recurring Fee Invoice Entity [A_CS]' will be created.

    • If Entity Administration component is not activated -  an invoice with document type 'Recurring Fee Invoice Non-Entity [A_GE]' will be created.

    When the options are disabled, the invoice will be created with the type 'Debit Note [DR_N]'.

 

Invoice lines

Lists all individual lines of the selected draft invoice.

 

Command buttons

Click the relevant command button to perform the necessary.

  • New - Click this to add a new line for the selected draft. You may repeat this step as many times as necessary.

    Enter the details for the detail line. Click Update when done.

  • Edit - Click this to edit the selected line for the draft.

  • Delete - Click this to delete the selected line for the draft.

  • Line Source - Click this to view lines pulled from Time and Disbursements. This is only available if the draft invoice contains such lines.

    The Source screen will be displayed separately to show the lines. These detail lines that originate from Time and Disbursements can still be edited when they are in the draft invoice stage.

    Note that certain fields such as number of units and unit price are disabled and cannot be changed.

     

    Add

    Click to add another entry into the selected draft invoice. This is useful if the entry was only approved after the draft invoice has been created. Note that the entries must be from the same Billing File, Matter and is of the same Service Code as the draft invoice line.

    This displays approved entries that were not included in the draft invoice. Tick the entry or entries to include and click Select.

     

    Edit

    Click to edit the selected line.

     

    Remove

    Click to remove the selected item.

     

    Close

    Click to close the Source screen

  • Non Chargeable - Click this to view or add non-chargeable time entries. The date for time entries must be earlier or the same as the Date - to of the invoice and the time entries must be approved. This is done through the Non Chargeable window.

    1. Click Add.

    2. Select the time entry by ticking the checkbox.

    3. Click Select.

 

Checkbox

  • Add Administrative Mark-up - Tick this to include Administrative Mark-up to the invoice line. Enabling this option will create Administrative Mark-up as a separate invoice line. This checkbox is available when the ‘Add Administrative Markup’ is checked in the Practice Management settings.
