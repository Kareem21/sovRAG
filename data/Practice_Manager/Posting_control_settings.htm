Posting Control Settings

Before you start to enter time and disbursement entries or post draft invoices, you need to specify the posting window. To do this, access the Posting Control Settings screen through:

  • Viewpoint - Home screen:

    • Click Sales Ledger Ribbon button and select Sales Ledger > Settings (PM) > Posting Control Settings from the navigation listing in Sales Ledger Overview.

    • Click Purchase Ledger Ribbon button and select Purchase Ledger > Settings (PM) > Posting Control Settings from the navigation listing in Sales Ledger Overview.

    • In Section Menu, click Home > Practice Management > Posting Control Settings.

 

Practice Management

Tick this to update items for Practice Management. This is seen only if you have the Practice Management licence.

  • Journal - Date Open, Date Close - Enter the date range. The system will perform validations and if applicable, you will be informed of the following:

    • Unposted invoices, receipts and payments outside the date range.

    • Posting window of revenue entities that are outside this date range.

  • Update to other Open/Close Dates - Tick this to apply the date range for Journal to WIP - Time and Disbursements as well.

  • WIP Time - Date Open, Date Close - Enter the date range in which you can post time entries.

  • Disbursements - Date Open, Date Close - Enter the date range in which you can post disbursement entries.

 

Accounting

Tick this to update items for Account File(s). This is seen only if you have the Accounting licence.

  • Journal - Date Open, Date Close - Enter the date range for posting of journals to Account Files. The system will perform validation of the Account File(s) current date range.

  • Open Date must be after Last Year-End - Tick this to have the system assign the Date Open to the day after the Account File's Last Year-End date.

  • Apply to - Select the option to specify the Account File(s) where the changes will be applied.

    Option

    Explanation

    Selected Account File

    This allows you to select only one Account File and apply the changes to this Account File. The default value in the Account File will be the default revenue entity.

    All Revenue Entities

    This will apply the changes to all revenue entities.

    Administrator

    This allows you to select a user and apply the changes to all Account Files where the user is the Account File Administrator. By default, this is the logged-in user.

    Manager

    This allows you to choose a user and apply the changes to all Account Files where the user is the Account File Manager. By default, this is the logged-in user.

    Controller

    This allows you to choose a user and apply the changes to all Account Files where the user is the Account File Controller. By default, this is the logged-in user.

    Client/Group

    This allows you to select a Master File as the Client/Group and apply the changes to all Account Files that belong to the Client/Group.

    Administrator Group

    This allows you to select an Administrator Group and apply the changes to all Account Files in that Administrator Group. By default, the Administrator Group is the Administrator Group of the logged-in user.

    Chart of Account

    This allows you to select a Chart of Account template and apply the changes to all Account Files using that Chart of Account template.

    All Account Files

    This applies the changes to all Account Files.

     

  • Load Grid - Click this to display the Account File(s) that will be updated.

  • List of Account Files - This shows the Account File(s) where the changes will be applied. The information includes current date range and the new date range as specified. If applicable, you will be informed of:

    • Posting Date Open before Last Year-End.

    • Unposted transactions.

    • Account File's posting window that is outside the date range defined for Practice Management's Journal - Date Open and Date Close.

  • Select All - Click this to select all Account Files in the list.

  • Deselect All - Click this to undo all selection.

 

Command buttons

Click on the relevant command button to perform the necessary.

  • Update - Click this to update the changes.

  • Cancel/Close - Click this to cancel the changes. If no change was made, click this to exit the screen.
