Checklist Item - Help/Description

This is where you can provide information for the checklist item in the Help column and it can be:

  • Plain text including letters, numbers, symbols and special characters.

  • A tag to retrieve existing help texts.

  • HTML tag.

Tag to Retrieve Help Text

Help texts corresponding to standard Viewpoint fields can be retrieved to show in the Help column by adding the following tag:

[SHOWHELP=<ScreenID>.<DBField>]

where

  • <ScreenID> refers to the screen where the help text exists.

  • <DBField> refers to the field of the help text.

To obtain the ScreenID,

  1. Go to the screen containing the field. If the screen contains a grid, click New.

  2. Right-click and select Copy.

  3. Right-click again and select Paste.

  4. The Viewpoint Clipboard appears to show the ScreenID.

To obtain the DBField, go to Configuration > Settings > Customise > Field Settings.

 

Example - Step to retrieve the Help Text for 'Use Name Format' field

Enter the tag in the Help/Description column of the checklist item and the help text for the item will be seen in workflow as configured.

 

HTML Tag

HTML tags can be added to allow for the following:

  • Formatting the Help Text such as bolding/italicising/underlining the text:

    • Bold - By including the syntax <strong>...</strong>.

    • Italic - By including the syntax <em>...</em>.

    • Underline - By including the syntax <u>...</u>.

  • Including a website link in the Help Text by adding the <a> tag with the href attribute and the website link within "…".
