Account Movement Codes

This screen lists the Account Movement Codes according to the Account Type and this is where you can customise the movement codes.

 

Filters

The filters apply to the account codes list.

  • Type - You can choose to view account codes specifically for an account type or all.

  • Customised setting - You can choose to view account movement codes that have been customised or not.

 

Edit button

Click this to make changes to the selected account movement code.

  1. Select the movement code.

  2. Click Edit.

  3. Customise as desired. You can do the following:

    • Customise description for the movement code

      Type the new description over the current one. The change will be seen the journal entry screen and Analyses screen.

    • Enable/disable Tax and Quantity option

      If Tax is ticked, in journal scripts, the Tax option will also be ticked. In manual journals for this account movement code, the field Tax account will be seen.

      Note: If the journal script requires a tax account to be selected or contains a line that has Tax enabled, then the Tax option will be ticked. For manual journals, even if the Tax option has been disabled for the account movement code, if the Account File has Tax account sub-ledger, the field Tax Account will still be seen.

       

      If Quantity is ticked, in journal scripts, the Quantity option will also be ticked. In manual journals for this account movement code, the field Quantity will be seen. However, note that this also requires sub-ledger to have the option 'Control Quantity' ticked.

      Note: Quantity is applicable to sub-ledgers of the Fixed Assets, Securities, Investment, Share/unit Holder and Products/Services categories

    • Customise captions for fields in journal entries

      Type the new caption for the relevant field over the current caption. The change will be seen the journal entry screen after you select the movement code.

  4. Click Update.
