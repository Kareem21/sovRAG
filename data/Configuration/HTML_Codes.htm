HTML Codes

This refers to standard HTML codes used to define the design elements of the template. When adding HTML codes, follow the conventional syntax where the code will be enclosed within '<>'. The use of HTML codes allow for the following in the template:

  • Defining the style used in the template

  • Adding headings

  • Adding links

  • Defining table properties

For links, typically, this is to a website address but it can also be used as a link to Viewpoint screens by specifying SH.<ScreenID> e.g. <a href="SH.FSEAMEET" class="btn btn-link">

 

Commonly-used HTML Codes

The default CSS framework is Bootstrap CSS framework version 3.3.1 and the more commonly-used HTML codes are described below.

<h#>Heading</h#>

where # refers to 1, 2, 3 or any number specified in the style sheet, and Heading is the title that will appear in the screen.

<h1>Master File Information Sheet – (+ftype=evw|HD.FullName+)</h1>

 

<div>grouped information</div>

This is used to group information together, such as bookmarks. See class=”style” for example.

 

id="heading ID", name=”name”

These codes are used to identify the header, and can be referred to in a bookmark.

<h2 id=”favGen” name=”favGen”>General Information</h2>

 

<a href="#heading ID">Hyperlink text</a>

This code is used create hyperlinks.

This code will link to an id in the template:

<a href="#favGen">General Information</a>

This code on the other hand, is prefixed with SH. and will link to a screen in Viewpoint

<a href="SH.FMMCOMPN" >Compliance Details</a>

You can get the screen ID that you want to link to, from the VpSysMenu table in the database.

 

class="style"

This is used to determine the style used for that section.

<a class="btn btn-link"><span class="glyphicon glyphicon-share-alt"></span> For more details see Client information</a>

 

Important note:

Use data-role="collapsiblePanel" to allow a section to expand/collapse where the data-target must point to the id of the panel body.

<div class="panel panel-info" id="favClient">

      <div class="panel-heading">

         <h3 class="panel-title" data-target="#pnlClient" data-role="collapsiblePanel">Client Information</h3>

      </div>

      <div id="pnlClient" class="panel-body">

         <content>

      </div>

</div>

 

<table>, <tr>, <td>

These codes are used for tables. Here, <table> refers to table, <tr> refers to table row, and <td> refers to table data. In the following example:

  • width=”x%” is the percentage of the table width for the column

  • colspan=”3” means that the selected column is to be spread over 3 columns

  • &npsb; is used to place a blank space in the column if the query used for that section returns a blank

  • Align=”center” is used to align the information to the centre of the cell.

<table class="table table-hover table-bordered">

    <tr>

        <td width="22%" align="center"  class="active">Company name</td>

        <td width="78%">(+ftyp=evw|HD.EntCode\#AD-0-0+)</td>

    </tr>

    <tr>

        <td width="22%" class="active">(+ftyp=evw|DT.AddressMF+)</td>

        <td width="78%">&nbsp;</td>

    </tr>

    <tr>

        <td width="22%" class="active">Residential Address</td>

        <td width="78%" colspan="3">(+ftyp=evw|HD.RefCode\#AD-42-1+)</td>

    </tr>

</table>
