Document Templates

You can specify different templates for the various invoices, statements, notes and vouchers that each revenue entity will use. These templates are set up in the Billing Templates section in the Document Library. If you need to issue these documents in multi-languages, this can also be set up here. This means that you can issue invoices in Greek or German to clients who prefer their documents in such languages instead of English.

 

Languages

All active languages from the Language Codes [LANG] lookup value table are listed here, and each available revenue entity is repeated under each language. This is so that you can specify exactly which template is to be used for each of the revenue entities for each of the languages present. You need only to specify the templates for the languages that you use.

 

Template Type

The available template types will be listed in the appropriate sections. In the Sales & Receivables section, the active templates in the Invoice Types (sales) [INVT] lookup value table will be listed. Similarly, in the Purchase & Payables section, the active templates in the Purchase Order Types [PURT] lookup value table will be listed.

  1. Select the Language and Revenue Entity you are setting up the templates for.

  2. In the Template Type table, click the Draft/PO/Template field next to the document type. This will enable a drop down list of the available templates you can link to the document type.

  3. Once you have set the templates for each of the document types, click Save and then Close to exit the screen.

 

Command buttons

Click on the relevant command button to perform the necessary.

  • Save - Click this to save the changes made.

  • Cancel - Click this to cancel the changes.

  • Close - Click this to close the screen.
