Service Tax Codes

This screen is to map the tax code to the service for the different revenue entities. It is inactive by default. If you are using the option 'Use Accounting Tax Sub-ledger as Tax Codes', ensure that the screen is accessible as it will be activated once the option is enabled.

To access this screen, go to Configuration > Business Parameters > Practice Management > Service Tax Codes.

 

Options

Select the relevant option to specify the mapping for Sales Invoicing (for Sales Ledger) and Supplier Invoicing (for Purchase Ledger).

 

'Copy from Revenue Entity' Options

Tick this to copy the tax codes mapping from one revenue entity to another.

  1. In Service Tax Codes screen, select the current revenue entity.

  2. Tick 'Copy from Revenue Entity' Options.

  3. Select the revenue entity that you want to copy from.

  4. Click Update Sheet.

  5. Click OK when complete.

  6. Click Update.

 

Revenue Entity

Select the revenue entity whose tax codes mapping you want to view or edit.

 

Mapping table

The mapping table comprises the following:

  • Description - This lists your services by the Tax Charge Type (set up in Tax (VAT/GST etc) Charge Types [TCHT] lookup table and the Service Class. Click on the plus (+) sign to expand.

  • Code - This is the service code.

  • Revenue Entities - All active revenue entities will be listed. For each revenue entity, select the relevant tax code for:

    • Tax Charge Type - Click on the arrow down button for the Tax Charge Type and select a tax code. This applies the same tax code for all services in the Tax Charge Type.

    • Service Class - Click on the arrow down button for the Service Class and select the tax code. This applies the same tax code for all services under the Service Class of that Tax Charge Type.

    • Service - Click on the arrow down button for the service. This applies the tax code ONLY to that service within the Tax Charge Type.

      Note: The available options are tax sub-ledgers that have been set up for the revenue entity. See also Sub-ledger Accounts - Setup.

 

Command buttons

Click on the relevant command button to perform the necessary.

  • Update  - Click this to save changes made. This button is activated after you have edited a field in the table.

  • Cancel - Click this to cancel changes made. This button is activated after you have edited a field in the table.

  • Copy Sheet - This button is disabled when you amend the information in the table and is enabled after you have either saved or cancelled the change. The information copied follows what is displayed in the table. Paste the information to an Excel spreadsheet for further action.
