Convert SQL function

The convert SQL function streamlines SQL statements for both MS Access and MS SQL and is performed only when you refresh a form using Refresh/Options function. Use this to refresh your forms when you upgrade from MS Access to MS SQL.

The Convert SQL function will NOT perform the following:

  • Convert bit values like: "First Officer" to "First Officer<>0", "FirstOfficer=True" to "FirstOfficer<>0".

  • Convert the usage of Format functions in MS Access.
