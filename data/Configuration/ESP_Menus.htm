Menus

This is where configuration of menus for CP-WP is done.

 

Filters

The filters apply to the menu list.

  • Menu status - You can choose to view only active, inactive or all menus.

  • Menu Group - You can choose to view menus from a specific Menu Group or all.

 

Menu list

The list shows the menus available in ESP.

 

Command buttons

Click on the relevant command button to perform the necessary.

  • New - Click this to set up a new menu for ESP.

  • Edit - Click this to edit properties of the selected item.

    You can edit the following:

    • Caption

    • Description

    • Tag Values

    • Apply to File Where

      This setting is only available for Menu Group 'File Menus [12]'.

      It defines whether the menu will be shown in the ESP for Master File with the respective component enabled in the Back-office.

      For example, Master File XYX has Entity Administrator component enabled in the Back-office and menu A has the Entity Administrator component enabled. Hence, in the ESP, menu A will show when Master File XYZ is opened.

      The following options (components) can be selected to allow the menu to be shown in the ESP:

      • Entity Administrator

      • Time & Billing (Sales Ledger)

      • Supplier (Purchase Ledger)

      • Accounting

      • Non-Managed File

      Note: When upgrading from a lower version (below version 8.0), you need to run 'Engage Servicing Portal - Entity Update [CPEU]' job in the Front-office for the new default settings to take effect.

    • Others

      This setting is only available for Menu Group 'File Menus [12]'.

      • Relation - Tick this to enable this menu when the Master File is a Relation file.

      Note: When upgrading from a lower version (below version 8.0), you need to run 'Engage Servicing Portal - Relation Update [CPRU]' job in the Front-office for the new default settings to take effect.

    To retain your setting and to avoid it from being overwritten during an upgrade, tick the Customised option.

    To hide the menu from appearing in the CP-WP, untick the Active option.

     

     

  • Delete - Click this to remove the selected menu. You cannot delete a Viewpoint standard menu.
