Quarantined Documents

This screen is part of the Document Virus Scanning feature. To view this screen, please ensure that the right authority is selected in the User Role Authority Settings. This screen displays documents that have been scanned and are detected as containing virus.

 

Command buttons

Click on the relevant command button to perform the necessary. The buttons in this screen are only available for users with '8 - Power User' authority.

  • Copy Document(s) - This allows users to save the quarantined documents to a selected location on the hard drive. Clicking this will prompt a warning message to remind users to store the documents in a safe location.

  • Mark as Safe Document - There may be scenarios where quarantined documents contain no virus. Users who are sure of documents that are safe may choose to mark them as such and the documents will no longer be quarantined.

  • Delete - This will delete unsafe documents from the holder. If the holder has multiple documents, the safe or unscanned documents will not be deleted. If the holder only has one document, then the holder will also be deleted. A warning message will appear when this button is clicked. Click Yes to confirm deletion.
