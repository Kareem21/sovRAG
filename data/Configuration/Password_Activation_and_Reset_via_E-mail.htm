Password Activation and Reset via E-mail

Previously, when creating a new Viewpoint Connect User, administrators can choose to set a default password for the new user. In version 8.0, new users can activate their passwords using links sent via e-mails. This feature also allows password resets via e-mails for existing users.

This feature is only applicable with the following configurations:

  • Database Connection - Common Db user.

  • Connect Application Login Mode - Login Detail Required.

  • Viewpoint IIS Configuration is installed.

  • Viewpoint Application Server is configured.

  • Viewpoint Task Scheduler Notification Agent is configured.

  • SMTP Server Configuration is configured.

 

E-mail Templates and Job Types

For this feature, two new e-mail templates have been added to the Email Notification Templates [D03] document type in Document Libraries:

  • Connect Login Activation E-mail [VPACTPWD] – This is sent to new users to activate their passwords.

  • Connect Login Reset Password E-mail [VPRESPWD] – This is sent to users to reset their passwords.

The e-mails containing links for user activation or password reset will be sent out using the following job types:

  • Connect - Password Activation [VPAP]

  • Connect - Reset Password [VRSP]

You may set the number of hours for the password activation or reset token of the e-mail links to remain valid via the 'Token Expires In' field in Security screen. When the number of hours set is exceeded, the token will expire and users will not be able to use the links sent.

To set the token expiration,

  1. From Viewpoint – Home, go to Configuration > Settings > System Defaults > Security.

  2. In the Token Expires In field, enter the number of hours for tokens to remain valid before expiring. The value for this field must be numerical. The default value is 5.

  3. Click Update.

 

Password Activation/Reset E-mail Links

Password Activation

To send the password activation link to a new user,

  1. After creating a new User, the Assign/Reset Password window will automatically pop-up. Tick the ‘Auto generate password and e-mail user login link’ checkbox. The ‘Require password change on next login’ checkbox will automatically be ticked.

  2. Click Update. The new user is now set to Inactive and will receive the Viewpoint Password Assistance e-mail to activate the account.

 

Password Reset

To send password reset link to an existing user,

  1. From Viewpoint – Home, go to Configuration > Users & Security > Users.

  2. Select the user.

  3. Click Password.

  4. In the Assign/Reset Password window, tick the ‘Auto generate password and e-mail user login link’ checkbox. The ‘Require password change on next login’ checkbox will automatically be ticked.

  5. Click Update. The user will receive the Viewpoint Password Assistance e-mail to reset the password.

 

Using the Forgot Password Option

The Forgot Password option in the Viewpoint Connect login screen allows users to reset their passwords. After completing the following process, the reset link will be e-mailed to the user,

  1. In the login screen, enter the User ID.

  2. Click on Forgot Password.

  3. Click Yes in the reset confirmation message prompted.

  4. An e-mail sent confirmation message will appear. Click OK to close the message.

 

Using Password Activation/Reset URL

Clicking the link in the e-mail will open the URL using the default web browser. The password activation/reset page is similar in design with the Engage Servicing Portal. All links will remain valid until the token expires.

To create or change the password,

  1. Enter the password in the New Password field.

  2. Enter the password again in the Confirm Password field.

  3. Click Submit and a password updated confirmation will appear.

  4. Users can now proceed to log in to Viewpoint Connect using the new password. Users that are Inactive will now become Active.
