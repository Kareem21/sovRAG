Ribbon Settings

The Ribbon Settings screen allows user to customise the Ribbon caption.

 

Screen

This lists all screens in Viewpoint with a Ribbon. Also listed are the Opened Master File and Context Sensitive Master File tabs.

 

Button list

This lists all function buttons for the selected screen's Ribbon.

 

Edit

Click this to change the caption of the selected function button.
