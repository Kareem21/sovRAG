Web Menu

The Web menu is where you set the access levels for the menus, screens and functions for Viewpoint .WEB.

 
Edit

Select the screen you want to edit from the table and click the Edit button to modify the screen settings. Find out more here.

Caption

This is the caption that is displayed in the menu and on the tops of the screen.

 

Description

This describes the screen function.

 

Authorisation Method

This field does not have any effect on the system at the moment.

 

User Group Authority

By default, this is ticked but it only takes effect if Security Model 3 is used. If it is not ticked, then the Access Group Security is applied for the screen.

 

Customised

Tick this to prevent your customised settings from being overwritten in an upgrade.

 

Active

Tick or un-tick to activate or deactivate the screen. If the screen is deactivated, it will not appear in the User Roles Authority screen and will not be seen in Viewpoint.

 

Update

Click to save the changes.

 

Cancel

Click to cancel your changes.
