Windows Menu - Tags

For a Dashboard or a Summary screen, it is necessary to add tags to:

  • specify the widgets for the Dashboard or Overview screen

    [/Widgets=<WidgetID>/]

    You can obtain the Widget ID from Dashboard Widgets or use the Tag Values Wizard. For multiple widgets, separate the widget IDs with a comma. For example:

    [/Widgets=VPEAJUR001,VPEAOFF001,VPEAREL001/]

  • hide the Selection Setup area from the screen

    [/EnableSelection=0/]

    (Optional) Only applicable to Home level Widgets. Apply this tag to hide the Selection Setup area.

    Note: Scope selection is shown by default.

  • the template for the Summary screen

    [/Summary=[<Condition>=<FormCode>]/]

    For each Summary screen, it is possible to set different templates based on condition so that only relevant information is shown in the Summary screen where:

    • <Condition> can be

      • MF.<TypeCode> to set condition based on Master File type e.g. MF.I, MF.C

      • ET.<JurisdictionCode>-<EntityTypeCode> to set condition based on entity's jurisdiction and type e.g. ET.HK-CG

      • OS.<OwnershipStructureCode> to set condition based on entity's ownership structure e.g. OS.SA

      • JU.<CountryCode> to set condition based on jurisdiction e.g. JU.HK

      • DF to set as default template for the Summary screen

    • <FormCode> refers to the form code of the template

    The caption for conditional widgets will be ‘Multiple widgets’. Users can change this by adding the [Caption=] tag to define a new caption.

  • link a workflow to a screen

    [/WF=<WorkflowKey>/]

    You can obtain the Workflow Key from Workflow Library or use the Tag Values Wizard.

Some dashboards contain default widgets. While these widgets can still be removed, a reset dashboard function in Viewpoint Management can restore the default widgets.

[/DF.Widgets=/]

Default widgets will appear with this tag.

 

Tag Values Wizard

This screen is used to generate and insert the tags based on the selected widget and/or workflow for the screen. To access it, click on the Browse button next to the Tag Values field.

 

Tabs

The available tabs allow you to select the relevant widget and workflow template.

 

List of available items

This shows the available widgets and workflow templates in the relevant tab that you can select.

 

List of selected items

This shows the widgets/workflow templates that you have selected.

 

Command buttons

Click on the relevant command button to perform the necessary.

  • >> - Click this to add to the list of selected items

  • << - Click this to remove from the list of selected items

  • Move Up - Click this to move the selected item up the list

  • Move Down - Click this to move the selected item down the list

  • < Back - This is enabled in Select Workflows tab and will return to the Select Widgets tab.

  • Next >/OK - Click Next > to go to the Select Workflows tab. Click OK after you have completed your selection so the tags are added to the Tag Values field.

  • Cancel - Click this to exit the screen without adding any tags.
