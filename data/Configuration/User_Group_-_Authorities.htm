User Group - Authorities

These are additional authorities that can be assigned to a User Group.

 

Address Card Maintenance

  • Disallow Address Card Editing - This removes the group's ability to edit address cards.

  • Allow Access to Confidential Card - This gives the group access to Address Cards that have been marked 'Confidential'.

  • Allow Edit of Restricted Card - This allows the group to edit Address Cards that have been marked 'Restricted'.

  • Allow Create Address Card on File Maintenance - This allows the group to create address card on File Maintenance. This option is enabled by default.

 

Engage Servicing Portal

  • Allow Configure ESP File Chart - This allows the ability to configure the file chart on ESP. If this is unticked, you would not be able to see the option 'Save as ESP Configuration'.

  • Publish User Profile to ESP - This allows for user profiles in the User Group to be loaded to ESP.

 

Enquiry

  • Apply 'MF Security Model' to Enquiry access - This is applicable when the security model is not model 1. When ticked, the group can only access File Maintenance screen, File Enquiry screen, Ownership enquiry screen and Relations enquiry screen for Master Files that they have access to; as opening the screen will depend on the User Group logic.

  • Access to Ownership Information - When ticked, this gives the group access to the following to the following:

    • In Ownership Enquiry screen - ownership information such as (interest holder from the Interests screen, beneficial owner from Shareholders screen).

    • In Interests screen - Master File information in the grid display.

  • Access to Identity Details/CDD - When ticked, this gives the group access to Identity Details/CDD information in the View Details option  and CDD button in the Ribbon of the Relations Enquiry screen and Ownership Enquiry screen.

  • Allow to use 'Table Nodes' in Charts - This is to restrict access to the 'Table Node' and 'Add. Info' option in ownership chart in Ownership Enquiry screen. When ticked, the group will have access to the two options.

 

General

  • Allow Export of Selections/Grid/Reports/Clipboard - This gives the group the ability to export selections, grids and reports. Tick this to enable the Export, Print and Copy functions in the right-click mouse menu. In summary screens, this will also enable Save function in the right-click mouse menu.

  • Deliver Word/Excel Reports to Document Queue - This will only allow the group to send reports to Document Queue. The other generation options will not be visible to the group when this option is ticked.

  • Allow Toolbar/Grid Customisation - This gives the group the ability to customize grids and to save their grid layout. Tick this to enable the Customise, Save this Layout and Customize Tasks and Checklists features in the right-click mouse menu.

  • Allow Access to API Call (See API Documentation) - This allows the User Group to access API calls. This is required for Data Replication jobs.

  • Bypass 'Updates through Workflow Only' - This sets all screens that have been set as read-only based on the Windows Menu edit option to return to the original state. Users can perform functions in the screens again. However, this does not change the condition of the ‘Updates through Workflow Only’ checkbox for individual screens and has no effect on other normal screens.

 

System Information

  • Disallow Access to Confidential System Information - Tick this to hide certain information such as the database and server name from the System Information screen as well as on the bottom right-hand corner.

 

Time and Disbursement

  • Allow Selection of Other/All Administrators - Tick to give the group the ability to select Administrators other than themselves in the Select folder.

  • Allow Selection on Billing File/Client/Service - Tick to allow the group to select on Billing File, Client File or Service in the Select folder. Otherwise, the selection defaults to Administrator.

  • Allow Access to 'Special' Function - Tick to give the group access to the Special Functions screen in Time Entries and Disbursements folder.

  • Approve/Undo Approval of Own T&D - Tick to allow the User Group to approve or unapprove Time and Disbursement entries created by the users themselves. If the ‘Enable Time Management’ is ticked in the Configuration > Settings > System Defaults > Time and Disbursements screen, this option also allows user to approve or reject the Timesheet submitted by the user themselves.

  • Allow Changing of Matter Rate List - Tick to allow the group to change the Rate List of the selected Matter in the Time/Disbursement Entries screen.

  • Allow Edit of Time Unit Charges - Tick to allow the group to edit the Charge/Unit time in the Time Entries screen.

  • Allow Edit of Time Unit Cost - Tick to allow the group to edit the Unit Cost field in the Time Entries screen. The Unit Cost is set up in the Administrators & Rates screen.

  • Allow Edit of Time Total Charge - Tick to allow the group to edit the Total Charge in the Time Entries screen.

  • Allow Edit of Disbursement Unit Charge - Tick to allow the group to edit the Charge/Unit for Disbursement - normal [DI] services or the Markup field for Disbursement - markup [DM] services in the Disbursements screen.

  • Allow Edit of Disbursement Unit Cost - Tick to allow the group to edit the Unit Cost field in the Disbursements screen. This is applicable only for Disbursement - normal [DI] services.

  • Allow Edit of Disbursement Total Charge - Tick to allow the User Group to edit the Total Charge in the Disbursements screen. This is applicable only for Disbursement - normal [DI] services.

 

User Field Maintenance

  • Allow Deletion of Templates - Tick to enable the Delete button in the User Field/Template Selection screen.

  • Allow Creation/Edit of Templates - Tick to enable the View/Edit button in the User Field/Template Selection screen.

  • Allow Deletion of Fields - Tick to enable the Delete and Delete All buttons in the User Fields screens.
