User Field Codes

User fields are retrieved using specific codes as described below.

 

Syntax

Forms

Code

Explanation

(+DU.FLDKEY|FLDSELECT\Format+)

Retrieves the user field for the Entity related to the form. Please refer to FLDKEY and FLDSELECT, below, for more information.

(+EVT.FileCode\#DU-FLDKEY|FLDSELECT\Format+)

The #DU format options retrieves the specified code for the Master File code specified by FileCode.

For example: (+EVT.EntCode\#DU-CYOCC|VALTEXT+)

(+QueryName.FileCode\#DU-FLDKEY|FLDSELECT\Format+)

As above. The format option works for all Master File codes retrieved by QueryName.

For example: (+AllEntities.EntCode\#DU-MF01|VALUE\#AD-40-0+)

 

Word reports, Correspondence and Billing Templates

Code

Explanation

(+QueryName.FileCode\#DU-FLDKEY|FLDSELECT\Format+)

As above.

 

FLDKEY

This is the value of the Key field as defined in the User Field set up. Use the following letters as the prefix to the value of the Key.

Prefix

Explanation

M*

All Master File user fields.

N*

All Compliance user fields

C*

All Company user fields.

T*

All Trust user fields

F*

All Fund user fields

S*CLASS

All Share Class-related user fields, where CLASS is the share class code.

B*

All Billing File user fields

R*

All Matter Account user fields

A*

All Client Account File user fields

L*

All Sub-ledger account user fields

P*

All Portfolio user fields

*

All user fields related to the Master File

 

The following are format codes for field keys returning multiple values (FLDKEY* options).

Prefix

Explanation

FMT01

Return the field values for a two-column table where the first column holds the field caption and the second column the field values (in the VALTEXT format). The code must be placed in the left column of a two-column table.

FMT02

Same as above except that the code must be placed in a three-column table and the code will retrieve the Note field in the third column.

For example: (+HD.RefCode\#DU-C*|FMT01+)

 

FLDSELECT

This is the value of the other fields as defined in the User Field set up. The following options for FLDSELECT explains their corresponding field in the User Field setup.

Option

Field

CAP

Field caption

VALTEXT

Formatted value of the user field. See VALTEXT below, for the format applied to various user field types.

VALUE

Value of the field. For Master File/User/Lookup value fields, this option returns the code.

VALDESCR

Returns the Name/Description for the Master File/User/Lookup value fields. For other field types, it returns the VALUE.

NOTE

Note field

DFROM

Date from

DTO

Date to

TABVAL

Code of the Lookup/User table value for Master File + Lookup/User table fields.

TABVALDESCR

Name/Description of the Lookup/User table value for Master File + Lookup/User table fields.

TABVALCAP

Caption text of the Lookup/User table field.

For example: (+HD.RefCode\#DU-CODE|VALTEXT+)
