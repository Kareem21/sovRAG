Dashboard Widgets

In this screen, you will see a list of all the available widgets that can be added to Dashboard screens.

 

Filters

The filters apply to the widget list.

  • Widget status - You can choose to view only active widgets, inactive widgets or all widgets.

  • Menu Group - This shows the grouping of the Dashboard screens.

  • Application - This shows the application area of the Dashboard screen.

 

Widget list

The list shows all existing widgets based on the filters applied.

 

Command buttons

Click on the relevant command button to perform the necessary.

  • New - Click this to add a new widget.

    A widget is defined using tags that are to be entered in the Tag Values field. When you select the Type for the widget, the relevant tags will be added but you need to update these correctly.

  • Edit - Click this to edit settings for the selected widget.

  • Select All - Click this to select all listed widgets.

  • Deselect All - Click this to clear all selection.

  • Delete - Click this to remove the selected widget.

  • Import - Click this to import widgets.

    1. In the Import screen, browse to the folder containing the widget.

    2. Click Import - Load List.

    3. Tick the widget(s) you want to import.

    4. Click Import.

  • Export - Click this to export the selected widget.

    1. Once you click Export, browse to the folder to store the exported widget(s).

    2. Click Export.

    Note: Each widget will be exported as a .txt. file.

  • Add to Screen - Click this to add the selected widget to a dashboard-configurable screen.

    1. Tick to select the widget(s) and click Add to screen.

    2. In the Select Menu screen, click to select the screen to add the widget.

    3. Click Select.
