Export

The functions in this screen are used to create export files.

 

Export Bills to Sage

Use this to create a .csv file that you can import into Sage software.

 

Upgrade Master File Archives

Click this to move all documents to their appropriate Master File folders. For example, if your Master File Viewpoint Ltd. is numbered 89, this function will move a document from \\<server>\Viewpoint\Archive to \\<server>\Viewpoint\Archive\MFILES\00000089. This function is used only if you do not utilise Document Management and the 'Archive by Master File' option is enabled in System Defaults > General/Master File.
