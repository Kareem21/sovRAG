Import/Export Document Templates

All Word templates in the system are exported as two different files, a Word file and a text file. The Word file contains the text of the document (Filename.doc) and the text file (Filename.def) contains instructions for system generation. Both the files will be placed in the export directory and should be kept together because you will need both files in order to import the template into another environment if required.

 

Exporting document templates

  1. Click Export.

  2. Browse to the location you want to save the template/templates. The path is shown in the Folder field.

  3. If you are only exporting the selected document, then select the Select Document radio button. To export a category or jurisdiction, select the Select Category/Jurisdiction radio button. This will display the available categories and jurisdictions in the table. Tick the categories and jurisdictions you want to export. If you want to export specific templates from the selected categories and jurisdictions, then select the Select Document radio button at this point. This will load the templates in the selected categories and jurisdictions.

  4. Tick the check boxes next to the templates you want to export. You may use the Select All or Deselect All buttons to select/deselect all the available templates.

  5. Click Export.

  6. Click Close when the process is complete to close the Export Forms screen.

 

Importing document templates

  1. Click Import.

  2. Browse to the directory where the forms are located.

  3. Click Import - Load List. The text files (.def) will appear for selection.

  4. Select the relevant options:

    • Import Document Attributes - If you tick this, attributes for the templates will be imported as well. This applies to templates in Form Library, Word Reports and Correspondence.

    • Perform Refresh - If you tick this, the template will be refreshed during the import.

  5. Select the forms you want to import. You may use the Select All or Deselect All buttons to select/deselect all the available templates. If an existing template with the same code already exists in this environment, this will be indicated with the message 'existing' in the Message/Result column. You need to tick each of the customised templates that you want to over-write.

  6. Click Import.

  7. Click Close when the import process is complete to close the Import Forms screen.
