Coding a Word Template

This section will provide some basic information on coding templates.

  • Field Codes - These refer to the codes that are used to retrieve information from the database.

  • Format Options - These will format field codes into meaningful information.

  • User Field Codes - User field codes are used to retrieve information from user fields.

  • Key Information - This is where you place SQL queries needed to pick up specific information for the template.

  • BLOCK Statements - BLOCK statements are used when you want the information shown only if a certain set of requirements are met.

  • E-mail Tags - These are used to pick up information when you create an e-mail from Viewpoint.

  • Popup Fields - These require input from the user at the point when the document is generated.

 

When coding the template, you can also use the following:

  • Field Wizard - This is a function to obtain field codes and format options for the template.

  • Index Wizard - This is a function to obtain Document Management index tags to add to the template.
