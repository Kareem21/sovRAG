Nominal Accounts

In Viewpoint Practice Management, you are allowed to maintain a chart of accounts and automatically post invoices and receipts against the correct account codes. To set up this feature you will first need to input your chart of accounts into the system and then select the appropriate account codes when you set up your services. You would not use this feature if you are using the Accounting component.

 

Filter

The filter applies to the account list.

 

Account list

This shows all the accounts that you have set up.

 

Command buttons

Click on the relevant command button to perform the necessary.

  • New - Click this to add a new nominal account.

  • Edit - Click this to edit an existing account.

  • Delete - Click this to delete an account. If you do not want to delete an account permanently, you can deactivate the account.

  • Deactivate/Re-activate - Click this to deactivate or re-activate a selected account.
