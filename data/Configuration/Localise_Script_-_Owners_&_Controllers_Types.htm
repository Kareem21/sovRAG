Localise Script Tags - Owners & Controllers Types

Tag is available to allow new Owners and Controller Types to be added using Localise Script where the type will have the Customised checkbox ticked by default.

 

Adding New Type

When using Localise Script to add a new type, you need to specify the required fields and values for the type. Each field will be separated with the pipe '|'.

 

Syntax

OCTYPE:<CountryCode>=TypeDescription=<Desc>|<DBField>=<Value>|...

 

where:

  • <CountryCode> refers to the code of the country the type is for

  • <Desc> refers to the description for the type

  • <DBField> refers to the fields required for the type and comprises User Table fields, Tag Values field and Active checkbox

    • UTCODE_EOTP - Register Type - User Table field and is mandatory

    • UTCODE_EORS - Exemption Reason - User Table field

    • UTCODE_EOCO - Capacity - User Table field

    • UTCODE_EONI - Nature of Interest - User Table field

    • UTCODE_EOST - Status - User Table field

    • UTCODE_EOX1 - Extra 1 - User Table field

    • UTCODE_EOX2 - Extra 2 - User Table field

    • UTCODE_EOX3 - Extra 3 - User Table field

    • UTCODE_ACTP - Action Type - User Table field and is mandatory

    • TagValues - Tag Values field

    • IsActive - Active checkbox

  • <Value> is the value for <DBField>

    • For <DBField> that are User Table field, the value must be the code of the User Table

    • For Tag Values field, the <Value> are tags to define the field settings of non-User Table fields and can include

      • <br> for a line break

      • <pipe> for a | symbol

 

Example

OCTYPE:GG=TypeDescription=Guernsey|UTCODE_EOTP=GG_EOTP|UTCODE_ACTP=GG_ACTP|TagValues=[RefDirect.Visible=Y][RefDirect.Caption=Direct Owner][RefDirect.Mandatory=Y]

 

Deleting a Type

The tag below will delete the Owner and Controller Type for the specified jurisdiction.

 

Syntax

OCTYPEDEL:<CountryCode>

where <CountryCode> refers to the country code of the jurisdiction.

 

Example

To delete the Owner and Controller type for Guernsey: OCTYPEDEL:GG
