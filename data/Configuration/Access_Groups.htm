Access Groups

Access Groups are used only if you have selected Access + User Group Model [3] as your security setup. An Access Group is combined with an Administrator Group and a User Group to create an Access Group Security.

 

Filter

The filter applies to the Access Group list.

 

Access Group list

The list shows all existing Access Groups.

 

Command buttons

Click on the relevant command button to perform the necessary.

  • New - Click this to create a new Access Group.

    1. Enter details for the Access Group.

    2. Click Update.

     

  • Edit - Click this to edit the details of an existing Access Group.

  • Delete - Click this to delete an existing Access Group.
