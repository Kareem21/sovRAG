Workflow Designer Screen

In the Workflow Designer, you will be able to see the following:

 

Ribbon

The Ribbon allows you to access to various functions such as change the viewing size of the workflow diagram and print the workflow steps.

 

These buttons can be seen in the Ribbon:

 

General Group

    • Windows Manager - Click to open Windows Manager.

    • Notifications - Click to open the Notifications screen.

    • Go to Home - Click to return to Viewpoint - Home screen.

Diagram Group

    • Fit All - Click to show the whole workflow diagram.

    • Actual Size - Click to view the diagram in its actual size.

    • Zoom In - Click to zoom in.

    • Zoom Out - Click to zoom out.

    • Copy - Click to copy the diagram to clipboard.

    • PDF - Click to save the diagram as a PDF file.

    • Visio - Click to save the diagram as a Visio file.

Print Group

    • Workflow - Click to print the workflow design.

    • Step - Click to print the workflow step.

Properties Group

    • Header - Click to open the Edit Header Properties screen.

    • Verify - Click to verify the Workflow Template setup.

    • Increase Version - Click to update the Version Description of the Workflow Template.

 

 

Workflow diagram

This is used to define the step sequence for the workflow template. When you first open the Workflow Designer, the diagram contains only Start and Completed symbols with an arrow link.

 

Step list

This lists all the steps that you have added to the workflow design. For workflow templates, you can have:

  • Decision steps - this type of step allows for two-way branching in the workflow. A decision step is automatically added to all workflow template but you can change its caption and values.

    Note: If the workflow contains multiple decision steps – one after another – and the decision is automated, the output refers to an item in the checklist step before the first decision step.  

  • Checklist steps - These are steps that contain items which you need to set up.

 

Step details

When you select a step from the Step list or the workflow diagram, its configuration is seen here.

 

Complete with Next Step

Tick this option to indicate that the next step is parallel to the selected step. When you have steps that are in parallel, these steps are all considered as the 'current step'.

 

Allow Copy Step

Tick this to allow the step to be copied. When working with the workflow, there will be a Copy Step button that users can click to get another instance of the step.

 

Optional Step

Tick this to make the step optional. If not required, it can be removed by using the Remove Optional button from the Ribbon.

 

Completion %

Enter the completion percentage between 0 to 100 for each step. This field is enabled when a step is selected from the workflow diagram and the value should be entered in cumulative percentage from one step to the percentage of another step. If not required, this can be left empty. For value out of the acceptable range, an error message will be prompted and the value will be cleared.

 

Command buttons

Click on the relevant command button to perform the necessary.

These buttons apply to the workflow diagram:

  • Add to Link - Click this to add the selected step to the selected link. This button becomes inactive when there is no link selected.

  • Remove - Click this to remove selected step from workflow diagram.

These buttons apply to the steps in the step list.

  • Add Step - Click this to add steps for the workflow template. This opens the Select Workflow Steps screen.

    In this screen are the following:

     

    Checklist/Workflow step List

    This is to show all existing workflow steps that is in your Viewpoint system in a grid format. By default, the workflow steps are listed according to the Step Number in an ascending order.

    The following options can be seen by right-clicking on the grid:

    • View Details - Click this to view details of the selected record from the grid.

    • Refresh - Click this to refresh the grid.

    • Print Preview - Click this to open the Page Setup screen to change the page settings before printing.

    • Print ... - Click this to print the grid.

    • Copy - Click this to copy all records.

    • Export - Click this to export all or selected grid information which can include hidden columns. You can also copy the information to Viewpoint Clipboard.

    • Find - Click this to open the Find function.

    • Customise/Select - Click this to change the grid display such as filtering the grid or hide/show step details.

    • Alternate Row - Click this to apply highlight to alternate rows of the grid.

    • Reset - Click this to reset the display to default settings.

    • Save This Layout - Click this to save changes made to the grid.

    • Group Column - Click this to group by selected column.

     

    Command buttons

    Click on the relevant command button to perform the necessary.

    • New Step - Click this to create a new step without exiting from this screen. This is to create the workflow step record.

    • Copy Step - Click this to copy the selected checklist/workflow step. This will copy all its checklist items as well.

      1. Enter details for the copy.

      2. Click Update.

       

    • Select - Click this to select the steps that have been ticked.

    • Cancel - Click this to exit the screen.

  • Edit Properties - Click this to make changes to the step properties without exiting Workflow Designer.

  • Edit with Editor - Click this to open the Workflow Step Editor to amend selected workflow step items.

  • Remove Step - Click this to remove step from the list.

These apply to the workflow template.

  • Update - Click this to save the workflow template.

  • Cancel - Click this to cancel changes made and exit the Workflow Designer screen

Note:

  • Only one Workflow Designer screen is allowed to open at one time for each Workflow template.

  • Changes made to the Workflow Designer screen will trigger the Workflow library – Edit with Designer [WFTF] change code and the changes will be recorded in the Changes screen.

  • If you want to update the version of the workflow template, click Increase Version button in the Ribbon. The version description, latest update date and last modified user will be updated and displayed in the Workflow Library screen.
