Relations

This is where you customise information displayed in Relation Enquiry on Engage Servicing Portal.

 

Filters

The filters apply to the relations list.

  • Status - You can choose to view only active, inactive or all relations.

  • Customised - You can choose to view types that have been customised or not.

 

Relations list

The list shows the relations that will be included in ESP's Relation Enquiry.

 

Edit button

Click this to edit properties of the selected item.

You can edit the following:

  • Parent Caption

  • Child Caption

  • Record Caption

  • Tag Values - This is to define the template for the display. Just enter the formcode for the template. By default, it is CPMFSS.

To retain your change, tick the Customised option and click Update.
