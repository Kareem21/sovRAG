User Groups

User Groups are used to separate users into specific groups depending on their job functions. For example, a company's secretarial department is divided into two groups, and only one group is responsible for generating the Annual Return and AGM minutes. You can create two user groups: one group with authority to perform the Annual Return and AGM functions, the other group without. A user can only be assigned to one User Group.

The access rights that you allocate should be carefully considered in order to protect your data. If in doubt about what authority level to give a user group, give them the minimum rights - you can always give them higher access rights later, if necessary.

 

Filter

The filter applies to the User Group list.

 

User Group list

The list shows all existing User Groups.

 

Command buttons

Click on the relevant command button to perform the necessary.

  • New - Click this to create a new User Group.

    1. Enter details for the User Group.

    2. Click Update.

     

  • Edit - Click this to edit the details of an existing User Group.

  • Delete - Click this to delete a User Group. This can only be done if there are no users assigned to the group.

  • Assign Roles - Click this to assign User Role(s) to a selected User Group. These roles determine the group's access to screens and functions. When there are overlapping authorities, you can see the final authority by clicking the View Authority button. Tick on checkbox 'Show Assigned Only' to display only those items being checked. Unchecking the checkbox will display back all items.

  • Authority - Click this to assign authorities to the selected User Group. These settings over-ride any User Role authority that might be assigned to the group. Tick on checkbox 'Show Assigned Only' to display only those items being checked. Unchecking the checkbox will display back all items.

    1. In the User Group Authority Settings screen, tick the relevant authority.

    2. Click Update.

     

  • Document Management - Click this to assign Document Management access and authorities to the group.

    1. In the Document Management Authority Settings screen, define the authority for Global Document and Filing Cabinets by ticking the relevant option.

    2. Click Update.

     

  • Widgets - Click this to assign access to widgets in dashboards for the group.

    Widget Viewing Authority in User Groups

    1. In the Widget Settings screen, select the Menu Group.

    2. Select the Dashboard. Only dashboards with defined widgets in Windows Menu will be displayed.

    3. Select the widgets from the Available Widgets area.

    4. Click >> or double-click the selected widget to move it to the Assigned Widgets list. Click << or double-click a widget from the Assigned Widgets list if you want to move it back to the Available Widgets list. he default sequence of the widgets appearance in the dashboard will be determined by the order of the widgets added to the Assigned Widgets area.
      Users can change the sequence by selecting a widget and click the Move Up or Move Down buttons.

    5. Tick on Mandatory or Show. Widgets that are set as ‘Mandatory’ or ‘Show’ will be displayed in the assigned dashboard screen for all users in the User Group. ‘Mandatory’ widgets cannot be removed, edited, resized, or moved while users can perform these actions on ‘Show’ only widgets. Setting a widget as ‘Mandatory’ will also set it as ‘Show’.
      Note: Mandatory widgets should be placed above non-Mandatory widgets as these widgets cannot be resized.

    6. Click Update.

     

    Reset User Dashboard

    As all dashboards are user-based and can be configured by each user, the Reset User Dashboard button will clear all the users’ customisations and the dashboard will revert to the default sequence and view of the widgets.

     

  • View Authority - Click this to view the overall authority levels for the group. This is useful if you have assigned User Roles with overlapping authorities to a particular screen.

    The information is displayed in the User Group Access Level - <User Group> screen where you will see:

    • Tabs - this shows the Menu Group like Home, Reporting, Configuration, File, Billing/CA Enquiry. Click to select.

    • Menu Type options - There are three options:

      • Windows Menus - this lists screens for Viewpoint Connect

      • Web Menus - this lists screens for Viewpoint 8.0.WEB

      • All Menus - this lists screens for both the Viewpoint Connect and .WEB

    • Authority list - this shows the screens and the highest authority conferred for the User Group. You can double click on the authority column for a specific screen to see the access level per each User Role assigned to the User Group.
