Grid Fields

Customising grid fields allows you to change the standard/default captions in Viewpoint as well as the size of the grid. This is only applicable for tables in Viewpoint. To identify the grid you want to customise, right-click in any area in the grid and select View Details. You will see the Grid Code at the bottom of the detail screen.

 

Grid Fields Table Listing

This displays the codes and descriptions of grids.

 

Grid Fields

This shows the information of columns in the grid you selected from the Grid Fields Table Listing.

 

Edit

Click this to make changes to the selected grid column. You can do the following:

  • Change the caption - Enter the new caption for the column header in the Caption field.

  • Change the width - Enter the width value.

  • Change the display format - Select if you want to see just the code, the description or both. Note that this is not available for all grid columns.

  • Control display of the grid column - Tick the applicable option(s) for the grid column:

    • Default

    • Not Used

    • Customised
