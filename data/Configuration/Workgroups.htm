Workgroups

Workgroups is a user type that users can assign work items to when they do not have a specific user to assign work to. Users can pick up work items from the Workgroups and reassign the work items to other users or themselves when they want to edit or complete the work items.

 

Filters

The filters apply to the workgroup list.

  • Workgroup status - You can choose to view only active workgroups, inactive workgroups or all workgroups.

  • Administrator Group - You can choose to view workgroups that are assigned to a specific Administrator Group.

 

Select button

This allows for additional criteria to apply to the workgroup list.

 

Workgroup list

The list shows all existing workgroups.

 

Command buttons

These apply to the workgroup list. Click on the relevant command button to perform the necessary.

  • New - Click this to create new workgroups.

    1. Enter the necessary details for the new workgroup.

    2. Click Update when done.

  • Edit - Click this to view/change details of an existing workgroup.

  • Delete - Click this to remove an existing workgroup.

  • Reassign - Click this to reassign items for the selected workgroup to another.

    1. In the Workgroup Reassign screen, select the types of item that you want to reassign.

    2. Enter the workgroup or user to whom the items are being reassigned to.

    3. Click Reassign.

    4. Click Yes to confirm.

    Note: If you are transferring only one item type, then you can choose to transfer all of the item type or only selected ones. If more than one item type, then all of each item type will be included.

     

  • Relation Enquiry - Click this to view all Master Files where the selected workgroup has been set up as a relation. This will be shown in a separate User Relations enquiry screen.
