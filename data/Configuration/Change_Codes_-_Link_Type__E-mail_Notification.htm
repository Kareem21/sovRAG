Link Type: E-mail Notifications

This link type allows users to choose whether or not to send an e-mail notification for the linked change, for example, to inform of information updates such as when a charge is discharged or when a bank account is closed. This requires the following to be done:

  • Application Server must be installed and running.

  • Task Scheduler and Notification Agent must be installed and running.

  • SMTP mail server must be configured in Configuration > Settings > System Defaults > SMTP Server Configuration screen.

To set up this linked item, do the following:

  1. In Configuration > Business Parameters > General > Change Codes screen, select the relevant Change Code.

  2. Click New.

  3. Specify the following details:

    • Link Type - Select E-mail Notifications.

    • Description - Enter a description for the e-mail notification.

    • Tag Values - Enter the relevant tag values to define the e-mail details:

      • Message body

      • Subject line

      • Recipient(s) of the e-mail where different recipient can be specified for To, CC and BCC

      • E-mail address to use for replies

  4. Click Update.

Note: To be notified of work assigned, use the Notifications Link Type.
