Defining Reminder Rules

New Reminder Rules are added directly to the ToDoCodes database table. The definitions used are described below.

Field Name

Description

Example TodoCode (Alert)

TodoCode

To do code

Ensure that the hash (#) symbol is used as the first character of the ToDoCode to prevent the rule being overwritten during an upgrade.

Note: If you modify any of the database fields related to standard Viewpoint Reminder Rules, ticking the Customised checkbox does not prevent the rule from being overwritten during an upgrade. In such an instance, deactivate the standard Viewpoint Reminder Rule and create a new Reminder Rules with a ToDoCode starting with hash (#).
