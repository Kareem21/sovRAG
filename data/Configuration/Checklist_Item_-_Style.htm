Checklist Item - Style

Users can apply a style to an item so that it will be displayed differently from other items. The style will be applied to the Caption, Complete Field and Help columns and can be applied to all field types.

The available styles are:

  • Heading 1

  • Heading 2

  • Warning Text

  • Red

  • Green

  • Amber

To apply the style, select it from the Style column.

Note: System default style will overwrite the style formatting applied by user. E.g. Blue coloured font for mandatory fields, purple coloured font for required fields, bold font for group header, yellow background at Complete Field column for the ‘File Review’, ‘Macros’ and ‘Workflow Link’ field types, and style of ‘Task’ field type that change according to the status of the task.
