Localise Script Tags - User Fields

Three tags are available for Localise Scripts -  for deleting an existing user field, creating a new user field and then defining the characteristics of the user field.

 

Deleting a User Field

The tag below is to delete an existing User Field.

 

Syntax

FUDEL:<FKey>

 

where <FKey> is the user field code that is obtained from Configuration > Business Parameters > User Fields.

 

Example

To delete the field Reference [KYC120]: FUDEL:KYC120

 

Adding and Defining a User Field

When using Localise Script to add a new User Field, tags are required to create the User Field and specify its settings.

 

Syntax

To add a user field:

FUADD:<FKey>+<FCat>=<Caption>

 

To define the user field:

FUVAL:<FKey>+<DBField>=<Value>

 

where:

  • <FKey> is the user field code that is obtained from Configuration > Business Parameters > User Fields.

  • <FCat> is the Category/Jurisdiction/Type/Classification of user field. Enter ALL if it is a General Field.

  • <Caption> is the Caption of the user field.

  • <DBField> is the database field for the user field setting, and can be obtained from Configuration > Settings > Customise > Field Settings.

    • MfType - to indicate the User Field Type

    • AllowDup - Allow Duplicates option

    • AutoAdd - Automatically Add option

    • FldType - Field Type field

    • FldCaption - Caption field

    • DefValue - Default Value field

    • DefNote - Default Note field

    • KeyValues - Key Values field

  • <Value> is the value for <DBField>

    • For MfType, <Value> is a code to indicate the User Field Type:

      User Field Type

      Code

      Master File

      M

      Identity Details

      N

      Tax Details

      O

      Company

      C

      Trust

      T

      Fund

      F

      Share Class

      S

      Billing File

      B

      Matter Account

      R

      Client Account File

      A

      Sub-ledger Accounts

      L

      Portfolios

      P

      Fixed Assets

      X

 

Example

To set up a user field as shown below:

FUADD:KYC120+KYG=Reference

FUVAL:KYC120+MfType=M

FUVAL:KYC120+AllowDup=-1

FUVAL:KYC120+AutoAdd=0

FUVAL:KYC120+FldType=71

FUVAL:KYC120+DefValue=

FUVAL:KYC120+DefNote=

FUVAL:KYC120+KeyValues=[TABLECODE=REID][SHOWCOMBO]
