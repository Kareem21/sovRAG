Teams

The teams function can be used if you require additional separation of your existing Administrator Groups. This screen is switched off under normal conditions and can be activated through Configuration > Settings > Customise > Windows Menu. Although you can enable the screen, you must enable the feature before you can assign users to a team. Please refer to Viewpoint Support for more information on this.

When you enable the feature, the team is assigned to a Master File in the New Master File wizard.

 

Filter

The filter applies to the Team list.

 

Team list

The list shows all existing teams.

 

Command buttons

Click on the relevant command button to perform the necessary.

  • New - Click this to create a new Team.

    1. Enter details for the Team.

    2. Click Update.

     

  • Edit - Click this to edit the name of an existing Team.

  • Delete - Click this to delete an existing Team.
