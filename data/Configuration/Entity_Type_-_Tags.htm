Entity Type - Tags

These tags are to be entered in the Tag Values field for the Entity Type.

 

Change template for Print Registers options in Registers screen

The Print Registers options are coded to be based on default Viewpoint templates. However, this can be changed by changing the form code using the tag: [WREP.<DefaultFormCode>=<NewFormCode>]

where:

  • <DefaultFormCode> refers to the code for the default template.

  • <NewFormCode> refers to the code of the template you want to use.

Filter Field Values

For fields where users can select from a list of values, a tag can be added to filter the list so that users see only values that are applicable to the entity type. Refer to Filtering List Values.

 

 

Define File Review

This is to define the item to check on when performing a File Review for the following scopes:

  • Setup - refer to Setup/Statutory Review - Tags.

  • Statutory - refer to Setup/Statutory Review - Tags.

  • Meetings - refer to Meetings, Filings, Reviews - Tags.

  • Filings - refer to Meetings, Filings, Reviews - Tags.

  • Reviews - refer to Meetings, Filings, Reviews - Tags.

 

Specify sections to include for Jersey Annual Return

The following tags control what is included int he Jersey Annual Return.

Tag

Explanation

[AR.MEMBER=ME]

This will show only Guarantor Members section for Guarantee Companies in the Jersey Annual Return (JEC20).

[AR.MEMBER=UM]

This will show only Unlimited Members section in the Jersey Annual Return (JEC20).

[AR.DIRLIST=Y]

This will show the Particular of Directors section in the Jersey Annual Return (JEC20).

 

Enable Special Configuration for Shares

Tag

Explanation

[SHVAL_ISSUEPRICE=1]

This is to allow for overpayment and underpayment of shares in the Call(s) Paid screen. With this tag, you can enter an amount resulting in the total paid-up/share to be higher or lower than the share's nominal value when performing Call(s) Paid.

[SHFLD_ISSUEPRICECHANGE=1]

This is to allow changes to the share's issue price in Call(s) Paid screen.

[SHVAL_AUTHZERO=1]

This is to allow for unlimited issuance of shares for companies with the Ownership Structure of 'Shares - Authorised Capital [SA]'. When creating the share class, users can enter zero in the Number of Shares field.
