Localise Script Tags - User Table Values

Similar to lookup tables, there are also tags which can be used in Localise Scripts for user tables. In this case, tags are available to create new tables and add values to the table.

 

Adding a Table

The tag below is to create a new user table using Localise Script.

 

Syntax

UTADD: <TCode>=<TName>

Or

UTADD:<TCode>=Name=<TName>|ElNote=<TNote>|TagValues=<TTag>|WithRating=<Value>  

 

where:

  • <TCode> is the code for the table.

  • <TName> is the name for the table.

  • <TNote> is the note for the table.

  • <TTag> is the tag value for the table.

  • <Value> is to indicate the risk rating option for the table.

 

Example

To add the table FATCA Classification:  UTADD:FCST=FATCA Classification or UTADD:FCST=Name=FATCA Classification|ElNote=For FARCA Classification||TagValues=[SUBSCRIBER=GEN.ASA.EUR]|WithRating=1

 

Note: Part of the Value will be missing in the result if fields value contain the pipe symbol (|).  

 

Adding a User Table Value

Once the user table has been created, then the tag below can be used to add values to the user table.

 

Syntax

UVADD: <TCode>+<VCode>=<VName>

 

where:

  • <TCode> is the user table code obtained from Configuration > Business Parameters > General > User Tables/Values.

  • <VCode> is the code of the User Table value to be added. There should not be duplicates of any (existing) code.

  • <VName> is the name of the User Table value.

 

Example

To add the value shown above: UVADD:FCST+0001=Reporting Model 1 FFI

 

Adding a User Table Value with Note and Tag Values

If the User Table value is to contain note or tags, then the syntax is:

UVADD:<TCode>+<VCode>=Name=<VName>|ElNote=<VNote>|TagValues=<VTag>    

 

where:

  • <TCode> is the user table code obtained from Configuration > Business Parameters > General > User Tables/Values.

  • <VCode> is the code of the User Table value to be added. There should not be duplicates of any (existing) code.

  • <VName> is the name of the User Table value.

  • <VNote> is the note for the User Table value.

  • <VTag> is the tag value for the User Table value.
