Revenue Entities - Tags

For a revenue entity, sometimes there is a need to impose extra charge for its invoices and this is done by using the tags below in the Tag Values field of the Revenue Entity record. You can have up a maximum of nine charges.

Note: Replace X with a number e.g. [Charge1_ Service=...].

 

[ChargeX_Service=<ServiceCode>]

This sets the service as an additional charge and will create an additional invoice line for the service.

 

[ChargeX_Percent=<Percentage>]

This specifies the percentage for the additional charge. If a negative percentage is specified, this will be subtracted instead.

 

[ChargeX_Amount=<Amount>]

This sets a fixed amount for the additional charge. If a negative amount is specified, this will be subtracted instead.

 

[ChargeX_Descr=<Text>]

Use this tag to have an alternative description for the charge. Without this tag, the description will follow the description in the Service Master.

 

[ChargeX_Cur=<CurrencyCode>]

Use this tag to define the currency of the charge. Values for <CurrencyCode> can be actual currency code e.g. EUR or the following:

  • IC - Invoice currency.

  • BC - PM base currency.

  • AC - Currency defined for the Account File of the Revenue Entity.

Without this tag, the currency will follow the invoice currency.

 

[ChargeX_AllowChange=<Value>]

This tag controls whether or not users can delete the charge or edit the Discount % and Total Line fields. <Value> can be:

  • Y - Yes

  • N - No

Without this tag, the charge cannot be deleted.

 

[ChargeX_Exempt=<TaxChargeType>]

This tag exempts the charge for Matters with the Tax Charge Type specified.

 

[ChargeX_Excl_ServCode=<ServiceCode>]

This tag excludes the specified service code from the additional charge.

 

[ChargeX_Excl_Class=<ServiceClass>]

This tag excludes service(s) belonging to the specified service class from the additional charge.

 

[ChargeX_Exclude=<ServiceType>]

This tag excludes service(s) belonging to the specified service type from the additional charge.

 

[ChargeX_Incl_ServCode=<ServiceCode>]

This tag includes the specified service code for the additional charge.

 

[ChargeX_Incl_Class=<ServiceClass>]

This tag includes service(s) belonging to the specified service class for the additional charge.

 

[ChargeX_InclTax=<Value>]

<Value> can be:

  • Y - Yes

  • N - No

This tag will include the VAT tax and the service's total for calculation of the additional charge if value is Y. By default, the value is Y.

If this tag is not defined, the VAT will be included by default.

 

[ChargeX_TaxOnly=<Value>]

<Value> can be:

  • Y - Yes

  • N - No

This tag will take only the VAT tax amount for the calculation of the additional charge if value is Y.

 

[ChargeX_Rounding=<Method>]

This tags is to specify if the calculated charge should be round up or down. Values for <Method> can be:

  • N - Normal rounding according to the currency and whether or not 'Round VAT Down to 5 cents Multiples' is enabled.

  • U2 - Round up to nearest cent.

  • D2 - Round down to nearest cent.

  • N0 - Normal rounding to whole number.

  • U0 - Round up to whole number.

  • D0 - Round down to whole number.

 

[ChargeX_Min=<Amount>]

This tag allows you to specify a minimum amount for the additional charge. To fix the charge in a currency, add the currency code e.g. [Charge1_Min_EUR=50]. If a negative percentage is specified, this will be subtracted instead.

 

[ChargeX_Max=<Amount>]

This tag allows you to specify a maximum amount for the additional charge. To fix the charge in a currency, add the currency code e.g. [Charge1_Max_EUR=500]. If a negative percentage is specified, this will be subtracted instead.

 

Note: For Purchase Invoice, include prefix 'P' for the tag, example:

[PChargeX_Service=<ServiceCode>]

[PChargeX_Percent=<Percentage>]

[PChargeX_Amount=<Amount>]

[PChargeX_Descr=<Text>]

[PChargeX_Cur=<CurrencyCode>]

[PChargeX_AllowChange=<Value>]

[PChargeX_Exempt=<TaxChargeType>]

[PChargeX_Excl_Class=<ServiceClass>]

[PChargeX_Excl_ServCode=<ServiceCode>]

[PChargeX_Incl_Class=<ServiceClass>]

[PChargeX_Incl_ServCode=<ServiceCode>]

[PChargeX_Exclude=<ServiceType>]

[PChargeX_InclTax=<Value>]

[PChargeX_TaxOnly=<Value>]

[PChargeX_Rounding=<Method>]

[PChargeX_Min=<Amount>]

[PChargeX_Max=<Amount>]
