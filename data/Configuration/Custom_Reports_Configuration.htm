Custom Reports

All custom reports in Viewpoint are managed in this screen.

 

Custom report list

The list shows the existing standard reports.

 

Command buttons

Click on the relevant command button to perform the necessary.

  • Edit - Click this to update the report settings such as the report title and the authority required to access the report. Users without access to the report will not be able to create or run the report.

  • Import - Click this to import a custom report created using Custom Report Designer.

    1. In the Import Custom Reports screen, specify the folder containing the custom report.

    2. Click Import - Load List.

    3. Tick to select the report you want to import.

    4. Click Import.

  • Export - Click this to export the selected custom report.

    1. In the Export Custom Reports screen, select the folder where the exported report is to be stored.

    2. Click Export.

  • Delete - Click this to remove a custom report created using Custom Report Designer.

  • Attributes - Click this to set up attributes for the selected template. When exporting this custom report, you will also get the .dtr file which contains the attributes information. Refer to Attributes for more information.
