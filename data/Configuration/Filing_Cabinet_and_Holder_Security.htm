Filing Cabinet and Holder Security

There are several ways you can control access to the Filing Cabinets and document holders in Document Management.

 

Filing Cabinet Security

A user's Access to Filing Cabinets is determined by his User Group access. When setting up a User Group's Document Management Authority Settings, it is important to consider whether or not the individual users in that group need to have access to the documents that are stored in that cabinet. If all users need access to the Filing Cabinet, then access is granted to the group in the Document Management Authority Settings screen. You may control the types of functions they can perform on the cabinet, such as adding new holders and tasks, by selecting the appropriate options in the Document Management Authority Settings screen.

If the User Group does not require access to a cabinet, then no access is granted, and they will not see it when they are performing a search or when they are creating new holders.

 

Holder Security

Once you have granted a User Group access to a Filing Cabinet, you can limit the document holders that a user can see and/or access, by using the holder security feature. This feature is set up in Viewpoint - Home > Configuration > Settings > System Defaults > Document Management.
