Exchange Rates (AD/PM)

The Exchange Rates screen contains frequently used conversion rates in the form of a table. The rate is picked up when you are working in Entity Administration, Practice Management or Accounting.  It is available in Viewpoint - Home screen. To access it:

  • Using Ribbon - Click EA Global Ribbon button, select Global Change > Exchange Rates (AD/PM)

  • Using Section Menu - Click Home > Entity Administration >  Exchange Rates (AD/PM)

There are two different exchange rate tables:

  • Exchange Rates (AD/PM) - This is used for Entity Administration and Practice Management.

    To access:

    • In Viewpoint - Home screen, click Home > Entity Administration > Exchange Rates (AD/PM); or

    • In Viewpoint - Home screen, click Home > Practice Management > Exchange Rates (AD/PM)

  • Exchange Rates (CA) - This is used for Accounting.

    To access, in Viewpoint - Home screen, click Home > Client Accountant > Exchange Rates (CA).

    The most recent rate entered on (if available), or before the journal date is then picked up when entering journals or when reconciling them. You can manually update the exchange rates, or use an Excel spreadsheet to import the latest rates.

If you wish to use the same exchange rate table for Practice Management and Accounting, tick the Use Accounting Exchange Rates option in Configuration > Settings > System Defaults > General/Master File.

 

Rate Type

Click to switch the rate type. This is seen in both Exchange Rates (AD/PM) and Exchange Rate (CA) screens if the option 'Use Accounting Exchange Rates' is enabled in System Defaults. Otherwise, you will see this ONLY in Exchange Rates (CA) screen. The rate types are set up in Account Exchange Rate Types [AEXT] lookup table.

Note: Different rate type is only applicable for Accounting component where you can set different rate type for different Account File. In Entity Administration and Practice Management components, the rate type will always be Default Rates [0] even if you choose to use Exchange Rates (CA) throughout Viewpoint.

 

Exchange Rate table

This lists the rates that you have set up.

 

Command buttons

Click on the relevant command button to perform the necessary.

  • New - Click this to create a new exchange rate.

    1. Enter the details for the new exchange rate:

      • Base Code - Select the base currency.

      • To Code - Select the conversion currency.

      • Rate - Enter the rate of conversion.

      • Date of Rate - Enter the date for the rate.

    2. Click Update.

    Note: Once you have created an exchange rate, when you click on it, the top of the table will show you the reverse rate.

    If you are using multi-currency rate list, you need to ensure that the reverse rate for each currency exchange is set up. This is because the system cannot utilize the reverse rate displayed at the top.

  • Delete - Click this to delete the selected exchange rate.

    Click Yes to confirm the deletion.

  • Update - Click this to update changes made to exchange rate(s).

    1. In Exchange Rate Update screen, click in the Rate column of the exchange rate that you want to update.

    2. Enter the new rate.

    3. The Date of Rate will default to today's date but you can change this if necessary.

    4. Repeat steps 1 - 3 for all other exchange rates you want to update.

    5. Click Update.

    6. Click OK.

    Note: If the difference between the old rate and the new rate is more than ten percent, a prompt will appear. Click Yes to continue with the change. If you click No, the new rate will not be saved.

    This is due to Exchange Rate reasonableness check that will compare the new exchange rate with the existing one.

  • Cancel - Click this to cancel changes made.

  • History - Click this to view previous rates for the selected exchange rate.
