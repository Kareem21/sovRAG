Service Rates

The Service Rates screen is where you update the Administrator rates, service prices and VAT rates. Additionally, if you have enabled the 'Multi Rate/CCY List' feature, you can also update the prices for each currency in this screen. You can access this screen through Viewpoint - Home screen by clicking Configuration > Business Parameters > Practice Management > Service Rates.

 

Unit Costs

Tick this to display the Cost - Time and Cost - Expenses columns in the table.

Note: The Cost - Time and Cost - Expenses columns for Administrators and Administrator Rate Level will be disabled if both the 'Multi Rate/CCY List' and 'Multi Rate/CCY List Cost of Services' options are selected. Cost - Time and Cost - Expense will have the same number of columns as Service Rates. (Value will be saved in the Service Rates table)

 

VAT/Tax

Tick this to hide/show the VAT/TAX column. This displays the VAT rates for all services but does not apply to Administrators and Administrator Rate Levels. If the option 'Use Accounting Tax Sub-ledger as Tax Codes' option is enabled in Configuration > Settings > System Defaults > Practice Management, this option will be disabled.

 

Charge Rates

Tick this to hide/show the different rate levels or currency columns in the table.

 

‘Copy Rates’ Options

Tick this to show the copy function that you can use to simplify the data entry process for the various rate and currency columns. Find out more here.

This function copies details from one column to another, hence if your existing Service Master comprises of services in different currencies, some information will be over-written in the destination column.

 

Source Column

Select the source column. The selections available depend on whether you have chosen to display a Rate List or a Currency. If you have selected a Rate List, then all available currencies for that rate list will be shown. If you have selected a currency, then all rate lists that use that currency will be displayed.

 

Copy to Column

Select the destination column. As with the Source Column, the selections available depend on whether you have chosen a Rate List or a Currency.

 

Multiply by

Enter the multiplication factor here. This will convert all rates in the Source Column by the multiplication factor, and the results are placed in the destination column.

 

Round

This determines whether the multiplication result will be rounded up or not. For example, if the multiplication result is 32.68:

  • select #,#00.00 to round up to the nearest hundred. In this instance, the result is 0.00.

  • select #,##0.00 to round up to the nearest ten, i.e. 30.00.

  • select #,###.00 to round up to the nearest one, i.e. 33.00

  • select #,###.#0 to round up to the nearest tenth, i.e. 32.70.

  • select #,###.## if you do not want any rounding up. The final result will remain as 32.68.

 

Update Sheet

Click this to perform the copy. The results of the copy can be seen in the destination column, and you can make amendments to any of the rates by overwriting the figure. You need to click Update before the results are saved.

 

Rate List

Select a rate list from the field to display the rates and charges for that list.

Note: This field will only be available if 'Multi Rate/CCY List' feature is enabled in the Configuration > Settings > System Defaults > Practice Management.

 

Currency

Use this to show all rates that use that currency.

Note: This field will only be available if 'Multi Rate/CCY List' feature is enabled in the Configuration > Settings > System Defaults > Practice Management.

 

Service & Administrator Rate List

The information displayed is determined by whether you have enabled the 'Multi Rate/CCY List' feature in the Configuration > Settings > System Defaults > Practice Management. Use this to show all rates that use that currency., and the options you have selected in the Select/Show section of the screen.

 

Command buttons

Click on the relevant command button to perform the necessary.

  • Update - Click this to save changes made. This button is activated only after you have edited details in the list.

  • Cancel - Click this to cancel the changes made. This button is activated only after you have edited details in the list.

  • Copy Sheet - This button is disabled when you amend the information in the table and is enabled after you have either saved or cancelled the change. The information copied follows what is displayed in the table, so if you have hidden the Unit Costs information, this will not be copied. Paste the information to an Excel spreadsheet for further action.
