Windows Menu

In this screen, you will see a list of all the screens for the Windows version of Viewpoint 8.0 where you can do the following:

  • Activate/deactivate a screen.

  • Set the authority required to access an active screen.

  • Set up custom menu.

 

Filters

The filters apply to the screen list.

  • Screen status - You can choose to view only active screens, inactive screens or all screens for a specific Menu Group.

  • Menu Group - This shows the available grouping for the screens. Select a Menu Group to view its screens.

  • Dashboard Configurable - This is used to filter the list of screens by the ability to add widgets to the screen.

 

Screen list

The list shows all existing screens for the selected Menu Group.

 

Command buttons

Click on the relevant command button to perform the necessary.

  • Edit - Click to edit settings for the selected screen.

    This includes:

    • Description - By default, this is the same as the Caption but it can be changed to describe the function of the screen.

    • Tag Values - Additional tags can be entered or copied from the Tag Values Wizard. See Windows Menu - Tags for more details.

    • Service Office Authorisation - This is to specify the authorisation method for the Service Office Model [8] is used. See Security Models for more details.

    • User Group Authority - When this is ticked, access to the screen will be based on the user's User Group. This will override other access control. By default, this is ticked for screens in the Menu Group 'Home [1]', 'Reporting [2]' and 'Configuration [4]'.

    • Updates through Workflow Only - When this is ticked, the screen will be set to read-only and all functions can only be performed via Workflows. For some screen types, the buttons will be hidden. This option is disabled for Section Menu, Folder, or Collapsed Folder types.

    • Customised - This will be ticked automatically when you make changes to the screen settings.

    • Active - This is to be ticked if you want to make the screen available for use.

  • Add External Link - Click to add an External Link item. The External Link is a section in the Section Menu that is not active by default. There is one for Home and File Menu Group. External Links are used to set up links to external applications so that these can be launched from Viewpoint. Two types of applications are supported - standard programs and ActiveX executables.

    Prior to adding the item, you have to ensure that the External Links section is active.

     

    Activating External Links

    1. In Windows Menu screen, select the Home or File Menu Group.

    2. Locate External Links and click Edit.

    3. Tick the Active option. If necessary, you can also change the Caption and Description.

    4. Click Update.

     

    Add links to External Applications

    1. In Windows Menu screen, select the Home or File Menu Group.

    2. Click Add External Link

    3. In the Description field, enter the path and name of the application:

      • Standard programs

        APP:<application path>\<app.exe><parameter1>,<parameter2>,<parametern> /msg=<warning message>

        For example: APP:C:\Temp\ViewPointDemo\viewpointdemo.exe/mf=(-MF-)/sr=(-SR-)/uc=(-UC-)/db=(-DB-) /msg=

      • ActiveX executables

        RVP:<ProgramID>,<FunctionName> /msg=<warning message>

        For example: RVP:ViewPointDemo.MyInfo,ExecuteMe /msg=<warning message>

    4. Click Update.

    Note: The following supported parameters are optional for standard programs, but are mandatory for ActiveX executables. The ActiveX function (API) must have the four parameters listed in the order shown below:

    • (-SR-) - database server name

    • (-DB-) - database name

    • (-UC-) - current logged on user

    • (-MF-) - current selected Master File

    The  /msg=<warning message> parameter will prompt <warning message> when you launch the external application. If no message is provided, no message will be prompted, or you will get a dash '-'. If you remove the parameter, multiple applications might be launched.

  • Delete External Link - Click to delete external link created earlier.

  • Edit Caption - Click this to update the Windows Menu caption and its related Ribbon or To Do Summary caption.
