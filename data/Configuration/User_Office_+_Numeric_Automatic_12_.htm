User Office + Numeric Automatic[12]

This method will result in Master File codes with a User Office code prefix followed by a number. The User Office code prefix follows the User Office of the user who created the Master File. In an environment where users may be from different jurisdictions, this coding method makes it easy to identify which User Office the Master File was created for.

 

Format of the Master File code

The first two characters of the code is the User Office code while the rest is from the 'Master File Codes [ADR]' system running number

e.g.

 

 

For the Master File Codes [ADR] system running number, this starts out at number 1 and increases so the length of a Master File code can be:

  • As short as three characters comprising the 2-character User Office code and a single-digit value for Master File Codes [ADR]; or

  • As long as ten characters comprising the 2-character User Office code and eight-digit value for Master File Codes [ADR]

If you want the length of Master File codes to be uniform, you can set a format for Master File Codes [ADR]:

  1. In Viewpoint – Home, go to Configuration > Settings > Customise > System Running Numbers.

  2. Select the document type ‘ADR’.

  3. Click Edit.

  4. In the Prefix & Format field, enter a ‘0’ (zero) for each digit.

    For example:

    • If you want the Master File code to be 8 characters, enter 000000 to get Master File codes like NL000012 and NL001234.

    • If you want the Master File code to be 10 characters, enter 00000000 to get Master File codes like NL00000012 and NL00001234.

  5. Click Update.

 

Using 'User Office + Numeric Automatic [12]' coding method

In order to use this coding method, ensure that the following is done:

  • Values for User Office is set up

  • User is assigned to a User Office

Once the above is done, you can then select this coding method in Configuration > System Defaults > General/Master File.

 

Setting up User Office Values

  1. In Viewpoint – Home, go to Configuration > Business Parameters > General > Lookup Values.

  2. Select ‘USOF – User Office’.

  3. Click New.

  4. Enter the following:

    • Element Code – this will be the User Office prefix and it can be any two alphanumeric characters.

    • Name – this is the name of the User Office.

  5. Click Update.

  6. Repeat until you have entered all required values for User Office.

 

Assigning User Office to Users

  1. In Viewpoint – Home, go to Configuration > Users & Security > Users.

  2. Select the user account and click Edit.

  3. Assign the relevant User Office to the User.

  4. Click Update.
