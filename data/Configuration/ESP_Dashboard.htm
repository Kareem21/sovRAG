Dashboard (ESP)

The Dashboard screen in CP-AD controls the settings for the widgets that appears in the CP-WP.

 

Filters

The filters apply to the Dashboard list. You can select either Active, Inactive or All.

 

Notification list

The list shows the widgets available in the ESP.

 

Command buttons

Click on the relevant command button to perform the necessary.

  • New - Click this to create a new widget.

    When creating a new widget, it is mandatory to enter both the Caption and the Supported Area (either in Home dashboard or Entity dashboard).

    You can only create widget of the following types:

    • e-View [104] - this widget type requires an e-View template to display the data retrieved from the Back-office. This is to be specified using a tag. Refer to ESP Dashboard - Tags for details.

    • RSS Feed [105] - this widget type will populate the widget with feeds from online site. Refer to ESP Dashboard - Tags for details.

  • Edit - Click this to edit the selected widget.

    You can edit the following:

    • Caption - to change the caption for the widget.

    • Tag Values - to change the properties of the widget. Refer to ESP Dashboard - Tags for details.

    • Customised - when ticked, changes made to the widget will be retained. Upgrading your environment will not overwrite the changes.

    • Active - when ticked, the widget is available to the user so it can be selected and added to the Dashboard.
