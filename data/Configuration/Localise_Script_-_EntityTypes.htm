Localise Script Tags - Entity Types

For Entity Types, tags are available for Localise Scripts to be used to:

  • Deactivate existing entity types

  • Add new entity types

  • Edit existing entity types

For new entity types, the entity types will have the Customised and Active checkboxes ticked by default.

 

Deactivating Entity Types

The tag below will deactivate entity types within a specified jurisdiction:

 

Syntax

ETYPE:<JurisdictionCode>+<EntityTypeCode>=InUse=0

 

where:

  • <JurisdictionCode> refers to the code of the jurisdiction for the entity type.

  • <EntityTypeCode> refers to the reference code of the entity type.

To deactivate multiple entity types from the same jurisdiction, separate each <EntityTypeCode> with a comma.

Note: Entity types that are configured as Viewpoint standard will be reinserted on the next upgrade if they have been deleted.

 

Example

Tag

Explanation

ETYPE:VG+PL=InUse=0

 This will deactivate the Limited Partnership [PL] entity type from the British Virgin Islands jurisdiction.

ETYPE:VG+PA,PL=InUse=0

 This will deactivate the International Limited Partnership [PA] and Limited Partnership [PL] entity types from the British Virgin Islands jurisdiction.

ETYPE:VG+PL,PL|KY+PA,PL=InUse=0

 This will deactivate the International Limited Partnership [PA] and Limited Partnership [PL] entity types from both the British Virgin Islands and Cayman Islands jurisdictions.

 

 

Adding and Updating Entity Types

The following is used to add a new entity type as well as to update an existing one. If the specified jurisdiction and entity type does not exist, then the localise script will create a new entity type. However, if it already exist, then the current configuration will be replaced by the configuration of the localise script.

 

Syntax

ETYPE:<JurisdictionCode>+<EntityTypeCode>=<DBField>=<Value>|<DBField>=<Value>|...|InUse=1|NavIDs=<NavigationID>

 

where:

  • <JurisdictionCode> refers to the code of the jurisdiction for the entity type.

  • <EntityTypeCode> refers to the reference code of the entity type.

  • <DBField> refers to the database field for the entity type and they are:

    • ShortName - Short Name field

    • LongDescr - Full Name field

    • Ownership - Ownership Structure field

    • ST_ReqSec - Secretary field

    • ST_ReqAud - Auditor field

    • ST_ReqAR - Annual Return field

    • ST_ReqAGM - AGM field

    • ST_MinShareHolders - Minimum Shareholders or Partners field

    • ST_MinDirs - Minimum Directors field

    • TagValues - Tag Values field

    • EtNote - Note field

  • <Value> refers to the value for the <DBField>. Use 1 to tick a checkbox and 0 to untick the checkbox.

  • <NavigationID> refers to the screen menu navigation ID to include for the entity type.

 

Example

ETYPE:VG+PG=ShortName=General Partnership|LongDescr=General Partnership| Ownership=PA|ST_ReqSec=-|ST_ReqAud=-|ST_ReqAR=-|ST_ReqAGM=-|ST_MinDirs=1|ST_MinShareHolders=1|TagValues=[FILTER:INT.TYPE=PLE,UBO]|InUse=1|NavIds=FSECOPYE,FSEGENER,FSERELAT,FSESTATC,FSEUSRFC,FSEPARTN,FSEPARPT,FSEPAPOS,FSEPAMEM,FSEPATRA,FSEOPINT,FSEREGIS,FSEFRREG,FSEBKACC,FSECHARG,FSENAMES,FSEDIVID,FSEACCTS,FSEACTCR,FSEAMEET,FSEAFILG,FSEAREVW

 

This will create a new entity type in the Entity Types screen where:

  • Jurisdiction: British Virgin Islands

  • Type Code: PG

  • Short Name: General Partnership

  • Full Name: General Partnership

  • Ownership Structure: Partnership [PA]

  • Secretary: Not required [-]

  • Auditor: Not required [-]

  • Annual Return: Not required [-]

  • AGM: Not required [-]

  • Minimum Shareholders or Partners:1

  • Minimum Directors: 1

  • Tag Values: [FILTER:INT.TYPE=PLE,UBO]

The navigation screen menu for this entity type includes:

  • Copy From Existing Screen [FSECOPYE]

  • General [FSEGENER]

  • Relations [FSERELAT]

  • Statutory [FSESTATC]

  • User Fields - Company [FSEUSRFC]

  • Partners [FSEPARTN]

  • Posting [FSEPARPT]

  • Classes [FSEPAPOS]

  • Members [FSEPAMEM]

  • Transactions [FSEPATRA]

  • Interests [FSEOPINT]

  • Registration [FSEREGIS]

  • Foreign Registrations [FSEFRREG]

  • Bank Accounts [FSEBKACC]

  • Charges [FSECHARG]

  • Name Changes [FSENAMES]

  • Dividends [FSEDIVID]

  • Accounts [FSEACCTS]

  • Activities - Company [FSEACTCR]

  • Meetings [FSEAMEET]

  • Filings [FFSEAFILG]

  • Reviews [FSEAREVW]
