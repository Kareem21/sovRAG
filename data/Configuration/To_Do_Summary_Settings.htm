To Do Summary Settings

The Summary Settings screen allows user to customise the captions of items in Summary screens.

 

Category

This lists two categories - File and Home referring to the Summary screen in Viewpoint - Home and Master File screens.

 

Item list

This lists the items in the selected Summary.

 

Edit

Click this to change the caption of the selected Summary item.
