Bank Accounts

Creating a list of revenue entities' bank accounts in the system will allow you to select the appropriate bank account from the list each time you enter a receipt in Practice Management.

 

Revenue Entity

This shows all the existing revenue entities. Select the revenue entity to view its bank accounts, or to set up a new bank account.

 

Bank Account list

This shows all the bank accounts set up for the selected revenue entity.

 

Command buttons

Click on the relevant command button to perform the necessary.

  • New - Click this to add a new bank account.

    1. Enter details for the bank account.

    2. Click Update.

  • Edit - Click this to edit an existing account.

  • Delete - Click this to delete an existing account. This is possible if the bank account is not linked to any receipt and there is no mapping for the bank account to an account.

Tag Values

Users can use the tag values in Bank Accounts to define the Unit Code 2 value in the Accounting Journals tab when posting invoice to journals. The following shows the tags that are available for sales invoice, purchase invoice and matter receipts.

 

If Receipt Base Amount >/= 0

If Receipt Base Amount < 0

Sales Invoice

[R-SR=<Unit Code2>]

[R-SP=<Unit Code2>]

Purchase Invoice

[R-PR=<Unit Code2>]

[R-PP=<Unit Code2>]

Matter Receipt

[R-MR=<Unit Code2>]

[R-MP=<Unit Code2>]

Note: The tags are only applicable to users who have selected B(ank) Sub Ledgers Only [3] for Unit Code 2 in Accounting > Account File > Unit Codes in File Maintenance.
