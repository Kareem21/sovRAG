Localise Script Tags - Field Settings

Tags are available for Localise Scripts to set the caption and authority required for update for Viewpoint standard fields.

 

Setting Field Caption

When using Localise Script to set the caption, this apply also to Custom Reports and Grid Fields. If you have customised the caption earlier, this will also be over-written by the caption defined by the Localise Script.

 

Syntax

FIELDCAP:<FieldDef>+<DBField>=<Caption>

 

where:

  • <FieldDef> can be retrieved from Field Definition in Configurations > Settings > Customise > Field Settings.

  • <DBField> is retrieved from the Database Field.

  • <Caption> is the caption for the field.

 

Example

To set the caption for CDD Note as ‘Additional Information’: FIELDCAP:KYC.Note+KYC_Note=Additional Information

 

Setting Update Authority

The tag below allows you specify the value for Auth. Req. for Update setting of the field.

 

Syntax

FIELDAUTH:<FieldDef>+<DBField>=<Authority>

 

where:

  • <FieldDef> can be retrieved from Field Definition in Configurations > Settings > Customise > Field Settings.

  • <DBField> is retrieved from the Database Field.

  • <Authority> is the value for the Auth. Req. for Update field.

 

Example

To set the authority for CDD Note to Regular User [6]: FIELDAUTH:KYC.Note+KYC_Note=6

Note: The Auth. Req. for New should not be changed without consulting Viewpoint.
