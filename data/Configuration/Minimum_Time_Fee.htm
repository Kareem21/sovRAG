Minimum Time Fee

There are two Minimum Time Fee service classes:

  • Minimum time fee - with carry forward [RM]

  • Minimum time fee - w/o carry forward [RN]

Take note of the following when you are setting up recurring fees of the service class Minimum Time Fee.

 

Configuration

In Configuration > Settings > System Defaults > Time and Disbursements. It is compulsory for you to tick 'Use Recurring Services for Time Entries' option. The reason is that you need to use the same service for time entry.

 

Practice Management - Sales Ledger

When setting up the Recurring Fee the following fields must be complete as specified:

 

Currency

If you are not using the multi-rate list feature, then this must be the same as your base currency. If you are using the multi-rate list feature then this must be the Matter currency.

 

Number of Units

Set this as 1.

 

Unit Price

Enter the minimum amount to charge the client. If the total time charged is less than this amount, the program will add a time charge for the shortfall, using the date entered in the - to field in the Posting of WIP screen. This time charge is reversed the next day if the service class is Minimum time fee - with carry forward [RM].

Yes, you can, by customising the information in the Tag information field of the service in the Service Master.
