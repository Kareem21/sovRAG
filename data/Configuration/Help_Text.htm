Help Text

Help text refers to explanation for Viewpoint fields which can be seen when the user mouse over the Help Text icon (information icon) next to the field. To enable the Help Text, tick the option Enable 'Help Text'  in Configuration > Settings > System Defaults > General/Master File.

To enter the explanation, this can be done in:

  • Using the Help Text button in Configuration > Settings > Customise > Field Settings.

  • Though the Help Text tab.

When entering the help text, you can also include the following codes:

  • <BR> - This will add a line break.

  • >?> - This will add a bullet point to the front of the line
