Verify Settings

The Verify Settings function checks the for errors in the following areas:

  • General: Document numbers (such as form queue number), duplicate Master File number and duplicate Document File Index Sequence.

  • Practice Management: Document numbers for all Revenue Entities.

  • Selected Account File: Draft and journal numbers for the selected Account File.

If the system finds that there is an overlap in the document range, the 'Repair selected' option enables the user to automatically set the document number to the last number found in the range.

 

General/Practice Management/Selected Account File

Select the area you want to check. For Selected Account File, specify the Account File that you want to check.

 

Verify Settings

Click to begin the verification process.

 

Repair Selected

Click to repair the selected records. This can only be done for items that are in red. Tick the Repair? check box next to these items, and then click Repair Selected.
