Folder Templates

Folder templates can be created to reduce the time taken to store a document as they store commonly-used information for index fields. You can create as many templates as you need to represent the Master Files and document descriptions that you commonly use.

Each folder template is specific to the Filing Cabinet it was created for and can be used by any users. Note that you need to first enable the use of Folder templates to use this feature.

 

Filing Cabinet structure

This shows the filing cabinet structure that has been set up. Select the filing cabinet the folder template is for from the list.

 

Folder Template list

This list shows the folder templates for a selected filing cabinet.

 

Command buttons

Click on the relevant command button to perform the necessary.

  • New - Click to create a new folder template for the filing cabinet that you have selected.

    1. Complete the index fields for the folder template:

      • Template Title - Enter the title for the template. This value will appear in the Select Template list.

      • Description - Enter description for the document. This is optional and the value here will be the Holder's Description if the template is selected.

      • Master File - Select the Master File the template is created for. You may also use the MF.FOUND tag in any Master File index field type to display the last selected Master File.

      • Document Date - Enter the document date. You may select {NOW} for today's date or {NOW+0} for today + n number of days.

    2. Double-click the line under Select Template. Enter a description for the template. This will be displayed when selecting the template. If you skip this step, the text entered for the Description index field will be displayed instead.

    3. Click Update.

  • Edit - Click this to edit selected folder template.

  • Delete - Click this to delete selected folder template.

  • Close  - Click this to exit the screen.

 

Using Folder Template

If you have created folder templates for a filing cabinet, you will be able to select the folder template when you store a document.

  1. Choose to store the document.

  2. Select the Filing Cabinet.

  3. Select the folder template you want to use.

  4. Click Select. Index fields that have been specified in the folder template would have been completed.

How do I enable folder templates?

In Viewpoint - Home screen, go to Configurations > Settings > System Defaults > Document Management. Tick the option 'Enable Folder Templates'.
