Exception Log

Exception errors in Viewpoint are recorded here if System Defaults > Security has the option Log to Database [2] set for Exception Log. When you first access the screen, the filter criteria list is displayed with Log Type = 50. However, you can further filter the log by:

  • Sequence No.

  • Description

  • Log

  • Log Date

  • User Code

  • Application

If you want to view the full log, click Apply without entering any value for the criteria.

 

Select button

This displays the filter criteria list.

 

Exception Log

This shows exception errors that occurred in Viewpoint.

 

Command buttons

Click on the relevant command buttons to perform the necessary.

  • Select - Click this to filter the exception log.

  • E-mail - Click this to send a support e-mail for the selected error.

  • Delete - Click this to deleted the selected error record.
