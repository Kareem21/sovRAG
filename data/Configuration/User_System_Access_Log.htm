User/System Access Log

This is a record of user activity in Viewpoint related to Viewpoint access. This record is maintained only if the option 'Maintain User/System Access Log' in System Defaults > Security is ticked. When you first access the screen, the filter criteria list is displayed so that you can filter the log by:

  • Log Type

    Values for the log type can be:

    • 1 - Login Successful/ Login Successful - password change failed

    • 2 - User logged out.

    • 3 - Invalid user ID or password/ User ID was deactivated/ Login successful - password changed/ Login failed, concurrent login is not allowed

    • 6 - Activities in Viewpoint Management for Viewpoint Connect upgrade.

      The activities logged are:

      • Table pre-upgrade - This is logged after a Pre-upgrade process has completed.
      • Pre-upgrade Aborted - Pre-upgrade cancelled by user - This is logged after the Pre-upgrade process was cancelled by a user.
      • Upgrade Started - This is logged after the Upgrade process has started. This log will only appear during the upgrading process. This will not appear when the upgrade is complete.
      • Upgrade Completed - This is logged after the Upgrade process has completed.
      • Upgrade Aborted - Upgrade cancelled by user - This is logged after an Upgrade process was cancelled by a user.
      • Upgrade Aborted - Initialisation failed - This is logged after an Upgrade process fails due to issues with the upgrade package (e.g. missing files in the package)

    • 50 - System Exceptions - Records for this log type can only be viewed in the Exception Log screen.

  • Description

    This refers to the description for the log type. Refer to Log Type for sample description.

  • Log Date

  • User Code

  • Application

  • Log details

  • Machine logged on user

  • Machine name

  • Machine IP address

If you want to view the full log, click Apply without entering any value for the criteria.

 

Select button

This displays the filter criteria list.

 

User/System Access Log

The log shows the date and time of user activity.

 

Command buttons

Click on the relevant command button to perform the necessary.

  • Print - Click this to print the User/System Access Log.

  • Delete - Click this to delete a selected entry of the User/System Access Log.
