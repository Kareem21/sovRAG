User Reset

This function is used to reset passwords for users who have been locked out of the system. This function can also be done in the Users screen if you have sufficient authority to access that screen.

 

Filters

The filters apply to the user account list.

  • User account status - You can choose to view only active user accounts, inactive user accounts or all user accounts.

  • Administrator Group - You can choose to view user accounts that are assigned to a specific Administrator Group.

 

Select button

This allows for additional criteria to apply to the user account list.

 

User account list

The list shows all existing user accounts.

 

Command buttons

Click on the relevant command button to perform the necessary.

  • Password - Click this to reset the password and/or change the user account status.

    To change password:

    1. In the Assign/Reset Password screen, tick the Change password option.

    2. Enter the new password in New Password and Confirm Password field.

    3. Click Update.

    Other Options

    • Auto generate password and e-mail user login link - This will send a password reset e-mail to the user. This option is only available in Engage Servicing Portal > Configuration> Users screen for ESP Front-Office.

    • Require password change on next log-on - Tick this to prompt the user to change the password on the next log in.

    • User login active - Tick this to set the user account as active. If this is not ticked, the user account becomes inactive.

    • Allow Select - When ticked, the user's name will be displayed for selection in Viewpoint.

    • Administrator Rate - When ticked, the Administrator Rate for the user is set to active.

    • Reset secret key - Tick this to reset the secret key of the two-step verification feature. This option is only available if two-step verification is enabled in Configuration > Settings > System Defaults > Security.

     

  • Reset Connection - Click this to reset connection in case the user account has been 'locked' and user is no longer able to login to the application.
