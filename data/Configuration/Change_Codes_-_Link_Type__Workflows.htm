Link Type: Workflows

This link type is to allow for a workflow to be included in the Change Processing screen.

To set up this link item, do the following:

  1. In Configuration > Business Parameters > General > Change Codes screen, select the relevant Change Code.

  2. Click New.

  3. Specify the following details:

    • Link Type - Select Tasks.

    • Description - Enter a description for the workflow that will appear in the Change Processing screen.

    • Tag Values - Enter tag values to specify the task details. Refer to Change Codes: Link Item - Tags.

    • Help Text - Enter additional text that will also be displayed, if relevant.

  4. Click Update.
