System Running Numbers

Incremental numbers are assigned to documents and records in Viewpoint and the most current value for a document/record type can be seen in the System Running Numbers screen.

 

Item list

This lists the document/record types in the Viewpoint environment with its current value and any prefix or format set for the document/record type.

 

Command buttons

Click on the relevant command button to perform the necessary.

  • New - Click to add the document/record type. Note that the document/record type are in the Document Number Codes [DOCT] lookup table.

    1. Enter the details for the new document/record type.

    2. Click Update.

  • Edit - Click to edit details of an existing document/record type.

  • Delete - Click to delete the selected document/record type.
