Link Type: Macros

This Link Type can be used to perform a predefined action to replace the need for user to do it manually. When the macro is executed, the result will be the same as a manually-completed action. For instance, a macro to resign a statutory officer will update the record (changing the officer status from Appointed to Resigned) and display the Change Processing screen for user to select the relevant linked items for the resignation.

To set up this link type, do the following:

  1. In Configuration > Business Parameters > General > Change Codes screen, select the relevant Change Code.

  2. Click New.

  3. Specify the following details:

    • Link Type - Select Macros.

    • Description - Enter a description for the macro.

    • Tag Values - Enter the tags to define the action. For further information on the tags, you can refer to Macros - Tags.

  4. Click Update.
