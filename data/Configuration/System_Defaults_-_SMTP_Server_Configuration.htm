System Defaults - SMTP Server Configuration

If you are using Viewpoint Task Scheduler and Notification Agent, this is where you configure the SMTP Server for e-mail notifications.

 

From Address

Enter the e-mail address that will be used to send the e-mail notifications.

 

From Display Name

Enter the display name for the e-mail address.

 

SMTP Server

Specify the IP or server name of the mail server.

 

SMTP Server Port

Specify the port number of the SMTP Server.

 

Enable SSL

Tick this if you will be using SSL.

 

Authentication Username

Enter the authorised user name. Leave this blank if authorisation mode is turned off.

 

Password

Enter the password for authentication. Leave this blank if authentication mode is turned off.

 

Confirm Password

Enter the password one more time.

 

Command buttons:

Click on the relevant command button to perform the necessary.

  • Update - Click this to save changes.

  • Test E-mail Setting - Click this to test the settings.

  • Cancel - Click this to cancel the changes.
