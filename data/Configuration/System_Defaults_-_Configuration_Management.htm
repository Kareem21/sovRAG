System Defaults - Configuration Management

This applies to Master Configuration Environment (MCE) and Data Replication and is used to specify the environment as either the Master Configuration Environment or subscriber environment. By default, this screen is not active.

 

Set as Master Configuration Database

Tick this to set the environment as the Master Configuration Environment. When this option is ticked, the 'Subscriber Groups' textbox will be disabled.

 

Source Folder Path

This field is enabled when 'Set as Master Configuration Database' is ticked. Specify the source folder path. Folder path validation is disabled when Viewpoint Connect is set up in Cloud Solution ('Enabled Viewpoint Connect to connect through Application Server' is ticked in Viewpoint Management with Application Server running).

 

Subscriber Groups

Enter the code of the subscriber group(s) that the environment belongs to. The codes are based on the values in Replication Subscriber [RPSB] lookup table. For multiple codes, separate each code with a semi-colon e.g. AME;EUR.

Note: The codes will be displayed as uppercase characters.

 

Generate Replication Log File

Tick this to have the system generate a log file after an import job has been completed. This applies only to subscriber environments.

 

Command buttons:

Click on the relevant command button to perform the necessary.

  • Update - Click this to save changes.

  • Cancel - Click this to cancel the changes.
