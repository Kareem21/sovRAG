Tag Values

When necessary, you can use tags (which are special codes that provide instructions to the program) to define certain values to further configure the program to your needs. These tags can be applied to a variety of items ranging from services to reports.

Depending on the function, the tags will differ but are generally entered in either Tag Values or Tags field. At the moment, tags are available for the following:

  • Action Management

  • Approval Cards

  • Change Codes

  • Chart of Accounts - Nominal Accounts

  • Checklists/Workflow Steps

  • Checklist items

  • Entity Types

  • ESP Menus

  • ESP Dashboard

  • Excel Statements - Design

  • File Review

  • Identity/AEOI Register Types

  • Journal Scripts

  • Localise Custom Script

  • Lookup Tables

  • Payment Types & Export Batches

  • Relationship Types

  • Revenue Entity

  • Review Rules

  • Screen Settings

  • Service Master

  • Score Assignment for checklist and workflow

  • Task Scheduler

  • Tasks

  • Users

  • User Tables/Values

Note: For Entity Types, Lookup Table values and Review rules. if you require multiple tag configuration, then you can use  Referencing Tags.
