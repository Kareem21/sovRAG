Drop-down Values

Drop-down tables are fields where you can enter a value or select one from the drop-down list. For example, the Title field in the Master File - General screen is a drop-down list that can be over-written. The entries in these fields can be populated so that distinct new entries are added to the database.

 

Drop-down Table list

This shows all the available drop-down tables in Viewpoint. Click to select the table you require.

 

Command buttons

Click on the relevant command button to perform the necessary.

  • New - Click this to add a new value to the selected table when you do not want the table to be updated by user input. This requires setting the mask value 1 for the field that utilises the table. For Nationality Individual Codes [NATI] table, you will be able to add the value and link it to a country from the Country Codes [CNTR] lookup table.  

    For example, the drop-down table 'Titles for Individuals [ITIT]' is used for the Title field in Master File - General screen for Master Files of Individual Master File type.

    To prevent update to this table by user input, do the following:

    1. Go to Settings > Field Settings.

    2. Select the screen by specifying:

      • Menu Group - File [8]

      • Section - General [FTDMAINF]

      • Screen - General [FMMGENEI]

    3. Select the Title field.

    4. Click Edit.

    5. In Mask, enter 1.

    6. Click Update.

     

  • Edit - Click this to edit the selected value.

    1. Make the change.

    2. Tick Global Update to globally update the name from the previous one to the new one.

    3. Click Update.

     

  • Delete - Click this to delete the selected value.

  • Populate - Click this to update the selected table with new values that may have been entered elsewhere in Viewpoint.
