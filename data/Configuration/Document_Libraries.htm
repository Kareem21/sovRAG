Document Libraries

The Document Libraries is where document templates and widgets used throughout Viewpoint are maintained.

 

Document Templates

For document templates, these are sample templates that are pre-installed but these are specimens and should be approved by your organisation before use. Viewpoint accepts no liability for any errors in the documents.

These templates can be:

  • Word document templates - which are created in Microsoft Word program in either .doc or .docx format. The default output is also as .doc or .docx file. For each Word template, you will have:

    • .doc or .docx file - This is the Microsoft Word file of the template.

    • .def file - This contains the template properties.

  • PDF templates - These will generate as PDF files by default.

  • HTM/HTML templates - These are used for Summary Screens in Viewpoint Connect. .

The templates are categorised under the following heading:

  • Form Library - This library contains templates for statutory forms and minutes which can be generated only when triggered by the linked change(s).

  • Word Reports - This library contains templates for reports and registers that can be generated as and when necessary.

  • Correspondence - This library contains the templates that are used for correspondence.

  • Billing Templates - This library contains templates for billing documents such as debit notes, credit notes, receipts and statements.

  • Other Document Templates - This library contains templates for the index page of Binders feature and e-mail notifications from Task Scheduler.

  • HTML Templates - The templates used for Summary Screens are located here.

  • Standard Reports - This library contains template for Viewpoint reports, SSRS reports and Analysis Reports.

Widgets

A widget is graphical user interface element that displays information in a specific way. In Viewpoint Connect, widgets in the Dashboard Widgets screen can be added to Overview screen and Dashboard screen.
