Audit Trail Setup

Once the audit trail has been enabled in System Defaults > Document Management, you need to select the areas you want to trail.

 

Audit type list

This list the audit types that can be included in the audit trail. Tick to select the types to include.

 

Command buttons

Click on the relevant command button to perform the necessary.

  • Update - Click this  to save your changes.

  • Cancel - Click this to cancel any changes.

 

Note: Viewpoint will only start keeping an audit trail from the time the feature is enabled. If the feature is switched off and then on again, there will be no records kept for the time the trail was switched off.
