Account Mapping - Client Accounts

The Account Mapping is completed by selecting the Account Type followed by a specific nominal account from the Chart of Accounts (CoA) template in Accounting. This mapping must be completed for each CoA template used in Client Account Files.

Sales Invoice Received

C.SC

Creditor

  1. When a sales invoice is posted, the invoice amount will be posted to this account as a Credit entry.

  2. When a receipt or a settlement is recorded on the sales invoice subsequently, the amount will be updated to this account as a Debit entry.

1 - Assets Posting

2 - Liability Posting

3 - Expenses Posting

4 - Income Posting

C - Creditor

2 - Liability Posting

Select this option if you wish all posted invoices to be recorded directly in a posting account.

C - Creditor

Select this option if you wish to maintain a separate Creditor sub-ledger for each invoice currency and/ or each Revenue Entity.

C.SL

Exchange Difference Loss

When a receipt or a settlement is recorded on a sales invoice, any exchange differences in term of losses will be updated to this account.

1 - Assets Posting

2 - Liability Posting

3 - Expenses Posting

4 - Income Posting

A loss account can be used.

C.SR

Rounding Difference

Rounding differences may occur when a sales invoice is posted, depending on which option is used when posting the journals.

1 - Assets Posting

2 - Liability Posting

3 - Expenses Posting

4 - Income Posting

A profit/loss account can be used.

C.SH

Bank Charges

When a receipt is recorded on a sales invoice and includes bank charges, the bank charges will be updated to this account as a Debit entry.

1 - Assets Posting

2 - Liability Posting

3 - Expenses Posting

4 - Income Posting

3 - Expenses Posting

C.SE

Exchange Difference

When a receipt or a settlement is recorded on a sales invoice, any exchange differences will be updated to this account.

1 - Assets Posting

2 - Liability Posting

3 - Expenses Posting

4 - Income Posting

A profit/loss account can be used.

C.SW

Write Off

When a receipt or a settlement is recorded on a sales invoice and includes an amount written off (i.e. expenses written off), the amount written off  will be updated to this account as a Credit entry.

1 - Assets Posting

2 - Liability Posting

3 - Expenses Posting

4 - Income Posting

A profit/loss account can be used.

C.SV

VAT Invoiced

When a sales invoice is posted, the sales tax (e.g. VAT/GST) invoiced will be posted to this account as a Debit entry.

1 - Assets Posting

2 - Liability Posting

3 - Expenses Posting

4 - Income Posting

A balance sheet account can be used.

Receipt/Payments

C.GB

Payment/Receipt (clearing account)

When a receipt is recorded on a sales invoice, the amount will be updated to this account as a Credit entry

1 - Assets Posting

2 - Liability Posting

3 - Expenses Posting

4 - Income Posting

The clearing account can be set up as a balance sheet or a profit/loss account.

If set up as a profit/loss account, ensure that you have cleared it before the financial year-end as any balances will not be carried forward.

C.GM

Matter Account Balances

  1. When money is received on a Matter account (e.g. client advances cash), the amount will be updated to this account as a Debit entry.

  2. When a settlement is recorded on a sales invoice or money is paid out of a Matter account (e.g. payments out of client cash advances), the amount will be updated to this account as a Credit entry.

1 - Assets Posting

2 - Liability Posting

3 - Expenses Posting

4 - Income Posting

C - Creditor

D - Debtor

1 - Assets Posting

Select this option if you wish all Matter transactions to be recorded directly in a posting account.

D - Debtor

Select this option if you wish to maintain a separate Debtor sub-ledger (i.e. prepayment account) for each Matter currency and/ or each Revenue Entity.

C.GC

Bank Charges

When a settlement is recorded on a sales invoice, money is received on or paid out of a Matter account (e.g. client advances cash or payments out of client cash advances), the bank charges will be updated to this account as a Debit entry.

1 - Assets Posting

2 - Liability Posting

3 - Expenses Posting

4 - Income Posting

3 - Expenses Posting

C.GE

Exchange Difference

When money is paid out of a Matter account (e.g. payments out of client cash advances), any exchange differences will be updated to this account.

1 - Assets Posting

2 - Liability Posting

3 - Expenses Posting

4 - Income Posting

A profit/loss account can be used.

C.GL

Exchange Difference Loss

When money is paid out of a Matter account(e.g. payments out of client cash advances), any exchange differences in term of losses will be updated to this account.

1 - Assets Posting

2 - Liability Posting

3 - Expenses Posting

4 - Income Posting

A loss account can be used.

C.GR

Rounding Difference

Rounding differences may occur when a receipt or a settlement is recorded on a sales invoice, depending on which option is used when posting the journals.

1 - Assets Posting

2 - Liability Posting

3 - Expenses Posting

4 - Income Posting

A profit/loss account can be used.

Services

C.SA

Expense Accounts

When a sales invoice is posted, the invoice amount will be posted to this account as a Debit entry.

1 - Assets Posting

2 - Liability Posting

3 - Expenses Posting

4 - Income Posting

1 - Assets Posting

Select this option if the service is billed in advance.

3 - Expenses Posting

Select this option if the service is billed in arrears.
