Using Change Codes for (Invoice) Document Generation

This option allows for billing documents such as invoices to be generated using change codes by linking  the template to the relevant change code(s) instead of specifying the template in Configuration > Business Parameters > Practice Management > Setup > Document Templates.

When using this method, you need to have separate template for draft invoice and posted invoice; and ensure that each template is then linked to the correct change codes.

If you currently have one single template used for both draft and posted invoice, this must now be separated so that:

  • Template for draft sales invoice has SQL queries to ProformaHeader and ProformaItems tables.

  • Template for posted sales invoice has SQL queries to InvoiceHeader and InvoiceItems tables.

  • Template for draft purchase invoice has SQL queries to PurchaseHeader and PurchaseItems tables.

  • Template for posted purchase invoice has SQL queries to InvSupHeader and InvSupItems tables.

 

Configuration Required

For Viewpoint Connect program, you need to enable the option 'Using Change Codes for (Invoice) Document Generation' in System Defaults > Practice Management. For the templates, you need to add the change code as Change Links for the template. This can be done through Change Codes or Document Libraries.

 

Link through Change Codes

  1. In Viewpoint – Home, go to Configuration > Business Parameters > Change Codes.

  2. Select Practice Management [2].

  3. Select the Change Code.

  4. Click New.

  5. In the Change Link screen, specify the following:

    • Link Type - Select Word Reports.

    • Word Reports - Select the relevant template.

    • Tag Values - Add [HD-SEL=] tag e.g. [HD-SEL=ProformaNr=(+CC.ProformaNr+)].

    • Additional change parameters if applicable.

  6. Click Update.

 

 

Link through Document Libraries

  1. In Viewpoint – Home, go to Configuration > Document Libraries > Billing Templates.

  2. Select the Document Type containing the template.

  3. Select the template.

  4. Click New Change.

  5. In the Change Link screen, specify the following:

    • Change Code -  Select the relevant change code.

    • Tag Values - Add [HD-SEL=] tag e.g. [HD-SEL=ProformaNr=(+CC.ProformaNr+)].

    • Additional change parameters if applicable.

  6. Click Update.

 

Generating the Template

Once the template has been linked, you will be able to see it in the Change Processing screen when the linked change occurs. To select, ensure it is ticked and click Queue.

In Document Queue, select the item and click Generate.

 

Change Codes

Change Codes that have been added for use with this feature are as follows:

 

Change Code

Occurs when you click...

PCDI – Print Draft Sales Invoice

  • Invoice - In Recurring Fees screen (Sales Ledger).

  • Prepare Draft(s) - In Posting of WIP screen (Sales Ledger).

  • Print - In Draft Invoices screen (Sales Ledger).

  • Print - In Create/Edit Invoice screen (Sales Ledger).

  • Print - In Settlements – Drafts tab (Sales Ledger).

Note: In Recurring Fees and Posting of WIP, the option ‘Print Document (to document queue)’ must be ticked.

PCOI – Post Sales Invoice

  • Invoice - In Draft Invoices screen (Sales Ledger).

  • Invoice - In Create/Edit Invoice screen (Sales Ledger).

  • Invoice - In Settlements – Drafts tab (Sales Ledger).

Note: In all screens, the option ‘Print Document (to document queue)’ must be ticked.

PCSI – Print Sales Invoice

  • Reprint Invoice - In Receivables screen (Sales Ledger).

  • Print - In Settlements – Invoice tab (Sales Ledger).

PMPR – Post Receipt

  • Enter Receipt - In Receivables screen (Sales Ledger).

  • Receipt/Payment - In Settlements – Invoice tab (Sales Ledger).

  • Settle to Matter - In Settlements – Invoice tab (Sales Ledger).

  • Receipt/Payment - In Settlements – Drafts tab (Sales Ledger).

  • Settle to Matter - In Settlements – Drafts tab (Sales Ledger).

  • Receipt - In Settlements – Matters tab (Sales Ledger).

  • Cancel - In Settlements – Transactions tab (Sales Ledger).

Note: In all screens except Settlements – Transactions tab, the option ‘Print Voucher’ must be ticked.

PMPP – Print Receipt

  • Print - In Settlements – Transactions tab (Sales Ledger).

PPDS – Print Debtors Statements

  • Print Statement - In Receivables screen (Sales Ledger).

PPMS – Print Matter Statements

  • Print - In Settlements – Matters tab (Sales Ledger).

PSDI – Print Draft Supplier Invoice

  • Print - In Purchase Orders screen (Purchase Ledger).

  • Print Order - In Create/Edit Purchase Orders screen (Purchase Ledger).

  • Print - In Settlements – Purchase Orders tab (Purchase Ledger).

PSOI – Post Supplier Invoice

  • Invoice - In Purchase Orders screen (Purchase Ledger).

  • Invoice - In Create/Edit Purchase Orders screen (Purchase Ledger).

  • Invoice - In Settlements – Purchase Orders tab (Purchase Ledger).

Note: In all screens, the option ‘Print Document (to document queue)’ must be ticked.

PSSI – Print Supplier Invoice

  • Reprint Invoice - In Payables screen (Purchase Ledger).

  • Print - In Settlements – Invoice tab (Purchase Ledger).

PMPS – Post Payment

  • Settlement - In Payables screen (Purchase Ledger).

  • Payment/Receipt - In Settlements – Invoice tab (Purchase Ledger).

  • Payment/Receipt - In Settlements – Purchase Orders tab (Purchase Ledger).

  • Payment - In Settlements – Matters tab (Sales Ledger).

Note: In all screens except Settlements - Matters, the option ‘Print Voucher’’ must be ticked.

PMPQ – Print Payment

  • Print - In Settlements – Transactions tab (Purchase Ledger).

PPCS – Print Creditor Statements

  • Print Statement - In Payables screen (Purchase Ledger).
