Link Type: Notifications

This link type is to create notification items to inform users of changes made or work assigned to the user. To set up this link item, do the following:

  1. In Configuration > Business Parameters > General > Change Codes screen, select the relevant Change Code.

  2. Click New.

  3. Specify the following details:

    • Link Type - Select Notifications.

    • Description - Enter a description for the notification.

    • Selection - Selecting Notifications in Link Type will default to the option 'Mandatory - selection not visible [X]'. This option is required for notifications to work.

    • Tag Values - Enter the tags to configure the notification such as the header, message related screen and the intended recipient of the notification.

  4. Click Update.

 

Localise scripts are also available to have certain notification items created by running the localise script. Refer to Localise Script - Link Type (Notifications).
