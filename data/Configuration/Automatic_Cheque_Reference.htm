Automatic Cheque Reference

For Bank Accounts journal scripts, the cheque reference can be automated. To do this, you need to:

  • Set up a user field with the key 'CHQREF'

    1. In Viewpoint - Home screen, go to Configuration > Business Parameters > User Fields > Account File, click New.

    2. Tick General Field.

    3. Enter the following information:

      • Key - Type CHQREF.

      • Field Type - Select Number[13].

      • Caption - Enter a suitable caption e.g. Cheque Reference.

    4. Click Update.

     

  • Retrieve the user field in Accounting > Account File > User Fields - Account File and enter the cheque number

Once you have done this, the value will be picked up in Bank Accounts Journal Scripts using the code {UFV.CHQREF} code.
