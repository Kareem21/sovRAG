Localise Script Tags - Risk Ratings

Localise Script for Risk Ratings is to add records to the Risk Ratings screen where:

  • If the table or type does not have the Risk Rating option ticked initially, running the script will tick this.

  • Effective date will be the date of when the localise script is run.

 

Syntax

RRATE: SET=RateGroup=<RateGroup>|TableType=<TableTypeCode>|KeyCode=<KeyCode>|KeyValue=<KeyValue>|Values=<Values>

 

where:

  • <RateGroup> refers to the code of the Rating Group. Rating Groups are maintained in the Risk Rating Group [RRGR] lookup table.

  • <TableTypeCode> refers to the code that is made up of the item and the rating type with the first character denoting the item and the second one denoting the type.

    For item, the code can be L - for Lookup Table, U - for User Table, R - for Relationship Type, I - for Identity Register Type, S - for Service Master.

    For type, the code can be C - for Current, R - for Prepare Revision, S - for Scenario Evaluation.

  • <KeyCode> refers to the code for the item but does not apply to Identity Register Type and Service Master. This is required for:

    • Lookup Tables - Enter the Lookup Table code.

    • User Tables - Enter the User Table code.

    • Relationship Type - Enter the code of the Relation Class.

  • <KeyValue> is the code of the value.

  • <Values> is the score to be set for the value.

 

Example

To set a score of 10 for the Female [F] value in the Gender [GEND] Lookup Table under Current type for Standard Rating Group [STDR]: RRATE:SET=RateGroup=STDR|TableType=LC|KeyCode=GEND|KeyValue=F|Values=10

 

To set a score of 20 for the Male [M] value in the Gender [GEND] Lookup Table under Current type for Standard Rating Group [STDR]: RRATE:SET=RateGroup=STDR|TableType=LC|KeyCode=GEND|KeyValue=M|Values=20

 

To set a score of 30 for the Unknown [U] value in the Gender [GEND] Lookup Table under Current type for Standard Rating Group [STDR]: RRATE:SET=RateGroup=STDR|TableType=LC|KeyCode=GEND|KeyValue=U|Values=30
