Localise Script - Link Type (Notifications)

 

Viewpoint provides two localise scripts that you can run to create certain notification items for the Notifications feature. The localise scripts are:

  • Add Notifications for Data/Information Changes - This will add notification items to inform users about changes made to file details such as change of entity status.

  • Add Notifications for Assignments - This will add notification items to inform users about new work assigned such as a new task. This type of notification items is linked to new Document Management change codes that have been added. To include these notification items, ensure that the option 'Enable Audit Trail' and 'Add Master File Code to Master File Name Display' are enabled in System Defaults > Document Management and you have selected task and scan batch audit types in Business Parameters > Document Management > Audit Trail Setup.

Note: The localise script only supports Notifications Link Type.

 

Adding Notifications Link Type

When adding a notification item, you also need to specify the Change Code, Link Item and Tag Values.

 

Syntax

NOTIFICATION:|IMPORT_METHOD=UPGRADE_EXISTING||EventCode=<Code>||Description=<Text>||Condition=<Parameters>||TagValue=<Tags>|

 

where:

  • <Code> refers to the change code.

  • <Text> refers to the description for the link item.

  • <Parameters> refers to the Change Link Parameters for form generation.

  • <Tags> refers to the tag to configure notification such as the header, message related screen and the intended recipient of the notification. For more information, refer to Change Codes - Tags.

 

Example

To set up a notifications link type for a change code as shown above:

NOTIFICATION:|IMPORT_METHOD=UPGRADE_EXISTING||EventCode=CGC||Description=Notification for Jurisdiction Change||Condition={Condition:^^LinkType=SQL.RefMaster^^LinkField=*^^LinkOper==^^}  {Condition:^^LinkType=CC^^LinkField=IncorpPlace^^LinkOper=@HCH^^} ||TagValue=[/NOTIFY_CAT=MF/][/NOTIFY_SYSTEMFLAGS=1/][/NAVIGATION_KEY=FSEGENER/][/PRIORITY=0/][/NOTIFY_HEADER=Jurisdiction Changed/][/NOTIFY_MSG=New Jurisdiction: (+CC.IncorpPlace\#LT-CNTR+)./] /][/NOTIFY_ADMINGP=(+SQL.RefMaster.AdminGroup+)/][/SQL.RefMaster=SELECT AdminGroup FROM RefMaster WHERE RefCode='(-EntCode-)'/]|
