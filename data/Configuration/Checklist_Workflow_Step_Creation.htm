Checklist/Workflow Step Creation

Each checklist or workflow step is made up of:

  • Checklist/workflow step record

    For the checklist/workflow step record, define the following items:

    • Key – This is automatically assigned. You can set a different prefix in Configuration > Settings > Settings > Document Numbers for the Workflow Step Key Prefix ID [WFS] document type.

    • Caption – Enter the caption for the step. This will be seen only in the workflow diagram. When user updates the caption in the Edit Step Properties screen, it will be automatically reflected in the diagram.

    • Description – Enter a description for the step. This is not seen in the workflow diagram.

    • Category – Select the category to be used to classify the step. New categories can be set up in Workflow Categories [WFCA] lookup table. This is only for recording purposes.

    • Type – Select the type classification for the step. New types can be set up in Workflow Step Type [WFPT] lookup table. This is only for recording purposes.

    • ESP Publish Option – Select the option to indicate whether or not the step is to be seen in Engage Servicing Portal (ESP):

      • Set to 'Show [1]' to allow the step to be visible in Engage Servicing Portal for status tracking. ESP users will not be able to make any changes to the step.

      • Set to ‘Action [2]’ to allow step to be actioned in Engage Servicing Portal.

      • Set to 'Don't Show [4]' to hide the step from Engage Servicing Portal.

    • Administrator Assign –  Specify the assignee of this step. Refer to Assign To Codes.

    • Time Days – Enter the number of days to be allocated for the step. This will be used to calculate the Date Due for the step.

    • Tag Values – Enter the applicable tag values. These are additional instructions for the step.

    • File Note Linking - This is to set up the step as a File Note checklist which can be selected by clicking the File Note button from the Ribbon. This requires ticking the option Show in File Note Selection and selecting the category of the File Note. If no category is selected, it will be displayed under the heading ‘No Group’.

    • Menu Linking – This is to set up the step as a Menu Checklist which is seen in the Checklist tab in Tools section when the linked menu is selected. This requires you to specify the Menu Id by specifying the Menu Group and Section and then clicking Find to select the menu.

     

  • Checklist items

    These refer to items for the checklist or workflow step and are to be added in the Workflow Step Editor screen or Edit Checklist screen.

 

Create New Checklist/Workflow Step in Step Library Screen

This method can be used when creating checklist or workflow steps. For workflow steps, this is when you choose to first set up all the workflow steps before you add them to the workflow template.

  1. In Viewpoint - Home, go to Configuration > Business Parameters > Workflow > Step Library.

  2. Click New.

  3. Enter information for the checklist/workflow step record.

  4. Click Update.

  5. Proceed to add items for the checklist/workflow step by clicking Edit with Workflow Step Editor.

  6. Click Update when you are done. This is to save the changes and close the Workflow Step Editor screen. If you choose to close the screen without clicking Update, you will be prompted with a message.

 

Create New Workflow Step from Workflow Designer Screen

This method can be used if you prefer to the create the workflow step as you work on your workflow template. Use this method once you can created your workflow record.

  1. In Configuration > Business Parameters > Workflow, select the workflow record and click Edit with Workflow Designer.

  2. Click Add Step.

  3. Click New Step.

  4. Enter information for the workflow step record.

  5. Click Update.

  6. The new step is seen in the list and ticked by default. Click Select.

  7. Click Edit with Editor to add items for the workflow step.

 

Create Checklist using Right-click

This method will create checklists that are linked to a screen in the Section Menu (usually in File Maintenance). To do this, the User Group authority 'Allow Toolbar/Grid Customisation' is required.

  1. Right-click on the screen the checklist is for.

  2. Click Customise Tasks and Checklists.

  3. Right-click on the screen again. This time, click Add new Checklist.

  4. In the Edit Checklist screen, enter the following:

    • Title - Enter a title for the checklist.

    • Category - Select the category for the checklist.

    • Type - This defaults to Menu Checklists.

  5. Add the checklist items.

  6. Click Update.
