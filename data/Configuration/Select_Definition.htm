Select Definition

The Select Definition screen allows you to change the captions of fields seen in the Information screen for a Master File (accessible by clicking Details in the Select Master File screen or View Details option in Relations/Ownership Enquiry screen). You can also choose to show or hide a field from the Master File Information screen.  

The following keys correspond to each tab in the Information screen. When a tab is selected, the lower grid will display the fields available in the tab.

  • SYS015 (Entity Information) - for ‘Entity File’ tab.

  • SYS065 (Identity Details Corporations) – for ‘Identity Details/CDD’ tab.

  • SYS100 (File Information) – for ‘Master File’ tab.

  • SYS290 (Billing File) – for ‘Billing File’ tab.

  • SYS291 (Supplier File) – for ‘Supplier File’ tab.

  • SYS495 (Account File Setup (GLOBAL)) – for ‘Account File’ tab.

Note: You are encouraged to customise only the above keys. To customise other items with keys not beginning with ‘SYS’, please consult Viewpoint before performing the changes.

Enter additional search criteria to filter the list.

After you have entered the criteria, click Apply.

 

Select Fields Listing

The list shows the available fields listing in Viewpoint Connect.

 

Select Fields

The list shows the available fields based on the selected listing in the Select Fields Listing grid.

 

Filter

Filter the list of fields based on the active/inactive status or to show all.

 

Edit

Click this to edit the settings for a field.

There are three functions available:

  • Caption - This text field displays the caption of the selected field. You can customise this caption.
  • Not Used - Tick this to hide this field from being displayed.
  • Customised - This will be ticked automatically when you make changes to the field's settings.
