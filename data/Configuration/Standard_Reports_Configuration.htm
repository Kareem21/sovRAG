Standard Reports

Standard Reports in Viewpoint are managed in this screen.

 

Filters

The filters apply to the standard report list.

  • Component - You can choose to view only standard reports by its scope or view all existing standard reports.

  • Type - You can choose to view only Viewpoint reports, SSRS reports, Analysis reports or all types.

 

Standard report list

The list shows the existing standard reports.

 

Command buttons

Click on the relevant command button to perform the necessary.

  • New - Click this to add a record of a new SSRS or Analysis report to Viewpoint.

    1. Enter the required details for the new report.

    2. Click Update.

     

    For Analysis Reports, refer to Analysis Reports.

  • Edit - Click this to update the report settings.

  • Delete - Click this to remove selected record for the SSRS or Analysis report.

  • Import - Click this to import an SSRS or Analysis report.

    1. In the import screen, specify the folder containing the SSRS or Analysis report.

    2. Click Import - Load List.

    3. Tick to select the report you want to import.

    4. Click Import.

     

  • Export - Click this to export the selected SSRS or Analysis report.

    1. In the export report creen, select the folder where the exported report is to be stored.

    2. Click Export.

  • Attributes - Click this to set up attributes for the selected report. When exporting this SSRS or Analysis report, you will also get the .dtr file which contains the attributes information. Refer to Attributes for more information.

  • Check In - The Check In button is only visible when users are connected via the Cloud Solution. The button is disabled when there are no templates being checked out locally in the client-side. When you open a template file, the Check In button will be enabled. Once completed, close the document and click Check In. The template will be uploaded to the server-side and the local copy in client-side will be deleted.
