Link Type: Notes

This link type is to allow for user-customisable notification that will appear in the Change Processing screen. This can be information, warning or error message.

To set up this linked item, do the following:

  1. In Configuration > Business Parameters > General > Change Codes screen, select the relevant Change Code.

  2. Click New.

  3. Specify the following details:

    • Link Type - Select Notes.

    • Description - Enter the notification message.

    • Tag Values - Enter the tag value to indicate if the notification is information, warning or error message. Refer to Change Codes: Link Item - Tags.

    • Help Text - Enter additional text that will also be displayed, if relevant.

  4. Click Update.
