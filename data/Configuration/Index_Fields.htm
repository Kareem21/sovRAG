Index Fields

Index fields increase the accuracy of document storage and also allow you to narrow down a search to more specific details. There are two default indices that come with the Document Management component: Master File and Document Date. These cannot be changed and are mandatory for completion when storing a document holder. Additional index fields can be created as required and assigned to the appropriate filing cabinets so that you can enter the information during document storage.

 

Index Field list

The list shows the existing index fields on the left of the screen. Select an index to view or edit its details.

 

Command buttons

Click on the relevant command button to perform the necessary.

  • New - Click this to add new index field.

    Name

    Enter the name of the field. This field, when assigned to a Filing Cabinet, can be renamed specifically for that cabinet. For instance, if you create an index field called 'Correspondent', you may re-caption this in the Billing cabinet to 'Sent To'.

     

    Key Tag

    Provide a unique code for the index field. For e-mail storing purposes, you can use the following codes to populate the fields with the corresponding e-mail addresses:

    • VPEMTS - to populate field from Sender/Recipient list.
    • VPEMCC - to populate field with addresses from the Cc list.
    Mandatory

    Tick if the field is mandatory for completion.

     

    Help/Note

    Enter any information on the usage of the field. This information will appear when the user is creating a new document holder in Document Manager, and he selects the index field.

     

    Field Type

    Select the field type from the available options.

     

    Format/Mask

    Select the format of the information you want to appear in the index field. The formats or masks that appear for selection depend on the selection made in the Field type (see above).

     

    Allow duplicate index relations for this field

    Tick this if you want to allow multiples of this index field in any document holder.

     

    Update

    Click this to save the entry.

     

    Cancel

    Click this to cancel the entry.

  • Delete - Click this to delete the selected index field. If the field is already in use you would not be able to do this. In this case, you will need to use the Inactive Global button (see above) to deactivate the field.

  • Move Up and Move Down - Click to move a selected field up or down one place in the list. This determines the arrangement of the fields in the document holder.

  • Apply Global - Click this to activate the index field in all Filing Cabinets.

  • Deactivate Global - Click this to deactivate the index field in all Filing Cabinets.

  • Update - Click this to save your changes.

  • Cancel - Click this to cancel any changes.

  • Close - Click this to exit the screen.
