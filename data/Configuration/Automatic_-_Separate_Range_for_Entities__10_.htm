Automatic - Separate Range for Entities[10]

The Automatic - Separate Range for Entities [10] coding method will assign a code that is made up of prefix and a running number. The prefix will differ for Master Files that are managed entities and those that are not.

Setting the prefix is done in Configuration > Settings > Customise > System Running Numbers where:

  • Master File Codes [ADR] is for non-managed entities (the Entity Administration option is not ticked)

  • Entity File Codes [ADE] is for managed entities (the Entity Administration option is ticked)

Format of the Master File code

The first few characters will be the prefix that you have specified and the remaining will be the number for the Master File Codes [ADR] or the Entity File Codes [ADE]

Both the numbers are running numbers that will start at 1 and increases so the length of the Master File code will depend on the prefix that you defined as well as the number.  

 

Using 'Automatic - Separate Range for Entities[10]'  coding method

To start using this coding method, do the following:

  • Select the coding method in System Defaults > General/Master File

  • Set up the required System Running Number types

Select Coding Method

  1. In Viewpoint – Home, go to Configuration > Settings > System Defaults > General/Master File.

  2. Under Coding Method for Master File, select Automatic - Separate Rage for Entities [10].

  3. Click Update.

 

Set up System Running Number Types

The three System Running Number types may not yet be available in your system. If you are not able to locate it, then you need to add the type by doing the following:

  1. In Viewpoint – Home, go to Configuration > Settings > Customise > System Running Numbers.

  2. Click New.

  3. In Document Type, select the required type e.g. Master File Codes [ADR].

  4. Specify the Current Doc. Number as 0.

  5. Specify the required prefix in the Prefix & Format field.

  6. Click Update.

  7. Repeat for the other type - Entity File Codes [ADE].

If the type is already available, you can select it and click Edit to specify the required prefix.
