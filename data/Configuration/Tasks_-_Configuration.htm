Tasks

This will open the Task Setup screen so that you can create tasks that can be:

  • Assigned to stored documents through Add Task button. If necessary, tags can be added to create follow-up task.

  • Assigned to Master File through Add Task button or via checklist/workflow (where checklist item's field type is 'Task'). If necessary, tags can be added to create follow-up task.

  • Assigned to documents in Document Queue to use the Delivery Manager function through Add Task button or Document Management index tags.

Tasks that have been assigned will be seen in the Assigned Task screen.

 

 

Task list

The list of tasks types already created are seen on the left of the screen. Select the task to edit the details of the task type except the Key Tag. Tasks can also be created using Localise Script - Document Management.

 

Command buttons

Click on the relevant command button to perform the necessary.

  • New - Click this to create a new  task type.

    1. Click New.

    2. Enter the required information. The Tag Values field is to be left blank unless:

      • The task requires a follow-up task. This requires tags to specify details for the follow-up task.

      • The task is a Delivery Manager task. This requires tags to specify the task as a Delivery Manager task.

    3. Click Update to save the new task.

     

  • Delete - Click this to delete a task type. This can only be done if the task type is not in use.

  • Update - Click this to save your changes.

  • Cancel - Click this to cancel your changes.

  • Close - Click this to exit the screen.
