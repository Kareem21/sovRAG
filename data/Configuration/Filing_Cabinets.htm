Filing Cabinets

Filing Cabinets are created to contain document holders and their associated documents. Each of these holders will be indexed according to the relevant information related to the documents. Filing Cabinets have to be created before documents can be saved. Once your Filing Cabinet structure has been set up, link the relevant index fields to each cabinet. These index fields should be relevant to the documents that are meant to be stored there.

Filing Cabinets are usually set up only once, during installation. You can create groups, cabinets and sub-cabinets depending on how you want to categorise your document holders.

Command buttons

Click on the relevant command button to perform the necessary.

  • Edit Cabinet - Click this to edit the details of the selected cabinet and add index fields to the cabinet.

    Index fields for a cabinet can be specified during the cabinet creation or after the cabinet has been created. If the latter, in Filing Cabinet Setup screen, select the cabinet and click Edit Cabinet. From the list of existing index fields, you can define the following options:

    • Use - Tick this if you want to apply the index field to the selected cabinet.

    • Active - This will be ticked automatically when you tick Use. If the index field is no longer applicable, you can untick this.

    • Mand. - Tick this if you want the index field to be mandatory for the holder. This means that users must complete the index field before they are able to store the holder.

    • Dup. - This depends on the setup of the index field. It will be ticked if the index field was set up to allow duplicates.

    • Help Text - This is the Help Text entered for the index field. To add or edit the Help Text, first click on the Help Text field to bring up the ellipsis button (...) in the field's right corner. This button is only available for index fields that are currently being used.

  • New Cabinet - Click this to add new cabinet.

    You should have the structure of your Filing Cabinets ready at hand before you begin this process.

    1. If the cabinet you are about to create belongs to a group, you will need to add the group first. See Add Group, below. This will create a group under the main Filing Cabinets. If you are not using groups, proceed to Step 4.

    2. Enter the name of the group and click OK. To create a sub of this group, select the Group name from the list and click Add Group again.

    3. Select the group you want to create a cabinet in.

    4. Click New Cabinet.

    5. Enter the details of the cabinet.

      • Cabinet Name  - This is the name of the cabinet. When creating a cabinet, give it a meaningful name such as Correspondence.

      • Key Tag - Enter a unique ID for the cabinet.

      • Help Text - Enter a description of the usage of the cabinet. This text is displayed at the bottom of the document holder when the cabinet is selected.

      • Retention Policy - Select from the retention policy for the cabinet. This list can be customised in the Document Retention Policy [DRTP] lookup value table. This is used for archiving purposes.

      • Closed for new documents - Tick this to prevent new documents from being added into the cabinet. This is normally done after you decide to retire the cabinet.

      • Closed for searching document - Tick to disallow searching of documents in the cabinet.

      • Note - Add any further explanation about the Cabinet, if necessary. This field is used to define cabinets for ESP.

    6. Now, select the index fields that are relevant to the cabinet. All mandatory index fields are automatically selected from the list.

    7. Click Update to save the new cabinet.

    Note: You can tick the option 'Hide unused index fields' to remove index fields that do not apply to the filing cabinet.

  • Delete Cabinet - Click this to delete a selected cabinet. You cannot delete a cabinet that contains documents. In this case, it would be better to edit the cabinet and tick the Closed for new documents and Closed for searching documents check boxes.

  • Add Group - Click this to add a group or sub-group. Any group without a cabinet will be automatically deleted when you exit the Filing Cabinet Setup screen.

  • Rename - Click this to rename the selected cabinet.

    1. Type the new name for the filing cabinet.

    2. Click OK.

  • Close - Click this to exit the screen.
