Reminder Rules

The Reminder rules are used by Viewpoint to check and create system reminders for a Master File.

 

Filter

The filter applies to the Reminder Rule list allowing you to view only active rules, inactive ones or all.

 

Reminder Rule list

This shows all the available Reminder Rules in Viewpoint.

 

Command buttons

Click on the relevant command button to perform the necessary.

  • Edit - Click this to edit the selected Reminder Rule.

  • Activate - Click this to make an inactive Reminder Rule active.

  • Deactivate - Click this to deactivate the selected rule if you do not require it. Inactive rules are ignored when Viewpoint checks for reminders.

  • Options - Click this to set options for the review of the reminders.

    1. In the Reminders Options screen, specify the following:

      • Review Horizon - This is in weeks and is used to indicate how far ahead you want Viewpoint to check for reminders.

      • Statuses to be included - Tick to include the statuses for the Reminders review. You can select from Master File status, Entity File status, Billing File status, Supplier File status and Account File status. Any status that is not selected can be seen in the System Settings screen under the VPCOSEC application for the following Keys:

        • DRBFST - Billing File Status

        • DRCAST - Account File Status

        • DRCOST - Entity Status

        • DRMFST - Master File Status

        • DRSUST - Supplier Status

    2. Click Update.

  • Review Reminders - Click this to perform a Reminders review for all users. If you are using Task Scheduler and Notification Agent, you can automate the review by setting up the Reminders Review [TDRE] job.
