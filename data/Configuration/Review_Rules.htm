Review Rules

These are rules that are used when performing a review on Master Files. The rules are utilised in the File Review feature as well as the cloning function of Billing Files. The rules are classified under the following categories:

  • Account File

  • AEOI

  • Billing File

  • Compliance

  • Data Validation

  • Entity

  • Error

  • Filings

  • Government Fees

  • Incorporation invoicing

  • Master File

  • Matter Account

  • Meetings

  • Nominees

  • Others

  • Recurring Fee

  • Registered Office

  • Relations

  • Reviews

  • Share Register

  • Trust

Filter

The filter applies to the Review rules list, allowing you to filter the list by the Review rule status.

 

Review rules list

This lists all review rules in Viewpoint.

 

Edit

Click this to change the rule settings. You can do the following:

  • Change the description of the rule - Enter the new description in the Description field. This will be seen in the results after you run File Review.

  • Add tags for the selected Review Rules - Enter the tags to define what to check in the Tag Values field. This applies only for Review Rules in the AEOI, Compliance and Data Validation categories. Refer to the below for the tags:

    • AEOI Review - Tags

    • Compliance Review - Tags

    • Data Validation Review - Tags

  • Activate/Deactivate a rule - Tick or untick the Active option. If a review rule has been deactivated, it is excluded from File Review and related review tags will be ignored.
