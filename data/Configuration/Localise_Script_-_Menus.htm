Localise Script Tags - Menus

The menu tags allow you to activate/deactivate and set caption for a menu (screen).

 

Activating/Deactivating a Screen

Screens in Viewpoint can be set to active or inactive using the tag below.

 

Syntax

MENUAUTH:<MenuID>=<Value>

 

where:

  • <MenuID> can be retrieved from the Navigation ID field in Configurations > Settings > Customise > Windows Menu.

  • <Value> is 1 to activate and 0 to deactivate.

 

Example

To activate the CDD screen: MENUAUTH:FMMKYCVV=1

 

Setting Captions

The tag below allows you to set the caption for the screen and will over-write previous customised caption.

 

Syntax

MENUCAP:<MenuID>=<Caption>

 

where:

  • <MenuID> can be retrieved from the Navigation ID field in Configurations > Settings > Customise > Windows Menu.

  • <Caption> is the screen caption.

 

Example

To set the caption for the CDD screen to ‘Due Diligence’: MENUCAP:FMMKYCVV=Due Diligence
