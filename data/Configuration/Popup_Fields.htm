Popup Fields

Popup fields prompt the user to select information when generating a form, and are stored with the change information. These fields are especially useful for information that (a) is not applicable to a certain change, and (b) is only known when the form is prepared.

Note: Popup fields can only be used in change-linked forms.

 

Document Fields

These are individual fields where the user can complete or edit the text or value. Document fields appear in the first folder of the popup screen.

 

Syntax

(+POP.Caption|Type&Default literal\format+)

or

(+POP.Caption|Type#Operator\format+)

 

Field

Explanation

Caption

This is a mandatory field and is used to identify the field in the popup screen.

Type

This is an optional field and can be omitted for the default TEXT field type. See Field Type and Description, below, for the available options.

&Default literal

This is an optional field. the text after the ampersand (&) is used for the default value.

Format

This is an optional field. Any valid change field formatting code can be used.

#Operator

This is an optional field. The default value will be completed per the operator code. See Operator and Result, below.

 

Field Type and Description

Field Type

Description

TEXT

Single line text entry

For example:

To prompt to enter some text: (+POP.The caption to show|TEXT+)

or to add a default: (+POP.Field with default|TEXT&Text default+)

NOTE

Multi-line text entry.

For example, see #UT-CODE below.

DATE

TIME

Date entry

Time entry

Values are validated and displayed using the field format setting configured in the Defaults screen.

For example:

The meeting was held on (+POP.Date of meeting|DATE+) at (+POP.Time of meeting|TIME+).

 

AMNT

AM4D

NR0D

NR4D

NR6D

PERC

Values are validated and displayed using the field format settings configured in the Defaults screen. The fields are as follow:

Amounts (2 d.p.)

Amounts (more than 2 d.p.)

Number without d.p.

Number - shares with d.p.

Number - others with d.p.

Percentages

CBE

Combo list where the user is allowed to edit the text. This code must be following with a hash (#) and items separated by a pipel ( | ).

For example:

(+POP.Select an item to edit|CBE#First item|Another one|and one more+)

Use the ampersand (&) as the first letter of the item to indicate this is the default item on loading of the screen.

For example: #Item 1|&Item 2|Item 3

CBF

Combo list where the user is NOT allowed to edit a selected item. (See CBE for more details.)

For example:

(+POP.Select an item|CBF#First item|&Another one|and one more+)

YESNO

Yes or No value

See also Defaults.

 

Operator and Result

Operator

Result

#UV-CODE

System will default with the value of the user defined field specified by CODE.

For example: #UV-SCAT2

#UT-CODE

As above, except that system defaults with the note of the user defined field specified by CODE.

For example:

(+POP.Key incorporation note|NOTE#UT-INCORPBY+)

#NOW

The system will default to today's date or time.

 

Selection Fields

This is a listing of specific Master Files, such as officers, shareholders or interest holders, and the user is required to select one or more of these files for the generated document. The displayed text (name) may be edited so that they appear as edited in the generated document. Each selection field appears in its own folder in the pop up screen.

 

Syntax

(+POP.Caption|Type|Preformat\PostFormat+)

 

Field

Explanation

Caption

This is a mandatory field and is used to identify the field in the popup screen.

Type

This is a mandatory field, see Field Type and Description, below, for the available options.

Preformat

This is an optional format for the Master File selection on the screen. Any valid Master File formatting option (the #AD-9-1 codes) can be used. If omitted, the list is formatted using names as displayed on the Master File selection screens (Last Name followed by the Initials).

When #AD formats are used, the program will save the selected Master File codes, separated by TAB, so that the formatted values will list through table cell (use the #LS formatting option to make it a list). In the following example, the names are shown on the selection screen, but the address is printed, not the names.

(+POP.Mail document to|L_OFN_D_0\#AD-5-0+)

PostFormat

This is an optional format applied on the selected Master File (codes) after the completion of the selection during the generation of the form. Any valid Master File code formatting option can be used.

If a post-format is used, the selection is stored for post-format processing as Master File codes, separated by TAB. If no post-format code is used, the selection is stored for processing as a list of the displayed (edited) values separated by line feeds.

 

Field Type and Description

Field Type

Description

L_OFx_yyy_9

Lists Officers of the entity per the following parameters.

x - specifies current, resigned or all officers. Valid options are:

A = Current BEFORE Date Document

B = Current ON Date Document

C = All Current from database

N = All (both current and resigned)

R = Resigned ON or before Date Document

Q = Resigned ON or after Date Document

yyy - Officer type code (or part of it) to be selected. Use D for Directors, S for Secretary or D:S for both Directors and Secretaries. You may also use M for all officer types starting with M.

9 - the number of Master Files required to be selected. Valid options are:

0 = one or more

1 = exactly one

2 = two or more

3 etc = three or more

Q = directors quorum as entered in the M&A fields

 

Example 1: To list the directors and secretary present at a meeting, you may place the popup field in a table:

Present at meeting:

(+POP.Directors present|L_OFA_D_Q+)

In attendance

(+POP.Secretary|L_OFA_S_1+), Secretary

 

Example 2: To list the directors in a paragraph:

(+POP.List Directors in paragraph|L_OFC_D_0\#LP-#AD-40-0+)

L_SHx_9

Lists shareholders of the entity per the following parameters.

x - specifies current shareholders. Valid options are:

A = Current BEFORE Date Document

B = Current ON Date Document

C = All Current from database

N = All from database

9 - the number of Master Files required to be selected. See L_OFx_yyy_9, above, for more information. The option Q is not valid for this field type.

For example:

(+POP.Shareholders List|L_SHA_0+)

L_INx_yyy_9

Lists interest holders of the entity. See L_OFx_yyy_9, above, for more information, replacing the interest type for the officer type.

For example:

(+POP.Beneficial Owners List|L_INA_B_0+)

(+POP.Ultimate Owners List|L_INA_U_0+)

L

Lists selection made for the popup field in another area of the form. The caption has to be the same as the popup field used for the selection.

For example:

(+POP.Chairman|L_OFA_D_1\#AD-40-0+) - displays the list of directors for user to select one of them as the Chairman.

(+POP.Chairman|L\#AD-40-0+) - user's selection for Chairman will be placed here.

L*

Display the selected values from the previous popup field in the same format:

For example:

(+POP.SelectDirector|L*+)

Note: F can be placed in front of the L_. If F is present, the values in the popup screen can be formatted and edited. If there is no F, the values in the popup screen will be displayed as default and cannot be changed if having the format option. For example, (+POP.Select any Director|FL_OFC_D_1\#AD-40-0+).

See also Format Options.

 

Optional Paragraphs

Optional paragraphs require the user to keep or discard paragraphs as required, and are useful for documents with multiple different clauses. If there are document fields within the paragraph, they will appear together with other document fields in the first folder of the popup screen.

 

Syntax

At the beginning of the optional paragraph text:

(+BM.BMNAME=Category name|Paragraph name\FORMAT+)

 

and at the end:

(+BM.BMNAME=END+)

 

 

Field

Explanation

BMNAME

Unique name given to the paragraph, used as the bookmark name.

Category name

Enables grouping of multiple paragraph under this heading.

Paragraph name

Description to be shown on the popup screen.

Format

When a optional paragraphs is loaded on document generation, all options are selected. Add NOCHECK to de-select non-default paragraphs.

For example:

(+BM.NNOTE=Paragraph|Item description\NOCHECK+)

 

Additional option in the template allowing you to define the number of options are allowed to be selected in the optional paragraph.

(+BM.IndName=Individual|Select this signatory is indvidual\NOCHECK\GRP1_1+)

(+BM.EntName=Entity|Select this if signatory is Entity\GRP1_1+)

(+BM.CorpName=Company|Select this if the paragraph is described on Company\GRP2_1+)

(+BM.TrustName=Trust| Select this if the paragraph is described on Trust\NOCHECK\GRP2_1+)

(+BM.PartnershipName=Partnership| Select this if the paragraph is described on Partnership\NOCHECK\GRP2_1+)

 

GRP1 is the example of grouping on the optional paragraph.

XXX_y where XXX is the grouping name and  y is indicating number of selections as the below:

0 = one or more

1 = exactly one

2 = two or more

3 etc = three or more

 

When refreshing, bookmarks are set around the start code and the END marker.

 

Examples

(+BM.NNOTE=Notes|Entitled to proxy form+)

NOTE: Any Member entitled to attend and vote at the Meeting is entitled to appoint a proxy to attend and (on a poll), vote on  his behalf; a proxy need not be a Member. The instrument appointing a proxy should be deposited at the Registered Office of the Company not less than 24/48 hours before the time fixed for holding the Meeting or adjourned Meeting. (+BM.NNOTE=END+)

(+BM.NTEXT=Notes|Notice text+) NOTICE IS HEREBY GIVEN that an Extraordinary General Meeting of the Company will be held on (+EVT.TransDate\DLB+) at (+DF.PRESEN\#AD-2-1+) for the purpose of considering and, if thought fit, passing the following resolution as an Special Resolution: (+BM.NTEXT=END+)

(+BM.FACCT=Accounts|Do file accounts+) It is decided to file accounts and to present them to the shareholders when the time is appropriate. (+BM.FACCT=END+)

(+BM.NOACCT=Accounts|Accounts presented at a later date+) It is decided not to file any accounts as they are not ready. (+BM.NOACCT=END+)

 

Tip: To review how Word has set the bookmarks during the form 'Refresh', go to Tools - Options - View and tick 'Show Bookmark'. You will see square brackets around the book-marked text. This function is implemented for Word 97/2000 and above.
