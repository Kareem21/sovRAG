Prefix

There are two available prefixes that can be added to field codes and this depends on the information being retrieved and how it will be displayed:

  • ftyp=evw - this is used for retrieval of single and multiple details. When the information is multiple records that are to be placed within rows and columns, HTML codes are required to define the table properties in the template's .htm file.

  • fgrd - this is used for retrieval of multiple details to be displayed in grids with grid elements like the ability to sort, set the number of records per page, etc. When using this prefix, you need to have Grid Statements in the Key Information to define the grid properties.
