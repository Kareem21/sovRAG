Settings

The Settings section is divided into: System Defaults, Customise and Tools & Utilities

 

System Defaults

This is used to set up configurations for Viewpoint environment. Click here for more information.

 

Customise

The Settings menu options allow you to customize the system to reflect your needs and to suit your requirements. We do, however, recommend that you consult a Viewpoint representative before you make any changes to this because certain changes can affect the functionality of the system. For example, when a form with the names of Directors is generated in Administrator, the system will look for all officers whose element code begins with 'D'. If you change the element code to another letter, the system will not recognize the code and will not be able to find the officers' names.

  • Windows Menu - This is where you set up the screens for Viewpoint.WIN.

  • Web Menu - This is where you set up the screens for Viewpoint.WEB.

  • Screen Settings - This is where you set up summary screens.

  • Field Settings - This is where you set up fields for Viewpoint screens.

  • System Running Numbers - This shows the running numbers used in your environment.

  • System Settings - This records values for system settings that should not be changed without prior instruction from Viewpoint.

  • Filing Settings - This screen shows the standard settings for different filing types.

  • Custom Report - This records the minimum authority required to generate Custom Reports.

  • Grid Fields - This is where you customise grids in Viewpoint.

  • Select Definition - This is where you change the captions of fields seen in the Information screen for a Master File.

  • Relations Enquiry - This is where you customise information displayed in Relations Enquiry.

  • Review Rules - This section lists the rules used for the File Review function in Billing.

  • Ribbon Settings - This screen allows you to customise the Ribbon caption.

  • To Do Summary Settings - This screen allows user to customise the captions of items in To Do Summary screens.

 

Tools & Utilities

The tools in this section are made up of the following and are generally used to check the system for errors.

  • Localise - This is used to set up Viewpoint based on local requirements.

  • Verify Settings - This contains functions that will check for errors in Viewpoint.
