Notice Board

The Notice Board is used to post news and information on the CP-WP. This screen is used to create Notice Board items.

 

Notice board item list

This shows all the notice board items that have been set up.

 

Command buttons

Click on the relevant command button to perform the necessary.

  • New - Click this to create new Notice Board item.

    1. Enter the required details.

    2. Click Update.

  • Edit - Click this to edit the selected Notice Board item.

  • Delete - Click this to remove the selected Notice Board item.
