Access Group Security

Access Group Security must be set up if you have selected Access + User Group Model [3]  as your security model. Each Access Group Security combines an Access Group with a User Group and an Administrator Group which results in the following:

  • A user assigned to that Access Group gains access to the Master Files of the Administrator Group. The user's own Administrator Group(s) is ignored for all Master File-related screens.

  • If there is more than one Access Group Security with the same Access Group, then the user will have access to the Master Files of ALL the linked Administrator Groups.

  • The user's authority on a Master File is determined by the User Group of the Access Group Security. This means he may have different access on Master Files that belong to different Administrator Groups.

The setup of an Access Group Security is done in this screen.

 

Access Group Security list

The list shows all existing Access Group Securities.

 

Command buttons

Click on the relevant command button to perform the necessary.

  • New - Click this to create a new Access Group Security.

    1. Select the Access Group, Administrator Group and User Group that makes up the Access Group Security.

    2. Click Update.

     

  • Delete - Click this to delete an existing Access Group Security record.
