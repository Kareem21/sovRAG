Checklist Item - Field Type: Yes/No

This Field Type will accept only a Yes or No value but there are various ways to allow user to specify a Yes or a No value:

  • Checkbox - There is just one checkbox available with a ticked checkbox indicating a Yes and an unticked checkbox indicating a No. This is the default format.

  • List - This will show a list with a Yes value and No value for user to select.

For workflows that will be published to Viewpoint Engage, the format can be as:

  • Radio buttons - This will show a Yes radio button and a No radio button for user to select.

  • Checkbox -  There is just one checkbox available with a ticked checkbox indicating a Yes and an unticked checkbox indicating a No. This is the default format.

  • Slider - This will show a slider where if a Yes value is indicated by sliding the button to the right.

The configuration for the different format is done in the Show List field in Values/Default where the following can be entered:

  • Yes|No to show as List.

  • #R to show as radio buttons (only for workflows published to Viewpoint Engage, otherwise, it will show as a drop-down list).

  • #S as slider (only for workflows published to Viewpoint Engage, otherwise, it will show as checkbox).
