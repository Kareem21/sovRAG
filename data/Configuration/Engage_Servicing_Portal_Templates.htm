Engage Servicing Portal Templates Library

This screen lists templates used for screens in Engage Servicing Portal (ESP). The templates are HTML templates and need to be linked to the ESP screen through Configuration > Engage Servicing Portal > Global > Menus.

 

Document Type

Select the category from the list to filter the available templates.

 

Templates table

The available templates are listed here.

 

Command buttons

Click on the relevant command button to perform the necessary.

  • New - Click this to add new template to the template library.

  • Edit - Click this to view the properties of the selected template.

    Document Type

    This field shows which category a template belongs to.

     

    Code

    This is the unique reference code for the template.

     

    Description

    This is the title of the document.

     

    Type

    The form type selections are maintained in the Form Type [FRMT] lookup value table.

     

    Customised (Don't overwrite in upgrade)

    Tick to prevent customised templates from being overwritten when you upgrade Viewpoint or import a template with the same code.

     

    Language

    Select the default language of the template. This does not currently have any impact on the usage of the report. Languages are maintained in the Language Codes [LANG] lookup value table.

     

    Select Definition

    This determines the header statement of the template.

     

    Print No. Of Copies

    Specify the number of copies to be printed, i.e. the number to be printed if the document is sent directly to the printer.

     

    Path & File Name

    Enter or browse to the location where the template is saved. All templates should be stored in the Forms directory specified in Settings > System Defaults > Folders - Template Forms. Remember that if you link the wrong template, you will end up editing the wrong template. When copying an existing template, the path will default to the category sub-folder in the Forms folder.

     

    Notes

    The notes field is for reference only. For example, you can enter a date on which the template was last reviewed.

     

    Help Text

    Enter any relevant information related to the usage of the report. Viewpoint can be customised to show this information to the user so that they know the purpose of the report before they select it for generation.

     

    Key Information

    This section contains the SQL (Standard Query Language) statements which are used to retrieve the information that is displayed in the template. Double-click on the field with the left mouse button to display the full contents.

     

    Field Details

    This displays the field codes contained in the template. Double click on the field with the left mouse button to display the full contents. Do not update this field as the contents are automatically updated every time you click on Refresh and should not be edited manually.

     

  • Delete - Click this to delete the selected template.

  • Copy - Click this to make a copy of a selected template. This will copy all the information of the selected template, including the layout. You will need to give the copy a new code, and this code will be used as the default name of the template.

  • Edit Document - Click this to open the template in your HTML editor.

  • Refresh - This button is disabled.

  • Field Wizard - Click this to launch the Field Wizard, which provides help on the field codes for the template.

  • Check In - The Check In button is only visible when users are connected via the Cloud Solution. The button is disabled when there are no templates being checked out locally in the client-side. When you open a template file, the Check In button will be enabled. Once completed, close the document and click Check In. The template will be uploaded to the server-side and the local copy in client-side will be deleted.

  • Import - Click this to import template.

    1. In the Import screen, browse to the folder containing the template files.

    2. Click Import - Load List.

    3. Tick the templates you want to import.

    4. Click Import.

     

  • Export - Click this to export template.

    To export selected template:

    1. Once you click Export, browse to the folder where the exported files are to be stored.

    2. Click Export.

     

    To export more than one template:

    1. Once you click Export, browse to the folder where the exported files are to be stored.

    2. Click Select Category/Jurisdiction.

    3. Tick the category or jurisdiction.

    4. Click Select Document to see all templates in the selected category or jurisdiction.

    5. Tick the templates you want to export.

    6. Click Export.

  • Refresh/Option - This button is disabled as the functions provided here are not applicable to Web templates.
