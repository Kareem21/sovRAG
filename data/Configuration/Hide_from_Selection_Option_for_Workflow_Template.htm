'Hide from Selection' Option for Workflow Template

This option is seen in the properties of a workflow template in the Workflow Library and is used to restrict the workflow from being started as an independent workflow. Workflow templates where the 'Hide from Selection' option is ticked can only be created from within a workflow.

Note: The parent workflow will be updated with the information specified in the workflow flagged as ‘Hide from Selection’ even though the status of the workflow is still in-progress.

 

Define Workflow Template as 'Hide from Selection'

  1. Go to Configuration  > Business Parameters > Workflow > Workflow Library.

  2. Select the workflow template and click Edit.

  3. Tick the ‘Hide from Selection’ checkbox.

  4. Click Update.

Note: For new workflow template, ensure the 'Hide from Selection' option is ticked when defining the properties.
