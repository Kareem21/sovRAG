Field Wizard

The Field Wizard can be used to determine the field codes you need to use to retrieve and display information in generated documents. It also provides the format options that you can use to translate the field codes into recognisable information. The Field Wizard can be accessed via the Field Wizard command button in the Document Libraries.

 

Related to

For form templates, this lists all the change codes that are linked to the template. Select the change you want to code for. This will refresh the Change fields in the Select from: section.

For other templates, this lists the Select Definition option for the template which will determine the available Header Fields in the Select from: section.

 

Select from:

Select one of the sections to display the available fields in that category.

 

Format field as:

This lists the format options available for the field. For example, you can specify whether you want a date to appear as a long date (Monday, 12 December, 2011), a medium date (12-Dec-2011) or a short date (12/12/2011). The format options change depending on the type of field. You will notice the result displayed in the Code: section changing as you change your format field selection.

 

Special Features:

The additional formatting options available are listed here. For example, for an address field, the option "comma's on end-of-line” would display an address with a comma at the end of each address line.

 

List multiple changes - use as the format:

This is available for form templates with combined changes where you can choose how you want the multiple details retrieved to be displayed.

 

Result of Your Selection

Code:

The resulting code, based on your category, format and special feature selections, is displayed here. This is the code that is to be copied and added to the document template.

 

Example of information retrieved:

This is available only for Field Wizard in Forms Library where an example of how the data will be displayed using the selected format is shown here.

 

Command buttons

Click on the relevant command button to perform the necessary.

  • Copy Code to Clipboard - Click this to copy the code to the clipboard from where it can be pasted into your word template. If you are using Microsoft Word, you may also use Ctrl+C to copy and Ctrl+V to paste.

  • Close - Click this to close the wizard.
