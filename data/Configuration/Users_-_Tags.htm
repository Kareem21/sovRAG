Users - Tags

These tags are to be entered in the Tag Values field of the user record.

 

Set Default Values for Time and Disbursements

Tag Explanation

[TIMECODE=<ServiceCode>]

This will set a default service code for the user's time entries.

[DISBCODE=<ServiceCode>]

This will set a default service code for the user's disbursement entries.

[TIMEBF=<BillingFileCode>]

This will set the default Billing File for the user's time entries when creating time entries using the service command buttons in Timesheet screen.

[TIMEMATTER=<MatterCode>]

This will set the default Matter for the user's time entries  when creating time entries using the service command buttons in Timesheet screen.

 

 

Create a Custom Menu List

This allows for a customised Section Menu for the user. Refer to Custom Menu List.

 

Filter Field Values

For fields where users can select from a list of values, a tag can be added to filter the list so that users see only values that are applicable to the entity type. Refer to Filtering List Values.
