Revenue Entities

A revenue entity must be created for the invoice generation process if you are utilising the Practice Management component. You can set up as many revenue entities as you require. The most commonly used one can be set as the default revenue entity in Configuration > Settings > System Defaults > Practice Management.

 

Filter

The filter applies to the revenue entity list.

 

Revenue entity  list

The list shows all existing revenue entities.

 

Command buttons

Click on the relevant command button to perform the necessary.

  • New - Click this to create a new revenue entity.

    1. Enter the details for the revenue entity. Ensure there is no overlapping for the document numbering.

    2. Click Update.

    Note: When there is more than one revenue entity, ensure that the number range for the billing documents do not overlap. Each revenue entity must have a unique starting number for the different types of document e.g.

    • All Billing documents for revenue entity A starts from 100000

    • All Billing documents for revenue entity B starts from 200000

     

  • Edit - Click this to edit the selected revenue entity.

  • Delete - Click this to remove an existing revenue entity. Revenue entities which have invoices linked to them cannot be deleted.
