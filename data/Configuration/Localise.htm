Localise

The Localise feature is used to set up new Viewpoint systems with certain predefined settings. There are ten preset scripts for General, Viewpoint Document Management and Engage Servicing Portal, as well as jurisdictional scripts for:

  • England & Wales

  • Guernsey

  • Gibraltar

  • Hong Kong

  • Isle of Man

  • Jersey

  • Cayman Islands

  • Luxembourg

  • Malta

  • Malaysia

  • Seychelles

  • Singapore

  • Turks and Caicos Islands

  • British Virgin Islands

However, you can also write your own custom script to update the configuration.

 

Localise script list

This shows:

  • the preset localise scripts with the option to add your own custom script in the first row in the Task column

  • checkbox to select the script to run in the Select column

  • icon to browse to the location containing the source information to be uploaded during the localise process (required for certain script such as Load SSIC Principal Activity Codes in User Table for Singapore)

 

Test (No update for script items) option

Tick this to do a test run. Changes will not be made, and you can remedy any errors before you begin the actual process.

 

Command buttons

Click on the relevant command button to perform the necessary.

  • Start  - Click this to start the test/actual run.

  • Print - Click this to print the results of the localise script run.

  • Cancel - Click this to exit the screen or cancel the localisation process. Once localisation is complete, the button caption will change to Close.

 

Running Viewpoint Localise Script

  1. In Localise Viewpoint Setup, choose the settings you require by ticking the checkbox under the Select column. You should select only one jurisdiction setting. However, you can select more than one setting under a particular jurisdiction.

  2. Perform a test run by ticking the Test (no update for script items) option.

  3. Click Start.

  4. If the results are satisfactory, perform the run for real.

  5. When the run is complete, click OK to exit the screen.

 

Running Custom Script

  1. In Localise Viewpoint Setup, double-click [Double Click to add a Custom Script] in the first row.

  2. In the Edit script: <Script> screen, enter the script.

  3. Click OK.

  4. In the Localise Viewpoint Setup screen, the first row will now display the caption of your script and the corresponding Select checkbox is ticked.

  5. Perform a test run by ticking the Test (no update for script items) option.

  6. Click Start.

  7. If the results are satisfactory, perform the run for real. If there is any error, these will be highlighted in red and you need to correct the script and test again.

  8. When the run is complete, click OK to exit the screen.
