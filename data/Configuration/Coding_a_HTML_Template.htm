Coding a HTML Template

HTML templates are used for Summary screens in Viewpoint as well as screens for Engage Servicing Portal (ESP) and require both SQL and HTML knowledge. The list of available HTML templates are maintained in:

  • Viewpoint – Home > Configuration > Document Libraries > HTML Templates under the ‘Win Template [V8]’ document type category

  • Viewpoint – Home > Configuration > Document Libraries > Engage Servicing Portal Templates

 

For each HTML template, it will have a:

  • .htm file - which contains the following:

    • HTML codes - These are standard HTML codes to define the style and format

    • Prefix - These are prefixes that are added to Field Codes, BLOCK statements or the Security Code

    • Field Codes - These refer to codes used to retrieve information from the database

    • BLOCK Statements - These are used to include or exclude information based on whether the condition is met

  • .def file - which contains the properties of the template including contents of the template's Key Information
