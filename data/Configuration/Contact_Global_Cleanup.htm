Contact Global Cleanup

The Contact Global Cleanup screen is where you can see existing contact details and have them corrected. It is available in Viewpoint - Home screen. To access it:

  • Using Ribbon - Click EA Global Ribbon button, select Global Change > Contact Global Cleanup

  • Using Section Menu - Click Home > Entity Administration >  Contact Global Cleanup

In the screen are fields, options, command buttons and the list of existing contact details.

Fields in Contact Global Cleanup

Field

Explanation

Cleanup For

You can choose to include contact details with country specified or ONLY those without any country information.

All/Administrator Group

This filter refers to the Administrator Group so you can choose to get contact details of all Master Files or only Master Files in a specific Administrator Group.

If you choose Administrator, you need to specify the Administrator Group.

Page Size

This is to specify the number of contact details that you want to see in the screen once you click Refresh. The minimum number is 20 and the maximum is 300.

If there are more than 300 existing contact details, you would need to work on the clean up in batches of 300 i.e. validate and update the first 300 and repeat for the next 300.

Note that the number of contact details seen may exceed the number set for Page Size e.g. Page Size is 20 but 21 contact details are shown. This is due to the Contact Global Cleanup loading all contact details for a Master File together.

Strip leading characters

This applies to the international dialling code (a.k.a. country exit code) which may have been included for some numbers and not for others. To standardise this, consider removing the international dialling code by entering it in this field.

When you enter ‘00’ for Strip leading characters:

  • Suggestion for 00 60 3 8656 0661 will be +60 3 8656 0661

  • Suggestion for 00 65 6323 7034 will be +65 6323 7034

 

Options in Contact Global Cleanup

These options are for correcting the format of the contact details and by default, all of them are displayed and ticked.

Once you have decided on the options that you want to apply and you wish to hide them, untick the Show Options checkbox.

Option

Explanation

Auto Complete Prefix

This adds the country calling code if the number currently does not have any.

Example:

Suggestion for a Malaysian number of 03 8656 0661 will be +6003 8656 0661

Add Space after Prefix

This adds a space after the country calling code.

Example:

Suggestion for a Malaysian number of 03 8656 0661 will be +60 03 8656 0661

Strip ‘(0)’

This will remove '(o)' from the number.

Example:

Suggestion for a Malaysian number of (0)3 8656 0661 will be +60 3 8656 0661

Replace ‘Hyphen’, ‘Dot’, ‘Parenthesis’ with Space

This replaces the characters with space.

Example:

Suggestion for a Malaysian number of 60.3.8656.0661 will be +60 3 8656 0661

Suggestion for a Malaysian number of 60-3-8656-0661 will be +60 3 8656 0661

 

Command buttons in Contact Global Cleanup

The command buttons are used to perform certain functions.

Command Button

Explanation

Refresh

Click this to load the contact details that are within the scope specified.

Suggest

Click this to have Viewpoint suggest the country for the contact details and how the number should be.

Open File

Click this to open the File Maintenance screen for the selected Master File.

View Details

Click this to open the Information screen for the selected Master File.

Export to Excel

Click this to export the list as an Excel file.

Update

Click this to update the existing contact details with the suggested country and number that have been validated.

Cancel

Click this to exit Contact Global Cleanup screen.

 

Checkbox in Contact Global Cleanup

Show Validated Only - Tick this checkbox to show only contacts with the Validated checkbox ticked.

 

Contact Details List

The list is populated once you clicked on Refresh command button.

Column

Explanation

MF Name

This is the Master File the contact detail is for.

Countries in Addresses

This lists countries of addresses for the Master File.

Contact Type

This is a description of the contact detail.

Country

This is the value in the Country column. If the scope is 'Without Country' this will be empty for all listed records.

Old Number

This is the existing contact details and is retrieved from the Number/E-mail column in the Contact Details grid.

Validated

This checkbox is to indicate that you have reviewed the record and it can be updated.

If left unticked, the record will not be updated.

Suggested Country

This holds a value that will be updated to the Country column in the Contact Details grid. When you click Suggest, a value may be suggested based on information in the following columns:

  • Countries in Addresses

  • Old Number

If there is no value suggested, you can add the value manually by selecting from the list.

Suggested Number

This holds a value that will be updated to the Number/E-mail column in the Contact Details grid. When you click Suggest, a value may be suggested based on the existing contact details and the options that you have ticked.

If there is no value suggested, you can double-click in the cell to make the corrections manually.

 

Using Contact Global Cleanup

  1. Access the function in Viewpoint - Home screen.

  2. In the Contact Global Cleanup screen, specify the scope for the cleanup in terms by selecting the relevant choices for the below:

    • Cleanup For

    • All/Administrator Group

  3. Specify the corrections.

    • In Strip leading characters, enter the existing prefix that you want to remove

    • Tick the checkbox for the option(s) that you want to apply

  4. Click Refresh to load the contact details that is in scope.

  5. Click Suggest.

  6. Review the suggested country and number for each contact details. If correct, tick the Validated checkbox. If there is no suggested country, you can manually set the country by selecting it from the list.
    If there is no suggested number, you can double-click in the column to make the corrections manually.

  7. Click Update to update the validated contact details.

Note: The contact details that are loaded in this screen are those where the contact type has the [REQCNTR=Y] tag in the Contact Types [ADRC] lookup table.
