# Obligations
An individual's or company's obligations can be recorded in the Obligations 
 screen which can be accessed through:

	

- Viewpoint - Home screen, by clicking To Do &gt; Jobs &gt; Obligations.

	

- File Maintenance screen by clicking General &gt; To Do &gt; 
    	 Jobs &gt; Obligations.

Note that by default, this screen is not active.

Filter

The filter allows you to view obligations by its status.

&nbsp;

Obligation list

This shows obligations that have been created.

&nbsp;

Command buttons

Click on the relevant command button to perform the necessary.

	

- <span class="hcp2">New</span> - Click to [add](javascript:TextPopup(this)) 
    	 new obligation. This is available only in File Maintenance.
    
    	<div class="droptext" id="POPUP381152910" style="display: none;">
    		Enter the details for the obligation record and click Update 
    		 when done.
     </div>

	

- <span class="hcp2">Edit</span> - Click to edit 
    	 selected obligation.

	

- <span class="hcp2">Delete</span> - Click to delete 
    	 selected obligation.

	

- <span class="hcp2">Complete</span> - Click to 
    	 update the selected obligation as completed once the obligation has 
    	 been fulfilled. This updates the Date Close and Date Done fields to 
    	 the date when you click Complete and tick the Complete option.

	

- <span class="hcp2">Linked Info</span> - Click 
    	 to view records linked to the selected obligation.
