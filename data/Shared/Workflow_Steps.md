 and can be accessed through:

	

- Viewpoint - Home screen - Click Home &gt; To Do &gt; Action 
    	 Items &gt; Workflow Steps.

	

- File Maintenance screen - Click General &gt; Action Items &gt; 
    	 Workflow Steps.

Workflow Step list

This shows the current steps for workflows that have been started. Each 
 workflow step will have a unique Step ID in the following format - <span class="hcp2">&lt;WorkflowID&gt;.&lt;StepNumber&gt;.&lt;StepSequenceNumber&gt;</span>

where:

	

- &lt;WorkflowID&gt; is the number assigned to the workflow when 
    	 the workflow was started.

	

- &lt;StepNumber&gt; is the number assigned to the step when the 
    	 step was created.

	

- &lt;StepSequenceNumber&gt; is the number assigned to the instance 
    	 of the step in the started workflow.

The Step ID allows each instance of the workflow step to be identified. 
 It can be seen in the Workflow Summary and is included in the printout 
 of the workflow and workflow step.
