Exchange Revaluation

Exchange revaluations are done to calculate exchange gains or losses and revalue foreign currency sub-ledger accounts. The system can automatically create the revaluation journal to realise exchange gains or losses.

The properties of the control accounts linked to the sub-ledger accounts are used for the journal posting: Exchange Gains for exchange gains and Exchange Losses for exchange losses. These must be set up for the relevant nominal control account(s).

When you select Exchange Revaluation, you will see:

  • Select - allows you to create new valuation or locate existing ones.

Once you choose to create a new valuation or view a previously saved one, you will then see:

  • Valuation - displays the valuation details.
