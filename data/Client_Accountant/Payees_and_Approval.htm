Payees & Approvals

The screen lists payees for all Account Files and is automatically filtered by those awaiting approval.

 

Filter

The filter applies to the payee list.

 

Select button

This allows for additional criteria.

 

Payee list

This shows all the payees that have been set up.

 

Command buttons

Click on the relevant command button to perform the necessary.

  • New - Click this to create a new payee.

  • Edit - Click this to edit a payee. If an approved payee is edited, the payee will have to be re-approved before subsequent payments can be made to the payee.

  • Delete - Click this to delete the selected payee record.

  • Approval - Click this to approve the payee. Payees will be approved based on the payee approval cards set up in Configuration.

    This will open the Approval screen allowing you to review the details and approve the payee by clicking Approve.
