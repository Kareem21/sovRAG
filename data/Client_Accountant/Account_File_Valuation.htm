Account File Valuation

Account File valuations are done to value all assets and liabilities for the purpose of management reporting or net asset value (NAV) calculations. The system has the option to generate an accrual journal to reflect the valuation in the account books. A valuation of an Account File will create an accrual journal, which, when posted, is automatically reversed the next day. Note that the control account properties used for the journal posting are Unrealized Value Reval. and Unrealized Exchange Reval. which must be set up for the nominal control account(s).

When you select Account File Valuation, you will see:

  • Select - allows you to create new valuation or locate existing ones.

Once you choose to create a new valuation or view a previously saved one, you will then see:

  • Valuation - displays the valuation details.
