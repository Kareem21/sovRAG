Journal Reporting Options

This screen allows user to customise the Report option for Journal reporting. The option(s) will be available when editing the Journal Line and serves as filter in the Account Trial Balance, Excel Statements and Standard Reports.

The Journal Reporting Options screen is Active by default, but users need to grant authority to the screen through the User Roles.

 

Field

Description

Caption(0x)

where:

x is either 2,4,8,16,32

Tick this to enable the option. The enabled option will have its caption visible for selection in the Journal, Journal Line or filter (in Account Trial Balance, Excel Statements and Standard Reports).

Note: Caption(64) is reserved for future use.

Text box (with pre-defined caption)

The caption in the text box is pre-defined and user is not encouraged to change this.

Allow at Journal Line

Tick this to enable the Journal Reporting option to be selected and updated only at the selected Journal Line.

Include by Default

Tick this to set the option as default for filtering and included in Sub-ledger and Nominal Account Balance. The ‘Include by Default’ option will be enabled automatically when the ‘Allow at Journal Line’ option is ticked.

Note: When any of the checkbox is updated in the Journal Reporting Options screen, you will need to run the ‘Update & Recalculate’ function in File Maintenance > Accounting > Functions > Recalculate/Calendar to recalculate the account balance.

 

When the caption option is enabled, it will show in the Flags field in File Maintenance > Accounting > Journals screen as follows:

  • In Journals > Script tab - User can select the option(s) from the checkbox in the Flags field. When the option is selected, it will be shown in the column ‘Flags’ at Journal line.

  • In Journals > Journal tab - User can select the option(s) from the checkbox in the Flags field.

 

‘Allow at Journal Line’ Option

The ‘Allow at Journal Line’ option controls the flags behaviour at journal line level.

 

System Behaviour when the ‘Allow at Journal Line’ Option is Enabled/Disabled

The following example shows three Journal Reporting options that have been enabled in the setting screen. They are:

  • Caption(02) Hide on Client Statement

  • Caption(04) Correction Bookings

  • Caption(08) Audit Adjustments

The option ‘Allow at Journal Line’ is only enabled for Journal Reporting option ‘Hide on Client Statement’. This would mean that the option ‘Hide on Client Statement’ will only apply to the selected line and will not apply to the other journal lines.

 

Example scenario

In the Journals, let’s say the first journal line is created with all three Journal Reporting options flagged. When the journal is updated, it will have the Flags value ‘14’ which indicates that Caption(02), Caption(04) and Caption(08) are selected. (Value 14 = 02+04+08).

When a new journal line is created, only the options ‘Correction Bookings’ and ‘Audit Adjustments’ are seen. This is because ‘Allow at Journal Line’ option is disabled for ‘Correction Bookings’ and ‘Audit Adjustments’ in the setting screen, resulting in both options being available for all the journal lines.

 

 

Journal Reporting Options in Journal Quick Entry

In Journal Quick Entry, the Journal Reporting options are shown in the Flags column at the Entries tab.

User can select the Journal Reporting option(s) at the Journal line by clicking on Edit Line button at Journals tab (top grid) > Journal Lines tab (bottom grid).

The option(s) will be visible in the ‘Edit Journal Line’ window:

 

 

System Behaviour when the ‘Allow at Journal Line’ Option is Enabled/Disabled for Journal Quick Entry

The following example shows three Journal Reporting options that have been enabled in the setting screen. They are:

  • Caption(02) Hide on Client Statement

  • Caption(04) Correction Bookings

  • Caption(08) Audit Adjustments

The option ‘Allow at Journal Line’ is only enabled for Journal Reporting option ‘Hide on Client Statement’. This would mean that the Journal Reporting option ‘Hide on Client Statement’ will only apply to the selected line and will not apply to the other journal lines.

 

Example scenario

In the Journal Quick Entry, select all three Journal Reporting options in the first line. When the second line is entered, only the Journal Reporting options ‘Correction Bookings’ and ‘Audit Adjustments’ are seen as both have ‘Allow at Journal Line’ option disabled in the setting, hence allowing them to be applied to other journal lines.

When the Journal Reporting option ‘Hide on Client Statement’ is flagged or unflagged on the existing line, it will only apply to that particular line.

Whereas when the Journal Reporting option ‘Correction Bookings’ or ‘Audit Adjustments’ is flagged or unflagged at any line, it will apply to all the journal lines.

 

 

‘Include by Default’ Option

The selected Journal Reporting option(s) will be available for filtering in the following screens when the option ‘Include by Default’ is ticked:

  • In Billing/CA Enquiry > Accounting > Trial Balance.

  • When generating Excel Statements in File Maintenance > Enquiry & Reporting > Excel Statements.

 

System Behaviour when ‘Included by Default’ Option is Enabled/Disabled

The following example shows three Journal Reporting options that have been enabled in the Account File. They are:

  • Caption(02) Hide on Client Statement

  • Caption(04) Correction Bookings

  • Caption(08) Audit Adjustments

The option ‘Include by Default’ is only enabled for Journal Reporting options ‘Hide on Client Statement’ and ‘Correction Bookings’.

With this setting, the Journal Reporting options ‘Hide on Client Statement’ and ‘Correction Bookings’ will be selected by default in the Trial Balance or when generating an Excel Statement.
