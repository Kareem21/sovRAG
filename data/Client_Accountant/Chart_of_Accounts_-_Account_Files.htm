Account Files

The Account Files that are linked to a particular Chart of Accounts template are listed here.

 

Chart of Account

Select the template to view the Account Files that are linked to it.

 

Account File list

This list shows all Account Files that utilise the selected CoA template.

 

Command buttons

Click on the relevant command button to perform the necessary.

  • Activate - Click this to activate an Account File.

  • Deactivate - Click this to de-activate an Account File. When an Account File is deactivated, the Delete button is enabled.

  • Delete - This button is activated only for inactive Account Files. Click to delete the selected Account File.

  • Backup - Click this to back up the selected Account File to an Access database file. This button is disabled for inactive Account Files.

    Tables that will be backed up:

    Table

    AcctArchHeader

    Accounting archive - header

    AcctArchJournal

    Accounting archive - journal header

    AcctArchTrans

    Accounting archive - journal transactions

    AcctBalance

    Account balance by account (system build)

    AcctBatch

    Journal line collection table

    AcctItems

    Account file sub-ledger master

    AcctJournal

    Journal master headers

    AcctPortFolio

    Portfolio master

    AcctRuleArrearFees

    Rule run details for recurring fees in arrears accrual

    AcctRuleLinks

    Posting rule run history for defer Income

    AcctRuleRun

    Account rule - run header

    AcctTrans

    Journal line master

    AcctTransLinks

    Account settlement for sub-ledger

    AcctValDetail

    Account valuation header

    AcctValHead

    Account valuation header

    EntityAccount

    Account file master

    EntityPeriods

    Account file period definitions

    EntityPostRules

    Posting rules

    PeriodList

    Period list (system generated)

     

  • Recovery - Click this to restore the Account File information from a backup Access database file. This will overwrite all existing details.
