Prices

This screen captures the current price of all securities in the Securities Master, as well as their historical price changes. Updating the Last Price and Date Last Price fields in the Security Master screen will update the information here. You can also update the Last Price and Date Last Price of a selected securities record here, or use an Excel spreadsheet to import the latest securities prices.

 

Filter

The filter allows you to view security prices for active, inactive or all securities.

 

Select button

This allows for additional criteria to apply to the list.

 

Security Price list

The list shows the prices set for the securities. Follow the steps below to update the security prices:

  1. Select the record you want to update.

  2. Update the Last Price field. The Date Last Price field will be updated to today's date.

  3. Click Update.

  4. Click Close to exit the screen.

 

Command buttons

Click on the relevant command button to perform the necessary.

  • Update - Click this to save changes.

  • Cancel - Click this to exit the screen without saving any changes.

  • History - Click this to view historical price changes for the selected record. You can click:

    • Delete - To delete a selected price record.

    • Close - To close the History screen.
