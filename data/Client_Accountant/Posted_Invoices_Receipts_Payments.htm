Posted Invoices/Receipts/Payments

To find out if an invoice/receipt/payment has been posted to Accounting, right-click on the item and select View Details from the context menu.

If it has been posted to Accounting, the Journal section will display the related journal number.

  • Journal No. - This will display a journal number if you do not use Accounting, and instead use an external accounting software. You can set up Viewpoint so that the journals are recorded (only for the revenue entity), then create a report to show the information for manual entry into your accounting software.

  • Journal No. CA Rev. Entity - This displays the number of the journal created in the Revenue Entity's accounts in Accounting. If the journal has not been created, this will show 0. If you mark all journals before a certain date as 'posted' even though it is not posted, this will show -1.

  • Journal No. CA B.File - This displays the number of the journal created in the Billing/Account File's accounts in Accounting. If the journal has not been created, this will show 0. If you mark all journals before a certain date as 'posted' even though it is not posted, this will show -1.
