Valuation/Revaluation - Select Folder

The Select folder has two main functions: to locate previous valuations/revaluations and to create new ones.

 

Account File Valuation/Exchange Revaluation/Value Revaluation/Portfolio Valuation

The appropriate function is automatically selected for you, while the other functions are disabled. Additional fields are enabled depending on your selection here:

  • Account File Valuation - Select the account type(s) that will be using exchange rate from Exchange Rates (CA) as per the valuation date. For account types that are not selected, the exchange rate used for the valuation will be the same as existing sub-ledger exchange rate.

  • Portfolio Valuation - Select the portfolio from the list.

  • Exchange/Value (Sub-ledger) Revaluation:

    • Capital/Income - Select the appropriate option to search for or create revaluations on the capital or income component of the accounts. This field is activated only if the Chart of Accounts linked to the selected Account File has been enabled for capital and income segregation, as is required for Trust files. Remember to create revaluations for both the capital and income components of the accounts to get a complete revaluation amount for the account balances.

    • Account Types (Exchange Revaluation only) - Select the sub-ledger accounts you want to include in the revaluation.

 

Valuation/Revaluation Date and (search +- days)

Enter the valuation/revaluation date and, if you are looking for an earlier valuation/revaluation, enter the number of days before and after the valuation/revaluation date to list all valuations/revaluations done in that period. The maximum number of days you can enter is 10,000.

 

New Valuation/Revaluation

Tick this checkbox to create a new valuation/revaluation based on the date entered in the Valuation/Revaluation Date field (see above). This will enable the Create button and disable the Search button.

 

Create

Click this to create a new valuation/revaluation.

 

Search

Click this to search for a previous valuation/revaluation based on the date range entered in the Valuation/Revaluation Date and (search +- days) fields. The results will be shown in the bottom half of the screen.

 

Edit/View

Select a valuation/revaluation from the list to enable this button. Click this to look at the details of the valuation/revaluation, which will be displayed in the Valuation/Revaluation folder.

 

Delete

Click this to delete the selected valuation/revaluation.

 

Print

Click to print a report on the selected valuation/revaluation.
