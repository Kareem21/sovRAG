Sub-ledger Revaluation

Sub-ledger revaluations revalue the sub-ledger accounts using the valuation method selected for the accounts; and calculate the gain or loss arising from differences in value. The system can automatically create the revaluation journal to realise the gains or losses. The properties of the control accounts linked to the sub-ledger accounts are used for the journal posting: Capital Gains for increases in value and Capital Losses for decreases in value. These must be set up for the relevant nominal control account(s).

 

When you select Sub-ledger Revaluation, you will see:

  • Select - Allows you to create new valuation or locate existing ones.

Once you choose to create a new valuation or view a previously saved one, you will then see:

  • Valuation - Displays the valuation details.
