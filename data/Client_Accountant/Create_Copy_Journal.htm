Create Journal

This is accessed from Account Enquiry allowing you to create journals by using the selected journal as a template.

 

Journal Type

Select the journal type to be created:

  • Reverse Journal - This will create a reversal to the selected transaction.

  • Use Journal as Template - This will create a new journal using the selected one as a template.

Reference

This defaults to the reference of the selected journal and is used in the Reference field of the new journal that you are creating. Change this if necessary.

 

Journal Date

This defaults to the journal date of the selected journal and is used in the Journal Date field of the new journal you are creating. Change this if necessary.

 

Clear the Reconcile and Line Dates

This checkbox is left unticked by default, so the Line Date and Reconciliation Date of the new journal picks up the information from the selected journal. Tick this to clear the fields and enter your own dates.

 

Get Exchange Rate at the specified Journal Date

Tick this to get the exchange rate for all the journal lines as at the specified journal date.

 

Open for 'Edit' in Journal screen

Select this option to display the lines of your new journal in the Journal screen. Edit where necessary before posting the journal.

See also:

  •  Journal Folder.

 

Save as 'Pending Journal'

Select this option to save your new journal as a pending journal.

See also:

  •  Pending Journals.

 

Post

Select this to post the new journal without further editing. If the option 'Get Exchange Rate at the specified Journal Date' is also ticked and results in a journal that does not balance, this will open the Journal screen so that you can edit the journal.

 

Command buttons

Click the relevant command button to perform the necessary.

  • Create - Click this to create the journal.

  • Cancel - Click this to cancel the process.

 

Note: When the accrual journal is edited, posted as final or deleted, the reverse journal will be edited, posted as final or deleted as well.
