Main/Sub Groups

Accounting uses a universal account classification which consists of Main groups and Sub groups ordered by their character (balance sheet or profit/loss item).

The Main and Sub group structure forms the backbone of the system and is set up globally for the whole database. The Main and Sub group structure is reflected in the trial balance and accounts enquiries, in financial statements as well as reports.

Command buttons

Click on the relevant command button to perform the necessary.

  • New - Click this to add new Main or Sub group. Alternatively, you can import the information from an Excel spreadsheet.

    You need to create the Main groups first, followed by the Sub groups.

    1. Enter the information required to set up the Main/Sub Group.

      • Group Type - Select the group type.

      • Main/Sub Group - The caption of this field changes depending on the selection made for Group Type. Enter the group code.

      • Main Group Code - This field is activated only when the Group Type selected is Sub Group [S]. Select the Main group that the Sub group belongs to.

      • Name - Enter the group name.

      • Carry Balance Forward (Balance Sheet) - This field is enabled only when the Group Type selected is Main Group [M]. Tick to indicate that this group is to be used in the Balance Sheet, and to carry balances forward to the next accounting period.

      • Active - This field is automatically ticked. You may remove the tick at a later time if you wish to deactivate the group.

      • Local Account Code - If you are using the Chart of Accounts set by the holding company but use a local code in preparing local accounts, you can enter the local code here.

      • Local Account Name - If you are using the Chart of Accounts set by the holding company but use a local name in preparing local accounts, you can enter the local account name here.

      • Note - Enter any relevant note relating to the group.

    2. Click Update to save the entry.

  • Edit - Click this to edit the selected group.

  • Delete - Click this to delete the selected Main or Sub group. You cannot delete a group if it is in use, i.e. if it has a Sub group or accounts linked to it.
