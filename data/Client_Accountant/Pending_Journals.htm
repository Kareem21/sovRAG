Pending Journals

The Pending Journals screen is where you can view and edit pending journals that have been created. Pending journals are created when you save an unfinished journal for completion at a later time. This helps you to avoid recreating the journal lines due to incomplete data entry as you would have to cancel it otherwise before you can exit the screen.

Pending journals can also be saved as recurring journals which is useful for entries that are repeated on a regular basis, such as the input of salary payments or monthly bank charges.

There are three tabs in Pending:

  • Select

  • Script

  • Journal

Select tab

This is where you can see all pending journals for the Account File.

 

Pending Journals, Recurring Journals, Deleted Journals and Posted Journals Headers

Select either radio button to filter the list.

 

Select button

Click this to apply filter to the list.

 

Pending Journals list

The pending journals are listed here.

 

Command buttons

These apply to the Pending Journals list. Click on the relevant command button to perform the necessary.

  • View - Click this to look at the journal lines relating to the selected transaction. This opens a new screen.

    The top section of the screen displays all transactions related to the journal in the Transactions table while the bottom half of the screen has tabs showing Journal Line Details for the selected transaction. If you had made multiple changes to the journal and saved each of the changes as a draft, then the journal line details of each draft journal is displayed in the related Journal Detail tab. This way, you can track the progressive changes made to the journal before it is finally posted.

    • Double-click the transaction from the Transactions table in order to view Line Details. This will give you information such as the originating journal number and the originating transaction number.

    • Select a tab to view the journal line details for that draft. When you do this, the selection in the Transactions table will sync with the tab so that you can see the transaction related to the line details.

    • Enter a Journal No. to search for a specific journal number, or a Transaction No. to locate a specific transaction, then click Search.

  • Edit Header - Click this to edit the selected journal's header.

  • Delete - Click this to delete the selected transaction. Deleted journals can be viewed by selecting the Deleted Journals radio button.

Journal Script List

If the selected pending journal was created using a Journal Script, the script will be highlighted.

 

> Script

This is enabled if the journal was created using a Journal Script. Click this to make changes to the selected journal in the Script tab.

 

> Journal

This is enabled if the journal was entered manually. Click this to make changes to the selected journal in the Journal tab.

 

You can create links for the journals listed here to documents, binders, file notes and obligations. The links will still exist after the journals are posted.

What happens to the link if I delete the pending journal?

This will delete the link from the linked item. If the linked item is deleted, then the link will also be removed from the pending journal record in Pending screen.
