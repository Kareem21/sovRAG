Templates

A Chart of Accounts template is based on the defined global account classification. It contains specific nominal accounts for certain jurisdictions or business sectors. The jurisdiction and default currency must be defined for each template although these fields are for reference purposes only and will not force the Account Files using the template to use the same base currency.

Each template includes specified links to retained earnings, profits transferred, and system suspense accounts.

Viewpoint recommends that Retained Earnings and Profit Transferred be linked to a system control account (account type 9) instead of to a balance sheet and profit/loss account. The system will automatically calculate and update the account balance for retained earnings and profit/loss with each journal posting.

Suspense accounts are reserved for any journal bookings from interface programs, and from interfaces that cannot be placed or booked, e.g. if the nominal account is invalid, the suspense account can be used for these unknown entries.

 

Template list

This list shows all the Chart of Accounts templates that have been set up.

 

Command buttons

Click on the relevant command button to perform the necessary.

  • New - Click this to add new template. Alternatively, you can import the information from an Excel spreadsheet.

  • Edit - Click this to edit and amend or update the information for a template. This must be done on the initial set up as well.

  • Delete - Click this to delete a template. This will delete all (nominal) accounts that are linked to it as well.
