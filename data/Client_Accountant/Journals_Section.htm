Journals

This section contains journal screens arranged as follows:

  • Journal Quick Entry screen which allows for journal and sub-ledger maintenance in one screen.

  • General Journals screen which can be used to record entries directly to nominal posting accounts or to the various sub-ledger accounts.

  • Nine sub-ledger account screens where you can enter journals specifically for that sub-ledger type's accounts.

When entering journals, note that the program uses the double-entry bookkeeping method and therefore, each journal consists of two parts; both parts must balance before the journal can be posted. The journal date is the reference date for storage in a particular period.
