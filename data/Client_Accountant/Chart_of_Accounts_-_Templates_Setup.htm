Chart of Accounts - Templates Setup

  1. Enter the information required to set up a template:

    • Code - Enter the template code. This cannot be changed once the template is saved.

    • Capital/Income Segregation - Tick this box if the template you are creating requires Capital/Income Segregation, as is required for Trust accounting. This cannot be changed once the template is saved.

    • Description - Enter a description of the template.

    • Default Currency - Select the currency for the template. This field is for reference purposes only and will not force Account Files that are linked to this template to use this currency as their base currency.

    • Default Country - Select a country for the template. This field is used for classification purposes.

    • Note - Enter any relevant note relating to the template.

    • Copy Accounts from - Copy the account structure of an existing template and modify it to suit this new template.

  2. Click Update to save the entry.

  3. Click Edit to complete the setup.
