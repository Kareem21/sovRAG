Recode Security

This feature is used to recode a security. Unlike the Recode Sub-ledger function, which is used to recode a securities sub-ledger account for a particular Account File, this function will recode a security throughout the system.

  1.  Complete the required information:

    • Recode-type - Security Master Code is automatically selected.

    • Security Master Code - Select the security to be recoded.

    • New code - Enter the new code.

  2. Click Recode.

  3. Click Yes to confirm and begin the recoding.
