Budgets

The Budgets screen allows you to create budgets for the selected Account File. The budget will be based on the Chart of Accounts used for the Account File. You can access this screen through:

  • File Maintenance screen:

    • Click Accounting Ribbon button and select Processing > Budgets from the navigation listing in Accounting Overview.

    • In Section Menu, click File > Accounting > Processing > Budgets.

 

Budgets list

This shows the budgets for the Account File.

 

Command buttons

Click the relevant command button to perform the necessary.

  • New - Click this to create a new budget.

    1. For the new budget, specify the budget parameters:

      • Budget Type - Select the budget type from the dropdown list. By default, two types are available - Budget and Forecast. Additional budget types can be set up in Budget Types [BDGT] lookup table.

      • Budgeting Period - Select the period for the budget from the standard period selection.

      • Details for Unit Code - If Unit Codes have been set up, the dropdown list will list the codes and the budget can be created for the selected one.

      • Note - Enter additional remark for the budget.

    2. If necessary, select options for the budget:

      • Hide in-active accounts - Tick this to hide accounts that are not active from the Budgets screen,

      • Hide 0 balances accounts - Tick this to hide accounts with zero balance from the Budgets screen.

      • Compare with - When a selection is made for one of the dropdown values, a column of figures corresponding to the selection will appear in the screen. This allows for comparison of the budget against that column. Note that 'Actual' means the actual value of the journals.

        Note: After the first budget has been created, the dropdown menu will include selections to compare with previous budgets.

      • Distribute 'Sum/Total' Edits - Select the distribution method:

        • Evenly - when a figure is entered in a cell in the Total row, that figure will be divided equally amongst the cells in that row. Similarly, if a figure is entered in a Total column, that figure will be equally divided along all cells that constitute the Total of that column. Where the Total is not equally divisible, the system will add the 'leftover' amounts one by one to the cells until there is no remainder.

        • Proportional - the same percentage by which the Total figure was increased or decreased will be applied to all cells in the related row or column.

      • Show variation - Select the display format for the difference between the budget and item of comparison. If the budget is being compared, i.e. a selection other than None was made for the Compare with field, depending on the selection for the Variation, a column will show the difference between the figure being compared and the figure being created either as an Amount or a Percentage.

      • Copy from 'Compare' - Select the format to apply to figures copied from item of comparison to the budget.

    3. Click Create. This will display the Budgets screen.

    4. Enter details for the budget in the Budgets screen cells.

      The budget is in the form of spreadsheet with editable cells. After entering details for the budget, click Update to save the information.

      1. By default, all accounts are displayed so care must be taken when allocating a budget to inactive accounts.

      2. Credit amount Income figures are shown as positive figures while debit amount Expenses figures are shown as negative figures. However, these figures are stored correctly as credit and debit amounts in the database tables and will be retrieved as such in reports.

      3. Cells that are yellow will calculate the totals for the related row or column. If a figure is entered in a Total cell, the value will be distributed according to the option selection for Distribute 'Sum/Total' Edits field.

      4. Figures in bold represent the sub total for the section.

      Note: Due to rounding of figures within the budget spreadsheet, the Balance Sheet may at times not be balanced.

    5. Click Update.

    6. Click Close to exit.

    After I have created a budget, can I print or copy it?

    Yes, this is possible. In the Budgets screen cells, right-click and you can select:

    • Copy

    • Export to Excel

    • Print Preview

  • Edit - Click this to edit the selected budget.

    1. Click Load.

    2. Make the necessary change for the budget.

    3. Click Update once complete.

  • Delete - Click this to delete the selected budget.

    Click Yes to confirm the deletion.
