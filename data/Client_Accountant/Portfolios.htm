Portfolios

Portfolios are used in the valuation feature in Accounting, enabling users to value all asset and liability sub-ledger accounts included in a selected portfolio, for the purpose of management reporting. The process uses the portfolio currency as the base for valuation.

 

Portfolio list

This shows the portfolios for the Account File.

 

Command buttons

Click the relevant command button to perform the necessary.

  • New - Click this to add a new portfolio.

    Enter the details of the portfolio. Click Update when done.

  • Edit - Click this to edit an existing portfolio.

  • Delete - Click this to delete a portfolio. If you delete a portfolio, the sub-ledger accounts linked to the portfolio will not be deleted.

  • User Fields - Click this to add user defined fields if you need to record more information on the portfolio.

 

How do I link a sub-ledger to a portfolio?

To do this, select the sub-ledger and click Edit. Select the portfolio you want to link to in the Portfolio field.

 

Where do I create user fields for portfolios?

This is done in Configuration > Business Parameters > User Fields > Portfolios.
