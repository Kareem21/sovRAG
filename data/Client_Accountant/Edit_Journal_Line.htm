Edit Journal Line

This feature allows you to update or make corrections to fields that are deemed non-critical to the permanent record from an audit perspective. In order to edit the line, you must have the authority to edit draft journals. Additionally, the journal date must be within the posting window.

 

The fields that can be changed are:

  • Movement Code

  • Tax Account and Tax %

  • Line Date and Reconciliation Date

  • Description and Narrative

  • Journal Report Option

 

Where do I customise the Journal Report Option?

This is done in File Maintenance > Accounting > Account File > Journal Reporting Options screen. See Journal Reporting Options for details.
