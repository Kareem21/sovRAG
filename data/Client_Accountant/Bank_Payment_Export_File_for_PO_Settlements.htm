Bank Payment Export File for PO Settlements

If you would like a payment export file for PO settlements, you need to first do the following:

  • Enable the ‘Create Accounting Payment Record for Payments’ option.

  • Enable the ‘Remit from’ and ‘Remit to’ field in Supplier File – General screen.

  • Set up the Revenue Entity’s bank account.

  • Configure the Payment Export File.

  • Set up and approve Payee and Payer Bank Accounts records.

 

Enable the ‘Create Accounting Payment Record for Payments’ option

This option will automatically create payment record for a supplier payment.

  1. In Viewpoint – Home, go to Configuration Ribbon button > Settings > System Defaults > Practice Management.

  2. Tick the ‘Create Accounting Payment Record for Payments’ option.

  3. Click Update.

 

Enable the ‘Remit from’ and ‘Remit to’ fields

  1. In Viewpoint – Home, go to Configuration Ribbon button > Settings > Customise > Field Settings.

  2. Specify the following:

    • Menu Group = File [8]

    • Section = Purchase Ledger [FPPURCLG]

    • Screen = General [FPSGENER]

  3. Select the Remit from field and click Edit.

  4. Set the Authority Required to at least level 6.

  5. Click Update.

  6. Do the same for Remit to field.

 

Set up the Revenue Entity’s Bank Account

Ensure that the bank account has been added to the Revenue Entity in Configuration > Business Parameters > Practice Management > Setup > Bank Accounts:

 

Ensure that this bank account is mapped to the Chart of Accounts through Account Mapping:

 

Configure the Bank Export File

If you require export batch files to be sent to the bank for further processing, you need to define the export file configuration. This depends on the bank file is for.

You may send the bank’s requirement to Viewpoint for assistance on creating the export file configuration. However, ensure that the configuration is updated when the bank requirement changes.

 

Set up and approve Payee and Payer Bank Accounts records

Before you record the settlement, set up both the Revenue Entity’s bank account and a Payee record for the Supplier.

 

Revenue Entity Bank Account

In Home > Practice Management > Payee/Payer Bank Accounts screen, click New to create the Revenue Entity’s bank account record.

Ensure that you approve this record.

 

Supplier Payee Record

In Home > Practice Management > Payee/Payer Bank Accounts screen, click New to create the Supplier’s payee record.

Ensure that you approve this record.

 

In the Supplier File’s General screen, set the following:

  • Remit from – Revenue Entity’s Bank Account record.

  • Remit to – The Supplier’s Payee record.

 

Recording the Settlements

After the configuration is complete, record the settlements in the normal manner.
