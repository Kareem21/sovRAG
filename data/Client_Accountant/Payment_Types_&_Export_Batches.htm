Payment Types & Export Batches

Bank Master Files can be configured to create export batch files, which will be sent to banks for further processing. Once the payment batches are complete, prepare the export batch files (where applicable) for subsequent transmission to the bank and close the payment batches.

 

You can click on the following to minimise or expand the sections in this screen:

  • - To minimise this section and hide the command buttons. In the grid, only the selected row will be seen.

  • - To expand the section and show the command buttons.

 

Filter

You can choose to view only active bank export configurations, inactive ones or all.

 

Batch configuration list

This shows all the bank export configuration that have been set up.

 

Command buttons

Click on the relevant command button to perform the necessary.

  • New - Click this to create a bank export configuration.

    Export batches are configured by bank so that payments can be assigned to them. These batches are then exported and submitted to the bank for processing.

    1. Enter the information of the bank export batch file.

    2. Click Update when done.

    See also:

    • Payment Types & Export Batches - Configuration Codes

    • Payment Types & Export Batches - Sample Configuration

     

  • Edit - Click this to edit a bank export configuration.

  • Delete - Click this to delete the bank export configuration.

 

Batch list

This list shows batches for the selected bank export configuration.

 

Command buttons

Click on the relevant command button to perform the necessary. These apply to the selected batch.

  • Export/Close - Click this to export and/or close the payment batch.

    Once payment batches are complete, prepare the export files for subsequent transmission to the bank, and close the batches. Batch files are closed when the export file is created to prevent further payments being assigned to them.

    1. Enter the following information:

      • Export Folder - Select the location where the export batch files will be saved.

      • Note/Result - Enter any notes relating to the payment batch.

      • Close Batch - This checkbox is ticked and disabled; so that when you create the export file, the batch automatically closes to prevent any further addition to it.

      • Print Report - Tick this to print a report of the transactions included in the batch.

    2. Click Export/Update to create the export file and automatically close the payment batch.

  • Print - Click this to print information on the selected batch.
