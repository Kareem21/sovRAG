CONSFIN3 - Consolidated Statements (Investment Sub-ledgers)

You must set up Viewpoint in the following manner in order to generate this standard statement:

    1. Select the Account File of the holding company.

    2. Set up Investment sub-ledgers to represent each of the subsidiaries. For example, if you have three subsidiaries, you would have three Investment sub-ledgers.

    3. Ensure the following fields are completed:

      • Master File - Select the subsidiary Master File.

      • Currency Code - Enter the Base Currency of the subsidiary Master File in this field.

      • Interest % - Enter the percentage ownership in this field.

In the generated report (see sample here*), the trial balances of all related Account Files (the holding company plus any subsidiary) will appear as follows:

  • Consolidation sheet - Trial Balances are converted to the Base Currency of the holding company. The exchange rates are taken from the Exchange Rate Master as at the Report End date. Note that the same rate is used for both Balance Sheet and the P&L figures; therefore, amend the rate in the generated report, if necessary.

If intercompany loans are logged to the same Nominal Account, the amounts will automatically knock off on consolidation. This sheet may be used as a consolidation worksheet. Enter any adjustments in column AD.

  • BaseCCY sheet - Trial Balances appear in the Base Currency of the individual Account File.
