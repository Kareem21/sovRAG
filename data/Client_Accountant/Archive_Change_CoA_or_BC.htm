Archive/Change CoA or BC

The functions here provide for the archiving of old journals and the changing of the Chart of Accounts and/or base currency. The CoA and base currency can be changed if you did not make an appropriate choice of either during the setup process or if the requirements of the Account File have since changed. This should only be done if necessary, since it is a complex process of carrying forward account balances, account mapping and currency conversions.

How do I view archived journals?

The list of all available archives for an Account File can be seen in the Archives section. Follow the steps below to access the Archive section:

  1. In File Maintenance screen, click the Accounts Ribbon button.

  2. Select Account tab.

  3. Click the Browse button for Select Ledger.

  4. Select the archive from the Archive section.

  5. Click OK.

  6. The journals will be loaded for viewing.

 

Archive journals and create a summary journal for the opening balance

This option is used to archive existing journals and create a new opening balance for all accounts accept Account Type 3 (Expense Posting) and Account Type 4 (Income Posting).

Opening balance for Retained Earnings (Account Type 9 - System Control Account) will be transferred to the System Suspense Account defined in the Nominal Account Template. Refer to Chart of Accounts - Templates - Edit.

Selecting this will activate the Change of 'Base Currency' of account file option and the Change the 'Chart of Account' template - allow to change (map) of the nominal account option (see below). This option can be used in conjunction with any of the other options.

How do I use this option?

  1. Click Next - Set new details >.

  2. Enter the required information:

    • Base Currency - This field displays the existing base currency for the Account File, and is disabled for the archiving process.

    • Conversion Rate - This exchange rate field is disabled for this process.

    • Chart of Accounts - This field displays the Chart of Accounts currently used for the Account File, and is disabled for this process.

    • Calendar - This shows the calendar used for the Account File and is disabled for this process.

    • Archive 'cut-off' Date - Enter the cut-off date. The date entered must be greater than or equal to the journal date of the last draft journal or final journal. All journals entered on or before this date will be archived.

  3. Click Next - Archive and perform changes >.

  4. As the process cannot be reversed, you will need to confirm the action in order to proceed. Click Yes.

  5. Click Close to exit the screen when the process is completed.

 

Change the 'Base Currency' of the account file

Select this to change the base currency of the Account File. This option is only possible if the Archive journals and create a summary journal for the opening balance option (see above) has been selected as well.

How do I use this option?

  1. Click Next - Set new details >.

  2. Enter the required information:

    • Base Currency - Enter the new base currency.

    • Conversion Rate - Enter the conversion rate from the existing base currency to the new base currency.

    • Chart of Accounts - This field displays the Chart of Accounts currently used for the Account File, and is disabled for this process.

    • Calendar - This shows the calendar used for the Account File and is disabled for this process.

    • Archive 'cut-off' Date - Enter the cut-off date. All journals entered on or before this date will be archived.

  3. Click Next - Archive and perform changes >.

  4. As the process cannot be reversed, you will need to confirm the action in order to proceed. Click Yes.

  5. Click Close to exit the screen when the process is completed.

 

Change the 'Chart of Account' template - all the existing (used) accounts exist in the new template

This option is used to change the Chart of Accounts template currently used for the Account File without changing any of the existing accounts. The system will map the existing accounts to the accounts in the new template and flag the ones that do not match. This option cannot be selected together with the Change the 'Chart of Account' template - allow to change (map) of the nominal account option (see below).

How do I use this option?

  1. Click Next - Set new details >.

  2. Enter the required information:

    • Base Currency - This field displays the existing base currency for the Account File, and is disabled for this process.

    • Conversion Rate - This exchange rate field is disabled for this process.

    • Chart of Accounts - Select the new Chart of Accounts.

    • Calendar - This shows the calendar used for the Account File and is disabled for this process.

    • Archive 'cut-off' Date - This field is disabled for this process.

  3. Click Next - Validate Account Mapping >.

  4. If any of the accounts are mapped to a red code, it means that the new Chart of Accounts does not have this account listed. You must set it up before you can proceed. Alternatively, use the Change the 'Chart of Account' template - allow to change (map) of the nominal account option (see below).

  5. If all codes have been correctly mapped, click Next - Perform changes >.

  6. As the process cannot be reversed, you will need to confirm the action in order to proceed. Click Yes.

  7. Click Close to exit the screen when the process is completed.

 

Change the 'Chart of Account' template - allow to change (map) of the nominal account

This option is used to change the Chart of Accounts template currently used for the Account File, with the added flexibility of allowing you to map the existing accounts to the accounts in the new template. This option is only possible if you choose to Archive journals and create a summary journal for the opening balance (see above) as well.

How do I use this option?

  1. Click Next - Set new details >.

  2. Enter the required information:

    • Base Currency - This field displays the existing base currency for the Account File, and is disabled for this process.

    • Conversion Rate - This exchange rate field is disabled for this process.

    • Chart of Accounts - Select the new Chart of Accounts.

    • Calendar - This shows the calendar used for the Account File and is disabled for this process.

    • Archive 'cut-off' Date - Enter the cut-off date. All journals entered on or before this date will be archived.

  3. Click Next - Validate Account Mapping >.

  4. Viewpoint will automatically map the old accounts to the new accounts, with unmapped accounts displayed in red. Select the appropriate accounts from the list for the unmapped accounts and do the same if you want to change the automatically mapped ones.

  5. Once all the codes have been correctly mapped, click Next - Perform changes >.

  6. As the process cannot be reversed, you will need to confirm the action in order to proceed. Click Yes.

  7. Click Close to exit the screen when the process is completed.
