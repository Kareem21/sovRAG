General Journals

This screen is generally used to enter day-to-day journals to a nominal posting account or to a specific sub-ledger account. It is divided into the following tabs:

  • Select - This lists available journal scripts for nominal accounts. You can enter a transaction by:

    • selecting the available journal script for Nominal Account ledgers and clicking > Script; or

    • clicking > Journal

  • Script - This can be accessed by selecting a journal script and clicking > Script. It shows the individual journal lines for the selected journal script. You may complete or amend the lines as necessary before you proceed to post the journal. This folder is disabled if you do not have any scripts or if you do not select a script for use. The feature 'Accrual Next Day Posting' is available on Journal Script screen. When the checkbox 'Accrual Posting' is ticked, the accrual next day reverse journal will be created.

  • Journal - This can be accessed by clicking > Journal. If you have selected a journal script before you clicked > Journal, it will show you the journal lines for the journal script so you can edit and post a pending journal. If no journal script was selected, you may enter the journal manually.
