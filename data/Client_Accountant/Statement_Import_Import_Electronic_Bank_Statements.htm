Statement Import & Processing - Import Electronic Bank Statements

This step is done after you have imported the source files.

  1. In Global - Functions, select Statement Import & Processing. The Electronic Statement Import & Reconciliation screen will open as a separate window.

  2. Click Import.

  3. Browse to the location of the New Import Files Folder, where electronic files are placed prior to import.

  4. Click Load File List.

  5. All available electronic files will be listed, and the related source file, if it has already been imported, will be indicated. Otherwise, select the source from the list of available files in the Source column. If the required source file is not in the list, then you need to import it first.

  6. When all the statements that you want to import have been matched to its related source files, click Import. If you do not match an electronic file to a source, the file will not be imported.

  7. Click Close when finished to exit the screen.
