Statement Import & Processing - Select/Import Folder

The function and options in this folder are used for the import of the source files as well as electronic bank statements.

 

Transaction Summary

Select this to view a summary by Account File, of the transactions in the selected statement.

 

Non-Completed, Completed and All

These options allow you to filter the statements based on whether or not they have been processed, completed, or both.

 

Date - from and - to

Enter the date range to search for statements based on the options selected above.

 

Statement Type

Use this to filter the list by transaction or account balance.

 

Statement Provider

Select the statement provider from the list.

 

Account File

Select to filter the statements for a specific Account File.

 

Search

Click this to refresh the list after you have entered or changed your search criteria.

 

Import

Click this to initiate the import process.

 

 

Statement List

The list of available statements is displayed with information on the following:

  • Statement Date - This is the date of the statement.

  • Statement Reference - This shows the type of electronic statement it is, i.e. Transaction or Balance.

  • Processed - This shows whether the statement has been processed. 'Ignored' is considered processed.

  • Completed - This shows whether the lines have been committed. 'Ignored' is considered committed.

  • Imported Date - This is the date the statement was imported.

  • Imported By - This shows the user who did the import.

 

View/Process

Click this to begin processing the selected statement or view already processed items. This will take you to the Process & Reconcile folder.

Note: Users with authority level 8- Power User can process statements imported by other users. With authority level 6 - Regular User, can only process users' own imported statements.

 

Edit

Click this to look at the details of the selected statement. Edit the Reference and Note fields as necessary, as well as tick or untick the Process and Completed checkboxes to change the values of the Processed and Completed fields in the Statement list (see above).

 

Delete

Click this to delete a selected statement from the list.

 

Report

Click this to generate an Excel statement.
