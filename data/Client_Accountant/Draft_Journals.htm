Draft Journals

Draft journals are used when you want the ability to review and/or edit a journal before the final posting. The usage of draft journals is optional and is set up for individual Account Files. Hence, one Account File may use draft journals while another does not. A draft journal is assigned:

  • Transaction No. - This is a running number assigned by the system and will remain unchanged even if the draft journal is edited and posted again. It starts at 1000 for each Account File.

  • Journal Post - This is a negative running number assigned by the system. It starts at -1 for each Account File with the next one being -2 and so on. If the draft journal is edited and posted again, the draft journal will be assigned the next number.

  • Transaction Revision No. - This is a running number assigned by the system. For newly-created draft journal, this will be the same as the transaction number. If the draft journal is edited, then posted again, this number will be increased by one. The increment will happen each time the draft journal is edited and posted as a draft journal.

In the Draft Journals screen, when you double-click on a draft journal, you will be able to see the above information.

 

Viewing Draft Journals

The draft journals of an Account File are listed in the Draft Journals screen for editing or deleting before the final posting. The Draft Journals screen can be accessed through:

  • Viewpoint - Home screen:

    • Click Accounting Ribbon button and select Functions > Draft Journals from the navigation listing in Accounting Overview.

    • In Section Menu, click Home > Accounting > Functions > Draft Journals.

  • File Maintenance screen:

    • Click Accounting Ribbon button and select Journals > Draft Journals from the navigation listing in Accounting Overview

    • In Section Menu, click File > Accounting > Pending > Draft Journals.

You can create links for the draft journals listed here to documents, binders, file notes and obligations. The links will still exist after the journals are posted as final.

If I edit a draft journal with linked item and post it again, will I get duplicate linked item?

No, as there is still just one instance of the draft journal.

What happens to the link if I cancel the draft journal?

This will delete the link from the linked item. If the linked item is deleted, then the link will also be removed from the draft journal record in the Draft Journals screen.

 

Draft Journals in Viewpoint - Home screen

In Viewpoint - Home screen, you can view draft journals for multiple Account Files. You can click on the following to minimise or expand the sections in this screen:

  • - To minimise this section and hide the command buttons. In the grid, only the selected row will be seen.

  • - To expand the section and show the command buttons.

 

Filters

The filters apply to the draft journal list.

  • Administrator - Enter the code or name of the administrator

  • Account File - Enter the code or name of the Account File

  • From Date, To Date - Enter the date range

Once you click Search, it will display all draft journals that fulfil your search criteria in the draft journals list.

 

Select button

This allows for additional criteria to apply to the draft journal list.

 

Draft journal list

This shows the draft journals that match your search criteria. Click on a draft journal to select it. The journals lines will be displayed at the bottom.

 

Command buttons

Click on the relevant command button to perform the necessary. These apply to the draft journals.

  • View - Click this to view details of the selected draft journal.

  • Print - Click this to print a statement.

  • Select All - Click this to select all displayed draft journals.

  • Deselect All - Click this to deselect all selected draft journals.

  • Post as Final - Click this to post the selected draft journal.

  • Delete - Click this to delete the selected draft journal. This is available only if you have authority level 8 - Power User.

 

Print Vouchers

Tick this option to print voucher for the draft journal that will be posted.

 

Journal Lines

The journals line seen are for the selected draft journal.

 

Command buttons

Click on the relevant command button to perform the necessary. These apply to the draft journal lines.

  • Edit Line - Click to edit the non-financial details of the selected journal line. All changes will be saved to the Change Log under the change code 'JNLE'.

  • Edit Units - Click this to edit the Unit Codes related to the journal line. Only the active Unit Codes related to the Account File will be enabled for editing.

  • AEOI Reporting - Click this to open the AEOI Reporting function.

  • Validate AEOI - Click this to Validate AEOI. The status will then be displayed in the Regime column in the grid.

    After the 'Validate AEOI' is performed, it will display the following in the Regime column:

    • Display ‘X’ if there isn't any transaction linked to CRS or FATCA.

    • Display ‘C’ for CRS, or/and ‘F’ for FATCA, or/and 'G' for UK FATCA.

    • Display ‘!’ if the Percentage is not 100%.

 

Draft Journals in File Maintenance screen

In File Maintenance screen, the draft journals are only for the selected Account File.

 

Filters

You can filter by:

  • Period - Click on the Browse button at the end to select the accounting period.

  • Journal type - Click to select if you want to view Draft Journals, Posted Journals or Deleted Drafts.

 

Draft journal list

Depending on your selection, this shows either draft journals, posted journals or deleted drafts.

 

Command buttons

Click on the relevant command button to perform the necessary.

  • View - Click this to view details of the selected draft journal.

  • Edit Journal - Click this to edit a draft journal. This will take you to the Journal tab. This is only available when the selection is Draft Journals.

  • Copy - Click this to copy the journal.

  • Print - Click this to print a statement. Select the statement you want in the Prepare: <Statement name> screen.

  • Select All - Click this to select all displayed draft journals. This is only available when the selection is Draft Journals.

  • Deselect All - Click this to deselect all selected draft journals. This is only available when the selection is Draft Journals.

  • Post as Final - Click this to post the selected draft journal without further edit. This is only available when the selection is Draft Journals. Note that only users with authority level 8 - Power User will be able to post draft journals.

  • Delete - Click this to delete the selected draft journal. This is available only if you have authority level 8 - Power User. This is only available when the selection is Draft Journals.

 

Journal Lines

This shows the journal lines for the selected journal.

 

Command buttons

Click on the relevant command button to perform the necessary. These apply to the draft journal lines.

  • Edit Line - Click to edit the non-financial details of the selected journal line. All changes will be saved to the Change Log under the change code 'JNLE'.

  • Edit Units - Click this to edit the Unit Codes related to the journal line. Only the active Unit Codes related to the Account File will be enabled for editing.

  • Rebook - Click this to rebook the account code. Note: The new account code must be the same ‘type’. This way, there is no need to un-post the draft journal. Rebook is not allowed for posted journal.

  • AEOI Reporting - Click this to open the AEOI Reporting function.

  • Validate AEOI - Click this to Validate AEOI. The status will then be displayed in the Regime column in the grid.

    After the 'Validate AEOI' is performed, it will display the following in the Regime column:

    • Display ‘X’ if there isn't any transaction linked to CRS or FATCA.

    • Display ‘C’ for CRS, or/and ‘F’ for FATCA, or/and 'G' for UK FATCA.

    • Display ‘!’ if the Percentage is not 100%.
