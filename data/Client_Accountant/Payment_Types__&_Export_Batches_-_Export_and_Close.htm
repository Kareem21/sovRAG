Payment Types & Export Batches - Export and Close

Once payment batches are complete, prepare the export files for subsequent transmission to the bank, and close the batches. Batch files are closed when the export file is created to prevent further payments being assigned to them.

  1. Enter the following information:

    • Export Folder - Select the location where the export batch files will be saved.

    • Note/Result - Enter any notes relating to the payment batch.

    • Close Batch - This checkbox is ticked and disabled so that when you create the export file, the batch automatically closes to prevent any further addition to it.

    • Print Report - Tick this to print a report of the transactions included in the batch.

  2. Click Export/Update to create the export file and automatically close the payment batch.
