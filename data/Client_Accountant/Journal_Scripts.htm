Journal Scripts

Journal scripts are used to standardise the posting of common financial transactions; and of complex transactions. You can build a library of journal scripts for each sub-ledger type. Before you begin the process of building your library, draft the journals and define which sub-ledger type they relate to. Viewpoint recommends that the creation of journal scripts be restricted to trained and authorised users.

 

Customise Journal Scripts

The Customise Journal Scripts screen opens as a separate window and is split into ten folders: one for General journals and one for each of the nine sub-ledger account types.

 

Section

Each folder contains three sections.

  • Journal header section - in this section, you have the following command buttons:

    • New - Click this to create a new journal script

    • Copy (New) - Click this to copy an existing journal script. You may do this if the new script you are creating is similar to the original and requires minor changes. Copying a script will copy all the journal lines from the original script as well.

    • Delete - Click this to delete the selected script.

    • Down - Click this to move the selected script header down one row. This only affects the way the scripts are listed in the sub-ledger journal screens.

    • Up - Click this to move the selected script header up one row. This only affects the way the scripts are listed in the sub-ledger journal screens.

    • Import - Click this to import scripts. Imported scripts are added to ALL Charts of Accounts (CoA) templates by default. You will need to de-select the CoA templates you do not wish the scripts to be used with.

    • Export - Click this to export scripts.

  • Journal line section - in this section, you have the following command buttons:

    • Add Line - Click this to add a new journal line to the selected script.

    • Remove Line - Click this to remove the selected journal line from the script.

  • Template assignment section - in this section, tick the Chart of Accounts templates that the selected script is for.

 

Command buttons

Click on the relevant command button to perform the necessary.

  • Update - Click this to save changes.

  • Cancel - Click this to cancel any changes made.

  • Close - Click this to close the Customise Journal Scripts screen.
