Journal Quick Entry

This screen allows for the maintenance and creation of journal entries and sub-ledgers in a screen. It can be accessed through:

  • Viewpoint - Home screen - Click Journal QE Ribbon button in Master File in Context tab.

  • File Maintenance screen - Click Journal QE Ribbon button in Opened Master File tab / In Section Menu, click File > Accounting > Pending > Journal Quick Entry.

  • Account Enquiry screen - Click Journal QE Ribbon button.

  • Billing Enquiry screen - Click Journal QE Ribbon button.

It opens as a separate window where there are the following:

  • Ribbon - comprises of buttons to enable options or access functions. The Ribbon is made up of the following tabs:

    • Home

      General Group

      The functions here allow you to open Windows Manager, open Notifications window or return to Viewpoint - Home screen.

       

      Create Group

      Click a function to create a workflow, file note, task, time entry, document holder and correspondence, respectively for the current account file. Click Add to Favourites to add the current account to Favourites.

       
      Enquiry Group

      Click a function to view the File Maintenance, File Enquiry, Relations, Ownership, Documents, Billing Enquiry and Account Enquiry screens for the current account file.

       

      Functions Group

      Click this to launch the Recode Subaccount/Sub-ledger feature. See Recode Subaccount/Sub-ledger.

       

    • Opened Master File

      In this tab, you have the Ribbon options below and the list of Master Files that are either the owner of, or owned by the selected Master File:

      General Group

      This group contains the following:

      • Windows Manager - Click to open Windows Manager.

      • Notifications - Click to open the Notifications window.

      • Go to Home - Click to return to Viewpoint - Home screen.

      • Open File - Click to open the File Maintenance screen for the selected Master File.

      Create Group

      This group contains the following:

      • Workflow - Click to start a new workflow for the selected Master File.

      • File Note - Click to create a file note for the selected Master File. See also File Notes.

      • Task - Click to add a task for the selected Master File. See also Tasks.

      • Time Entry - Click to add a time entry for the selected Master File. See also Time Entries.

      • Document Holder - Click to create a document holder for the selected Master File. See also Document Holder.

      • Binder - Click to create a binder for the selected Master File. See also Create New Binder.

      • Correspondence - Click to create a correspondence document for the selected Master File. See also Correspondence.

      • Journal QE - Click to open the Journal Quick Entry screen for the selected Master File. See also Journal Quick Entry. This button is disabled by default.

      Enquiry Group

      • File - Click to open the File Enquiry screen for the selected Master File.

      • Relations - Click to access the Relations enquiry screen for the selected Master File.

      • Ownership - Click to access the Ownership enquiry screen for the selected Master File.

      • Billing - Click to access the Billing enquiry screen for the selected Master File.

      • Accounts - Click to access the Account enquiry screen for the selected Master File.

      • Documents - Click to view stored documents for the selected Master File. This will launch the Documents screen.

      • Changes - Click to access the Changes screen for the selected Master File.

      Add To Group

      • Outlook - Click to add the selected Master File to the Outlook Correspondence folder.

      • Favourites - Click to add the Master File to the Favourites list.

    • Journal Quick Entry - this is seen when you select the Entries tab

      General Group

      This group contains the following:

      • Windows Manager - Click to open Windows Manager.

      • Notifications - Click to open Viewpoint Notifications.

      • Go to Home - Click to return to Viewpoint Home - Screen.

      Journal Types Group

      The functions here allow you to select the journal types that you want to create.

      • Single - Click this to create a single journal entry.

        1. Click the Single button from the Ribbon.

        2. In the Entries tab, click in the cell for Journal Date and enter the date.

        3. Enter details for the journal.

        4. Once done, click

          • Post to post the journal entry.

          • Save Pending to save as pending journal.

        Note: A green summary/total line will be seen showing the total debit/credit.

         

      • Multiple - Click this to create multiple journal entries.

        1. Click the Multiple button from the Ribbon.

        2. In the Entries tab, click in the cell for Journal Date in the first row and enter the date. This will display the green line.

        3. Enter details for the journal.

        4. Click the cell for Journal Date for the next journal and enter the date.

        5. Enter details for the journal.

        6. Repeat steps 4 and 5 until you have entered all journals.

        7. Once done, click Post to post the journal entries. If any of the journal validation fails, no journal will be posted. You need to review and correct before posting again.

        Note: The green line is the summary/total line will be seen showing the total debit/credit as well as separates one journal from another.

         

      • Cash Book - Click this to create Cash Book journal type. This allows for creation of a single journal entry or multiple journal entries.

        1. Click Cash Book button from the Ribbon.

        2. In the Entries tab, select the Sub-ledger.

        3. Enter the details for each journal line. You only need to add the non-cash journal line.

        4. Once done, click Post to post the journals. If any of the journal validation fails, no journal will be posted. You need to review and correct before posting again.

        Note: In the Sub-ledger section, the Reconciliation tab is shown for you to check as you enter the journals.

         

      • Inter-Company - Click this to create intercompany journal.

        Selecting Intercompany enables you to enter journals for the two Account Files within Journal Quick Entry. There is no need for you to access one Account File first to enter the journal and then proceed to open and enter journal for the second Account File. When you enter the first line for the first Account File, the program will also create the corresponding first line of the journal for you in the second Account File.

        Ensure that the second Account File is set up as a sub-ledger for the first Account File and vice-versa. You must specify the Account File in the Master File field when creating the sub-ledgers.

        1. Click the Intercompany button from the Ribbon.

        2. In the Entries tab, click in the cell for Journal Date in the first row and enter the date.

        3. Select the sub-ledger. This will set the selected Account File (first) as the sub-ledger for the second Account File.

        4. Enter details for the first journal line. The details will be seen in the corresponding first journal line for the second Account File (this line cannot be deleted).

        5. Complete the journal for both Account Files.

        6. Once done, click Post to post the journal entries. If any of the journal validation fails, no journal will be posted. You need to review and correct before posting again.

        Note: If the sub-ledger does not exist, you can create the sub-ledger using the 'Add Sub-ledger' button. The 'Add Sub-ledger' button is only enabled if the Account Type is Debtor [D], Creditor [C], Bank Account [B] and Investment [I].

       

      Options Group

      The options in this group applies to the journals that you will be creating.

      • Accrual Next Day - Click this to allow Accrual Next Day posting. This will create the reverse journal and applies to Single, Multiple and Inter-company journal types only.

      • Ignore Warnings - Click this to show/hide warnings when journal entries are posted or validated.

      • Multi Currency - This will be enabled if the selected sub-ledger is in a different currency from the Account File's base currency.

      • Tax Account - Click this to enable/disable the Tax Account column to specify the tax account for the entry.

      • Automatic Balance - Click this to enable/disable automatic balancing where a new journal line will balance the entry.

      • Default Rec. Date - Click this to enable/disable defaulting the reconciliation date to the journal date.

      • Tab Key - Use ‘Tab’ key to navigate within entries.

      • Copy Reference - When clicked, it will copy reference to description column.

       

      Fields Group

      This group contains fields that you can choose to show/hide in the journal grid. They are:

      • Running Balance

      • Account Name

      • Movements

      • Narrative

      • Line Date

      • Reconciliation Date

      • Quantity

      • Account Type

      • Date Exchange Rate

      When the field is enabled, it appears as a column in the journal grid. If the column is grey, it is to show information only. Users can only enter information for the field when the column is white.

       

      Functions Group

      This group contains functions that you can enable/disable for the journals that you will be creating.

      • Validate Journal - Click this to have the program check the journal entry. This validation is also performed by posting the journal.

      • Recalc. Exch. Rates - Click this to retrieve the exchanges rates for the journal entries. The exchange rate is from the Exchange Rates table as on the reconciliation date, if any. Otherwise, the exchange rate is as on the journal date. You can show the Date Exchange Rate to view.

      Edit Group

      The functions here are to select existing journal for editing.

      • Pending Journal - Click to search for a pending journal by entering its transaction number and clicking OK.

      • Draft Journal - Click to search for a draft journal by entering its journal number and clicking OK.

      Create Group

      The functions here apply to the selected Account File.

      • Copy Journal - Click to create a new journal based on an existing journal. Select the existing journal by entering its transaction number and clicking OK.

      • Opening Balance - Click to enter the opening balances for the Account File. This will list all active nominal accounts and sub-ledger accounts.

  • Entries Panel - for management and creation of journal entries.

  • Ledgers Panel - for management and creation of sub-ledgers.

  • Linked item tabs - that will appear to allow linking the journal entry to documents, file notes and workflows. This is seen when:

Note: You can delete reversal draft journals in Journal Quick Entry screen. When deleting a reversal draft journal it would not delete the original journal. However, deleting the original journal will automatically delete the reversal journal.

 

Entries Panel

This section has four tabs:

  • Scripts - This tab shows the Chart of Accounts and sub-ledgers for the Account File at the top and available journal scripts at the bottom. When Journal Quick Entry screen is first opened, the Chart of Accounts tab is shown by default, together with the 'General' journal scripts.

  • Entries - This is where you enter new journal entries.

    The journals are to be entered in the journal grid. You can start by clicking in the first cell in the first row and enter the journal date for the first journal line and proceed with the rest. The Journal Quick Entry accepts zero-amount for journal line. A green line indicates the end of a journal entry.

    Within the journal grid, you can use your keyboard to:

    • Move from cell to cell - either by using the left and right arrow key; or use TAB key if you enable the option 'Navigation with Tab Key' from the Ribbon.

    • Add new journal line - the second line is always added automatically once you start data entry but you can also press the Insert key on your keyboard.

    • Remove information - if you click on a cell with value, pressing the Delete key will remove the value. If the cell is empty, pressing Delete key will delete a line. Note that if the cell is a non-editable cell, pressing Delete key will also delete the line.

    Note: The nominal account line currency will default to previous line's currency. For example, in the second line, the nominal account line currency will default to the first line's currency. The 'Reference' input field is limited to 40 characters and the 'Description' input field is limited to 50 characters.

     

    Available command buttons:

    • Post - Click this to post the journal(s) entered.

    • Save Pending - Click this to save the journal as pending journal. This is only applicable for Single journal entry.

    • Cancel - Click this to clear all information entered.

    • Add Line - Click this to add new journal line.

    • Delete Line - Click this to delete the selected journal line.

    Note: Journals for share split can be posted as well.

     

  • Journals - This shows existing journals for the Account File and you can filter to view Draft journals, Posted journals, both Draft and Posted journals, Pending journals, Recurring journals and Deleted journals. Journal listing can be filtered by period selection.

    Available command buttons:

    • View - Click this to view details of the selected draft journal.

    • Edit Journal - Click this to edit a draft journal. This is only available when Journal Type is Draft, Pending or Recurring.

    • Copy - Click this to copy the journal.

    • Print - Click this to print a statement. Select the statement you want in the Prepare: <Statement name> screen.

    • Select All - Click this to select all displayed draft journals. This is only available when Journal Type is Draft, Pending or Recurring.

    • Deselect All - Click this to deselect all selected draft journals. This is only available when Journal Type is Draft, Pending or Recurring.

    • Post as Final - Click this to post the selected draft journal without further edit. This is only available when the selection is Draft Journals. Note that only users with authority level 8 - Power User will be able to post draft journals.

    • Delete - Click this to delete the selected draft journal. This is available only if you have authority level 8 - Power User.

  • Transactions - This shows the transactions for the journal selected in Journal tab or the transactions in the opening period of the selected sub-ledger.

    Available command buttons:

    • View - Click to display the journal details.

    • Print - Click to print the selected journal.

    • Edit Line - Click to edit the non-financial details of the selected journal line.

    • Edit Units - Click this to edit the Unit Codes related to the journal line. Only the active Unit Codes related to the Account File will be enabled for editing.

    • Edit Journal - Click this to edit the journal.

    • Copy - Click to create a journal entry using the selected journal as a template.

    • Settle - Click this to settle the journal.

    • Rebook - This button will prompt with a screen for input of the new account code. Note: This works only for draft journals.

    • AEOI Reporting - Click this to open the AEOI Reporting screen.

    • Validate AEOI - Click this to Validate AEOI. The status will then be displayed in the Regime column in the grid. This button is disable for pending journal.

      After the 'Validate AEOI' is performed, it will display the following in the Regime column:

      • Display ‘X’ if there isn't any transaction linked to CRS or FATCA.

      • Display ‘C’ for CRS, or/and ‘F’ for FATCA, or/and 'G' for UK FATCA.

      • Display ‘!’ if the Percentage is not 100%

 

Ledgers Panel

This sections is made up of the following tabs:

  • Chart of Accounts - This shows the nominal accounts for the Account File. When you first access it, you can specify criteria to filter the list.

    Available command buttons:

    • New Subaccount - Click this to create a new sub-account. It is only available for the account types Asset, Liability, Expense and Income.

    • Edit - Click this to edit the name of the selected account or deactivate it.

    • Transactions - Click this to show all transactions in the opening period of the selected account. The transactions are shown in the Transactions tab in the Journal section.

     

    See also:

    • Chart of Accounts

  • Sub-ledger types - This shows the existing sub-ledgers for each type and you can edit or delete the sub-ledgers. If necessary, you can also create new ones from here.

    Available command buttons:

    • New - Click this to set up a sub-ledger account.

    • Edit - Click this to amend the information of the selected sub-ledger account.

    • Delete - Click this to delete the selected sub-ledger account. This is possible only if there is no journal posted to it. Otherwise, consider deactivating it.

    • Settlements - Click this to perform account settlement. This button is available only if the Settlement Method for the selected sub-ledger account is 'Automatically (FIFO Settlement) [8]' or 'Manual Settlement [9]'.

      1. Enter the details:

        • Journal Date - This defaults to today's date.

        • Reference - Enter the reference for the settlement.

        • Description - Enter description for the settlement.

        • Sub-ledger Type, sub-ledger account - Select the sub-ledger type and account where the settlement will be placed. Currently, only posting account's sub-ledgers are allowed.

        • Amount <settlement sub-ledger currency>- Enter the settlement amount. The currency is the currency of the settlement sub-ledger. You can select either DR or CR when entering the settlement amount.

        • Exchange - This is the exchange rate for the settlement based on the journal date. It converts the currency of the settlement sub-ledger to the Account File's base currency.

        • <Base currency> - This is the amount in the Account File's base currency.

        • (Bank) Charges - Enter the bank charges, if any. Note that when there is bank charges, the total amount settled is 'Amount - (Bank) Charges'.

        • Narrative - Enter the narrative for the settlement.

        • Settlement <currency for the selected sub-ledger> - This shows the amount in the currency of the selected sub-ledger but can be changed.

        • Exchange - This is the exchange rate for converting the currency of the selected sub-ledger to the Account File's base currency.

        • <Base currency> - This is the amount in the Account File's base currency.

        • Write off <sub-ledger currency> - Enter any write off amount.

        • Settle this Transaction - Tick this to perform the settlement. Note: When this is ticked, the column filters will be enabled.

      2. To post the journal, click on either

        • Post Journal; or

        • View & Post Journal - this will display the Journal Quick Entry screen and populate the journal details before posting. User can verify the journal details before posting.

    • User Fields - Click this to add user fields to the sub-ledger account.

    • Analyses - Click this to view the analyses done on the sub-ledger account.

    • Statement - Click this to print a statement of the sub-ledger account. Enter the period and select the statement you want printed in the Prepare: <Statement name> screen.

    • Depreciate All - This is available only for Fixed Assets sub-ledgers. Click this to have Viewpoint analyse and suggest depreciation posting for all fixed assets.

    • Rebalance All - This is available only for Securities and Investment sub-ledger. Click this to have Viewpoint perform rebalance for all Securities and Investments accounts.

    • Corp. Actions - Click to view the corporate actions, such as dividends, bonus issues, etc., that have been made with regard to the securities in the Security Master. This function relates to the Securities sub-ledger only.

     

    See also:

    • Sub-ledger Accounts

  • Journal Lines - This shows the journal lines for the selected journal in Journals tab.

    Available command buttons:

    • View - Click this to display the journal details.

    • Print - Click this to print the selected journal.

    • Edit Line - Click this to edit the non-financial details of the selected journal line.

    • Edit Units - Click this to edit the Unit Codes related to the journal line. Only the active Unit Codes related to the Account File will be enabled for editing.

    • Edit Journal - Click this to edit the journal.

    • Copy - Click this to create a journal entry using the selected journal as a template.

    • Settle - Click this to settle the journal.

    • Rebook - This button will prompt with a screen for input of the new account code. It works only for draft journals.

    • AEOI Reporting - Click this to open the AEOI Reporting screen.

    • Validate AEOI - Click this to Validate AEOI. The status will then be displayed in the Regime column in the grid. This button is disable for pending journal.

      After the 'Validate AEOI' is performed, it will display the following in the Regime column:

      • Display ‘X’ if there isn't any transaction linked to CRS or FATCA.

      • Display ‘C’ for CRS, or/and ‘F’ for FATCA, or/and 'G' for UK FATCA.

      • Display ‘!’ if the Percentage is not 100%

  • Reconciliation - This tab is seen only when the Journal Types is set to Cash Book (in the ribbon). It allows you to enter the statement date, statement opening balance and statement balance in yellow cells. As you enter the journals, you can refresh it to check if any transaction is missed out.
