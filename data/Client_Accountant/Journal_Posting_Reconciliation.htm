Journal Posting Reconciliation

You can use the journal posting reconciliation reports provided in the Practice Management component to check for discrepancies that may exist between the journals in Practice Management and the ones in Accounting. If the journals in both modules are balanced, the difference between the two totals will be nil.

In cases where there is a difference in the two totals, run the posting function to post any outstanding journals from Practice Management to Accounting. Any errors identified in the posting process should be resolved before the posting process is run again.

  1. In Viewpoint - Home screen, click Reporting Ribbon button and select Practice Management > Standard Reports > Journal Posting Reconciliation from the navigation listing in Reporting Overview.

  2. There is a total of five standard reports available to help you check for discrepancies:

    • Client Account File - Creditors Reconciliation Report - This shows the total invoice amounts raised for a Billing File. Amounts are displayed in invoice currency and base currency for Practice Management, and in the sub-ledger currency for Accounting. Differences are displayed in the sub-ledger currency.

    • Client Account File - Matter Balance Reconciliation Report - This shows the Matter balances for a Billing File. Amounts are displayed in Matter currency and base currency for Practice Management, and in the sub-ledger currency for Accounting. Differences are displayed in the sub-ledger currency.

    • Revenue Entity - Debtors Reconciliation Report - This report shows the total amount raised by debtors of a particular Revenue Entity. Amounts are displayed in invoice currency and base currency for the Practice Management, and in the sub-ledger currency for Accounting. Differences are displayed in the sub-ledger currency.

    • Revenue Entity - Matter Balances Reconciliation Report - This report shows the Matter balances for the Revenue Entity and is displayed in the base currency.

    • Revenue Entity - Creditors Reconciliation Report - This shows the total amount owed by the Revenue Entity to its creditors. Amounts are displayed in the purchase order currency and base currency for Practice Management, and in the sub-ledger currency for Accounting. Differences are displayed in the sub-ledger currency.
