Statement Import & Processing - Import Source File

The source files used to set up your environment for the Statement Import and Processing feature can be obtained from your Viewpoint Support representative. If you find that the feature no longer works as expected, it may be that the bank has changed the format of their electronic statements, and you need to forward a copy of the latest to your Viewpoint Support representative so that a new source file can be created and imported to your environment.

  1. In Global - Functions, select Statement Import & Processing. The Electronic Statement Import & Reconciliation screen will open as a separate window.

  2. Click Import.

  3. Browse to the location of the New Import Files Folder, where source files are placed prior to import.

  4. Browse to the location of the Import Done Folder, where source files are moved to after import.

  5. Click Load File List.

  6. All available source files will be listed. At this point, select the source files that you do NOT want to import, as the default setting will import all files. Do this by selecting No Processing in the Source column.

  7. Click Import.

  8. Click Close when finished to exit the screen.
