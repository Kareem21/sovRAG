Pending Payments

Payments can be made in Accounting and Entity Administration to approved payees; each payment must be approved according to the approval process set up in the approval cards before they can be released. Approved payments are then released to bank payment batches for inclusion in bank payment export files. Once released, the payment will automatically create a pending journal. You should set up your journal scripts before using this function. This screen is part of the Bank Payment feature, which also includes the Payees and Export Batches functions.

 

Pending Payments Screen

This screen can be opened through:

  • Viewpoint - Home screen

    • Click Accounting Ribbon button and select Bank Payment > Pending Payments from the navigation listing in Accounting Overview.

    • In Section Menu, click Home > Accounting > Bank Payment > Pending Payments.

  • File Maintenance screen:

    • Click Accounting Ribbon button and select Processing > Pending Payments from the navigation listing in Accounting Overview.

    • In Section Menu, click File > Accounting > Processing > Pending Payments.

 

Filter

You can filter based on the status or apply selection criteria using the Select button.

  • Waiting Approval - displays the payments that are waiting for approval.

  • Waiting Release - displays the approved payments that are waiting for release.

  • All - displays both Waiting Approval and Waiting Release payments.

 

Payments list

This shows payments that have been created.

 

Command buttons

Click the relevant command button to perform the necessary.

  • New - Click this to create new payment.

  • Edit - Click this to edit a payment. If an approved payment is edited, the payment will have to be re-approved.

  • Delete - Click this to delete the payment.

  • Approval - Click this to approve the payment. Payments will be approved based on the payment approval cards set up in Configuration.

    If you are authorised to approve the payment, you can click on the Approve button.

    Otherwise, you will see a message saying you are not authorised for the approval.

     

     

  • Release - Click this to release the payment to a bank payment batch. This button will be enabled after the payment is approved.

    1. Select the applicable batch option.

    2. Click Release.

 

Approve Payments

Payments must be approved by the required number of users before they can be released. The approval process is determined by the approval cards set up in Configuration - the cards determine who can approve a payment, and the number of approvals required to approve the payment.

The Approval screen opens as a separate window when you click the Approval button. The payment that had been approved will shown a green colour header at the upper side with note ‘Approved & Completed. You can also see the status of the payment in the ‘Approved’ column. If the payment has been approved, the status will show ‘Yes’ and vice versa.

 

Release Payments

When a payment has been approved, it must be released to either a manual batch or an exportable bank payment batch. On release, a pending journal will be created for the transaction.

  • Proceed to release payment by clicking on the Release button.

  • The record will be disappeared from the list upon released.

  • The record has now transferred to export step in Home > Accounting > Bank Payment Functions > Payment Types & Export Batches Select the Payee bank and the release record will be displayed at the bottom panel.

  • Click Export/Close to export the payment into batch file.

  • The ‘Bank File Export’ window appears when Export/Close button is clicked.

  • When you release a payment, you will be given the option to add the payment to a batch. The available batches are displayed in the Payment Release screen, which opens as a separate window.

  • Manual options should only be selected if you have not configured a batch file for this bank.

  • Once a payment has been released to the Manual - Completed or Cancelled Payments batch option, the process cannot be undone.

Note: The option ‘Export file’ is checked by default. You may uncheck the box if using manual method to export the file. The option ‘Close Batch’ will be disabled when ‘Export File’ is checked.
