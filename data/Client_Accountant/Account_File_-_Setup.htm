Setup

This screen is only active for new Account Files that need to be set up. Note that before you set up an Account File, you must have at least one Chart of Accounts template set up.

  1. Specify the following:

    • Base Currency - Select the base currency for the file. Be sure of the currency as changing this at a later time will require complex process of carrying forward account balance, account mapping and currency conversions.

    • Chart of Accounts - Select the Chart of Accounts to be used for the file.

    • Last Year-End - Enter the last year-end for the file.

  2. Click Create Accounts.

  3. Once complete, click Close. You can review and enter additional information for the Account File in the following screens:

    • General

    • Chart of Accounts

    • Account Information

    • User Fields

    • Portfolios
