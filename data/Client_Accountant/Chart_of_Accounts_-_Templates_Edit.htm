Chart of Accounts - Templates - Edit

  1. Select the template you want to edit or complete and click Edit.

  2. Enter the information required:

    • Retained Earnings - Viewpoint recommends this be linked to a system control account (denoted by '9') instead of to a balance sheet account. The system will automatically calculate and update the account balance for retained earnings with each journal posting.

    • Profits Transferred - Viewpoint recommends this be linked to a system control account (denoted by '9') instead of to a profit/loss account. The system will automatically calculate and update the account balance for profit/loss with each journal posting.

    • System Suspense - Select the relevant posting account.

  3. Edit other template details if required.

  4. Click Update to save the entry.
