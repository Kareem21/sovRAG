General

The General folder displays the information entered during the setup process. The fields seen in this screen are:

  • Standard fields - these are standard Viewpoint fields that can be activated or deactivated for use.

  • Custom fields - these are fields that can be added to the screen. Refer to Custom Fields for more information.

 

The default posting period is two years from the Last Year-End, Can I change the default posting period?

Yes, you can change the default period from two years to a time frame more suitable to your organisation. This can be done in Configuration > Settings > Customise > Field Settings.

Select File [8] Menu Group and Accounting Section. Click Find and select General.

Amend the information in the Open and Close fields. Enter a number, followed by the time period (d = day, w = week, m = month, q = quarter or y = year). For instance, 6m for 6 months.

Note that this is only applicable for newly-set up Account Files. Post setup, if you want to change the posting period, you will need to manually adjust the dates in the Open and Close Posting Control Dates fields. Subsequent Roll Year End functions will add a year to the dates.
