Account Enquiry

The Account Enquiry screen opens as a separate window and provides easy access to accounting information for the selected file. The screen contains the following:

  • Ribbon - You can use the Ribbon to access other enquiry screens, return to Master File Search, open the File Maintenance screen or create items for the selected file.

    Opened Master File tab

    [Opened Master File] - contains additional function buttons that allows you to access screens and functions for the opened Master File.

    General Group

    • Windows Manager - Click to open Windows Manager.

    • Notifications - Click to open the Notifications window.

    • Go to Home - Click to return to Viewpoint - Home screen.

    • Open File - Click to open the File Maintenance screen for the selected Master File.

     

    Create Group

    • Workflow - Click to start a new workflow for the selected Master File.

    • File Note - Click to create a file note for the selected Master File. See also File Notes.

    • Task - Click to add a task for the selected Master File. See also Tasks.

    • Time Entry - Click to add a time entry for the selected Master File. See also Time Entries.

    • Document Holder - Click to create a document holder for the selected Master File. See also Document Holder.

    • Binder - Click to create a binder for the selected Master File. See also Create New Binder.

    • Correspondence - Click to create a correspondence document for the selected Master File. See also Correspondence.

    • Journal QE - Click to open the Journal Quick Entry screen for the selected Master File. See also Journal Quick Entry.

     
    Enquiry Group

    • File - Click to open the File Enquiry screen for the selected Master File.

    • Relations - Click to access the Relations enquiry screen for the selected Master File.

    • Ownership - Click to access the Ownership enquiry screen for the selected Master File.

    • Billing - Click to access the Billing enquiry screen for the selected Master File.

    • Accounts - Click to access the Account enquiry screen for the selected Master File. This button is disable by default.

    • Documents - Click to view stored documents for the selected Master File. This will launch the Documents screen.

    • Changes - Click to access the Changes screen for the selected Master File.

    Add To Group

    • Outlook - Click to add the selected Master File to the Outlook Correspondence folder.

    • Favourites - Click to add the Master File to the Favourites list.

  • Section Menu - There are two options available, Account and Trial Balance.

 

Account

This will show you accounts for the selected Account File. When you select Account, you will see these four tabs:

  • Account

    This is where you see the actual balances of the Account File's sub-ledger accounts. The balances are calculated after each posting and include draft journals.

     

    Select Ledger

    Click the browse button to display all available ledgers and select the one you want to display. If a portfolio is selected, then only record that belongs to the selected portfolio will be shown. If 'All Portfolios' is selected, the screen shows all portfolio outlines with sub-ledger.

     

    Period

    Click the browse button to display the available periods, and then select the range you want.

     

    Hide 0 balance Accounts

    Tick to hide zero-balance accounts.

     

    Show Sub-Ledger Balance

    Tick to display sub-ledger balances.

     

    Show Sub-ledgers for Posting Accounts

    Tick this to display all sub-ledgers that were posted to the selected posting account (account types 1, 2, 3 and 4) where the specified sub-ledger is set to either 'Mandatory' or 'Optional'. If no sub-ledger is specified, '<Not Specified>' is shown. The Amount shown is for the selected period and in the Account File's base currency.

     

    Show related Nominal Account postings

    Tick this to display all the nominal accounts that were posted to the selected sub-ledger with the posting sum for the selected period.

     

    Show Latest Sub-ledger Balance

    Tick this to display all the sub-ledger balances up to the latest date.

     

    Refresh

    Click to reload the screen with the latest account balances.

     

    Details

    Click to display information on the selected nominal account or sub-ledger.

     

    Print Statement

    Click to print a statement on the account. Enter the details and select the statement you want printed in the Prepare: <Statement name> screen that appears.

     

    Analyses

    Click to display a summary of the account within a selected time period.

     

    See also:

    • Sub-Ledger Accounts - Analyses

     
    Refresh

    Click to update the records to the latest value.

     

  • Balance

    This displays the balance of the selected nominal or sub-ledger account for the period specified in the Accounts tab.

     

    Details

    Click to display information on the selected nominal account or sub-ledger.

     

    Print Statement

    Click to print a statement on the account. Enter the details and select the statement you want printed in the Prepare: <Statement name> screen that appears.

     

    Analyses

    Click to display a summary of the account within a selected time period.

     

    See also:

    • Sub-Ledger Accounts - Analyses

     
    Refresh

    Click to update the records to the latest value.

     

  • Transactions

    This is where transactions of the selected nominal or sub-ledger account is displayed.

     

    Nominal Account

    The nominal account that was selected in the Accounts folder is displayed here.

     

    Period

    This displays the period selected in the Balance folder. Click the browse button to change this and select the range you want.

     

    Journal Date

    Select this to display transactions where the journal date is within the Period selected.

     

    Reconciliation Date

    Select this to display transactions where the reconciliation date is within the Period selected.

     

    Line Reference Date

    Select this to display transactions where the line date is within the Period selected.

     

    Non-reconciled journals

    Select this to display ALL unreconciled journals.

     

    Running Balance

    Tick this to display the account balance after each transaction.

     

    Quantity - Running Total

    Tick this to display the quantity as at the transaction in the 'Quantity - Running Total' column. This checkbox is available only when the 'Control Quantity' checkbox is ticked for Securities, Investment, Share/Unit Holder and Products/Services sub-ledgers.

     

    Transaction table

    The account transactions are listed in the table. Select a transaction from the list and the journal lines are displayed in the Journal Line table. You can also click the Journals tab to view the individual journal lines.

     

    View

    Click to display the journal details. This is the same information as you would see in the Journals folder, except that this opens as a separate window.

     

    See also:

    • Viewing Journal Details

     

    Print

    Click to print the selected journal. Select the template you want to use in the Prepare: <Statement name> screen that appears.

     

    Copy

    Click to create a journal entry using the selected journal as a template.

     

    See also:

    • Create/Copy Journal

     

    Edit Line

    Click to edit the non-financial details of the selected journal line. All changes will be saved to the Change Log under the change code 'JNLE' and can be seen in Global - Documents - Change Log.

     

    See also:

    • Edit Journal Line

     

    Edit Units

    Click this to edit the Unit Codes related to the journal line. Only the active Unit Codes related to the Account File will be enabled for editing.

     

    See also:

    • General

    • Unit Codes

     

    Edit Journal

    This button is activated only if the selected journal is a draft. Click this to edit the draft, which will be displayed in the Edit Draft Journal screen. When the amendments have been made and posted, it will be posted as another draft.

     

    See also:

    • Draft Journals

     

    Settle

    Click this to settle the journal.

     

    Rebook

    This button will prompt with a screen for input of the new account code which must be of the same 'type' - the same sub-ledger type (and for sub-ledger, the currency must be the same); or if the originating is a nominal posting account, the new one must also be a nominal posting account.

    Note: This works only for draft journals.

     

    AEOI Reporting

    Click this to open the AEOI Reporting screen.

     

    Validate AEOI

    Click this to Validate AEOI. The status will then be displayed in the Regime column in the grid. This button is disable for pending journal.

    After the 'Validate AEOI' is performed, it will display the following in the Regime column:

    • Display ‘X’ if there isn't any transaction linked to CRS or FATCA.

    • Display ‘C’ for CRS, or/and ‘F’ for FATCA, or/and 'G' for UK FATCA.

    • Display ‘!’ if the Percentage is not 100%

Trial Balance

This allows you to build the Trial Balance for the selected Account File. The screen is divided into four folders:

  • Select

    In this folder, choose the period for which you want to view the trial balance.

     

    Select on

    Select whether you want to view the trial balance based on Periods or Dates. Dates can be used for periods not available in the period selection window.

    • Period - This field is enabled if you opt to view the trial balance by period. Click the Browse button to select the required period.

    • Date - from and - to - This field is enabled if you opt to view the trial balance based on selected dates.

     

    Build TB on

    Select the date type on which to build the trial balance.

     

    Show Sub-ledgers for Posting Accounts

    Tick this to display all sub-ledgers that were posted to the selected posting account (account types 1, 2, 3 and 4) where the specified sub-ledger is set to either 'Mandatory' or 'Optional'. If no sub-ledger is specified, '<Not Specified>' is shown. The Amount shown is for the selected period and in the Account File's base currency.

     

    Main Group

    Select the main accounts group that you want to view or leave blank to view all groups.

     

    Organisational Units

    Select a specific unit to build a trial balance for that unit.

     

    Journal Report Option

    Journal Reporting option(s) will be available when the option ‘Include by Default’ is enabled in Journal Reporting Options screen. The generated Trial Balance will display all journal transactions when there is no Journal Reporting option selected.

    Whereas when a Journal Reporting option is selected in the Trial Balance, for example ‘Correction Bookings’, it will display all transactions where ‘Correction Bookings’ is flagged, as well as the transactions that are without any Journal Reporting option flagged. See Journal Reporting Options for details.

     

    Build Trial Balance

    Click the Build Trial Balance button, or click the Trial Balance folder to look at the trial balance.

  • Trial Balance

    The Trial Balance folder displays the trial balance based on the selections made in the Select folder. Select a nominal or sub-ledger account from the list and click the Transactions folder to view the transactions of the account in the selected period.

     

    View

    Select whether you want to view a detailed trial balance or just a summary, with and without Local Account.

     

    Filter on

    Filter the information displayed based on a selected Main group, Sub group or account.

     

    Hide zero balance accounts

    Tick this to hide accounts with zero balances.

     

    Both Cap/Inc

    Filter to view both capital and income balances, just capital balances or just income balances. This field is only active if the Chart of Accounts linked to the selected Account File has been enabled for Capital/Income Segregation, as is required for Trust files.

     

    See also:

    • Capital/Income Segregation

     

    Details

    Click to display information on the selected nominal account or sub-ledger.

     

    Analyses - NA

    Click to display a summary of the selected account within a selected time period.

     

    See also:

    • Sub-Ledger Accounts - Analyses

     

    Analyses - SL

    This is enabled if the selected account has sub-ledgers. Click to display analyses for the selected sub-ledger.

     

    Statement

    Click to print a statement on the account. Enter the details and select the statement you want printed in the Prepare: <Statement name> screen that appears.

     

    Report

    Click to view or print the trial balance in a report format.

     

    Export

    Click to export the trial balance as a SAF-T (Standard Audit File for Tax) file. This is available if you have authority level 8.
