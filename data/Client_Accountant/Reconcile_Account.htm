Reconcile Account

Use this feature to reconcile journal entries against a statement. For instance, to reconcile a bank sub-ledger account against actual statements from the bank, or your investments against the monthly statement from your broker.

  1. Complete the fields in this screen to select the records you want to reconcile:

    • Sub-ledger type and account - Select the sub-ledger type and account.

    • Non-Reconciled Entries - Select this option to display only unreconciled items.

    • Select on Journal Date  - Select this option to display journals based on the data range entered in the From and to fields (see below). This will display all reconciled and unreconciled items.

    • Select on Reconciliation Date - Select this option to display journals that have been reconciled within the date range specified in the From and to fields (see below).

    • From and to - Enter the date range for the journals to be displayed. This fields are mandatory if Select on Journal Date or Select on Reconciliation Date options is being selected.

  2. Click Reconcile.

  3. Enter the reconciliation date for the journals. The Reconciled cell will be shown as 'Yes' to indicate that the journal has been reconciled. To remove the date, double-click on the Reconciled cell and it will revert to 'No'.

  4. Click Refresh to update the list. If you are looking at unreconciled items, this will remove journals that you have reconciled from the list.

  5. For details, valuations and movement of a sub-ledger account, click Analyses.

  6. To return to the selection screen, click Selection.

Note: If the Reconciliation Date is the same as the Journal Date, then you can also double-click on the Reconciled cell. This will update it to 'Yes' with the Reconciliation Date following the Journal Date.
