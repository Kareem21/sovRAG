CAJOBIM1 - Import Opening Balances Journal Sheet

This Excel Statement works as an import sheet so that you can import opening balance for a new Account File. Before you generate the statement, ensure that you do the following:

  • Active the Account File.

  • Configure the Chart of Accounts if necessary.

  • Create all sub-ledger accounts.

If the above has been done, you can generate the statement by doing the following:

  1. Open the Account File.

  2. In the File Maintenance screen, click Reporting Ribbon button > Excel Statements > Import/Work Sheets.

  3. Click CAJOBIM1 and click Generate.

  4. Enter the Opening Date in the Date statement field.

  5. Select Generate the document(s) and:

  6. Browse Location for the generated document(s) to be stored.

  7. Select Open the document(s) in Excel/PDF viewer.

  8. Click Start.

  9. Complete all 'bold' fields like Quantity and Amount in cells that are boxed.

  10. Click Save.

  11. From the Viewpoint menu, click Import Sheet.

  12. Select the row to import and click Import & Post.
