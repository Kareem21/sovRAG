Accounting

If you are keeping accounts for the selected file, you will be able to view this section. For new files, there are two available options to set up the Account File:

  • The Cloning folder allows for Account File setup to be copied from an existing Account File. Cloning will result in the new Account File having the same Chart of Accounts template, general information and Unit Codes of the existing Account File. However, this excludes dates for Last Year-End, posting period, the last journal as well as the draft journal numbers.

    To clone from an existing Account File:

    1. Enter the code or name of the existing Account File.

    2. Enter the date for the new Account File's Last Year-End.

    3. Click Clone.

    4. When complete, click Close.

  • The Setup folder to manually set up the Account File.
