Recode Subaccount/Sub-ledger

Subaccount/Sub-ledgers created for an Account File can be recoded without having to re-enter the details of the subaccount/sub-ledger. A recoding is normally done in the event of inappropriate naming of the subaccount/sub-ledger, during restructuring or to avoid misrepresentation.

  1. Enter the necessary information:

    • Subaccount/Sub-ledger type - Select Subaccount/Sub-ledger type.

    • Subaccount/Sub-ledger to recode- Select the Subaccount/Sub-ledger to recode.

    • New Subaccount/Sub-ledger type - This reflects the Subaccount/Sub-ledger type selected and cannot be changed.

    • New Subaccount/Sub-ledger code - Enter the new Subaccount/Sub-ledger code.

    • Nominal Account - Select the control account from the list of nominal control accounts, if this needs to be changed as well.

  2. Click Recode to begin recoding.
