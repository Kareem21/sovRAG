Approval Cards for Payees and Payments

Approval cards contain the rules for payee and bank payment approvals. These approval cards can be set up at various approval levels:

  • Global - The approval card applies to all Master Files.

  • Master File - The approval card applies to the named Master File.

  • Sub-ledger Account - The approval card applies to the named sub-ledger account in the Master File. This approval level applies to the bank payment approval cards only.

The approval level represents the hierarchy which will be used when applying the approval cards. Approval cards on the sub-ledger account level will be applied before those on the Master File and Global levels. Approval cards on the Master File level will be applied before those on the Global level.

  4. All other Account Files will use Payee card no. 2.

When approving bank payments:

  1. All INVEST payments up to EUR10,000 (Payment ID 20) will use Payment card no. 3.

  2. All INVEST payments more than EUR10,000 (Payment ID 21) will use Payment card no. 4.

  3. All INVEST payments NOT using EUR (Payment ID 22) will use Payment card no. 4.

  4. All other Account Files will use Payment card no. 4.

Approval cards are set up in Configuration > Users & Security > Approval Cards.
