Portfolio Valuation

Portfolio valuations are done to value all assets and liabilities included in a selected portfolio for the purpose of management reporting. The portfolio currency is used as the base for the valuation. No journals are posted when a portfolio is valued.

When you select Portfolio Valuation, you will see:

  • Select - Allows you to create new valuation or locate existing ones.

Once you choose to create a new valuation or view a previously saved one, you will then see:

  • Valuation - Displays the valuation details.
