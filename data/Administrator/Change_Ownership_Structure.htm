Change of Ownership Structure

Change of Ownership Structure may occur due to change in legislation or the company is converted to another entity type. For either scenario, this will affect the ownership structure of its existing share classes as the entity type determines the available structure resulting in:

  • only one ownership structure for all share classes (when ownership structure for the entity type is set to SA, SB, SN or SS)

  • a choice of two ownership structure for each share class (when ownership structure for the entity type is set to XA, XB, XN or XS)

  • a choice of four ownership structure for each share class (when ownership structure for the entity type is set to XC)

When the legislation is changed for an entity type, then you need to update the ownership structure value for the entity type. If the company is converted to another entity type with a different ownership structure, then you need to change the entity type accordingly in the company’s File Maintenance screen.

Once that is done, use the Capital Change function and the Share Structure Explorer to update the existing share class(es) for the affected company or companies.

 

In Configuration

 

Update Ownership Structure for Entity Type

  1. In Viewpoint – Home, go to Configuration > Business Parameters > Entity Administration > Entity Types.

  2. Select the affected entity type.

  3. Click Edit.

  4. Update the Ownership Structure accordingly.

  5. Click Update.

Note: If the change is from an ownership structure that starts with 'S' to another one that starts with 'S', in step 4, select an ownership structure that allows both ownership structures first. Once you have performed the changes in File Maintenance for the affected companies, then set the ownership structure correctly.

For example, ownership structure change from 'SA' to 'SB' - select 'XC' in step 4 to allow the changes to be updated to existing share class of affected companies. After that is completed, set the ownership structure to 'SB'.

 

In File Maintenance

Update Company’s Entity Type

  1. In Viewpoint – Home, open the company’s File Maintenance screen.

  2. Go to Entity Administration and select the General screen.

  3. Change the Entity Type.

  4. Click Update.

 

Perform Capital Change

This is used to correct the Ownership Structure of existing share class(es) and also Nominal Value if migrating from Par Value to No Par Value regime.

For the affected companies, you must perform a capital change for all relevant share class.

  1. Open the File Maintenance screen of the company.

  2. Go to Shares > Capital screen.

  3. Select the share class and click Capital Change.

  4. In the Options tab in the Capital Change screen:

    1. Select the Create new share class, cancel existing shares and issue new shares ('Capital Change') options.

    2. Select the new ownership structure for the New Ownership Structure field.

    3. Update details for Nominal Value where, if moving to No Par Value, set Nominal Value to 0. Note that this will update the Consideration/Share and Paid-up/Share value for all share transactions.

  5. Click Next.

  6. In the Transactions tab, check the values in white cells and correct them if required. For example, updating Nominal Value to 0 earlier will also result in Consideration/Share and Paid-up/Share columns being set to 0.

  7. Click Next.

  8. In the Summary tab, under 'After Change', you will see the existing share class which will be cancelled and the new share class that will be created with the relevant details.

  9. Click Update.

  10. Click Yes when prompted.

  11. Click OK.

 

Using Share Structure Explorer

For entity types where Authorised Capital has been abolished, you can remove the existing Number of Shares entered for the Share Class using Share Structure Explorer. This requires access to the Explore command button in Shares > Transactions screen.

  1. In Shares > Transactions screen, select the affected share class.

  2. Click Explore.

  3. In the Structure Explorer screen, select the Authorisation record for the share class and click Edit.

  4. Change the values for the following to 0:

    • Number of Shares

    • Balance of Shares

  5. Enter the reason for the change in the Note field to have it appear in the Conditions field for the share class. Alternatively, you may also add this at a later time by selecting the share class in Shares > Capital and clicking Edit Class.

  6. Click Update.

  7. Click Rebuild.
