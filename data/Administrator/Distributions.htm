Distributions

The distributions for a trust beneficiary are recorded here.

 

Distribution list

This shows all distributions that have been added.

 

Command buttons

Click the relevant command button to perform the necessary.

  • New - Click this to add distributions.

    Enter the details of the distribution. Click Update when done.

  • Edit - Click this to edit details of the selected record.

    1. Edit the necessary details.

    2. Click Update after you have completed the editing.

  • Delete - Click this to delete the selected distribution.

    Click Yes to confirm the deletion.

  • Payment (CA) - Click this to create a payment for the selected distribution. If the beneficiary/payee has not been set up as a payee, you will be asked if you want to create the payee record. This button is active only if the selected distribution has a value in the Payee field. To use this, the Entity File must be set up as an active Account File.

    1. Click Yes to create the payee.

    2. Enter information on the new payee:

      • Payee Master File - This defaults to the Master File in the Beneficiary field (if the Payee field is empty) or the Master File in the Payee field of the selected Distribution record.

      • Active - This check box is automatically ticked when you create a new payee. Remove the tick to inactivate the payee.

      • Payee Name - The name of the Master File selected above defaults to this field. Change this if necessary.

      • Address 1, Address 2, Address 3 and Country - The payee's business address defaults to these fields. Change if necessary.

      • Bank Master File - Select the Bank Master File to which you will be making payments.

      • Bank Name - The name of the Bank Master File selected above defaults to this field. Change this if necessary.

      • Address 1, Address 2, Address 3 and Country - The bank's business address defaults to these fields. Change if necessary.

      • Bank Account No - Enter the bank account number to which you will be making payments.

      • Bank ID/Sort Code - Enter the bank ID/sort code for the selected bank account.

      • Payment CCY - Select the payment currency.

      • Maximum Amount - Enter the maximum amount per payment transaction. This amount will be validated against subsequent payments made to this payee.

      • Note - Enter any notes relating to the payee.

      • CDD Classification - Enter a CDD classification for this payee. Note that this is NOT the same as the Risk Level assigned to the payee in Administrator.

      • CDD Rating Score - Enter a CDD score.

      • CDD Justification - Enter an explanation for the CDD Classification or CDD Rating Score given.

    3. Click Update.

    Note: Once created, the payee must be approved.

    Enter information on the new payment. Click Update when done.

  • AEOI Reporting - Click this to open the AEOI Reporting function.

  • Validate AEOI - Click this to Validate AEOI. The status will then be displayed in the Regime column in the grid.

    After the 'Validate AEOI' is performed, it will display the following in the Regime column:

    • Display ‘X’ if there isn't any transaction linked to CRS or FATCA.

    • Display ‘C’ for CRS, or/and ‘F’ for FATCA, or/and 'G' for UK FATCA.

    • Display ‘!’ if the Percentage is not 100%

 

Why is there no exchange rate value?

Check with your System Manager. The exchange rate for the selected currency and reporting currency may not have been set up in the Administrator/Practice Manager Exchange Rates table in Home > Entity Administration > Exchange Rates (AD/PM) or Home > Practice Management > Exchange Rates (AD/PM).
