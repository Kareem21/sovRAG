Share Structure Explorer

The Share Structure Explorer function allows for changes to be made to posted share transactions. It can be accessed by clicking the Explore button in the following:

  • Shares > Capital in File Maintenance screen.

  • Shares > Transactions in File Maintenance screen.

The ability to edit and delete share transactions is possible only if you access Share Structure Explorer through Shares > Transactions. From Shares > Capital, you can only view the details of posted transactions. When you open Share Structure Explorer, you see the following:

  • Ribbon

    Home tab

    General Group

    This group contains the following:

    • Windows Manager - Click to open Windows Manager.

    • Notifications - Click to open the Notifications window.

    • Go to Home - Click to return to Viewpoint - Home screen.

    Functions Group

    The functions here apply to the share transactions of the selected Master File.

    • Edit - Click to edit the selected share transaction. This is available only if you access Share Structure Explorer from Shares >Transactions screen.

    • Delete - Click to delete the selected share transaction. This is available only if you access Share Structure Explorer from Shares >Transactions screen.

    • Delete Branch - Click to delete the selected share transaction branch. This is available only if you access Share Structure Explorer from Shares >Transactions screen.

    • Rebuild DN - Click this to correct Distinctive Numbers if there is inconsistency of Distinctive Numbers in share register. This applies only to system-generated Distinctive Numbers.

    • Rebuild - Click to rebuild the share register.

    Display Group

    The functions here apply to the share transactions of the selected Master File.

    • Refresh - Click to refresh the display.

    • Grid Columns - Click to open Options. Tick the additional information to include in Share Structure Explorer.

    • Linked Interests - Click to display interest holders linked to the selected share class. The information is from the Interests screen and details include the interest type, the name of the interest holder, the Interest Sequence Number, Effective Date of when the interest holder obtained the interest and Cancel Date which is when the interest holder ceased to be. These will be read-only.

    • View Details - Click to view details of the selected transaction.

    Tools Group

    The options here allow you to copy, print and find the information in the screen.

    • Copy - Click to copy the information in the screen.

    • Print - Click to preview and print the information in the screen.

    • Find - Click to find the information in the screen.

    Opened Master File tab

    [Opened Master File] - contains additional function buttons that allows you to access screens and functions for the opened Master File.

    General Group

    • Windows Manager - Click to open Windows Manager.

    • Go to Home - Click to return to Viewpoint - Home screen.

    • Open File - Click to open the File Maintenance screen for the selected Master File.

     

    Create Group

    • Workflow - Click to start a new workflow for the selected Master File.

    • File Note - Click to create a file note for the selected Master File. See also File Notes.

    • Task - Click to add a task for the selected Master File. See also Tasks.

    • Time Entry - Click to add a time entry for the selected Master File. See also Time Entries.

    • Document Holder - Click to create a document holder for the selected Master File. See also Document Holder.

    • Binder - Click to create a binder for the selected Master File. See also Create New Binder.

    • Correspondence - Click to create a correspondence document for the selected Master File. See also Correspondence.

    • Journal QE - Click to open the Journal Quick Entry screen for the selected Master File. See also Journal Quick Entry.

     
    Enquiry Group

    • File - Click to open the File Enquiry screen for the selected Master File.

    • Relations - Click to access the Relations enquiry screen for the selected Master File.

    • Ownership - Click to access the Ownership enquiry screen for the selected Master File.

    • Billing - Click to access the Billing enquiry screen for the selected Master File.

    • Accounts - Click to access the Account enquiry screen for the selected Master File.

    • Documents - Click to view stored documents for the selected Master File. This will launch the Documents screen.

    • Changes - Click to access the Changes screen for the selected Master File.

    Add To Group

    • Outlook - Click to add the selected Master File to the Outlook Correspondence folder.

    • Favourites - Click to add the Master File to the Favourites list.

  • List of share transactions

    Share transactions for the selected Master File are listed here. To select, click on the transaction record.

Note that the button must first be enabled and the user has authority level 8 - Power User. Usage of this function to edit share transactions does not protect the data integrity and may corrupt your share register if changes are made incorrectly.

WARNING: Do not use this function unless you are knowledgeable of the share transaction structure and constraints. You are responsible for the changes made.
