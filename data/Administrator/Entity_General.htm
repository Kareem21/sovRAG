General

This screen captures the basic information of an Entity File and must be completed as part of setting up the Entity File. The fields seen in this screen are:

  • Standard fields - These are standard Viewpoint fields that can be activated or deactivated for use. Among these fields are:

    • Fields that determine screens for the Entity File:

      • Jurisdiction - This must be specified before you can select the Entity Type below.

      • Entity Type - The Entity Type determines the screens for the Entity File and the options you see here depends on the jurisdiction.

    • Fields that are duplicates of fields in other screens:

      • UserCombo3 and UserText2 - also in ...Continued screen.

      • IncorpDate and IncorpNr - also in Statutory - Company and Statutory - Trust screens.

      • IncorpText - also in M&A/Dates screen.

  • Custom fields - these are fields that can be added to the screen. Refer to Custom Fields for more information.
