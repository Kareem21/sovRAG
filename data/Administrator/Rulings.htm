Rulings

This screen allows you to enter rules or taxes that apply to the company. By default, this screen is not active.

 

Rulings list

This shows the rulings for the entity.

 

Command buttons

Click the relevant command button to perform the necessary.

  • New - Click this to add new ruling.

    Enter the details for the ruling. Click Update when done.

  • Edit - Click this to edit selected ruling record.

  • Delete - Click this to delete selected ruling record.
