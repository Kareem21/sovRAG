Fee Notes

The Fee Notes screen lists all fee notes that have been created and can be accessed from:

  • Viewpoint - Home screen - Click Home > Entity Administration > Annual Fees > Fee Notes.

  • File Maintenance screen - Click Annual Fees > Fee Notes.

Click this to view the Selection Setup to filter the records in the grid.

  1. In the Selection Setup, select your option from:

    • Favourites - Click the desired option.

      The following option is displayed when the condition is met:

      Name

      Condition

      Me

      This is displayed in the list by default.

      I am the Viewpoint Users' Manager

      If the login user is specified as the manager of other user(s).

      My primary Administrator Group

      If the login user's Group Access is greater or equal to 'Files where Administrator and in Group[2]' in Home > Configuration > Users & Security > Users.

      All my Administrator Groups

      If the login user's Group Access is greater or equal to 'Files where Administrator, in Group + Group Member[3]' in Home > Configuration > Users & Security > Users.

      My User Group

      If the login user's Group Access is equal to 'All Files[4]' in Home > Configuration > Users & Security > Users.

      All Users

      If the login user's Group Access is equal to 'All Files[4]' in Home > Configuration > Users & Security > Users.

    • Users - Tick to select the user(s).

    • Workgroups - Tick to select the workgroup(s).

    • Administrator Groups - Tick to select the Administrator Group(s).

    • User Groups - Tick to select the User Group(s).

    • Organisational Units - Tick to select the Organisational Unit(s).

    • User Relations- Tick to select the user(s).

  2. Tick Apply to all screens if you want your selection to be applied to all other screens in Viewpoint - Home screen.

  3. Click Select.

Note: The available tabs depend on the File Access set for the user account in Configuration > Users & Security > Users.

 

This is seen only in Viewpoint - Home screen.

Fee note list

This shows all the fee notes that have been created.

 

Command buttons

Click on the relevant command button to perform the necessary.

  • Settle/Edit - Click this to settle the selected Fee Note.

    1. Enter the settlement details:

      • Fee Note Total - This shows the currency and amount for the Fee Note and cannot be edited.

      • Date Paid - This defaults to today's date but can be changed to reflect the date payment was made.

      • Amount Paid - Enter the payment amount.

      • Written Off - Enter the amount written off, if applicable.

      • Outstanding - This shows the amount outstanding, if any.

      • Reference - Enter a reference for the settlement.

      • Note - Enter additional notes about the settlement.

      • System Warning - This displays warning from the system relating to the Fee Note.

    2. Click Update.

  • Print Receipt - Click this  to print receipt for payments made.

    The template for Receipt [ANNRECEIPT] can be found in Configuration > Document Libraries > Billing Templates where the Document Type is Debit/Credit Note [DC].

    Select the generation option.

    The options that are available depend on the document template:

    • Template with e-mail tags

      Deliver to the Document Queue

      Select this to send the report to Document Queue.

       

      Create an Outlook e-mail and: <option>

      This allows you to generate the document as an e-mail message and:

      • Send e-mail straightaway

      • Place the e-mail in Outlook's Drafts folder

      • Open the e-mail message

       

      View and/or Edit the generated document(s) in Word

      Select this to generate the document and open it in Microsoft Word program.

       

      Remove all e-mail tags and e-mail body from generated document(s)

      Select this to remove e-mail tags and section in the generated Word document. This function will be disabled if the 'Convert e-mail formats and...' or 'Add the document(s) to the E-mail Bin' options are ticked.

       

    • Template with Delivery Manager task

      Deliver to the Document Queue

      The document needs to be generated from the Document Queue screen.

       

    • Template without e-mail tags and Delivery Manager task

      Deliver to the Document Queue

      Select this to send the report to Document Queue.

       

      Generate the document and: <option>

      This allows you to generate the document and have the program:

      • Convert it to PDF format

      • Open the document for viewing

      • Add the document to E-mail Bin

      • Copy the document

      • Open the folder to where the document is sent to (Define this in Location)

       

      Print the generated document to the default printer

      Select this to send the document to be printed.

    See also:

    • Document Queue
