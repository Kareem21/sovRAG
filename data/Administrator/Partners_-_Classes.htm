Classes

This section is where you create different partnership classes. Classes can be created to differentiate the levels of partners that may exist within a partnership.

There are two sections in this screen and you can click on:

  • - To minimise this section and hide the command buttons. In the grid, only the selected row will be seen.

  • - To expand the section and show the command buttons.

 

Partnership Class list

This shows partnership classes set up.

 

Command buttons

Click on the relevant command button to perform the necessary.

  • New Class - Click to create a new partnership class.

    1. In Classes section, click New Class.

    2. Enter the details of the class.

    3. Click Update.

     

  • Edit Class - Click to edit the selected class.

    1. Edit the necessary details.

    2. Click Update after you have completed the editing.

    If you tick the option 'Freeze Share Class', this will disallow further share transactions on the selected share class as the following command buttons in these screens will be disabled:

    • Capital

      • Increase/Decrease

      • Capital Change

      • Issue

      • User Fields

    • Posting

      • Post

      • Delete

      • Ben. Owner

    • Shareholders

      • Transfer

      • Edit Cert.

      • Ben. Owner

      • Call(s) Paid

      • Redeem

      • Cancel Cert.

      • Forfeit

    • Transactions

      • Reverse

      • Edit

      • Explore

  • User Fields - This section allows you to retrieve additional fields to record specific information for the partnership class.

 

Partnership Class Transaction

This shows transactions for the selected class.
