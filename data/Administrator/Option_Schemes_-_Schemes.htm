Schemes

This section is where you will first create the option scheme before recording all related grants and certificates. There are two sections in this screen and you can click on:

  • - To minimise this section and hide the command buttons. In the grid, only the selected row will be seen.

  • - To expand the section and show the command buttons.

 

Option scheme list

This lists the different schemes that have been issued.

 

Command buttons

These apply to the option scheme list. Click on the relevant command button to perform the necessary.

  • New - Click this to enter a new option scheme.

    Enter the details for the scheme. Click Update when done.

  • Edit - Click this to edit details of an existing scheme.

    1. Edit the necessary details.

    2. Click Update after you have completed the editing.

  • Increase - Click this to increase the authorised number of options for the scheme.

    Enter the information for the increase. Click Update when done.

  • Delete - Click this to delete the selected scheme.

    Click Yes to confirm the deletion.

  • Close - Click this to close the selected scheme.

    1. Enter the effective date for the event.

    2. Click Update.

 

Transaction table

This lists the transactions for the selected option scheme.

 

Command buttons

These apply to the transactions. Click on the relevant command button to perform the necessary.

  • Confirm - Click this to enter the effective date for an outstanding transaction. If you have selected a confirmed transaction, the button will be disabled. You can also confirm outstanding transaction in Transactions section.

  • Edit Note - Click this to edit the note related to the transaction.

  • Undo - Click this to undo the transaction. Transactions can only be undone from the most recent event backwards.
