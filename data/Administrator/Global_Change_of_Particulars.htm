Global Change of Particulars

When the particulars of an individual or entity are changed, statutory filings may be required for companies/trusts/partnerships where the individual or entity is linked in a specific capacity. The ability to trigger the relevant forms for the filings is done through the Global Change of Particulars screen.

This screen will appear when the change takes place and you indicate that the statutory filings are required. The types of changes to the individual or entity’s identity and contact information are recorded in these screens:

  • File Maintenance > General > Master File > [Master File] > Setup

  • File Maintenance > General > Master File > [Master File] > General

  • File Maintenance > General > Master File > [Master File] > Addresses

  • File Maintenance > General > Master File > [Master File] > Compliance > Identity Register

  • File Maintenance > General > Master File > [Master File] > Compliance > Identity Details

Depending on the options ticked in System Defaults - Entity/Forms, the screen will appear if the individual/entity is:

  • appointed as a Statutory/Trust/Foundation Officer

  • appointed as an Entity Relation

  • a shareholder/partner

  • a beneficiary

  • an Interest holder

  • an Owner or Controller

The screen allows you to specify certain parameters that will be used to retrieve the list of related Master Files where statutory filings are required. In the screen, there are two tabs:

  • Settings - define the parameters through:

    • Change applies to Statutory Officers only - tick this to include only entities where the Master File is appointed as a Statutory Officer

    • Master File type - tick the Master File types to include

    • Entity status - tick the entity statuses to include

    • Form Selection - select Automatic if you want all related forms for the change to be sent to Form Queue. Select Prompt if you want to make the selection manually.

  • Selection - shows the results based on the parameters defined in the Settings tab. You can select the Master Files that require statutory filings by ticking the checkbox of the record; or click

    • Select All - to select all records

    • Deselect All - to deselect all records

After the selection is made, click Start. If Form Selection is set to Prompt, you will see a list of relevant forms for each Master File that you have selected.
