Filing Cabinet for Master File

This screen is available if you have ticked the 'Archive by Master File' option in Configuration > Settings > System Defaults > General/Master File. It is generally used in place of the Document Management component and serves as an alternative document management option for Viewpoint.

To access this screen, you can:

  • Click Documents from the File Maintenance Ribbon tab.

  • In Viewpoint - Home > To Do > Master File > Search screen, right-click on a Master File and select Filing Cabinets.

  • In Viewpoint - Home screen, click Enquiry > Master File > Documents.

  • In File Maintenance screen, click Enquiry & Reporting > Enquiry > Documents.

By default, only generated forms from Viewpoint will be automatically saved to the Filing Cabinet under the folder Forms Generated. However, it is possible to add other documents using:

  • Import - Click to import other documents to a folder in the Filing Cabinet.

    1. Click the folder for the document.

    2. Click Import, go to the location containing the document you want to import.

    3. Select the document.

    4. Click Open.

     

  • Scan - Click to scan and save to a folder in the Filing Cabinet.

    1. Load all pages of the document on your scanner.

    2. Click the folder for the document.

    3. Type name for the scanned document.

    4. Click OK.

    5. In Viewpoint.Win Scanner screen, select the scanner to use.

    6. Set the scan settings.

    7. Click Scan.

    8. When complete, click More Page(s) to scan more document or click Complete to exit the scan.

     

 

Adding Folder to the Filing Cabinet

  1. Right-click on Filing Cabinet.

  2. Select New.

  3. Type name for the new folder.

  4. Click OK.
