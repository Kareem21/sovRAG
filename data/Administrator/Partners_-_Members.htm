Members

This section allows you to assign members to the partnership class and record the contributions of each partner. You can filter by the partnership class as well as past and current members.

There are two sections in this screen and you can click on:

  • - To minimise this section and hide the command buttons. In the grid, only the selected row will be seen.

  • - To expand the section and show the command buttons.

 

Filters

 You can filter the list by selecting the partnership class and/or listing type - Current Holdings, Past Holdings or All.

 

Members list

This table lists members for each partnership class.

 

Transactions

This lists details of all transactions that have occurred for a selected member.

 

Command buttons

Click on the relevant command button to perform the necessary. You need to first select a member.

  • New Member - Click this to add a new member to the selected partnership class.

    1. Select the partnership class.

    2. Click New Member.

    3. Enter details for the new partner:

      • Class - This defaults to the selected class. If you have selected the wrong class, click Cancel.

      • Description - This defaults to the Member New [MN] but you can change the description.

        More descriptions can be added in Partners: Transaction Types [S_PT] lookup value table.

      • New Member - Select the Master File for the new partner.

      • Designated Member? - Tick this to indicate if the partner is a designated member. In certain jurisdictions, like in the UK, designated members hold extra responsibilities such as appointing an auditor, signing accounts on behalf of members, and so on.

      • Transaction Date - This defaults to today's date but you can change this if necessary.

      • Effective Date - Enter the effective date for the partner's addition.

      • Contribution Promised - Enter the amount promised by the partner.

      • Contribution Made - Enter the contribution made by the partner at the time of the addition.

      • Paid - Select the mode of payment made by the partner.

      • Certificate No. - If you are using certificates for the partnership class, this defaults to the first available number.

      • Location - Enter the location of the certificate or the partner.

      • Note - Enter any relevant notes pertaining to the addition of the partner.

    4. Click Update

     

  • Contribute - Click this to increase contribution promised or made by a partner.

    This is possible if the appointment of the partner has been posted with an effective date.

    1. In Members section, select the partner.

    2. Click Contribute.

    3. Enter the details for the increase in contribution:

      • Description - This defaults to Contribution Increased [CI] but you can change the description.

        More descriptions can be added in Partners: Transaction Types [S_PT] lookup value table.

      • Transaction Date - This defaults to today's date.

      • Effective Date - Enter the date the transaction will take effect. The total amount of increase will not be reflected in the members list until this date is entered.

      • Contribution Promised - This displays the total contribution promised by the partner.

        • increased by - If the member is increasing the amount of contribution promised, enter the amount of increment here.

        • to - The total contribution promised after the increase is calculated by the system and displayed here.

      • Contribution Made - This displays the total contribution paid up to date by the partner.

        • increased by - Enter the increase in the contribution made by the partner.

        • to - The total contribution paid up after the increase is calculated by the system and displayed here.

      • Paid - Select the mode of payment.

      • Certificate No. - This will default to the first available number if you are using certificate number for the partnership class.

      • Location - Enter the location of the certificate or partner, if applicable.

      • Note - Enter any relevant notes for the increase in contribution.

    4. Click Update.

  • Return - Click this to decrease contribution promised or made by a partner.

    1. In Members section, select the partner.

    2. Click Return.

    3. Enter the details for the increase in contribution:

      • Description - This defaults to Contribution Returned [CR] but you can change the description.

        More descriptions can be added in Partners: Transaction Types [S_PT] lookup value table.

      • Transaction Date - This defaults to today's date.

      • Effective Date - Enter the date the transaction will take effect. The total amount of increase will not be reflected in the members list until this date is entered.

      • Contribution Promised - This displays the total contribution promised by the partner.

        • reduced by - If the member is decreasing the amount of contribution promised, enter the amount to be reduced by here.

        • remainder - The total contribution promised after the decrease is calculated by the system and displayed here.

      • Contribution Made - This displays the total contribution paid up to date by the partner.

        • reduced by - Enter the decrease in the contribution made by the partner.

        • remainder- The total contribution paid up after the decrease is calculated by the system and displayed here.

      • Paid - Select the mode of payment for the return.

      • Certificate No. - This will default to the first available number if you are using certificate number for the partnership class.

      • Location - Enter the location of the certificate or partner, if applicable.

      • Note - Enter any relevant notes for the decrease in contribution.

    4. Click Update.

     

  • Transfer - Click this to transfer a partnership to another member.

    1. Enter the relevant information for the transfer.

    2. Click Update.

    3. Tick the form(s) you require and click Queue.

    Note: The remainder for Contribution Promised must be more than the remainder for Contribution Made.

  • Cancel  - Click this to cancel the selected contribution.

    Enter the details for the cancellation. Click Update when done.
