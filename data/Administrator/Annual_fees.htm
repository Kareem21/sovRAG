Annual Fees

This feature is an additional option and is only available if included in your licence.

 

The Annual Fees feature allows for procedures such as storing client billing details, generating multi-currency annual fee invoices, formulating lump sum or itemised billing and generating annual fee invoice reminders. This section can be used to bill clients if you are not using the Practice Management component.

To use this, you need to enable the following screens:

  • File Setup

  • Fees Setup

  • Fees Due

  • Fee Notes
