Global Change of Registered Office - Entity

This is to change addresses linked to the Entity File in Statutory - Company. It is available in Viewpoint - Home screen. To access it:

  • Using Ribbon - Click EA Global Ribbon button, select Global Change > Registered Office Address - Entity

  • Using Section Menu - Click Home > Entity Administration >  Global Change > Registered Office Address - Entity

 

When accessing the function, you will see the following:

  • Settings tab - This is to specify the Address Cards and change parameters.

  • Selection tab - This will show results based on the parameters set.

 

Performing Global Change of Registered Office - Entity

  1. Access the function in Viewpoint - Home screen.

  2. Enter the resignation/replacement information in the Settings tab:

    • Current Address Card - Select the address card that needs to be replaced.

    • New Address Card - Select the new address card.

    • Effective Date - This defaults to today's date and refers to the effective date of the change. This date will appear on the appropriate forms.

    • Include location of Registers held at the above address - Tick to include location of all statutory records.

    • Form Selection - Select Automatic if you want all related forms for the change to be sent to Form Queue. Select Prompt if you want to make the selection manually.

    • Include Master File type(s) - Tick the Master File types to apply the change to.

    • Include Entity Status(es) - Tick the Entity statuses to apply the change to.

  3. Click Next.

  4. In Selection tab, ensure that the change you want to perform is ticked. You can click:

    • Select All - To tick all records. This is the default.

    • Deselect All - To untick all records.

  5. Click Start.
