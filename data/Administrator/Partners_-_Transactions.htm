Transactions

In this section, you will be able to see the transactions for all existing partnership classes and make certain corrections if required.

There are two sections in this screen and you can click on:

  • - To minimise this section and hide the command buttons. In the grid, only the selected row will be seen.

  • - To expand the section and show the command buttons.

 

Partnership Class

This lists the partnership classes.

 

Transactions

This lists details of all transactions that have occurred for a selected class.

 

Command buttons

Click on the relevant command button to perform the necessary.

  • Reverse - Click this to reverse the selected transaction.

    Once you have selected the transaction and clicked Reverse, you need to click Yes to continue or No to cancel.

    You are advised to reverse from last entered transaction (Last In First Out order) until you reach the incorrect transaction that you want to reverse.

     

    Note: For transactions that are related e.g. partial transfer resulting in 'Transfer' and 'New Certificate' transactions, reverse the 'Transfer' transaction and the related 'New Certificate' transaction will be reversed as well.

  • Rebuild - Click this to rebuild the partnership register. Once you have clicked Rebuild, the system will initiate the process. Click OK when the process is complete This can be used to correct certain data inconsistencies.
    However, for errors that cannot be corrected, you will see these displayed in the Share Transaction Analysis and Rebuild screen. In this screen, are the following buttons:

    • Open Log - This displays the log file for the rebuild.

    • Print Report - This enables you to print the log.

    • OK - Click to close the screen.

  • Edit - Click this to change description of the transaction or add/edit transaction notes.
