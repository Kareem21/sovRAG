Relations - Entity

The relations set up here are those that are connected to the selected Entity File and are not Statutory Officers. For example, the law firm of an entity would be recorded here and not in the Statutory Officers screen.

If you have enabled detailed relationship roles in the Setup folder, then you can only appoint Master Files that has been allowed to hold a certain role to that position. For example, if Charles Brown has been allowed to hold only Personal Assistant roles, he cannot be assigned to the role of Employer. Detailed relationship roles are set on or off in Configuration.

Like Master File relations, entity relations can also have reciprocal relations.

 

Relation status

Use this filter to hide or show active/resigned/proposed or all relations.

 

Relations list

This shows the relations added for the Entity File.

 

Command buttons

Click the relevant command button to perform the necessary.

  • Appoint - Click this to add a new relation.

    1. Select the Relation Type from the list.

    2. Enter the relevant information. The required information will depend on the relation type.

    3. Click Update to save the entry.

    4. You will be informed of the reciprocal relation, if any, that is created for the appointed Master File.

    Note: The relation type may require selection of a Master File or user depending on the configuration of the relation type.

  • Resign - Click this to resign the selected relation. This button is disabled if there are no relations listed.

    1. Enter the required information.

    2. Click Update to confirm the resignation. Any reciprocal relation will be resigned as well.

    Note: You may have additional fields depending on the set up of the relationship type.

  • Re-elect - Click this to re-elect the selected relation. This button is disabled if there are no relations listed; and if re-election is not enabled for the relationship type, you will get a message informing you of this.

    1. Enter the required information.

    2. Click Update to save the changes.

  • Confirm - Click this to Confirm the appointment. This will update the status from Proposed to Appointed. The command button is available only if you filter by Proposed status.

  • Edit - Click this to edit the relationship details of the selected relation. This button is disabled if there are no relations listed.

  • Delete - Click this to delete the selected relation. Any reciprocal relation will be removed as well. This button is disabled if there are no relations listed.

  • Move Up - Click this to move the selected relation up the list. The order of relations in the list can be reflected in reports.

  • Move Down - Click this to move the selected relation down the list. The order of relations in the list can be reflected in reports.
