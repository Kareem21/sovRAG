Meeting Attendance

In this screen, you can first create the attendance list, record attendees' response and attendance of the meeting. The screen is divided into two.

 

Attendance Type List

This is seen when the option 'Show Attendance Type' is ticked. The Attendance Type follows the type selected for the Meeting Type. You can tick the Master File listed to add them to the meeting's Attendee List.

If the 'Show Attendance Type' option is not ticked, this list is not displayed.

Note: If the Attendance Type for the meeting is set to 'Any (Master File) [ANY]' or left blank, this list will be empty.

 

Attendee List

This list can be populated by the Attendance Type List or by clicking Add button. For each attendee record, you can specify:

  • Position - If the attendee is from the Attendance Type List, this will follow the attendee's position. Otherwise, this will be blank but you can double-click to enter the position.

  • Master File Name - This is the name of the selected Master File for the attendee.

  • Status - Click to select the status of the attendee. The options are maintained in Meeting Attendee Type [MTAT] lookup table.

  • Represented by - If applicable, select who will be representing the attendee. Note that if status is 'Proxy Representation', Represented By can be an individual or non-individual Master File.

  • Confirm - Tick to indicate confirmation of attendance.

  • Note - Enter additional notes relating to the attendee.

 

Command buttons

Click on the relevant command button to perform the necessary. The buttons apply to the Attendee List section.

  • Add - Click this to add new attendee.

  • Remove - Click this to remove selected attendee. To select an attendee, tick the checkbox for the attendee.

  • Update - Click this to save changes made in this screen.

  • Cancel - Click this to cancel changes made and exit the screen.
