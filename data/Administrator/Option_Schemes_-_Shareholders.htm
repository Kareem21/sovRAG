Shareholders

The Shareholders folder lists all members for all the schemes that have been issued including past members. There are two sections in this screen and you can click on:

  • - To minimise this section and hide the command buttons. In the grid, only the selected row will be seen.

  • - To expand the section and show the command buttons.

Filters

Use the filter to select the list of members you want to view by selecting the scheme and/or status of the member.

 

Option Class Members list

This table lists members for the selected option scheme and status.

 

Transactions

This lists all transactions for the selected member.
