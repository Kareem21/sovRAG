Issue and Tranches

The Issue and Tranches screen contains two tables - Issue table and Tranches table and you can click on:

  • - To minimise this section and hide the command buttons. In the grid, only the selected row will be seen.

  • - To expand the section and show the command buttons.

 

Filter

You can filter the issue list by its status.

 

Issue list

This lists all issues by the company and can be filtered by the status.

 

Command buttons

Click the relevant command button to perform the necessary.

  • New - Click this to add new issue.

    Enter the details for the issue. Click Update when done.

  • Edit - Click this to edit selected issue record.

  • Delete - Click this to delete selected issue record.

 

Tranches list

This lists the tranches for a selected issue.

 

Command buttons

Click the relevant command button to perform the necessary.

  • New - Click this to add a new tranche.

    Enter the details for the tranche. Click Update when done.

  • Edit - Click this to edit selected tranche record.

  • Delete - Click this to delete selected tranche record.
