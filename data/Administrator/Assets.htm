Assets

You can record assets under any one of the following:

  • Bank/Broker Accounts - Information on an entity's bank/broker accounts and transactions are recorded here. Bank/broker accounts entered through Bank Accounts & Signatories will be reflected here and vice versa.

  • Assets - Information on an entity's fixed assets and its procurement or disposal are recorded here.

    Details for fixed assets are as follows:

    • Reference - Enter the asset reference. This should be enabled and set to Mandatory to allow for User Fields - Fixed Assets to be linked to the right fixed asset record. Note that this field should be unique.

    • Asset Type - Select the asset type. The types are maintained in the Fixed Asset Types [FATP] lookup table.

    • Asset Sub Type - Select the asset sub-type. The sub-types are maintained in the Fixed Asset Sub Types [FAST] lookup table.

    • Description - Enter a description of the asset.

    • Related Party - This field can be used to link the record to related Master File.

    • Location of Asset - This field can be used to record where the asset is. Values are from the Country Codes [CNTR] lookup table.

    • Currency Code - Select the currency used for the acquisition of the asset.

    • Purchase Value - Enter the value of the asset when it was purchased.

    • Facility Charge - Enter any facility charge amount if applicable.

    • Date Last Valuation - Enter the date of when the asset was last valued.

    • Date of Acquisition - Enter the date the asset was purchased.

    • Date Sold - Enter the date the asset is sold.

    • Date Closed - Enter the date of when the asset record is closed.

    • Valuation - Enter the valuation details of the asset.

    • Method Acquisition - Enter how the asset was acquired.

    • Document of Acquisition - Enter information for documents relating to the acquisition of the asset.

    • Insurance - Enter information on the insurance coverage for the asset.

    • Encumbrance - Enter details of any encumbrance for the asset.

    • Management - Enter details of management of the asset.

    • Note - Enter additional notes for the asset.

    Can I add other asset types?

    This is possible in Viewpoint - Home, go to Configuration > Business Parameters > General > Lookup Values. Select the Fixed Asset Types [FATP] lookup table and add new asset types.

  • Open Items - Information on assets that are not suitably recorded in the other folders can be recorded here.

    Details for open items are as follows:

    • Type - Select the type. The options are maintained in Open Item Type [OITP] lookup table.

    • Reference - Enter reference for the item.

    • Description - Enter a description for the item.

    • Related Party - Select the Master File of the related party. This could be the bank giving the loan, for instance.

    • Date Opened - Enter the date the item was opened.

    • Date Closed - Enter the date when the asset record is closed.

    • Date Due - Enter the due date of the item.

    • Currency Code - Select the currency code for the item.

    • Revaluation - Select the revaluation frequency.

    • Note - Enter additional notes for the item.

  • Securities - Information for investment items owned by the entity are recorded here.

    Details for securities are as follows:

    • Security Reference - Enter reference for the security.

    • Security - Select the Master File for the security.

    • Currency Code - Select the currency for the security.

    • Date Acquired - Enter the date the security was acquired.

    • Date Closed - Enter the date the asset record is closed.

    • Registered Holder - Select the Master File for the security's registered holder.

    • Description of Value - Enter the value of the security.

    • Location of Scrip - Enter where the scrip is kept.

    • Note - Enter additional notes about the security.

For each screen you can click on:

  • - To minimise this section and hide the command buttons. In the grid, only the selected row will be seen.

  • - To expand the section and show the command buttons.

If summary screen template has been set up, clicking Assets in Section Menu will give an overview of all assets recorded. The records can also be displayed in the chart in Ownership screen using Table Nodes.

 

Filter

This is only available for Bank Accounts. You can filter the list of bank account by the bank account status.

  • Open - Click to view only open bank accounts.

  • Closed - Click to view bank accounts that have been closed.

  • All - Click to view all bank accounts.

 

Asset list

This list the assets for the selected asset type.

 

Command buttons

These apply to the asset list. Click the relevant command button to perform the necessary.

  • New - Click this to add asset record.

    1. Enter information for the asset.

    2. Click Update when done.

  • Edit - Click this to edit details of selected asset record.

  • Close - Click this to close the selected bank account. This is only available in Bank Accounts folder.

    1. Enter the Date Closed and Note (if any).

    2. Click Update.

  • Delete - Click this to delete the selected asset record.

    Click Yes to confirm the deletion.

  • User Fields - Click this to retrieve user fields for the selected asset record. This is only available for Fixed Assets.

  • Securities Master - Click this to access the Security Master. This is only available for Securities.

    1. Define your selection criteria to filter the securities listing.

    2. Click Apply.

Valuation Details

Tick to display the valuation details for the selected asset.

 

Transaction Record

For each of the asset, you can also record its related transactions.

 

Command buttons

These apply to the asset transaction. Click the relevant command button to perform the necessary.

  • New - Click this to enter a transaction.

    Enter the details for the transaction. Click Update when done.

  • Edit - Click this to edit details of existing transaction.

  • Delete - Click this to delete the selected transaction.

    Click Yes to confirm the deletion.

  • Recalculate - Click this to recalculate the value for the asset.

  • Print Voucher - Click this to generate a voucher for the selected transaction.

    1. Select the generation method.

    2. Click Generate.

    Note: You may edit the Asset Transaction Voucher [ASSETVCH] template in 'Others (Entity Administration) [F13]' report folder of Word Reports Library.

  • Payment (CA) - Click this to create a payment for the selected transaction. This button is active only if the selected transaction has a value in the Payer/Payee field. If the Payee is not valid, you will be prompted to create the payee record. By default, this button is not enabled. To use this, the Entity File must be set up as an active Account File.

    1. Click Yes to create the payee.

    2. Enter information on the new payee.

    3. Click Update.

    Note: Once created, the payee needs to be approved.

    Enter information on the new payment. Click Update when done.

  • Print Statement - Click this to generate a statement for all transactions of the asset.

    1. Select the generation method.

    2. Click Generate.

    Note: You may edit the Asset Statement [ASSETSTA] template in 'Others (Entity Administration) [F13]' report folder of Word Reports Library.

  • AEOI Reporting - Click this to open the AEOI Reporting function.

  • Validate AEOI - Click this to Validate AEOI. The status will then be displayed in the Regime column in the grid.

    After the 'Validate AEOI' is performed, it will display the following in the Regime column:

    • Display ‘X’ if there isn't any transaction linked to CRS or FATCA.

    • Display ‘C’ for CRS, or/and ‘F’ for FATCA, or/and 'G' for UK FATCA.

    • Display ‘!’ if the Percentage is not 100%
