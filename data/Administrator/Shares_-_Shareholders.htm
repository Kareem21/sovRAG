Shareholders

The Shareholders folder lists all members for all the share classes that have been authorised including past members. The certificates held by members are also listed with the cancellation dates if the certificates have been redeemed, forfeited or cancelled.

There are two sections in this screen and you can click on:

  • - To minimise this section and hide the command buttons. In the grid, only the selected row will be seen.

  • - To expand the section and show the command buttons.

Filters

 You can filter the list by selecting the share class and/or listing type - Current Holdings, Past Holdings or All.

 

Select button

Click to filter using search criteria.

Enter the search criteria and click Apply.

If you are unsure of how to enter the value, place your mouse within the Search for.. column to view tooltip.

Tooltip will appear to guide data entry for the following field type:

Field Type Tooltip Message
Number Enter 'number and number' for a range (example: 100 and 200)
Date User a number for a date relative to today (example -1 for yesterday) and 'and' for a range (example: -7 and 0)
Search control Start with the '+' sign to search on the code (example: +CODE) or the start of the name
Text Use the * in the search text for wild-card selections

Command buttons

Click on the relevant command button to perform the necessary. You need to first select a shareholder or share certificate first.

  • Transfer - Click this to transfer the shares to another shareholder. Shares can be transferred from one shareholder to another existing shareholder or to a third party. Transfers are permitted only for shares that are fully paid up.

    1. Enter the relevant information for the share transfer.

    2. Click Update.

    3. Tick the form(s) you want to generate.

    4. Click OK.

    Note: When 'Use Distinctive Numbers' option is ticked, the Distinctive Numbers of shares transferred to transferee will be based on Last in First Out (LIFO) order.

    For share class without Share Certificate number but 'Use Distinctive Numbers' option is ticked, multiple transactions will be recorded as one transaction under the following conditions:

    • same Master File for both transferor and transferee

    • same transaction date

    • same share class type

  • Edit Cert. - Click this to edit the details of the selected certificate.

    1. Make the necessary change to the details.

    2. Click Update.

  • Ben. Owner - Click this to assign a beneficial owner to shares held by the selected shareholder.

    1. Select the Master File for the beneficial owner.

    2. Click Update.

    See also Tips:Beneficial Owner

     

  • Call(s) Paid - Click this to:

    • Update the amount paid on the shares held by the selected shareholder. If the entity type does not contain the [SHVAL_ISSUEPRICE=1] tag, this command button is enabled only if the share certificate still has outstanding payments. Refer to Entity Type - Tags for details.

      Enter the details of the call, which can be a further call or a final call. Click Update when done.

    • Update the share premium for shares issued at a premium. This requires the entity type to contain the [SHFLD_ISSUEPRICECHANGE=1] tag. Refer to Entity Type - Tags for details.

      Increase in share premium

      Change the default Description to 'Increase in Share Premium [PI]' and specify the value in the Change in Issue Price field e.g. 0.5 to increase the premium by 0.5.

       

      Decrease in share premium

      Change the default Description to 'Decrease in Share Premium [PD]' and specify the value as a negative value in the Change in Issue Price field e.g. -0.5 to decrease the premium by 0.5.

  • Redeem - Click this to redeem shares. Shares can be redeemed by its shareholders at a later date as long as non-redeemable shares have already been issued. Redeemable shares cannot be redeemed unless they have been fully paid. Normally holders of these shares have no voting rights.

    Enter details of the redemption. Click Update when done.

  • Cancel Cert. - Click this to cancel the selected share certificate.

    Enter the details for the cancellation. Click Update when done.

  • Forfeit - Click this to forfeit the selected shares.

    Enter the details for the forfeiture. Click Update when done.

     

  • Proxies - Click this to appoint an administrator as proxy for one or all shareholders for the selected Entity File.

    In the Proxies screen:

    1. Select the administrator and click <<Add. This will add the administrator to the proxies list.

    2. Tick the options that will apply:

      • Apply to all Shareholders (proxies are added/delete as per this Shareholders proxy list) - Tick this if you want the list of proxies to apply to all shareholders of the selected entity.

      • Confirm Effective Date - Tick this if you want to confirm the date of appointments of the proxy/proxies. If not, the event will be treated as outstanding and be listed in the To-Do list.

      • Trigger Proxy event (for document generation) - Tick this if you want to trigger forms for the event.

    3. Click OK.

 

Can I modify share allotment/transfer date or total consideration?

The Edit Cert. command does not allow these details to be changes. To revise these, you would have to reverse the share transaction that creates the certificate and then redo the transaction with the correct detail.

 

Why am I not able to see the Ben. Owner command button?

This command button may be turned off to have the beneficial owner recorded in Interests screen..

To check, go to Configuration > Settings > Customise > Field Settings and specify the following:

  • Menu Group = File

  • Section = Entity Administration

  • Screen = Shareholders

Why am I not able to see the Proxies command button?

This command button may be turned off. To check, go to Configuration > Settings > Customise > Field Settings and specify the following:

  • Menu Group = File

  • Section = Entity Administration

  • Screen = Shareholders

How do I set a default proxy?

  1. In the Proxies screen, select the administrator you want to set as default from the proxies list.

  2. Click Default.

  3. Click OK.

Can I remove an administrator from the proxies list?

  1. In Proxies screen, select the administrator.

  2. Click Delete.

  3. Click OK.
