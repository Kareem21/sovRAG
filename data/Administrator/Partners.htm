Partners

The Partners folder is split into four sections and is used to record partnership holdings.

 

Posting

This section lists incomplete partnership transactions. Find out more here.

 

Classes

This section is where all classes of partners are created and maintained. Find out more here.

 

Members

This section is where partners are assigned and records are kept on the contributions and returns made by each partner. Find out more here.

 

Transaction

This section lists all transactions for the selected partnership class. Find out more here.
