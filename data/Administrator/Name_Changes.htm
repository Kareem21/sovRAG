Name Changes

Any name change for the entity should be maintained in this folder. Both the name changes applied for and approved for the entity can be recorded.

If the entity is a shareholder, share transactions involving the entity must be rebuilt to reflect the new name.

 

Name list

This shows records of name changes for the entity.

 

Command buttons

Click the relevant command button to perform the necessary.

  • Apply - Click this to enter details for the name change application.

    Enter the details. Click Update when done.

  • Confirm - Click this to confirm the applied name change. The first name entered in the name requested will be selected as the new name.

    Enter details for the confirmation. Click Update when done.

  • Edit - Click this to edit details of the name change application.

  • Delete - Click this to delete the name change record.

    Click Yes to confirm the deletion.
