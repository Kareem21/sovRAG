Accounts

The accounting information for an Entity File is entered in this folder. The information here is used to generate next year's accounts and filing.

 

How do I set up default values for the fields in this folder?

This is done in Viewpoint - Home > Configuration > Settings > Customise > Field Settings:

  • Menu Group: File [8]

  • Section: Entity Administration [FESTATU]

  • Screen: Accounts [FSEACCTS]

Select the field, click Edit and enter the default value in Default field.
