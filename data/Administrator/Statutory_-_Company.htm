Statutory - Company

The Statutory folder for entity files records the incorporation details and location of the statutory books and common seal.

 

Date of Incorporation

Enter the date of incorporation for the entity. If you used the Copy From Existing Entity function, the date entered in the Incorporation Date field will be reflected here.

 

Incorporation Number

Enter the incorporation number for the entity.

 

Statutory Addresses

Address Type

This allows you to:

  • View addresses linked to the Master File - Select Master File Addresses.

  • View and add statutory locations links to the Entity File - Select Statutory Address/Location.

  • View both addresses and statutory locations - Select All.

Note: New addresses can be added if the one you want to use does not already exist in the system. It is advisable to first check the Address Cards for an address, before you proceed to create a new one.

 

Current / Past / All

Select if you want to see only current address, past address or all.

 

Addresses List

This shows the address cards linked to the file.

 

Command buttons

Click on the relevant command button to perform the necessary.

  • New - Click this to link a new address to the Entity File.

    This function is used to create a new address type link. These will be address types where the code starts with 'S'.

    You can only have one address type link per Master File unless the address type code is S*.

    1. Click New.

    2. Look for the address you want to link. If it cannot be found or has not been created, then proceed to create the address.

    3. Select the address type it is used for.

    4. Click OK to link the address.

    5. Click Update.

     

    For Hong Kong companies, section 14 of the Hong Kong Annual Return will show the following:

    • N/A - when all address types are the same

    • Error : Please check your inputs in Statutory Location - when there are no other address types apart from 'SR'.

    If there are multiple statutory locations and they differ from the 'SR' address, then these addresses will be added in the section 14 of the Hong Kong Annual Return unless the address type is configured to be excluded using the [EXCOMREC=1] tag in the Address Types [ADRT] lookup table.

     

    How do I create a new address?

     

  • Edit - Click this to edit a selected Address Card link. The program will show the address link detail on the Address Card Master screen.

    This function is used only to correct address card links such as the address type and effective date or to amend address details. Use the Change function if the address is being changed.

    1. Select the address to be edited.

    2. Click Edit.

    3. Amend the details and click OK.

    4. Click Update.

    What is an address type and can I create more types?

    Address types can be used to group or differentiate addresses linked to a Master File. You can create more address types in the Address Types [ADRT] lookup table. Remember the following:

    • Address types that start with 'S' are meant for use in the Statutory screen.

    • Address types starting with 'R' are reserved for the Master File > Addresses screen.

  • Delete - Click this to delete the address link. The address will remain in the Address Card Master until it is deleted from there.

  • Change - Click this to change the selected address. The Address Card Master screen will appear for you to select a new address card and to complete the effective date. The existing address link record will be deactivated, and the From date will be filled with the effective date of the change. The effective date of the address change will affect the address displayed on forms and reports, i.e., if the date of the address change is after a particular event, the generated form will display the previous address.

    This function is used to record new and archive old address records when a Master File address has changed. For instance, if an individual has changed his place of residence, or if an entity has changed the location of its registered office, using the change function allows for the recording of the effective date of the change, which is used for statutory filing purposes.

    1. Select the address type that is being changed.

    2. Click Change.

    3. Select the new address from the Address Card Master or create it if it does not already exist in the database.

    4. Click OK.

    5. Click Update.

    My office address has changed. How do I change the addresses of ALL Master Files that have been using the old address to the new one?

    To do this, you can use Global Change > Registered Officer Address - Entity.

     

  • Undo - Click this to undo ALL changes made to the address card links before you click the Update button.

  • Update - Click this to update the information.

  • Cancel - Click this to cancel any changes made.

Note: If you add or change address types with codes that start with 'R' or 'S'  for a Master File that acts as an officer for an entity, you will be prompted to file the changes. If the option 'Trigger Global Change of Particular for Shareholders' in Configuration > Settings > System Defaults > Entity/Forms is enabled, then this prompt is also seen if the Master File is a shareholder.
